# **课程作业：构建行星运动数值模拟与误差分析**

### **一、 背景**

在天体物理中，描述行星位置有两种主流路径：**分析理论 (VSOP)** 与 **数值历表 (DE)**。

- **VSOP (Analytical Theory)**：利用三角级数展开 $\sum A_i \cos(B_i + C_i t)$，体现了数学截断与近似。

- **DE (Numerical Ephemeris)**：利用高阶切比雪夫插值和数值积分，代表了当代数值计算的最高精度。

  本作业要求你利用 AI 编程工具，对比这两类模型的计算特征，并将其封装进你的“学习助手”智能体中。

------

### **二、 任务**

#### **任务 1：误差来源的理论映射**

阅读《数值计算的误差》PPT与《行星运动模型》文档，完成以下映射分析：

1. **模型误差**：分析 VSOP 模型在忽略小行星带质量时产生的误差属于哪一类？
2. **截断误差**：VSOP 级数只取前 $N$ 项，请给出其截断误差的数学表达式。
3. **舍入误差**：在进行 $\sum A_i \cos(B_i + C_i t)$ 万级规模的累加运算时，如何避免舍入误差的严重累积？

#### **任务 2：AI 辅助算法实现 **

通过AI coding 工具编写 Python 程序：

1. **VSOP87 模块**：实现地球坐标计算。要求 AI 编写高效的 `Parser` 读取系数表，并利用 `NumPy` 进行向量化运算。
2. **DE 插值模块**：实现切比雪夫多项式插值。
   - 公式：$x(t) = \sum_{i=0}^n c_i T_i(\tau)$
   - 要求：助手需能读取二进制数据并进行高阶插值。

#### **任务 3：误差传播与稳定性实验 **

对比 $\text{VSOP87}$ 与 $\text{JPL DE430}$ 计算得到的地球位置：

1. **时间演化实验**：计算从 $J2000.0$ 历元开始，往前后推 500 年，观察两模型之间的绝对误差 $\varepsilon$ 随时间 $t$ 的增长曲线。
2. **精度判定**：根据有效数字定义，判断在计算 1000 年后的位置时，$\text{VSOP87}$ 还能保持几位有效数字？

#### **任务 4：智能体集成 **

将上述功能集成到你的 **“科学计算助手”** 中：

- **功能需求**：用户输入“查询 2050 年火星位置，要求精度小于 $10^{-6}$”，助手需自动判断该调用哪个模型，并给出误差估计。
- **可视化**：需自动生成轨道差异对比图。
