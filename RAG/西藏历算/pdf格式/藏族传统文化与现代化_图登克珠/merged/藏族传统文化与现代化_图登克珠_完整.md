# 藏族传统文化与现代化

# 青年藏学工作者学术研讨会综述

图登克珠

由西藏自治区社会科学院主办，我区部分青年藏学工作者发起的《藏族传统文化与现代化学术研讨会于1992年11月10日至15日在西藏自治区首府拉萨市举办。来自北京、青海和西藏各地区的青年藏学工作者共60余人出席了研讨会，其中 $90 \%$ 以上是藏族青年代表。会议收到青年学者们有关西藏的政治、经济、文化、社会、历史、宗教、哲学、语言、文学、教育、医学、天文历算等各学科的藏、汉文研究论文50余篇。从青年学者所提交的论文来看大都以藏文为主。与会者在理论与实践相结合的基础上，从不同的角度探讨了在新的历史时期藏族传统文化与现代化观念的冲撞、融合等，历史与时代的选择问题上进行了大胆的探索并展开了热烈的讨论。现将其中一些观点综述如下：

# 一、关于藏族传统文化的核心及现代化

众所周知，“文化”作为一种历史和时间的范畴，是人类社会发展过程中创造的物质财富和精神财富的总和。目前国内外诸多学者认为“文化”有广、狭二义。广义的文化概念包含着物质和精神的两大部分，而狭义的文化概念是指一种精神形态的范畴·如社会、政治、哲学、宗教、教育、科学及伦理道德等领域中思想意识形态和相应的制度与组织，以及所产生的精神财富和精神产品。除此之外，还存在着物质因素与精神因素二者之间兼备的文化现象，如衣、食、住、行等等。因此，从整个文化的概念判断，与会代表一致认为：世界上几乎没有无文化的民族，同样文化作为历史和时间的范畴不具备民族特色的文化也是不存在的。由于人类社会历史发展的多样性及各民族所处的人文地理因素和自然条件等原因，各民族也就存在着自身的“文化圈”即民族文化圈。而所谓的民族文化就是指某一民族在物质和精神财富的创造上与其他民族所不同的文化模式。因此，从某种意义上讲也可以理解为该民族不同于其他民族的生活方式和价值取向。

与会代表一致认为：藏民族作为深受佛教文化影响的民族，在其自身文化形态上，以本土苯教文化作为根基，形成与印度佛教文化和我国中原地区的汉地佛教文化种种不同的内容和形式。从而形成了高原独特的“藏传佛教文化圈”。有学者提出了藏民族文化为“高原文化”的新观点。

与此同时，青年藏学工作者们对“高原文化”（这里所说的“高原文化”主要是指苯教文化与佛教文化合壁的文化模式）即藏族传统文化的历史界限和文化成份进行了进一步的探讨和研究。有学者提出藏族整体文化五千年和历史文化三千年的新观点。也有学者提出藏族文化从七世纪初开始到十九世纪又可分为几个不同的发展阶段的观点，因而，参加本次研讨会的大多数青年学者建议：藏族传统文化与现代文化的时间界限应划在五十年代的马克思列宁主义的传播即社会主义思想意识的确立。当然也有学者提出；

藏族传统文化与现代文化的界限不应仅仅以时间为绝对的分水岭，而要看藏民族不同的精神文化在某一特定的历史时期所处的地位来确定其主导地位。

根据藏民族整体各文化要素的主导地位，本次研讨会所争论的焦点大都集中在藏族传统文化的核心问题上。有学者建议：在当前探讨藏族传统文化的核心问题时必须在与其它民族文化进行比较的基础上，首先了解藏民族传统文化的基本内容和它所包含的各个时期不同文化的组成部分，然后进行科学的分析与组合，提炼出最精华的精神（即能够左右该民族价值取向的文化价值观），才能推断出其文化的整体核心。什么是藏族传统文化的核心问题？本次研讨会上大致有以下几种观点：（1）藏族传统文化主要因佛數思想的影响，从人治出发，是人的文化，其突出特点表现在人与自然的和谐和人与人的和谐两方面；（2）藏族传统文化的核心是由苯教文化与佛教思想相结合的一种文化，（3）藏族传统文化的核心是以藏族自已的特殊文化为基础的藏传佛教文化；（4）藏族传统文化是以苯教文化为基础，佛教文化为主导，吸收了周边地区其它民族文化的文化，因此藏族传统文化有其自身的特征，这就是以苯教入世思想为基础的对自然的依赖感和以佛教的出世思想为基础的因果报应、慈悲思想、众生平等等是藏族传统文化的核心；（5）有学者从藏族哲学的深层次论述了藏族传统文化的核心，认为藏族传统的哲学思想是以大乘佛教思想为主，而大乘佛教思想的整个合理内核是构成藏民族传统文化的精髓，因此其文化的核心首先应该体现在精神文化领域中，如：菩提慈悲、众生平等、普渡众生、无限的爱等都可以成为藏族传统文化的核心，因为这种思想从根本上讲，是佛苯两种文化自我谐调的结果；(6)有学者从文化比较学的角度通过对西方基督教文化、汉民族儒家文化和藏民族传统的佛苯文化的比较，提出：西方的基督教文化注重的是人与自然之间的“对抗”，其文化的主体不是人而是自然界。因此，把整个自然界作为其文化的主体和对象。显然，经济和科学技术是整个西方基督教文化(或新教文化）的两大支柱和核心。汉民族的儒家文化却往往注重的是人与自然之间的“和谐”或人与人之间的“和谐”，如“天人合一”思想，其文化的主体不是自然界而是人本身，把人作为文化的主体或对象。显然，人与人之间的关系和伦理道德的社会价值体系是整个汉民族儒家文化的核心，这种文化价值体系的核心又可以被认为是整个东方文化的一个特点。当然藏民族传系的佛苯文化虽属东方文化这个大系统，但是，除了具有东方文化的特点之外，跟以上两种文化形态所不同的一点，“悲”是其文化的核心。

除此之外，在谈到传统文化与现代化的关系问题时，大多数青年学者认为：传统文化从一般意义而言，注重的是人类精神形态的文化，而现代化（即现代文化或现代化文明）却在更高程度上注重以物质形态为基础的经济文化。所以，在当前提倡继承和发扬民族传统文化时，应该注意顺应历史潮流以民族优秀文化来丰富该地区物质文化和精神文化,从而使该民族、该地区的现代化具有时代特征和民族特点。同时，也有一些学者在结合学习邓小平同志南巡讲话精神和学习党的十四大提出的有关重大理论问题的關时，根据西藏的实际情况，进一步阐述了市场经济在西藏如何发育以及西藏现代化道路上的人才问题和教育等问题，提出了不同的看法。

# 二、关于西藏的人才培养与现代化道路

人是作为一种社会动物和文化的再显，一方面是社会和文化主体，另一方面又是社会和文化的客体。因而从某种意义上讲，无论传统文化还是现代文明无不都是人的能动性在物质和精神上的体现。所以，一个民族的存在与发展在很大程度上取决于它能否与整个时代的节奏成正比，那么如何去适应这个节奏呢？与会者一致认为：教育是适应速个节奏的关键，也就是说教育是实现整个社会现代化的关键。有学者用发达国家的教育投资跟发展中国家的教育经费作比较后提出：人的现代化是社会现代化的关键，要实现经济现代化，科学技术现代化，政治制度民主化，归根到底还是要靠人的现代化。所以人的现代化是社会现代化最重要的标志之一，也是社会现代化最深层次的问题。

这次研讨会除了对藏族传统文化的核心问题进行各种讨论之外，对教育问题和藏语规范化问题也进行了不同程度的研究和探讨，特别是有关民族教育问题。从整个学术研讨会收到的50多篇论文中看，三分之一是有关民族教育问题和藏语、藏文规范化问题的论文。如有关民族教育问题上有学者提出：开放的世界是充满竞争的世界，其竞争是一种综合实力的竞争，竞争的焦点是民族整体素质。民族素质差，是造成经济落后、制约社会发展的主要因素。与此同时，有学者提出由于我区藏民族素质普遍偏低和教育的落后形成了两大民族性格。一种是乞求—“感恩型”，另一种是自大一“闭锁型”。所以，有些学者认为这种民族素质普遍偏低的现状是使得西藏经济迟迟不能发展的重要因素，中央给西藏的一系列特殊政策和投资没有得到有效的利用，这样就相对地加大了西藏同内地，特别是沿海地区的距离。若不改变这种“效益上的贫血症”，西藏就难以加入全国的市场经济行列。当然，从表面上看西藏的这种“效益上的贫血症”与管理者的能力固然有关，但是，如果从深层次的角度来考虑不能说跟整个民族的教育水准无关，因此，学者认为：西藏的落后不再是资源和设备的落后，而是“人的落后”。西藏要发展，教育要先行。

有的青年学者建议：西藏要发展，教育要先行，必须是“双语教育”先行。但是，也有一些学者认为：目前对“双语教育”有种种不同的理解，究竟什么是“双语教育”？这是西藏要发展，教育要先行的首要问题。如果能正确解决“双语教育”的内容和形式以及方式方法，才能使西藏的民族教育沿着正规化的道路发展。因此又进一步提出：“双语教育”的新概念，是指两种教育体制并存，同时加授两种语言文字的一种教学形式。与此同时，也有一些学者对目前我区个别重点中学藏语授课“试点班”学生的前途和我区在内地办西藏中学等问题提出了一些不同的看法。总之，与会者一致认为：西藏的现代化道路首先应该选择一条自尊、自强、自立、自主、自爱的民族教育思想，才能培养出适应历史潮流和具有时代精神的西藏社会主义建设事业的接班人和建设者。

在这次青年藏学工作者研讨会上，值得注意的一点还有关于藏语言文字标准化和规范化等问题。从整个学术研讨会所收到的论文来看，涉及到有关语言学学科的论文占整个论文总数的五分之一。这就说明一些青年藏学工作者已经意识到，在继承和发扬藏民族优秀的传统文化时，藏语言文字的标准化和规范化以及推广藏语普通话的重要性，同时也有一些学者感到，推广藏语普通话是加快藏族地区的改革步代和建设社会主义市场经济极为重要的一项战略任务。语言的统一和标准化是任何一个民族走向开放社会的一个内部条件。有学者根据政治、经济、文化的中心提出：按照藏族地区三大方言（即卫藏方言、康区方言、安多方言)的特点，以卫藏方言为基础的拉萨话为藏族地区的普通话设想。但也有一些学者提出：目前拉萨话中存在着不属于三大方言的外来词较多,如：有汉语、印度、甚至还有英语的，使整个拉萨话较为混杂，因而，在藏族地区推广拉萨话为普通话，首先解决拉萨话中的外来语音和词汇的消除。然后再来解决各地区方言中的语音问题。如果藏族地区各大方言中的语音能够统一，推广藏语普通话并不是件很困难的事，因为三大方言的语法和词汇都是同一种语言的语法和词汇，只是语音不同而已。有关藏语词汇中特别是拉萨话里的外来词的消除,有学者认为必须通过两种途径，一是通过学校教育形成一套从幼儿教育到大学教育的一种完整统一的藏语言文字的教育体系；二是通过新闻，特别是广播、电视和电影等大众传播媒介来建立一套完整的社会语音规范化系统。当然，更多的与会者认为：藏语普通话的推广和藏语言文字的标准化、规范化的实现有利于藏族地区经济文化的发展，特别是科技文化领域的发展有着非常重要意义。

长期以来，无论是国外的藏学研究还是国内的藏学探讨在很大程度上由于受某种客观环境的影响，跟藏区的经济发展，特别是西藏的经济建设和社会主义现代化建设或多或少存在着一些脱离的现象，因此一些社会科学工作者和包括一些藏学研究人员不得不陷入了一种历史与时代之间的困境。自然，他们的研究成果也就很难为西藏的经济发展和社会主义现代化建设提供有力的理论武器。

而这次由青年自已发起并由自已参与的学术研讨会在我区还属首次。从整个会议的中心议题“藏族传统文化与现代化”来看，是我区社会科学工作者特别是藏学研究领域的一项由过去的务虚回到当前的务实的理论大研讨。同时这次研讨会无疑为我区今后的藏学研究和经济发展起到了一个解放思想、更新观念、鸣锣开道的先锋作用·
