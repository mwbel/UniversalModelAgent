# 第十一章 坐标系摄动

# 11.1 坐标系摄动问题的提出

前面各章采用的是地心赤道坐标系，正如1.3中所说的，由于岁差章动和地极移动现象的存在,地心赤道坐标系有多种选择。涉及坐标系选择的量有两类：一是运动天体(人造卫星)和摄动天体（日、月)的位置矢量，另一种是地球引力位(包括潮汐形变和自转形变)的表达式，通常总是选择历元地心惯性系，80年代前习惯用J1950.0系统，而近年来已改用J2000.0系统.在此坐标系中，人卫和日、月位置矢量的表达较方便，但是地球引力位的表达式是.在地固坐标系中建立的，因此这一部分就涉及坐标转换问题．若分别记 $\pmb R$ 和 $\scriptstyle { \vec { r } } ^ { \prime }$ 为历元地心惯性系和地固系中的卫星位置矢量，则根据第一章 $\ S \operatorname { 1 . 3 }$ 中给出的关系，有

$$
\begin{array} { r } { \pmb { r } ^ { \prime } = ( H G ) \pmb { R } , \pmb { R } = ( H G ) ^ { \pmb { r } } \pmb { r } ^ { \prime } , } \end{array}
$$

其中转换矩阵 $( H G )$ 的具体形式见(1.40)式。地球引力位记作

$$
V { = } V \left( r ^ { \prime } \right) { = } V _ { 0 } \left( r ^ { \prime } \right) { + } V _ { \varepsilon } \left( r ^ { \prime } \right) ,
$$

其中 $\boldsymbol { V } _ { \varepsilon }$ 为摄动函数，即非球形部分和各种形变引起的相应部分，不难导出

$$
\stackrel {  } { \boldsymbol { R } } = ( H G ) ^ { \tau } ( \stackrel { \hat { c } V } { \partial r ^ { \prime } } ) ^ { \tau } ,
$$

$\left( { \frac { \partial V } { \partial r ^ { \prime } } } \right)$ 就是地固坐标系中的地球引力位加速度。

根据上述讨论，若选择历元地心惯性系，在用数值方法计算人卫有摄星历表时，每一步都有上述转换问题，这虽然容易实现，但每步转换总嫌不便，至少需多占用一些机时．是否可用其它方法解决？为此，有必要搞清这种坐标选择引起的转换对卫星轨道影响的性质和程度．由于上述坐标转换必然引进与岁差章动、极移有关的附加部分（与地球自转有关的部分在5.7和5.8中已讨论过)，它将导致附加摄动的出现，通常就称这种附加摄动为坐标摄动.

# 4.两点说明

（1）从上述两种附加摄动的结果不难看出，早期为什么提出轨道坐标系，其原因已很清楚．在用分析方法建立卫星轨道摄动解时，显然采用轨道坐标要简单得多，尤其对于一般精度要求的一阶解，无需考虑岁差、章动和极移引起的附加摄动。

（2）在高精度要求下，往往是采用数值方法建立卫星的有摄星历表，在历元地心惯性系中，上述附加摄动是通过坐标转换解决的，即前面的(11.3)式．若采用轨道坐标系，是否会简单些？不妨作些具体分析．在此情况下，需解决两个问题，一是在以卫星位置矢量 $\pmb { r }$ 和速度矢量 $\dot { \pmb { r } }$ 作为基本变量建立的运动方程中，如何给出上述附加项;另一个问题是如何简单地将历元地心惯性系中的日、月位置转换到轨道坐标系中来。

关于第一个问题,即方程形式的转换问题．由

$$
\left. \begin{array} { l } { r = r ( a , e , i , \Omega , \omega , M ) , } \\ { \dot { r } = \dot { r } ( a , e , i , \Omega , \omega , M ) } \end{array} \right\}

$$

$$

这里的 $\pmb { \sigma }$ 就仅包含摄动部分。 $\frac { \partial r } { \partial \sigma } \mathfrak { F } \mathfrak { H } \frac { \partial \dot { r } } { \partial \sigma }$ 将在第十四章中给出,其中

$$

$$

$$
\begin{array} { l } { \displaystyle \frac { d r } { d t } = \dot { r } + \sum _ { j } \frac { \partial r } { \partial \sigma _ { j } } \boldsymbol { \phi } _ { j } , } \\ { \displaystyle \frac { d \dot { r } } { d t } = - \frac { r } { r ^ { 3 } } + \sum _ { j } \frac { \partial \dot { r } } { \partial \sigma _ { j } } \boldsymbol { \phi } _ { j } . } \end{array}
$$

根据第四章中引用常数变易法的原理(见(4.24)式)，除本章涉及的坐标系变化引起的附加摄动部分以外，有

$$
\sum _ { i } \frac { \partial r } { \partial \sigma _ { i } } \boldsymbol { \cdot } \boldsymbol { \sigma } _ { i } = 0 , \quad \sum _ { i } \frac { \partial \dot { r } } { \partial \sigma _ { i } } \boldsymbol { \cdot } \boldsymbol { r } _ { i } = \boldsymbol { F } _ { \varepsilon } .
$$

这里的 $\pmb { F }$ ，即除上述附加部分外的全部摄动加速度，因此，最后导出轨道坐标系中，以 $\pmb { r }$ 和 $\dot { \pmb { r } }$ 作基本变量的运动方程为

$$
\left. \begin{array} { l } { { \frac { d r } { d t } { = } \dot { r } + \Bigl ( \frac { \partial r } { \partial t } \Bigr ) , } } \\ { { \frac { d \dot { r } } { d t } { = } F + \Bigl ( \frac { \partial \dot { r } } { \partial t } \Bigr ) , } } \end{array} \right\}
$$

其中

$$
