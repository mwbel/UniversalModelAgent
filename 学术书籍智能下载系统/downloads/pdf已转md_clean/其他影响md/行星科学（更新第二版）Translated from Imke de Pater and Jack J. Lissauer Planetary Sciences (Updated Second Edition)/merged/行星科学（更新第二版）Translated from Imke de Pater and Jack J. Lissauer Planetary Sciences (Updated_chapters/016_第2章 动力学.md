$$
{ \frac { \mathrm { d } ^ { 2 } { \boldsymbol { r } } } { \mathrm { d } t ^ { 2 } } } = - { \frac { \mathrm { G } M } { r ^ { 2 } } } { \hat { \boldsymbol { r } } }
