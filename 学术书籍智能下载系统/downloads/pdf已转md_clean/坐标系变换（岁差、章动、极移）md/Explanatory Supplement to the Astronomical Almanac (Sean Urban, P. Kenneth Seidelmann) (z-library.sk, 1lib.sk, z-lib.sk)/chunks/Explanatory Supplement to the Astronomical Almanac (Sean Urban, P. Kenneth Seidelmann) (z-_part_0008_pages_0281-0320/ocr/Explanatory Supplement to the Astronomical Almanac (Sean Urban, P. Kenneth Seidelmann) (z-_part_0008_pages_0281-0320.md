for $x _ { p }$ and $y _ { p }$ in seconds of arc.The values $x _ { p }$ and $y _ { p }$ varyby,at most, $0 . { } ^ { \prime \prime } 8$ ,so the maximum radial displacement is approximately $2 5 ~ \mathrm { m m }$ ,and the maximum horizontal displacement is about $7 \mathrm { m m }$ .

$$
\left[ \begin{array} { l } { d X } \\ { d Y } \\ { d Z } \end{array} \right] = \mathbf { R } ^ { \mathrm { T } } \left[ \begin{array} { l } { S _ { \Theta } } \\ { S _ { \lambda } } \\ { S _ { r } } \end{array} \right]
$$

where

$$
\mathbf { R } = \left[ \begin{array} { c c c } { \cos \Theta \cos \lambda } & { \cos \Theta \sin \lambda } & { - \sin \Theta } \\ { - \sin \lambda } & { \cos \lambda } & { 0 } \\ { \sin \Theta \cos \lambda } & { \sin \Theta \sin \lambda } & { \cos \Theta } \end{array} \right] .
$$

Using $k = 0 . 3 0$ gives

$$

where $x _ { p }$ and $y _ { p }$ are in seconds of arc and are used instead of $m _ { 1 }$ and $- m _ { 2 }$ so that no mean value is introduced into $C _ { 2 , 1 }$ and $S _ { 2 , 1 }$ while making this correction.

# INTRODUCTION

This chapter provides for the reduction of celestial coordinates from one reference system to another.The coordinates are usually rectangular equatorial coordinates or equivalently right ascension,declination and hour angle.Inparticular the transformations fromaposition inthe Barycentric Celestial Reference System (BCRS)—or equivalently the International Celestial Reference System (ICRS)—to a geocentric apparent or intermediate position are described, but some other systemsand coordinates are discussed.

This chapter first summarizes the various reference systems,places,coordinates and notation used within the chapter. Then it describes the individual corrections of space motion, parallax,light-time,aberrationand gravitational light-deflection,which are required in the transformation from the BCRS to the Geocentric Celestial Reference System(GCRS).If the requirementis for precision better than milliarcseconds (mas),when General Relativity principlesareneeded,then thisis indicatedwhereappropriate.Finally,vector-matrixalgorithms are given for calculating various positions including,apparent,intermediate and topocentric places.Also analgorithm for calculating atmosphericrefraction isincluded.

Atthe current time all these procedures conform to IAU 2oo6 resolutions 1 and 2 as well as IAU 2000 resolutions B1.6,B1.7and B1.8 (IAU 2001,2008; URL[6]).

In simple terms givenaposition of an object $\mathbf { u _ { B } }$ with respect to the BCRS,then the transformation to $\mathbf { r }$ ，aposition of date in the International Terrestrial Reference System (ITRS),may be considered thus

$$
{ \bf r } = { \bf W } \ { \bf R } _ { 3 } ( - \beta ) \mathcal { R } _ { \Sigma } \ [ f \left[ g \left[ h \left[ { \bf u } _ { \bf B } \right] \right] \right]
$$

where,asappropriate to the object, $h [ 1 , g [ ]$ and $f [ ]$ are schematicand representthe functions forspace motion,parallax,aberration and light-deflection,producinga vector in the GCRS, and are discussed in detail in $\ S 7 . 2$

The transformation $\mathscr { R } _ { \Sigma }$ ,which includes rotations for frame bias,precessionand nutation, transformsposition vectors from the GCRS to the of date system defined by the celestial intermediate pole(CIP).The origin for right ascension,and thus the paradigm,is chosen via the parameter $\beta$ and is described in CapitaineandWallace (2Oo6).Thischoicewill dictate whether right ascensions are measured from the equinox or from the celestial intermediate origin (CIO). Included in $\beta$ ,and depending on the users choice of right ascension origin, will be either,Greenwich Sidereal Time (apparent or mean) or the Earth Rotation Angle, respectively.The last transformation $( \mathbf { W } )$ ,which includes polar motion,transforms from the Terrestrial Intermediate Reference System to the ITRS.The explanation of the theory behind the transformations from the GCRS to the ITRS,precession, nutation,polar motionand Earth rotation are discussed in detail in Chapter 6.

Equation 7.1 expresses the process in terms of the transformation from the celestial to the terrestrial,and this ordering is followed in much of this chapter.Other explanations,for example the International EarthRotationandReferenceSystems Service(IERS)Conventions (IERS 2O1O),adopt the reverse process of transforming from the terrestrial to the celestial. Thus,readers are advised to ensure that the transformations they select for their purpose produces the position in the appropriate reference system.

# 7.1.2 Places

There are various“names”that are used to specify different combinations of the system, the type of position and the location of the observer.These places together with the implied systemand location of the observerare shown in Table7.2.

Proper or virtual1,local and astrometric placesare all described in $\ S \ 7 . 3$ ，although astrometricplacesare consideredto be ina hybrid system.Theremainder are discussed in $\textrm { \scriptsize 8 7 . 4 }$ Itisimportant to stress that for the highest precision it is necessary to apply theappropriate transformations and effects in the correct sequence,evaluated using the appropriate timescale.

A mean place is used to label coordinates of objects outside the solar systemat a specific date.The coordinates represent the direction of the object as it would be observed from the solar system barycentre at the specified date if the masses of the Sun and other solar system bodies were negligible.Mean places are often used in star catalogues.

The terms apparent and intermediate place are used to label positions that are “geocentric of date".These positions include corrections for space motion,parallax,aberration,and lightdeflection,as appropriate to the object.They also include effects of precession.Anapparent place thatis“mean of date”ignoresnutation.The only difference betweenapparent and intermediate place is the location of the origin of right ascension.

Atopocentric place is an apparent or an intermediate place,where the location of the observer is the surface of the Earth and not the geocentre.A topocentric right ascension may be measured from the equinox or the CIO.However,usually, it is GHAorLHA that is being calculated.The algorithm for calculating topocentric places is given in $\ S 7 . 4 . 3$

The geometric place of an object isdefined bya straight line between the centre of the Earth (or the observer) and the object at a given time,without any other corrections.In many cases it is used for clarity as it confirms that no corrections have been applied.

# 7.1.3.1 Right Ascension and Declination

Right ascension is measured in the plane of the equator from an origin on that equator. This origin of the right ascension system may be one of six different locations;ICRS origin, J2000.O origin,GCRS origin,the mean equinox,the true equinox,or the CIO.When using the equator and equinox of date system (see item 5, $\ S \ 7 . 1 . 1$ ),rightascension ismeasured from the equinox and iscalled equinox right ascension; an apparent right ascension is always an equinox right ascension.When using the Celestial Intermediate Reference System (see item 6, $\ S \ 7 . 1 . 1$ ),rightascension ismeasured from the CIOand iscalled intermediate right ascension.Right ascensionand related quantities,such as Greenwich Apparent Sidereal Time (GAST),are often expressed in time-measure $1 ^ { \mathrm { h } } = 1 5 ^ { \circ }$ ,etc.）unless the contrary is indicated.

Declination is measured north and south of the equator.Declinations,apparent or intermediate,derived using either equinox-based or CIO-based methods,respectively,are identical. Declination and related quantitiesare expressed in angular measure.

Right ascension and declination $( \alpha , \delta )$ are transformed from an equatorial rectangular position vector $\mathbf { p } = ( \xi , \eta , \zeta )$ thus

$$
\alpha = \tan ^ { - 1 } \frac { \eta } { \xi } \qquad \delta = \tan ^ { - 1 } \frac { \zeta } { \sqrt { \xi ^ { 2 } + \eta ^ { 2 } } } .
$$

The quadrant of $\alpha$ is determined by the signs of $\xi$ and $\eta$ .Most computers have a double argument function forinverse tangent (e.g.,ATAN2) which willprovide the correct quadrant if the numerator and denominator are entered as separated arguments.Equation 7.2 above assumes that $\mathbf { p }$ is not necessarily a unit vector,and the distance is given by

$$
| { \bf p } | = p = \sqrt { \xi ^ { 2 } + \eta ^ { 2 } + \zeta ^ { 2 } } .

# 7.2.2 Parallax

Parallax is the term used to describe the difference in the apparent direction of an object as seen fromtwo different locations.For example,the Sun,Moon and planetsare observed from the surface of the Earth,and most almanacs publish their geocentric positions.The termisusually used in connection with a shift of origin from the centre of the Earth to the surface (geocentric,a.k.a.diurnal parallax),ora shift from the barycentre or heliocentre to thegeocentre (heliocentric,a.k.a.annualparallax).

The following is a simple and rigorous method of making corrections for both diurnal and annual parallax,and it is recommended that it be used in preference to approximate methods. The position vector of a body ${ \bf u } _ { E }$ ,with respect to the originE,is given by

$$
\mathbf { u } _ { \mathrm { E } } = \mathbf { u } _ { \mathrm { B } } - \mathbf { E } _ { \mathrm { B } }
$$

where $\mathbf { E } _ { B }$ and ${ \bf u } _ { B }$ are the position vectors of the new origin and the body,respectively,with respect to the old origin and referred to the same reference frame.

Approximate methods are given for calculating diurnal and annual parallax,which are useful when the appropriate data are not available; e.g.,the distance ofthe body is not known.

Aberration,because of itsdependence on therelative motionsanddistances,sometimes has complex effects where two or more bodies areinvolved,as,e.g.,in eclipses,transits,and the phenomena of satellite systems;and on some past occasions,the determination of these effects has presented an intricate problem.

Inatransit of a planet across the disk of the Sun,e.g.,the external contacts occur when the observer is on the conical surface that circumscribes the Sun and the planet and has its vertex between the Sun and the planet.The internal contacts occur when the observer is on the cone circumscribing the planet and the Sun having its vertex between the planet and the Earth.The observed contacts are at the instants when the apparent positions of a point on the limb of the planet and a point on the limb of the Sun are the same,i.e.,the ray of light from the Sun that reaches the geometric position of the observer at the instant $T$ of contact hasgrazed theplanet on theway.Thisrayleft the Sunataprevious time $T - \tau _ { 2 }$ and reached the planet at time $( T - \tau _ { 2 } ) + \tau _ { 1 }$ .The circumscribing cones are formed by the grazing rays; hence,the points on the Earth and the planet that lie in the same straight line on one of the cones at the instant of a contact are the geometric position of the observer at the time $T$ , and the geometricposition of the point on the planet at time $T - \tau _ { 2 } + \tau _ { 1 }$ .Therefore,in the formulae of the theory of transits,for any value of the time $T$ ,all quantities depending on the time must be derived from the values of the geometric coordinates $( r , l , b )$ of the planet at $T - \tau _ { 2 } + \tau _ { 1 }$ and the geometric coordinates $( \boldsymbol r ^ { \prime } , l ^ { \prime } , b ^ { \prime } )$ of the Earth at $T$

Similarly,in comparing observed positions of objects in the solar system with one another or with reference stars,in order to determine the coordinates of abody,great care is required in correcting the observations for aberration,according to the means of observation used,and the method of comparison.

In eclipsing binary systems,an apparent variation of the period may be produced by the variation in light-time with changing distance from the observer due to an orbital motion of the eclipsing pair with respect to a distant third component (Irwin 1959).

# 7.2.7.1 Refraction-Numerical Integration

In this section we give a precise algorithm for calculating refraction using numerical integration.The real accuracyis very dependent on the conditions,particularly forlarge zenith distances,and on the atmospheric model.

The atmosphere is assumed to be spherically symmetric and in hydrostatic equilibrium, and to obey the perfect gas law for the combined mixture of dry air and water vapour,and also for the dry air and water vapour separately. The two layers of the atmosphere are the troposphere and the stratosphere.The troposphere,the lowest portion,extends from the Earth to the tropopause,which isassumed to beat $1 1 \mathrm { k m } ( h _ { t } )$ .Inthisregion the temperature decreases at aconstant rate ( $\alpha = 0 . 0 0 6 5$ degrees Kelvin per meter) and the relative humidity is constant and equal to its value at the observer.In the stratosphere the temperature remains constant and equal to the temperature at the tropopause $( T _ { t } )$ and there is no pressure due to watervapour.Theupper height $( h _ { s } )$ of the stratosphere is taken as $8 0 \mathrm { k m }$ ,sinceabove this height refraction becomes negligible.

# 7.3.4 Differential Astrometry

Theproper place,the local place and the astrometric place are all suitable not only for differential astrometry,butalso for any kind of space astrometry where orientation of the Earth(precession and nutation) are irrelevant.

In differential work it is necessary to consider only effects which can alter the angles between the position vectors of the observed bodies,i.e.,arc lengths on the celestial sphere. The orientation of the coordinate system is not considered of fundamental importance since in most cases the celestial and instrumental coordinate systems are coupled.Inany event,in differential observing,the coordinate system is not established until afterthe fact,during the reduction of the observations.

For the reduction of high precision differential observations,the local place should be used.Withavailablecomputingpower itis straightforward to compute the local place of all objectswithina field and use the ensembleoflocal placesasthe startingpoint for the reduction procedure.

# 7.4 Algorithms for the Reductions of Positions

This section presents a setof algorithms for calculating apparent,intermediate and topocentric placesof celestial objects.These algorithms assume that the process starts from a position in the CRS,and in this chapter this is assumed to be the BCRS(ICRS).Other algorithms, such as those of the IERS(URL[5]),start with observations in the TRS,and thus great care is needed to ensure that the various transformations are implemented in the correct sense,i.e., celestialtoterrestrialorterrestrialtocelestial.

An apparent place is a proper place (see $\ S 7 . 3 . 1 $ )transformed toan“ofdate”position,i.e., transformed from the GCRS to a system where the origin of right ascension is the equinox on the true equator of date.If nutation is ignored,then the origin of right ascension will be the mean equinox rather than the true equinox.

An intermediate place is also a geocentric position that includes the same effects as those foranapparent place;however,the origin of right ascension is the CIO,thus the transformation from the GCRS is handled by a different transformation.

A topocentric place is an apparent or intermediate place where the origin of the system is the observer on the Earth's surface and not the geocentre.

All the above places may be reduced so that they are with respect toa terrestrial reference system,the Terrestrial IntermediateReference Systemandthe ITRS.Here itisessential to refer the right ascension (either mean,apparent or intermediate)to the TIO,and thus the appropriate angle,Greenwich Sidereal Time (mean or apparent),or Earth Rotation Angle, must be used.Thus,this will depend on the choice of paradigm,either equinox-based or the IAU recommenced CIO-based (see $\ S 7 . 1 . 3 . 2 $ method.
