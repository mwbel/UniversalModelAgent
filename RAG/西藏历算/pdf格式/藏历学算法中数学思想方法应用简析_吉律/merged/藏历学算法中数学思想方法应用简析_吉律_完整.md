# 藏历学算法中数学思想方法应用简析

吉 律（西藏大学理学院 西藏拉萨 850000）

摘要 藏族天文历算是通过数学运算方法,把各种天体运动的方位、时间等未知量用数字的方式表达，并进行计算的一门自然科学，其内容涵盖了数学、天文学、气象与地理学、医学和民俗学等学科内容。文章以《时轮历精要》和《马杨汉历要旨》的算法和算理作为基本依据，主要采用举例方法，从数学的视角分析和阐述了藏历学计算方法的科学原理和其中所蕴含的现代数学思想方法。

藏族天文历算学是藏族十明文化中最具有代表性的学科，它是藏民族的物候历、印度的时轮历和汉族历法的混合体，其内容涵盖了数学、天文学、气象与地理学、医学和民俗学等诸多学科内容。从数学科学的角度来讲，藏历学中时轮历计算属于代数系统，用加、减、乘、除等基本的运算方法来计算太阳、月亮和五大行星相关的基本数据，推算日、月、五星等七曜在以十二宫和二十七宿恒星为背景的宇宙空间中运动轨迹、时间与方位，以及它们之间的相互位置关系等，而藏传时宪历①的数学原理是球面三角学，以几何学和三角学理论为依据，推算日月食发生的食限标准、食甚时刻等天文学基本数据，确定日食、月食等天文现象，推断这些现象对地球的影响等。

本文以《时轮历精要》和《马杨汉历要旨》②的基本运算方法和原理为依据，从现代数学知识视角对藏历学中的一些常用计算方法进行科学分析，阐述其算法的科学性和所蕴含的数学思想。

# 1 带余除法

藏历学中计算基础数据时用重张位方法，把同一度量系统中不同单位的数码由上而下地分开书写，不同单位的共同乘数则相应地重复写几遍，且用整数表示。在一个度量系统的不同单位间进位时，往往需要将下位的商数加到上位，在商数和余数计算的过程中就能用到带余除法的思想。这种运算方法在藏历学计算中是十分常见的。

1.1 带余除法的定义及定理

定义1：已知整数 $a$ 和非零整数 $b$ ，求一对整数 $q , r$ ，使得 $a = b q + r$ （ $0 \leqslant r < | \boldsymbol { b } | )$ ,这种运算方法叫带余除法，可记为 $a \div b = q \cdots r$ ，其中 $q \mathrm { ~ , ~ } r$ 分别表示商数和余数。如， $1 4 = 4 \times 3 + 2$ ，其意为14除以 4所得商数为3，余数为2，可记作 $1 4 \div 4 = 3 \cdots 2$ 。

定理1：设 $a \ , \ b$ 是整数，且 $b \neq 0$ ，那么存在一对整数 $q \mathrm { ~ , ~ } r$ ，使得 $a = b q + r$ （ $\scriptstyle 0 \leqslant r < | b | )$ ，且满足此式的$q \mathrm { ~ , ~ } r$ 是唯一的。

1.2 带余除法在藏历算法中的应用例1.求第十七饶迥土兔年五月十五日的积月。

积月就是所求日所在之月的月初向前算，距历元时的总积月数。《时轮历精要》中计算积月和闰余的方法为：“从第十四饶迥的第二十一年火猪年起，计算已经过去的年数，乘以12，加上从（当年的）角宿（三月）月起已过月数，把这个数值重张两位。下位乘以2，恒加60，除以65，以其商数加于上位，得朔望月的积月，下位的余数为闰余”。

# 2 高斯函数和商余函数（mod函数）

2.1 高斯函数的定义

定义2：不超过实数 $x$ 的最大整数，记为 $[ x ]$ 。函数 $y = [ x ]$ ， $x \in R$ （其中 $R$ 为实数集）叫做高斯（Gaoss）函数，如 $[ \pi ] = 3$ ， $[ { \sqrt { 2 } } ] = 1$ ， $[ - 1 . 3 ] = - 2$ 。高斯函数被形象地称为“地板函数”，它是取整函数的一种形式，在计算机语言中用函数命令“floor( )”来表示。

2.2 商余函数（mod函数）的定义

定义3：商余函数（mod函数）是一个求余函数，其语法格式为 $\bmod { ( \mathrm { n E x p 1 } , \mathrm { n E x p 2 } ) }$ ，即两个数值表达式作除法运算后的余数。例如， $\mod ( 3 , 2 ) = 1 ; \bmod ( 9 , 3 ) = 0$ 等等。

2.3 高斯函数和商余函数在藏历学算法中的应用

藏历学中重张位计算方法的应用十分广泛，其中蕴含着带余除法和函数学思想方法。带余除法式$a = b q + r$ 可分解成取整函数 $q = f l o o r ( \frac { a } { b } )$ 和商余函数 $r = \mod ( a , b )$ 两个独立的函数。由此可见，人工的重张位计算过程和方法被计算机程序化替代是完全可行的。

# 3 辗转相除法

3.1 最大公约数的性质

定理2：若 $a = b q + r$ （ $0 \leq < | \boldsymbol { b } | )$ ，则 $a$ 与 $b$ 的最大公约数等于 $b$ 与 $r$ 的最大公约数，记作 $( a , b ) = ( b , r )$ 。如， $1 4 = 4 \times 3 + 2$ ，则 $( 1 4 , 4 ) = ( 4 , 2 ) = 2$ 。

3.2 辗转相除法的定义

定义4：设 $a$ ， $b \in N \ ( \ N$ 为自然数集）,若 $a = b q _ { 1 } + r _ { 1 } \left( \phantom { - } 0 \leq r _ { 1 } < \left| b \right| \right) ; b = r _ { 1 } q _ { 2 } + r _ { 2 } \left( \phantom { - } 0 \leq r _ { 2 } < \left| r _ { 1 } \right| \right) ; r _ { 1 } = r _ { 2 } q _ { 3 } + r _ { 3 } \left( \phantom { - } 0 \leq r _ { 2 } - \left| r _ { 1 } \right| \right) .$ （ $0 \leq r _ { 3 } < \left| r _ { 2 } \right|$ ）；… …； $r _ { n - 2 } = r _ { n - 1 } q _ { n } + r _ { n }$ （ $0 \leq r _ { n } < \left| r _ { n - 1 } \right|$ ）； $r _ { n - 1 } = r _ { n } q _ { n + 1 } + r _ { n + 1 }$ （ $0 \leq r _ { n + 1 } < \left| r _ { n } \right|$ ）；而$r _ { n + 1 } = 0$ ，则 $( a , b ) = ( b , r _ { 1 } ) = ( r _ { 1 } , r _ { 2 } ) = \cdots \cdots = ( r _ { _ n } , r _ { _ { n + 1 } } ) = r _ { _ n }$

3.3 辗转相除法在藏历学算法中的应用

《时轮历精要》中计算曜净行的方法为：“置整数，加所求日期，除以14，所得商数若为1和3，表示不均衡，若为0、2和4，表示均衡。均衡即顺序，不均衡即逆序。将此商数关押起来，以其余数查月离步数表的检步序数栏（见表 ），查得位置后，将此数码擦掉，将相应的盈缩积一栏的数值写在弧刻位上，用旁下步损益率乘零数。以上所求日除以14后，所得余数若为零，称为未过，其盈缩积也是零，其下步损益率则用表2中前步的第一行数值。检步序数旁下数，乘“零数”后，除以126，所得商数叫做净行（弧刻），商余乘60，再除以126，得净行分，商余乘6，除以126，得净行息，商余乘707，除以126，得子位，一定能除尽，再无余数。”

分析上述曜净行的算理发现，本算理形式上与定义4所描述的不是完全相同，但其本质仍是辗转相除法。

# 4既约分数

# 4.1 既约分数的定义

定义5：设 $^ { a , b }$ 均为正整数，若 $^ { a , b }$ 的最大公约数 $( a , b ) = 1$ ，则分数 $\frac { a } { b }$ 叫做既约分数（或称最简分数）。

4.2 既约分数在藏历学算法中的应用

时轮历在藏历学中长期居于主导地位，藏历学中的历算术语基本上是时轮历中的术语，较少使用时宪历内容。时轮历中的数学运算是分数运算，且基本数据和计算结果的记数方法采用既约分数，其主要目的一是为了便于记住一些重要的常用数据，二是为了便于表达和计算。如，体系派计算太阴日平均长度的一种方法为：“太阳日平行（长度）60漏刻乘以11135，得668100，除以11312，其商余数乘以60，及6，除以原除数11312，其商余除以16，或乘以707，再除以11312，皆得太阴日平均长度。”（繳纆繼་纉繴་臱་腸臥་肧纀་纈羏纊་纋། 耡繼་纆繳་輂纍་興繴纍་纀繲纈་臶臥་醙醓繻輨繳 纈輨繴་纈輄繻་輅繳་胑纍་醔醔醔醖醘繿軛纊་繿་纋། 耡་纀་纈輄繻་臟纍་醔醔醖醔醕繿羉纍་繽臥་轘繳 纀繲纈་臶་醙醓輂纍་醙繿軛纊་繿羉繻་輩繻་醔醔醖醔醕贂纍། 繿羉纍་轘繳་繿翯་輇繳་醔醙羆纍་繿羉纈纀། 纉繴་繼་臱་纀繲纈་臱纍་醜醓醜繿軛纊་繼纍། 繿羉繻་輩繻་醔醔醖醔醕贂纍་軻繿་轁纍་興繴纍་臫繼། ）

依据以上算法理论，计算出体系派太阴日平均长度为59漏刻3分4息 $\frac { 1 6 } { 7 0 7 }$ ，发现太阳日和太阴日平均长度间有关系：太阳日 × 11135 太阴日，即1太阴日 $= \frac { 1 1 1 3 5 } { 1 1 3 1 2 }$ 太阳日 ${ \approx } 0 . 9 8 4 3 5$ 太阳日。用辗转相除法$1 1 3 1 2 = 1 1 1 3 5 \times 1 + 1 7 7$ ， $1 1 1 3 5 = 1 7 7 \times 6 2 + 1 6 1$ ， $1 7 7 = 1 6 1 \times 1 + 1 6$ ， $1 6 1 = 1 6 \times 1 0 + 1$ ， $1 6 = 1 \times 1 6 + 0$ ，即$( 1 1 3 1 2 , 1 1 1 3 5 ) = ( 1 1 1 3 5 , 1 7 7 ) = ( 1 7 7 , 1 6 1 ) = ( 1 6 1 , 1 6 ) = ( 1 6 , 1 ) = ( 1 , 0 ) = 1$ ，由定义 5 可知， $\frac { 1 1 1 3 5 } { 1 1 3 1 2 }$ 是既约分数，为便于记忆体系派太阴日长度一般用既约分数 $\frac { 1 1 1 3 5 } { 1 1 3 1 2 }$ 来表示。

时轮历的时间和空间计量单位系统中息以下的单位没有单独的名称，计算时不用小数点而用分数来表示，且其分母不是固定不变的，尽量选用能使分子为整数不再有余数的数值做分母，必要时也可用繁分数。例如体系派恒星年的长度为365日16漏刻14分 $1 \frac { 1 2 1 { \frac { 1 2 1 } { 7 0 7 } } } { 1 3 }$ 息 $= 3 6 5 { \frac { 4 9 7 5 } { 1 8 3 8 2 } } = 3 6 5 . 2 7 0 6 4 5 1$ 日，其中分数$\frac { 1 2 1 } { 7 0 7 }$ 和 $\frac { 4 9 7 5 } { 1 8 3 8 2 }$ 都是既约分数。

# 5球面三角

从十六世纪起由于天文学、航海学、测量学等方面的发展，球面三角学逐渐发展成为一门独立学科，其思想方法在藏历学计算日月食等领域中得到了应用。

5.1 藏历学中月食的食限标准

藏历学各派经过长期的观测实践，结合观测点的地理位置情况，经验性地给出了各自判断有无交食的标准，称作食限。《时轮历精要》中按照《日光论》的方法对月食的食限是这样规定的：“看罗睺头尾哪个与十五日体系派的太阴日月宿相近，以小减大，余数宿位为零，弧刻在五十以下者必定有食”。（繻་肑་耡་肵繴་繳輄繴་纀耇繳་繻繴་།繿翴་軬臥་羕繿་輶纍་轁纍་纈罼繻་繳耡纍་། 繳繴་耤་纀繴་纋་耣繴་羆纍་腣繴纍། 轘繳་纀་罜纊་軸繳་醓翽་轂繻་輂། 纀繲纈་纈輨繴་醘醓纀繼་續纊་軥纍་繽纊་纈轃繼། ）这个食限标准中的五十弧刻相当于 $5 0 \div 4 . 5 \approx 1 1 . 1 ^ { 0 }$ ，即月亮出现在黄白交点（罗睺头尾）前后 $1 1 . 1 ^ { \circ }$ 的范围以内时必定有月食发生。

藏传时宪历推算月食可分五个步骤，其中第一步求入交月数是推算月食的最基本步骤，也是判断有无月食发生的关键，只有准确地判断该月有月食发生，才有进一步推算食分大小，食甚时刻等的必要。

《马杨汉历要旨》中判断有无交食的标准是：“实望罗月距（实交周）在十一宫十七度四十三分或五宫十七度四十三分以上，零宫十二度十七分或在六宫十二度十七分以下者有食，即自 $5 ^ { \mathrm { z } } 1 7 ^ { \mathrm { \circ } } 4 3 ^ { \prime }$ 至 $6 ^ { \mathrm { z } } 1 2 ^ { \mathrm { o } } 1 7 ^ { \prime }$ ，$1 1 ^ { \mathrm { z } } 1 7 ^ { \mathrm { o } } 4 3 ^ { \prime }$ 至 $0 ^ { z } 1 2 ^ { \circ } 1 7 ^ { \prime }$ 为必食之限”。（較་繼纍་臟་纈轃繼་繿聏繳་繽་纋། 軥纍་繽臥་迏་繳輄繴་繿纊་罺繻་罻纀། 繳臘་醔醔繻繴་纈羂繳་醘臫繼་纆繳་臱་繳臜繳纍། 醔醜軹繼་轂繻་腷་翽་醗醖纉繼་繶繻་纈輨繴་། 翽་舠繼་醓繱贕་軁་醙臫繼་繼། 纆繳་耡་醔醕軹繼་轂繻་臱་繳臜繳纍་醔醜轀繼། 輨繴་繼་臟་繿་纈轃繼་繽纊་繿續繻།）规定这个月食食限的标准是有数学道理的。
