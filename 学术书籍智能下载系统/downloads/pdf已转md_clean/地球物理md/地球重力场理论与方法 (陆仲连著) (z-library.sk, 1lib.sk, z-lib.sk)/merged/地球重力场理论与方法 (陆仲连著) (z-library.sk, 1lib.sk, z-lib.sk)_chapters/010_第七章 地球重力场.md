# 第七章 地球重力场

# 7.1地球重力位的级数展开式

地球重力场就是地球重力所作用的空间。在地球重力场中，重力的方向和大小随点位按一定的规律变化，也还有重力位、重力线和等位面等要素。地球重力场可以通过这些要素的物理特征和数学表达式来描述。本节将对地球重力位作详细的讨论。

重力位等于引力位加离心力位。由 $( 4 \cdot 7 6 )$ 式可以写出地球重力位为

$$
\mathbf { W } { = } \mathbf { f } \int _ { ( \mathbf { \overline { { M } } } ) } \frac { \mathbf { d m } } { \mathbf { r } } { + } \frac { \omega ^ { 2 } } { 2 } ( \mathbf { x } ^ { 2 } { + } \mathbf { y } ^ { 2 } )
$$

其中右端第二项为离心力位，地球以角速度 $\pmb { \omega }$ 匀速绕Z轴旋转；第一项为地球引力位，积分限为整个地球质量(M),r为地球上的单元质量至求位点的距离。显然，离心力位很简单，无需进一步的讨论。对于地球引力位，由于不规则的地球形状和不均匀的内部质量分布细节都没有精确的了解，要想严格完成积分运算是不可能的。通常将它改写成级数形式，以利于作进一步的分析讨论。

# $8 7 . 3$ 扰动位

由于地球形状的不规则和内部质量分布的不均匀，地球重力场是很复杂的。通过上两节的讨论可知，它可以分解成两部份。一部份是很有规则的，它可看作是一个形状和质量分布都很规则的质体所产生的重力场，而且它已占了地球重力场的绝大部份，通常称之为正常重力场；剩余的微小部份极不规则，称之为扰动重力场。只要求定了扰动重力场，加上已知的正常重力场，实际的地球重力场就可求得。由于扰动重力场中各种元素的量值是很小的，在讨论问题时只需取至线性项，这就给问题的解决带来很大的方便。这就是本书研究地球重力场的基本出发点。

在上一节中已得出,地球重力场中的水准面接近于旋转椭球面。另外，在大地测量中，也以旋转椭球而作为基本的参考面。因此，采用第六章所讲的水准椭球重力场作为正常重力场最为适宜。将产生正常重力场的水准椭球称为正常椭球，它所产生的重力位称为正常重力位，用U表示，而 $\pmb { \gamma }$ 则表示正常重力，利用第六章中的公式就可由正常椭球的已知参数 $( \omega , \mathbf { f } \mathbf { M } , a , \alpha )$ 计算出正常重力场中各种量的数值，如 $\mathbb { U } _ { \cdot ^ { 0 } \colon } \pmb { \gamma } _ { 0 } \ J _ { 2 } \ J _ { 4 }$ 等等。计算 $\pmb { \gamma _ { 0 } }$ 的(6·47)式以后称之为正常重力公式。

在引入正常重力场后，对于空间任意点，都存在两个重力位值：真正的地球重力位 $\mathbf { w }$ 和正常重力位U。这两者之差就是扰动重力场中的重力位值，称为扰动位，用 $\boldsymbol { \mathsf { T } }$ 表示之，则有

$$
\mathbf { T } = \mathbf { W } - \mathbf { U }
$$

由此式可得

$$
{ \bf { \Delta } } \mathbf { w } { = } \mathbf { { \dot { U } } } { + } \mathbf { { \dot { T } } }
$$

这就是说，地球重力位等于正常重力位加扰动位。

无论是地球重力位，还是正常重力位 $\dot { \mathbf { \Omega } }$ 都是由引力位 $\mathbf { v }$ 和离心力位 $\mathbf { Q }$ 两部份组成的。因此,由 $( 7 ~ { \cdot } ~ 3 4 )$ 式可以写出 ：

$$
\mathbf { T } { = } \mathbf { W _ { E } } { - } \mathbf { V _ { H } } { + } \mathbf { Q _ { E } } { - } \mathbf { Q _ { H } }
$$

该式右端字母右下角写E表示该量属于实际地球，而写H则表示该量属于正常地球。因为VH和QH不但与正常椭球的参数有关，也还与坐标有关。因此，扰动位不但取决于正常椭球参数的数值，也还与正常椭球与实际地球之间的相关位置有关。

为了使得扰动位单纯些，最好使QH=Qε,这就要求正常椭球的旋转轴(短半轴)与实际地球的旋转轴重合，同时两者的旋转角速度相等。此时，扰动位就只是两个引力位之差了，即

$$
\pmb { \mathrm { T } } \mathrm { = } \pmb { \mathrm { V } } _ { \Xi } \mathrm { - } \pmb { \mathrm { V } } _ { \Xi }
$$

这样，扰动位就可以理解为：由于地球质量的分布与正常椭球的质量分布不一致而引起的引力位差。若把这两种质量之差(注意，这里指的是每点的质量差，而不是总的质量差)称为扰动质量,那末扰动位就是由扰动质量所构成的质体的引力位。因此，对于外部点，扰动位是个调和函数。

既然扰动位是一个调和函数，也可用一个球谐函数级数式表示。由(7·36)式可知，扰动位的级数式可以由地球引力位和正常引力位的两个展开式相减而得。由(6·59)式，在坐标原点重合于正常椭球的质心,Z轴重合于正常椭球的短轴时，正常引力位VH的级数展开式可写为
