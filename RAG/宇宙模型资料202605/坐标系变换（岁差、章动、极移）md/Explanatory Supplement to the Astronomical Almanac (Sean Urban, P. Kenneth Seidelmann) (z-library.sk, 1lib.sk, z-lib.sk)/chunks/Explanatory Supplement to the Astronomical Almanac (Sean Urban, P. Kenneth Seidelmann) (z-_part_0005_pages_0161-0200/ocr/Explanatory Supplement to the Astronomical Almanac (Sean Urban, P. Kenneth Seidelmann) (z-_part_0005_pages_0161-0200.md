# 5.2 Terrestrial Reference Systems

# 5.2.1 Reference Systems and Reference Frame Concepts

Locating a point on or near the Earth's surface requires the definition of areference system that enables aset of coordinates to be unambiguously associated with or assigned to the point. It also provides a consistent means for identifying the location of one point with respect to another.Todothis,the conceptofaterrestrialreference systemis introduced.Suchasystem is generally defined by three orthogonal Cartesian coordinate axes $( x , y , z )$ fixed to the Earth's crust,thelocationof the origin,a specified orientation in space,anda setofmathematical and physical models,constants,conventions,and parameters.A geocentric terrestrial reference system is one in which the origin coincides with the geocenter-the center of mass of the solid Earth,oceans and atmosphere-and in which the coordinate axes rotate with the Earth. Thishas been adopted by the international scientific communityas the standard conceptual definition of a global geodetic reference system.The $z$ -axis is oriented along the rotationaxis of the Earth,the $x$ -and $y$ -axesare in the equatorial plane,the $x$ -axis intersects the equator at the point where areference(or zero)meridian crosses it,and the $y$ -axisis perpendicular to the $x$ -and $z$ -axestocompletearight-handed system.Astandard set of proceduresand models (conventions),prescribedby the International Earth Rotation and Reference Systems Service (IERS),formally relates this Earth-centered Earth-fixed reference system to the Celestial Reference System (see Chapter 4).The geocentric terrestrial reference system with these adopted conventionsis designatedas the International Terrestrial ReferenceSystem (ITRS) (IERS 2004).

The practical realization of a terrestrial reference system is accomplished by establishing a set of reference points on the topographic surface of the Earth and determining their 3- dimensional Cartesian coordinates and velocities from actual observations,currently done byusing space geodetic techniques and applying the reference system conventions.These physical points and theunderlying conventions then define the geocentric terrestrialreference frame.The more accurately the reference point positions can be determined,the better the reference framecan be realized.

Two of the most widely used geocentric terrestrial reference frames,the International Terrestrial Reference Frame (ITRF) and theWorld Geodetic System (WGS),are described in more detail below.Inaddition,South Americaand Europe have implemented regional realizations of the ITRS—Sistemade Referencia Geocentrico para Las Américas (SIRGAS) (Hoyeret al.1998;URL[1]) and the European TerrestrialReferenceFrame(ETRF),(URL[2]) respectively.By adopting the IERS Conventions and incorporating ITRF reference points (stations) into the individual reference frame realizations,WGS,SIRGAS and ETRF(and others)attempt to constrain theirframes to have the same origin,orientation and scale as the ITRF.

# 5.2.1.1The International Terrestrial Reference System

The International Terrestrial Reference System (ITRS) specifiesaset of physical constants, conventions and models to be used for estimating the coordinates of points with respect to the Earth.The ITRS is defined to be geocentric with its orientation traceable to that of the Bureau International deI'Heure (BIH) Terrestrial Systemat 1984.O; its standard unit of length is the meter(SI),and its time evolution is constrained toa conditionof no-net-rotation with regard to horizontal plate tectonic motions over the Earth asawhole (IERS2OO4,p.25).

Thephysical realization (or materialization) of the ITRS is the International Terrestrial Reference Frame (ITRF),definedby the 3-dimensional coordinatesand linearvelocities ofa global network of ground stations(on the Earth's crust) at a specific epoch in time. Thesecoordinates and velocities are derived from observations collected at the stations using four space-based techniques:Very Long Baseline Interferometry (VLBI),satellite laser ranging (SLR),the Global Positioning System (GPS),and Doppler Orbitography and Radiopositioning Integratedby Satellite (DORIS). The origin,coordinate system orientation andthe scaleareimplicitlydefined by theCartesiancoordinatesand velocities ofthe stations, and the solutions conform to the ITRS and IERS conventions and standards.This represents the fundamental scientific standard reference frame (Boucher and Altamimi1996).

The ITRS isa direct continuation of the BIHTerrestrial System (BTS),first established in 1985and continued through 1987.The BTS had in turn evolved fromthe BIH1968 System, which was established in 1969 by the BIHon the basis of a comparison of data collected by the BIHand the International Polar Motion Service (IPMS) during the mid-196Os.However, it should be emphasized that this was not a complete terrestrial reference system; it was a referencesystemfor orientation only,being based on the observed astronomical coordinates of the involved observing stations.The comparison of the BIHcollected data with the IPMS data allowed the BIH1968 System to be referenced to the IPMS reference pole.This pole is the Conventional International Originl which was the approximate average position of the true celestial pole ofthe Earth from19o0.O to 1905.O,but wasactually definedby the adopted latitudes of five International Latitude Service (ILS)stations.(For further information on the history of the terrestrial reference frame prior to 1988,see IERS (1989),or the BIH Annual Reports for the appropriate year(s),or see Mueller(1969) for further information on these systems for the pre-1969 period.)

The IERSReference Pole (IRP)and Reference Meridianare consistentwith the corresponding directions in the BIH Terrestrial System(BTS) within $\pm 0 . 0 0 5$ The BIH reference pole was adjusted to the Conventional International Origin in 1967.It was then kept stable independentlyuntil1987.Re-reductionof the available data (BIH1989,p.A-3)tied the BTS (and therefore the IRP) with the Conventional International Originto $\pm 0 \prime \prime 0 3$ .

Asnoted in the2O10 update to the IERSconventions (IERS 2O1O,chapter4),there have been 12 versions of the ITRF released since 1988.ITRF2o05 used,for the first time as input, time seriesof station positions and Earth orientation parametervalues from each of the space geodesy techniques.The most recent realization of the ITRF was computed by the IERS in 2008 with an epoch of 2Oo5.0 (Altamimi,Collilieux and Métivier 2O11).This solution used the same procedure as ITRF2oo5,incorporating observations from VLBI (29 years), SLR(26 years),DORIS (16 years) and GPS (12.5 years).ITRF2008 is constituted by 934 stations at 580 sites, with 105 of those sites supporting two or more of the four space geodesy techniques.Transformation parameters computed from ITRF20o8 to ITRF2005 show no rotational differences,about a $5 \mathrm { m m }$ difference in the location of the origins,and O.94 parts per billion scale difference (IERS 2010,Table 4.1).

# 5.2.1.2 The World Geodetic System1984

The terrestrial reference system used by the U.S.Department of Defense is the World GeodeticSystem 1984(WGS84) (NIMA1997).It isalso the standard reference system for the Global Positioning System(GPS).WGS 84 actually constitutes acoherent set of global modelsand definitions that include the WGS 84 terrestrial reference frame,aprescribed set of constants,models and conventions that are largely adopted from the IERS Standards (IERS 2004;IERS1996),an Earth Gravitational Model (currently EGM2008),and a World MagneticModel (currentlyWMM201O).For more detailsonEGM2008,seePavlis etal.(2012) and URL[4],and formore detailsonWMM2010,seeURL[5],URL[6],andMaus (2010). The Defense Mapping Agency was responsible for the initial development and maintenance oftheWorld Geodetic System,and this responsibility passed to its successor agencies,the National ImageryandMappingAgency (NIMA) and theNational Geospatial-Intelligence Agency (NGA).The World Geodetic System geocentric terrestrial reference frame Was first developed in 1960 with subsequent updates in 1966 (WGS 66),1972 (WGS 72),and 1984 (WGS 84).The original WGS 84 reference frame was realized from aset of Transit satellite (Doppler) stationcoordinates and aligned ascloselyas possible with the origin,scale and orientation of the BIH Terrestrial System.Transit-realized WGS84 coordinates were derived for the Department of Defense GPS monitor stations and were used until1994.Refinements were made to theWGS84 reference frame in1994,1996,and 2002and the reference frames are respectively designated asWGS 84(G730),WGS84 (G873) and WGS 84(G1150).The G'indicates that the reference frame is derived from GPS observations and the number is the GPS Week Number that the reference frame became operationally effective for GPS orbit production by NGA and the Air Force's GPS Operations.

TheWGS 84 reference frame is defined bya global network of GPS stationswhose coordinatesare closely alignedto the ITRF.This has beendonebyconstrainingthe coordinate solutions to the ITRF through the incorporation of up to several dozen independent GPS reference stations with well-defined ITRF positions and velocities in these solutions.Each realizationof the WGS 84 reference frame has improved the reference frame accuracy relative totheITRF.WGS84(G1150) wasaligned to ITRF200O to approximately $1 \ \mathrm { c m }$ ineach coordinate (Malysand Slater 1994;SlaterandMalys1998;Merriganetal.20O2).Another realignment of the WGS 84 referenceframe to ITRF2008 is expected inthe 2011-2012 time frame,as wellas adoption of relevant parts of the IERS Conventions (2OlO)(IERS 2010) into theupdatedWGS 84 reference system.

Since the GPS reference orbits computed by the Air Force are in the WGS 84reference frame,the broadcast satelite navigation message orbits generated from these are also referenced to WGS 84.Therefore,any position coordinates derived directly from the GPS broadcast orbits willalso bein the WGS 84 reference frame.Due to the few centimeter alignment of the WGS 84 reference frame with the ITRF,for most practical applications,WGS 84 and ITRFare considered equivalent,and no transformation is required between the reference frames.

Notethat,up to this point,there has been noexplicit need to introduce aconceptual model for the Earth's surface;however,most positioning and navigation applications need to convert Cartesian coordinates to more meaningful coordinates that refer to some representation of the surface fromalocal,regional or global perspective.Froma purely geometric perspective, thebest fiting figureof theEarth isarotational elipsoid since its shapecanconform to theflatteningof theEarthatthepolesand itisrelatively simplemathematically.(Amore complicated surface,such asa triaxial ellipsoid,will not greatly improve the representation, but will greatly complicate the mathematics involved in using it.) Latitude and longitude can be defined as horizontal coordinateswith respect to the elipsoid surface,and heights can bereferenced to the ellipsoid aswell. Conventionally,the longitude line passing through Greenwich,England was chosen as the zero longitude and the equator defined zero latitude.

Prior to the era of satellite geodesy,individual countries or geographic regions would adoptreference ellipsoids that seemed to best fit the country or region,but asa result, these ellipsoids were frequently not geocentric and not necessarily aligned with the Earth's rotationaxis and equator. For global applications and especially satellite operations,reference ellipsoids are defined to be geocentric and their semimajor and semiminor axes aligned with the axes of a geocentric terrestrial reference frame.Coordinates such as those derived from GlobalPositioningSystemobservationscanbereadilyconverted fromCartesian $x$ , $y$ ,zto latitude,longitudeand height above the ellipsoid or the reverse (see $\ S 5 . 2 . 4 )$ .Thechoiceofa reference ellipsoid is inherent in the definitions of geodetic datums (see $\ S 5 . 2 . 5 . 3 \}$ and map projections (see $\ S 5 . 2 . 5 . 2 \}$ which will be discussed laterin thischapter.

Although mathematically pleasing,the abstract nature of the ellipsoid makes it less than optimal for visualizing terrain or the sea surface,and it cannot be used to determine the direction that water willflow.A reference surface that is better suited for these purposes is onedefinedbyalevel surface of equal gravity potential,also calleda geopotential surface or geop.Note that gravity potential here includes the centrifugal potential and the gravitational potential.There are aninfinite number of geopotential surfaces that theoreticallycan be defined;however,the one that“best fits”the mean sea level ischosenasthe standard geopotential reference surface.This particular level surface is called the geoid.Conceptually, thegeoid can be thought of as an imaginary“surface coincidingwith the mean sea-level in the oceans,and lying under the land at the level to which thesea would reach if admited by smallfrictionless channels"(Bomford1971).Various otherdefinitions are givenin Heiskanen and Moritz(1967,pp.48-50);Moritz(1980,p.6);and Torge(1980,pp.2-3).Elevation can then be expressed as height above mean sea level or height with respect to the geoid, where mean sea level and the geoid define zero elevation.Atany point on the Earth,the gravity vector will be perpendicular to the level surface (geop) that passes through that point. Frequently,instruments or sensors operating on the topographic surface of the Earth need to bealigned with the local vertical,in effect the local gravity vector,and accomplish this through the use of plumb bobs,bubble levelsand poolsof mercury.The geopotential surface isalso a critical construct in hydrology,since water bodies conform to a level surface and water flows“downhill’from higher potential to lower potential.Itis important to realize thatthe geoid surface is irregular and determined by the distribution of the local underlying mass.In practice,gravity potential must be derived fromactual aerial and terrestrial gravity measurements,and satellite orbital data.On the other hand,the mean sea surface is derived from global satellite altimetry measurements of the sea surface and local coastal tide gauge observations.

# 5.2.3 Geodetic Datums

The set of parameters that defines the relationship between a specific reference ellipsoid and aterrestrial referencesystem iscalledageodeticdatum.A global geodeticdatumisone in which the reference ellipsoid is the“best fit”for the Earth as whole,its origin is geocentric, andits major and minor axes are coincident with the coordinate axes of the adopted terrestrial reference system.A local geodetic datum,on the other hand,is one in which the reference ellpsoid is frequently not geocentricand itssurfaceisabest fitonlytothelocal orregional areafor which it is intended to be used.The axes of the local reference ellipsoid also may not be aligned with the coordinate axes ofa geocentric terrestrialreference system.For mapping andsurveying applications,especiallypriorto satellite technology,datums were traditionally defined aseither horizontal datums or vertical datums.The horizontaldatumprovidesameans of identifying latitude and longitude of apoint on the surface of the reference ellipsoid,while the height of the point is separately determined with respectto local meansea level orageoid model (the vertical datum).

Alocal horizontal datum is defined by the size and shape of the reference ellipsoid and by the positioning of the ellipsoid relative to the figure of the Earth (represented by the geoid). The ellipsoid's size and shape are specified by two parameters,generally its semimajor axis $( a )$ and flattening $( f )$ (therelative difference in length of the semimajor and semiminor axes).An initial point (origin) is chosen near the surface of the ellipsoid (specified by its astronomical latitude and longitude, $( \Phi , \Lambda ) )$ ,andthedeflectionoftheverticalandgeoid undulation $( N )$ determined at the chosen origin along with an azimuth $( \alpha )$ from the origin to another point. See $\ S 5 . 2 . 5 . 3$ andFigs.5.2and5.3.The ellipsoid ischosen so as to represent the geoid aswellaspossible over the area of interest (a region or the entire Earth).The position of the ellipsoid with respect to the geoid is given by the deflection of the vertical (theangle between the local vertical gravity direction and the elipsoidal normal) and the geoid undulation.Historically,local datums were usually established in support of large triangulation networks.The origin was chosen to be at the center of the network in order to minimize errors in measurements relative to the origin as one moved outward in the network.In order to simplify the datum definition,the deflectionof the vertical and geoid undulation were frequently chosen to be zero at the origin,in effect forcing the ellipsoid and geoid to be coincident at that point.By imposing the condition that the vertical axis of the reference ellipsoid be parallel with the Earth's rotationaxis,the deflection of the vertical and geoid undulation can then be related to the offset between the center of the ellipsoid and the geocenter. Thisis equivalentto a translational shift in $x$ , $y$ and $z$ between a coordinate system whose origin isat the geocenter andacoordinate system whose origin isat the center of the local datum's ellipsoid. Specific datums and datum conversions are discussed in $\ S 5 . 2 . 5 . 3$ and $\ S 5 . 2 . 5 . 4$ .

Many maps are still based on local datums.In these cases the reference ellipsoid is used onlyas a horizontal (2-dimensional) datum,whose origin and orientation are defined by six topocentric parameters.North American Datum 1927 (NAD 27) is one example.Its origin isasurvey point inKansas.North American Datum 1983(NAD 83) replacedNAD 27 asthe offcial horizontal datum for the U.S.in 1989 (NOAA 1989).It was derived from Doppler satellite observationsasageocentric global datum.NAD 83 removed significant local distortions inNAD 27and replaced the Clarke 1866 reference elipsoid with the Geodetic Reference System 198O (GRS 80) elipsoid.However,it still retained some biases in the origin and orientation and wasused primarily as a horizontal datum.Beginning in 1994,to meet the growing demands and capabilities of geodetic surveying with GPS,the National Geodetic Survey organized a network of continuously operating reference stations that were positioned using GPS and tied to the ITRF.Additional stations have been added to the network over time and new 3-dimensional coordinates have been computed,tying these stations to the latest realizations of the ITRF(Snay and Soler 2000).

Avertical datum is the reference surface for orthometric heights.Vertical datums are traditionally definedasMean SeaLevel (MSL) oraveraged tidal observations based on low or high water (for example,Mean LowerLow Water).Many national and regional vertical datums are tied to a local mean sea level,which may differ significantly (decimeters) from global meansea level due to local effects such as river run off and extremes in coastal tidal effects.Thus,such nationaland regional vertical datumsaround the world willdifferfrom one another significantly when considered onaglobal basis.North American Vertical Datum1988 (NAVD 88)applied this concept by adopting the single tide gauge elevationatPoint Rimouski, Quebec,Canada as the continental elevation reference point and essentially references all other elevationsin the U.S.to this.NAVD88 was designated the official vertical datum of theU.S.in1993.Bycontrast,theU.S.National VerticalDatum1929(NGVD29)was fixed to a set of reference tide gauges,without correction forlocal variations in the sea state,as a method of defining the vertical reference.

# 5.2.5.1 Horizon System

Depending on the author,slight variations exist in the definition of any local horizon coordinatesystem.However,a more or less“standard”system is described as follows.

First,any such system must have at least two reference planes.The primary reference planehereisthe local horizon,and the secondary is the localmeridian (seeFig.5.6).The horizonis theplaneperpendicular to the observer’slocal vertical,while the meridianis the plane perpendicular to the horizon that passes through the north and south celestial poles,the observer and the zenith.The direction of any point $p _ { 2 }$ (from the observer at $p _ { 1 } )$ in thissystem is then given by its altitude and azimuth.Ifwe imagine a plane passing through $p _ { 2 }$ and the zenith $z$ ,theanglebetween the horizonand thepoint $p _ { 2 }$ inthisplaneis thealtitude $a$ (also sometimes knownas the elevation angle).The angle between the meridianand thisplane measuredclockwisefromnorth,asviewed fromabove,istheazimuth $A$ .(Some users in the past,particularly for geodetic purposes,have defined azimuth as being measured clockwise from south.This practice isno longer much in use but users of old observations should be aware of it.) Thealtitudeisdefined from $- 9 0 ^ { \circ }$ to $+ 9 0 ^ { \circ }$ (negative if the object is below the horizon and positive if above),and the azimuth ismeasured from $0 ^ { \circ }$ to $3 6 0 ^ { \circ }$ .Instead of this spherical coordinate system,a left-handed three-dimensional Cartesian coordinate system is also used,with the $z$ -axispositive up,the $x$ -axispositive north,and the $y$ -axispositive east (through $9 0 ^ { \circ }$ azimuth).

Afurther variation of this system is to use the zenith distance $z$ instead of the altitude. The zenith distanceis simply the complement of the altitude,i.e.,the angular distance of $p _ { 2 }$ from the zenith z.It has the advantage of always being positive,with values from $0 ^ { \circ }$ to $1 8 0 ^ { \circ }$ It is computed from altitude as

$$
z = 9 0 ^ { \circ } - a
$$

where the $z$ and $a$ are in degrees.

# 5.2.5.2 Grid Coordinate Systems

For computational convenience and for ease of use inlocal and regional areas,itis common to represent sections of the curved surface of the Earth with a plane.Various systems have been devised to do this,all based on given transformation equations from the sphere or ellipsoid to the plane.These transformations are,in fact,map projections,in many cases the same as those used for mapping large regional areas or the entire Earth.Conformal projections arecommonly used because they preserve the true shapes of small areas;relative angles (directions) about any point are approximately correct;and the local scale inall directions about any given point is constant.In these projections,the meridians and parallels of latitude intersect at right angles.

The coordinate systems based on such map projections are usually termed grid coordinate systems.Themostwidelyusedare theUniversalTransverseMercator(UTM),UniversalPolar Stereographic (UPS),and Lambert conformal conic (LCC) projections.In the United States, the state plane coordinate systems are based on UTMorLCC projections (Burkholder 2008, chapter1O) while the U.S.National Grid (USNG) System(FGDC 2001) is based on UTM and theMilitary Grid Reference System (seeUniversal TransverseMercator grid section below).Obviously many other systems exist for various countriesand areas,but the basic principles (and many times the projection formulae,with the exception of constants) are the same.

# 5.2.5.4 Datum Conversions

The subject of converting coordinates from one datum to another is a complex one,with a very extensive literature.Indeed,the problem almost always arises and isusually discussed whenever a new datum or reference system is formed,since there will usually be a need to transform the coordinates of points in other systems to coordinates in the new system.

Two general methods have been used in the past for datum transformations.The first isa differential method,where differential (in the partial-derivative sense) equations are formed that give the change in station coordinates $( \Delta \phi , \Delta \lambda )$ as a function of changes in datum/ellipsoid parameters (such as for changes in $^ a$ ， $f$ ,scale,etc.).Thistype of method has been most useful in the past where small differences exist between the two datums,or a minor change inan existing datum was being made.However,the derivation ofthe differential equations for a specific case and their application and accuracy is a complicated subject and will not be discussed further here.[Information on this type of method is available in Rapp (1980,pp.42-74),Bomford (1971,pp.199-207),Heiskanen and Moritz (1967,pp.204- 210),Molodenskii,Eremeev,and Yurkina (1962,pp.13-17),Torge(1980,pp.150-152), and DMA (1987,part 1,pp.7-1 to 8-8).]

A second method is to do a rigorous transformation between the systems of each datum. Thisisoftenrequired if highaccuracy isdesired,or thereare several or large differences between the datums.For any pair of datums,transformation parameters may have already been determined,at least for offsets and sometimes for rotation and scale.A three-step procedure is required to convert $( \phi , \lambda , h )$ from system1 to system 2

$$
\begin{array} { r } { ( \phi , \lambda , h ) _ { 1 } \Rightarrow ( X , Y , Z ) _ { 1 } } \\ { ( X , Y , Z ) _ { 1 } \Rightarrow ( X , Y , Z ) _ { 2 } } \\ { ( X , Y , Z ) _ { 2 } \Rightarrow ( \phi , \lambda , h ) _ { 2 } . } \end{array}
$$

The geodetic to Cartesian and Cartesian to geodetic transformations have already been described in $\ S \ 5 . 2 . 4$ .Ingeneral,the transformation of one Cartesian system into another Cartesian system requiresa translation of the origin and arotation of the coordinate axes of the first system toalign itwith the second system.This takes the form ofa7-parameter similarity transformation:

$$
{ \left[ \begin{array} { l } { X } \\ { Y } \\ { Z } \end{array} \right] } _ { 2 } = { \left[ \begin{array} { l } { \Delta X } \\ { \Delta Y } \\ { \Delta Z } \end{array} \right] } + { \left[ \begin{array} { l l l } { ~ D } & { - R _ { 3 } } & { R _ { 2 } } \\ { ~ R _ { 3 } } & { ~ D } & { - R _ { 1 } } \\ { - R _ { 2 } } & { ~ R _ { 1 } } & { ~ D } \end{array} \right] } { \left[ \begin{array} { l } { X } \\ { Y } \\ { Z } \end{array} \right] } _ { 1 }
$$

where $\Delta X , \Delta Y$ and $\Delta Z$ are the components of the origin translation; $R _ { 1 } , R _ { 2 }$ ,and $R _ { 3 }$ are the rotation angles (using small-angle formulae here); and $D$ is the scale factor (ISO 2009, p.343).Various forms of this 7-parameter transformation exist under various names (some notasrigorousasothers).(SeeKumar1972,Rapp1980,p.48,NIMA1997,Seeber2003, pp.27-28,and Hofmann-Wellenhof,Lichtenegger,and Collins2oo1.)It isalso possible that some of the parameters may not be known or can be assumed to be zero.

Insome cases,the transformationparametersarenotwel-determined or not knownat all. The user must then determine the parameters directly from observations that tie stations together between the two systems.If enough points have their coordinates determinedin both systems,Eq.5.57or similar equations can be solved(usuallyinaleast-squares solution) for thetransformation parameters.Of course,this may not takeinto account any deformations of either of the datums from their idealized reference systems (due to surveying errors).Any deformations willbe absorbed into the transformation parameters when they are solved for. Software for converting geographic coordinates among a wide variety of coordinate systems, map projections,and datums can be found at URL[9].

In comparing or transforming modern geocentric terrestrial reference frames,in contrast to classical datums,all the transformation parameters tend to be very small or close to zero. Thisis due to the adoption of common standards and conventions (e.g.,IERS conventions), use of common equipment,and similar data processng techniques.The realizations of the reference frames willdifferdue todifferences in the observing stations andthe software used, among other things.There are essentially two options in satellite geodesy forcomputing transformations between reference frames.One can compute the orbits of the same satelites (for example,GPS) in twodifferent reference framesand then determine7-parameter transformationsbetween the orbits.Alternatively,ifanetwork of stations isavailable with coordinates independently determined from both reference frames,then one can computea7-parameter transformation between the two sets of station coordinates.As the accuracy and precision of measurements and modeling have improved,it is now common to take into account for scientific purposes the time-dependency of the7 parameters,resultingin computationof 14-parameter transformations (see URL[3]).

To avoid transformations completely,one could measure the coordinates of all the desired points with a GPS satellite receiver—thus determining the coordinates of the points in some current reference system,bypassnganytransformationsentirely.Thiseliminates theneed to use the old datums or old observations.The coordinates can be measured in an absolute sense,relative to the center of the Earth at decimeter accuracy or betterif sufcient data arecollected (Leandro,Santos,andLangley 2Oo7;Bisnath and Gao 2Oo9;Zumberge et al. 1997).Coordinates can also be determined relative to known reference points that are close by( $1 ~ \mathrm { k m }$ orless) orlonger distances away(1Os to 1oOs of kilometers) to accuracies of a few centimeters or even millimeters (URL[1O];Dixon 2Oo6) by using the appropriate receiver equipment,data processing techniques and sufficient data,depending upon the application and theaccuracy required.The resultingcoordinates will be consistent with the geodetic reference frame and datum used for the data processing.

# 5.3 Gravity,Tides,and Motions of the Crust

In thissection,variousphenomenawill be covered that affect theEarth'sgravity field and coordinates of a given station.A very short summary of one method of modeling the Earth's gravity field (as a spherical harmonic series)is given,along with the IERS recommendations fora“standard”gravity field.Thisis followed by,almost verbatim,theIERSConventions (2010)recommendations on Earth tides,ocean tides,ocean and atmospheric loading,and plate motions (IERS2010,2O04).Finally,the effect of tides on the Earth's rotation itself is described,as given by the IERSConventions(2010).

# 5.3.1Modeling the Earth's Gravity Field

TheEarth'sgravitycan be considered as a force that is the sum of the Earth's gravitational force and the centrifugal force resulting from the Earth's rotation.

The Earth's external gravitational field can be considered a potential field whose strength $V$ fulfills theLaplaceequation

$$
\Delta V = 0
$$

where $\Delta$ is the Laplacian operator

$$
\frac { \partial ^ { 2 } } { \partial X ^ { 2 } } + \frac { \partial ^ { 2 } } { \partial Y ^ { 2 } } + \frac { \partial ^ { 2 } } { \partial Z ^ { 2 } }
$$

in a cartesian $X$ , $Y$ ,and $Z$ coordinate system.

It is possible to develop a solution to Eq.5.58 (HeiskanenandMoritz 1967,pp.15-35; Torge1980,pp.26-29) in terms of a series of Legendrepolynomials:

$$
P _ { n } ( t ) = \frac { 1 } { 2 ^ { n } \times n ! } \times \frac { d ^ { n } } { d t ^ { n } } ( t ^ { 2 } - 1 ) ^ { n }
$$

with

$$
t = \cos \psi = \cos \theta \cos \theta ^ { \prime } + \sin \theta \sin \theta ^ { \prime } \cos ( \lambda ^ { \prime } - \lambda ) .
$$

The variable $\psi$ is the spherical distance on a unit sphere between an atracted point (at spherical coordinates $r , \theta , \lambda )$ andanattractingpoint (at $r ^ { \prime } , \theta ^ { \prime } , \lambda ^ { \prime } )$

In order to express $P _ { n } ( t )$ as a function of the spherical coordinates $\theta$ and $\lambda$ ,itmaybe decomposed using associated Legendre Functions of thefirstkind of degree $n$ and order $m$ obtainedbydifferentiating $P _ { n } ( t ) m$ times

$$
P _ { n m } ( t ) = ( 1 - t ^ { 2 } ) ^ { m / 2 } \frac { d ^ { m } } { d t ^ { m } } P _ { n } ( t ) .
$$

It can then be expressed by the functions

$$
\begin{array} { l } { P _ { n m } ( \cos \theta ) \cos m \lambda } \\ { P _ { n m } ( \cos \theta ) \sin m \lambda } \end{array}
$$

known as surface spherical harmonic functions.
