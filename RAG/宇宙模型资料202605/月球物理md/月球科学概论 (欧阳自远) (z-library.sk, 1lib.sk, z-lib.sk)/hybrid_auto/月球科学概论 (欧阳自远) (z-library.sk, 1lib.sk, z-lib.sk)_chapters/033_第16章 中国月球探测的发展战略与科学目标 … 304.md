# 第1章绪论

自古以来, 人们就对月球寄予了真情的遐想和诗意的赞美。在古代, 当一轮皎洁如玉的明月挂在夜空, 人们只能靠肉眼观察月球并寄托自己的想象。直到16世纪望远镜发明以后, 人类才能够用望远镜观测月球。人们发现月球上有高山和广阔的平原, 并且布满了环形山, 但望远镜观测的空间分辨率大于 $10 \mathrm{~km}$ 。

对月球的真正了解是在20世纪50年代以后，月球探测进入深空探测阶段。月球是离地球最近的天体，亮度也仅次于太阳，历来是人类天文活动的首选目标，自然也是人类走出地球摇篮，迈向浩瀚宇宙的第一步。1958年至1976年，在人类对月球的各种探测活动中，苏联的Luna(月球)计划成功拍摄了月球背面的照片，第一次把月球的整个面貌展现在人们面前；1969年，Apollo(阿波罗)11号实现了人类的梦想，人类的足迹第一次踏上了月球，人们才全面了解到一个真实的月球。

# 1.1 早期探测

在冷战背景下, 美国和苏联展开了以月球探测为中心的空间竞赛, 掀起了第一次月球探测高潮。自1958年至1976年, 美国和苏联共发射108个月球探测器, 成功或部分成功52个。1969年7月, 美国Apollo11号飞船实现了人类首次登月, 紧接着Apollo12、14、15、16、17和苏联的Luna16、20和24进行了载人和不载人登月取样, 共获得了 $382\mathrm{kg}$ 的月球样品和海量的科学数据, 月球探测取得了划时代的成就。

月球探测是人类进行太阳系探测的历史性开端，大大促进了人类对月球、地球和太阳系的认识，带动了一系列基础科学的创新，促进了一系列应用科学的新发展。月球探测，尤其是载人登月是人类迈出地球摇篮的第一步，是整个人类历史进程的里程碑。人类在宇宙空间展示的智慧创举、超强能力和攀登精神，是人类开拓进取、求实创新的光辉范例，增强了人类探索宇宙、建设好地球家园的信心。月球探测成为人类历史上划时代的标志性事件。

# 1.1.1 第一次月球探测高潮

第一次月球探测高潮最主要的推动力是冷战和空间霸权争夺的政治需求。美国和苏联正是通过月球探测，建立和完善了庞大的航天工业和技术体系，有力地带动和促进了一系列科学技术的快速发展；月球探测技术在军事和民用领域得到延伸、推广和二次开发，形成了一大批高科技工业群体，包括微电子、计算机、遥感、遥测与遥控、微波技术、红外与激光、超低温、超高温和超高真空技术，以及冶金、化工、机械、电子视听声像和信息传递等，产生了显著的社会经济效益。据不完全统计，从Apollo计划派生出了大约3000多种应用技术成果。在登月后的短短几年内，这些应用技术就取得了巨大的效益——在登月计划中每投入1美元就可获得

4～5美元的产出。

# 1.1.2 月球探测宁静期

1976—1994年是月球探测的宁静期。自1976年以后的18年间，世界上没有进行过任何成功的月球探测活动。其原因主要是：

1）随着冷战形势的缓和并延续到苏联的解体，空间霸权的争夺有所缓解；  
2）新的月球探测活动需要总结前期经验，吸取以前月球探测耗资大、效率低、探测水平不高的教训，改变探测思路，提出新的发展战略；  
3）以月球探测获得的技术为基础，将月球探测技术向各领域转化、推广和应用，完善航天技术系统，研制新的空间探测技术，如地月间往返运输系统、高效探测仪器等，为进一步开发利用地外天体资源进行科学和技术储备；  
4）需要较长时间进行探测资料的消化、分析与综合，将月球科学研究提高到更高的理性认识阶段。

# 1.1.3 对月球的主要科学认识

人类通过飞越月球、绕月卫星、月球车、无人和载人登月探测与取样等探测手段，获得了极其丰富的科学数据，对月球的形状、大小、轨道参数、地月空间和近月空间环境、月球地质构造与特征、月球的岩石类型、矿物和化学组成、月球的资源与能源、月球的内部结构与演化历史等的研究取得了一系列突破性进展，对月球的起源和地月系的相互作用与影响获得了新的认识，主要表现在以下几个方面：

1）精确测定了月球的形状、大小和运行轨道。月球是一个南北稍扁、赤道略鼓的圆球体，极半径比赤道半径短约 $500\mathrm{m}$ 。月球的平均直径为 $3476\mathrm{km}$ ，相当于地球直径的 $27\%$ ；质量为 $7.35 \times 10^{22}\mathrm{kg}$ ，约为地球的 $1/81$ ；体积只有地球的 $1/49$ 。月球的表面积约为 $3800万\mathrm{km}^2$ ，只有地球表面积的 $1/14$ ，大约相当于中国陆地面积的4倍；月球的平均密度为 $3.34\mathrm{g/cm}^3$ ，比地球的平均密度 $(5.52\mathrm{g/cm}^3)$ 小得多；月球表面的引力也只有地球表面的 $1/6$ 。月球围绕着地球以椭圆形轨道运动，其远地点为 $406700\mathrm{km}$ ，近地点为 $356400\mathrm{km}$ ，与地球的平均距离为 $384403\mathrm{km}$ ，大约相当于地球赤道长度的10倍。  
2）月球表面基本上没有大气，表面气压仅为 $10^{-14}$ 地球大气压。由于没有大气的热传导，月表平均温度分别为白昼的 $107\mathrm{C}$ 和夜晚的 $-153\mathrm{C}$ ，向阳面与背阳面的温度分别为 $120\mathrm{C}$ 和 $-150\mathrm{C}$ ，夜晚和太阳不能照射到的阴影区的温度仅为 $-160\mathrm{C} \sim -180\mathrm{C}$ ，最高与最低温度分别为 $-180\mathrm{C}$ 和 $130\mathrm{C}$ ，最大温差可以达到 $300\mathrm{C}$ 以上。  
3) 尽管月球现在没有明显的磁场存在, 但月球的岩石有极微弱的剩磁, 磁化强度约为 $10^{-9} \mathrm{T}$ 量级), 这表明月球可能曾经有过较弱的全球性偶极磁场, 但这个磁场大约在 31 亿年前就消失了。  
4）月球表面的主要地形单元为月海盆地、高地和撞击坑。月海是指地形相对低洼的大型盆地，月球正面的月海约占正面面积的一半，背面月海分布极少；高地是月球表面最古老的地形单元，高地物质大部分是富含斜长石的深成岩；撞击坑是小天体撞击月表形成的，撞击作用导致大量岩石碎块溅射并堆积在盆地周围，形成围绕月海盆地的山系，如比利牛斯山脉、阿尔

泰山脉、阿尔卑斯山脉、朱拉山脉、喀尔巴阡山脉、科迪勒拉山脉等。据统计，月球表面直径大于 $1\mathrm{km}$ 的撞击坑总数在33000个以上，总面积约占月球表面积的 $7\%$ 到 $10\%$ ，直径大于 $1\mathrm{m}$ 的撞击坑总数高达3万亿个。

5）月球表面覆盖着一层由岩石碎屑、粉末、角砾和撞击熔融玻璃等组成的、结构松散、成分复杂的风化层——月壤，月壤中绝大部分物质是就地及邻近地区提供的。月海月壤平均厚度为 $4 \sim 5 \mathrm{~m}$ ，高地月壤平均厚度为 $10 \sim 15 \mathrm{~m}$ 。由于长期接受太阳风和宇宙射线的辐射，月壤明显富集由太阳风注入的挥发性元素及其同位素，如 $\mathrm{H}, \mathrm{He}, \mathrm{N}, \mathrm{C}$ 等，因而储存了独特的太阳辐射历史，完整地记录了40亿年太阳的活动，这对了解地球气候变化是非常重要的。

月壤中的氦-3( ${}^{3}\mathrm{He}$ )是入类未来可长期使用的、清洁、安全而廉价的可控核聚变燃料。提取 ${}^{3}\mathrm{He}$ 过程中产生的其他副产品，如氢、氮、二氧化碳等，也将是未来月球基地生命保障系统建设的重要资源。

6）月球表面没有水体，月球在地质演化历史中也没有或只有极微量的水参与，但月球南、北两极的永久阴影区可能存在水冰。它们可能是彗星撞击后残留下来的，并在撞击坑的永久阴影区中得以长期保存。

7）月球上没有生命，没有活动的有机体、化石或有机体固有的有机化合物。

8）月球和地球在成因上是相互联系的，它们都是由相同的物质“原料”（元素）以不同的比例“混合”形成的。虽然组成月球和地球的化学元素相同，但月球更富含难熔元素，强烈地亏损铁和挥发性元素。所有的月球岩石都是通过高温的内生过程(岩浆或火山作用)形成的。月球上的岩石可粗略地分为四类：玄武岩、斜长岩、克里普岩和角砾岩。

迄今为止，在月球上已发现100多种矿物，其中绝大多数矿物的成分和结构与地球的矿物相同，只有静海石等5种矿物在地球上未发现过。月球矿物普遍不含水，矿物中的变价元素多为低价元素，如Fe多为0价或 $+2$ 价铁，表明月球矿物是在缺水和还原的条件下形成的。

9)月球有一个厚的月壳 $(60\mathrm{km})$ ，一个相当均一的岩石圈 $(60\sim 1000\mathrm{km})$ 和一个部分液化的软流圈(深度在 $1000\sim 1740\mathrm{km}$ )，在软流圈的底部有可能存在一个小的铁核或硫化铁核。  
10）月球在总体形态上有轻微的不对称现象，这可能是地球重力影响的结果。月球背面的月亮较厚，而火山熔岩填充的大型月海盆地和质量瘤(大的质量密集区)大多存在于月球正面。月球内部的质量分布是不均匀的，质量瘤多位于大型月海盆地的表面之下。  
11）现在的月球是一个古老的、“僵死”的天体。月球的内部能量已近于衰竭，表面热流仅为 $2\mu \mathrm{W} / \mathrm{cm}^2$ 左右，内部的地温梯度也很小，月震释放的能量小于 $10^{6}\mathrm{J} / \mathrm{h}$ ，每年由月震释放的能量仅相当于地震的 $1 / 10^{8}$ 。自31亿年以来，月球没有发生过显著的火山活动和构造运动，因此，月球的“地质时钟”停留在31亿年之前，保留了其形成早期的历史。  
12）根据对月球各类岩石的成分、结构与形成年龄的研究，月球演化历史的重大事件可归纳为以下几个阶段：

- 月球的形成年龄约45亿年；  
- 月球形成后曾发生过较大规模的岩浆洋事件, 通过岩浆的熔离过程和内部物质调整, 于41亿年前形成了斜长岩月亮、以及月幔和月核;  
- 在40亿～39亿年前，月球曾遭受到小天体的剧烈撞击，形成广泛分布的月海盆地，称为雨海事件；  
- 月海泛滥事件, 在39亿~31.5亿年前, 月球发生过多次剧烈的玄武岩喷发事件, 大量

玄武岩充填了月海，厚度达 $0.5 \sim 2.5 \mathrm{~km}$

- 31.5亿年以来，月球内部的能源逐渐枯竭，再也没有发生大规模的岩浆火山活动和构造事件，但小天体撞击仍然频繁发生，形成大量撞击坑，使月面斑驳陆离、千疮百孔。  
13）月球起源的理论主要包括四种假说：捕获说、共振潮汐分裂说、双星说和大碰撞分裂说。近年来，大碰撞分裂说在大量证据的支持下得到了大多数学者的认可。

总之，现在的月球是一个古老的、“僵死”的天体，月球的内部能量已近于衰竭，月震和表面热流均极小。比较而言，火星是一个步入“老年”的天体，地球是正处于“壮年”期的生机勃勃的天体，而月球的“地质时钟”停留在31亿年之前，这对研究地球的早期演化历史具有重要意义。

# 1.2 重返月球

1986年，深空探测技术和月球科学研究进入了新的阶段，对月球进行科学的、“理性”的探测时机即将成熟，美国国家航空航天局(NASA)开始构思重返月球计划。1989年7月20日，时任美国总统乔治·赫伯特·沃克·布什(1989—1993年任美国第51届总统)宣布：“在即将到来的10年里，我们努力的目标是自由号太空船；然后，在新的世纪，我们要重返月球，重返未来，而且这一次要常驻下去”。“要常驻下去”开发和利用月球矿产资源、能源和特殊环境，建立月球基地，为人类社会的可持续发展服务，已成为新世纪月球探测的总体目标。

1994年和1998年，美国成功发射了“Clementine”（克莱门汀）号和“Lunar Prospector”（月球勘探者）号月球探测器，对月球的地形地貌、资源分布、水冰等进行了探测，标志着“又快、又好、又省”空间探测战略的实施，奏响了人类重返月球、建立月球基地的序曲。

2004年1月14日，美国总统乔治·沃克·布什(2001年至今连任美国第54、55届总统)宣布了雄心勃勃的新太空计划。其中提出，在2008年前开始发射无人月球探测器，进行系统的月球探测；2015—2020年重新载入登月，并在月球上长期居留；以月球为跳板，在2030年前把宇航员送上火星乃至更遥远的宇宙空间。对此，各国的评论一致认为，美国新太空计划的目标是控制能源、军事优先。欧州空间局(ESA)根据美国新的重返月球计划，迅速调整其月球探测战略，计划于2020年前进行不载人月球探测；2020—2025年实施载人登月，并在月球上长期居留；2035年把宇航员送上火星。

重返月球是在现有的和未来可以实现的技术条件下，重新对月球进行全球性、综合性和整体性的探测，进而载人登月，并开发和利用月球资源、能源和特殊环境，建立月球基地，以月球为跳板逐步开展深空探测。美国、欧洲空间局、俄罗斯、日本、乌克兰、奥地利、英国、德国、巴西和印度等国家和组织都制定了相应的月球探测计划，并在积极实施中。

美国计划于2008年发射月球勘察轨道器(Lunar Reconnaissance Orbiter)，勘察月球可能的资源并为重新载人登月选址；欧洲空间局于2004年提出新的“曙光”月球探测计划，希望通过发射一系列不载人月球探测器、月面软着陆器和月球车，为2020—2035年载人登月和2035年通过月球载人登上火星打好基础；俄罗斯于2004年提出新的“全月球”(Lunar Glob)探测计划，预计2009年发射月球轨道器，在轨道器上发送十几个大小不等的穿透器，记录月震和小天体撞击事件，研究月球的内部结构与演化；日本近期的月球探测主要包括“月球-A”(Lunar-A)和“月亮女神”(Selene)计划，“月球-A”计划将开展月球成像、月震监测、月表热流测量和月球内部结构研究等探测目标，“月亮女神”计划将开展月面成像、月球重力场探测、月面软着陆

等探测目标；印度提出的“月球-1”(Chandrayaan-1)探测轨道器，将进行月表高分辨制图和月表元素分布探测等，计划于2007年9月27日发射。

重返月球反映了科学技术发展的需求，特别是空间科学技术发展以及人类社会可持续发展的需求，这些需求包括：

1）重返月球是社会发展的需求。重返月球，建立永久基地，是人类开发太空资源，拓展生存空间至关重要的第一步。通过这一系统工程，人类可以学会如何离开地球，建立类似南极的永久研究基地，在地球以外的空间发展工业和生产产品，建设能够自给自足的地外家园。  
2）重返月球是科学和技术发展的需求。20世纪 $60\sim 70$ 年代的月球探测工程证实，空间探测是一个具有极高产出率的项目，它实现的真正价值往往远远高于工程本身。月球探测可以成为科学和技术的“孵化器”。  
3）重返月球是空间科学技术发展的需求，是空间探测发展的必然趋势。月球将是人类进行深空探测的前哨站和转运站。随着技术的不断成熟，可以在月球基地上建设空间加“油”站和发射场。月球是一个庞大的天然空间站或“太空实验室”，可开展一系列天文学、空间科学、近代物理学、生物工程学等的研究，研制和生产一系列特殊生物制品和特殊材料。  
4) 重返月球是空间军事活动发展的需求。月球将被用作对地球监视的基地或新的军事平台, 是实现制“天”权的重要步骤。  
5) 月球蕴藏有丰富的矿产资源和能源, 可为人类社会可持续发展提供资源储备, 这一因素是重返月球最主要的原动力。

# 1.3 月球资源与环境的开发利用前景

冷战时期，月球探测的政治意义和象征意义是第一位的，通过月球探测促进科学技术和经济社会发展则是次要的。当今世界，和平与发展成为主流，环境污染、能源短缺和人口压力成为人类面临的主要问题。因此，新世纪初月球探测活动的主要目标将集中于月球资源、能源和特殊环境的利用，更加关注月球在探测太阳系其他天体中的跳板作用，向着建立月球基地、从月球载人登陆火星和太阳系其他天体的宏伟蓝图前进，主要目标包括：

1）月球能源性资源的全球分布与利用；  
2）月球矿产资源的全球分布和利用；  
3) 月球特殊空间环境资源(超高真空、无大气活动、无磁场、地质构造稳定、弱重力、超洁净)的开发利用, 包括: 开展月基天文观测、建立特殊生物制品和特种新型材料生产基地、建设基础科学实验室等;  
4）建立月球长期居留基地并利用月球进行深空探测和载人登上太阳系其他天体。

# 1.3.1 月球能源开发利用前景

月球上可利用的能源主要有太阳能和核聚变燃料。由于月球表面没有大气, 太阳辐射可以长驱直人; 同时, 月球上的白天和黑夜都相当于 14 个地球日, 因此在月球表面建立全球性的并联式太阳能发电厂, 就可以获得极其丰富而稳定的太阳能。这不但解决了月球基地的能源供应问题, 还可以用微波将能量传输到地球, 为地球提供新的能源。

由于月球几乎没有大气层，月球表面长期受到微陨石冲击和太阳风粒子注入，特别是太阳风粒子的注入使月壤富含稀有气体。由于太阳风粒子注入暴露月表的月壤颗粒的深度一般小于 $0.2\mu \mathrm{m}$ ，因此，这些稀有气体在细粒月壤中的平均含量最高，有些细粒月壤中的稀有气体含量高达 $0.1\sim 1\mathrm{cm}^3 /\mathrm{g}$ （标准状态下），相当于 $10^{19}\sim 10^{20}$ 原子/立方厘米。在整个月球演化史中，由于外来天体对月球表面的频繁撞击，月壤物质几乎完全混合，在深达数 $\mathfrak{m}$ 的月壤中这些稀有气体的含量较均匀。

在月壤的稀有气体中, 最让我们感兴趣的是氦-3。因为, 相比目前正加速发展的利用氘和氚反应的热核聚变装置来说, 用氦-3进行核聚变反应具有比用氚作燃料有更多的优点, 主要表现在:

1)传统的氘核反应过程中，伴随核聚变能的产生，要产生大量的高能中子，而这些中子将对核反应装置产生广泛的放射性损伤；相反，若用氦-3作为反应物，则主要产生高能质子而不是中子，对环境保护更为有利；

2) 氪本身具有放射性, 而氦 -3 没有放射性。

月壤中氦-3的资源量对未来人类开发利用月球能源具有极重要的意义。由于月壤中氦-3的含量较为稳定，只要能够精确探测月壤的厚度，就可以估算出月壤中氦-3的资源量。以Apollo和Luna月球样品的实测结果为参考标准计算，月壤中氦-3的资源总量可达100万～500万t。而地球上可提取的氦-3只有 $15\sim 20\mathrm{t}$ 。若能实现商业化利用，月壤中的氦-3可供地球能源需求达数万年。因此，开发月壤中丰富的氦-3资源对人类未来能源的可持续发展具有重要而深远的意义。

据计算, 氦-3 的能量回报率为 270, 核裂变发电的能量回报率为 20, 而煤仅为 16。氦-3 作为一种清洁、高效、安全的核聚变发电燃料是无容质疑的。当前, 可控核聚变商业发电尚未实现, 从月球上运回氦-3 成本过高。由于目前技术条件、经济发展等诸多条件的制约, 利用月壤中氦-3 来进行发电似乎是难以想象的。但完全有理由相信, 随着可控核聚变发电的商业化, 航天科技的发展和航天运输成本的日益降低, 当地月之间的运输成本降低到可以接受的程度时, 利用月球上的氦-3 发电将成为历史的必然。人类要开发月球, 建立月球基地, 必然要在月球上获得维持生命所需的各种气体, 如 $\mathrm{O}_2$ 、 $\mathrm{CO}_2$ 、 $\mathrm{N}_2$ 等, 而氦-3 可以作为提取这些气体的副产品来进行开发, 这样会进一步降低成本。

# 1.3.2 月球金属矿产资源开发利用前景

月球表面分布着22个主要的月海，除东海、莫斯科海和智海位于月球的背面外，其他19个月海都分布在月球的正面。月海玄武岩中蕴藏着丰富的钛、铁等资源。 $\mathrm{TiO_2}$ 的含量从 $0.5\% \sim 13\%$ 不等。22个月海中所填充的玄武岩总体积约 $10^{10}~\mathrm{km^3}$ 。若假设月海玄武岩中钛铁矿含量为 $8\%$ ，即 $\mathrm{TiO_2}$ 含量为 $4.2\%$ ，则月海玄武岩中钛铁矿（ $\mathrm{FeTiO_3}$ ）的总资源量约为 $1.3\times 10^{15}\sim 1.9\times 10^{15}\mathrm{t}$ 。尽管这种估算带着很大的推测性与不确定性，但可以肯定的是月海玄武岩中丰富的钛铁矿是未来月球可供开发利用的最重要的矿产资源之一。

克里普岩(KREEP)是月球高地三大岩石类型之一，因富含K(钾)、REE(稀土元素)和P(磷)而得名。克里普岩在月球上分布很广泛。富钍、铀的风暴洋区的克里普岩被后期月海玄武岩所覆盖，克里普岩与月海玄武岩混合并形成了高钍、铀物质，其厚度估计有 $10\sim 20~\mathrm{km}$ 。

风暴洋区克里普岩中的稀土元素总资源量约为225亿～450亿t。克里普岩中所蕴藏的丰富的钍、铀也是未来人类开发利用月球资源的重要矿产资源之一。

此外，月球还蕴藏有丰富的铬、镍、钾、钠、镁、硅、铜等金属矿产资源，将为人类社会的可持续发展作出贡献。

# 1.3.3 月球表面特殊环境开发利用前景

月球几乎没有大气层, 属于超高真空状态, 因而月球表面不会有大气的吸收、反射与散射等; 由于没有大气的热传导, 月球表面昼夜温差极大; 月球没有全球性的磁场, 月岩只有极微弱的剩磁; 月球的内部能量已近于衰竭, 内部的地温梯度也很小; 月震释放的能量仅相当于地震的一亿分之一, 地质构造极其稳定; 距今31亿年以来, 月球没有发生过显著的火山活动和构造运动, 因此, 月球的“地质时钟”停滞在31亿年之前, 至今仍保留了其形成早期的历史状况; 月球表面还具有高洁净、弱重力的特征。

上述所有这些特征，都是地球上所不具备的。因此，在月球表面开展月基天文观测和研究有特殊优势。月球上的天文观测站将是月球基地的重要组成部分，它不仅可以对太阳系、银河系天体和星际空间进行观测研究，而且是进行太阳物理学、天体物理学、重力波物理学、中微子物理学观测和实验最有吸引力的场所。在月面建立月基对地监测站，可以对地球的气候变化、生态演化、环境污染和各种自然灾害进行高精度的观察和监视，为人类的可持续发展做出贡献。

月球的特殊环境为研制特殊生物制品和特种材料开拓了广阔而诱人的前景，目前科学家们已提出了庞杂的需要在月球基地内研制的生物制品与特种材料的清单。月球将成为新的生物制品和特种材料的研制、开发和生产的基地。

# 1.3.4 月球基地

月球是地球惟一的天然卫星, 是人类惟一的、庞大而稳固的“天然空间站”, 是人类开展深空探测的前哨阵地和转运站。在月球上建立永久性“月球基地”, 是人类向外层空间发展的第一个目标, 也是最关键的一步。而重返月球计划旨在建设一个具有生命保障系统的受控生态环境的月球基地, 进行建筑、运输、采矿、材料加工和各项科学研究, 为将来建设适于人类居住的月球村进行科研和技术准备, 使月球最终成为一个庞大、稳固而功能齐全的“天然空间站”, 成为人类共有的科学实验室、开展深空探测的试验基地和物资转运站、载人登上火星和太阳系其他天体的前哨阵地。

可以看出，月球的矿产资源、能源资源和特殊环境资源将对人类社会的可持续发展发挥长期稳定的支撑作用。

地月系不仅是一个统一的自然体系，而在人类社会的可持续发展方面，也将构成一个统一的整体。

# 1.4 月球科学研究中存在的主要问题和展望

早在1959年1月2日，苏联就发射了第一个月球探测器。到目前为止，人类已经成功地

发射了100多个月球探测器，包括环月飞行、在月球上软着陆和从月球取样返回的探测器以及载人飞船。在1994年和1998年，美国分别发射了两个小型月球探测器——克莱门汀号(Clementine)和月球勘探者号(Lunar Prospector)，它们携带了一些遥感仪器，探测了整个月球表面，月球勘探者号还绘制了月球的重力场分布图，取得了丰硕的成果。

人类对月球的探测虽然已经历了将近50年，但对月球主要科学问题的认识仍然是极其肤浅的。宇航员在月球上进行过实地勘察与取样的位置仅有6处，在月球表面不载人自动采样的位置仅有3处，加上人类对月球进行软着陆就位探测和不载人月球车巡视勘察，全部加起来也仅有18处。通过这些探月活动，人类对月球的表面形态、矿物分布和月球附近的空间环境等有了基本的了解；但是，这些样品都是在月球朝向地球一面的赤道及其附近地区获得的，而在月球的另一面和两极地区，还没有获得样品，而这些地区的矿物成分和分布特征，很可能与上述地区有很大不同。为了全面地了解月球物质组成和分布特征，需要对月球背面和极区进行取样研究。目前，限于技术水平的制约，人类的月球探测活动还不够深人，对月球的了解仍存在很大的局限性，对一些月球主要科学问题的认识，如月球大气层的来源与成因、地形地貌类型的划分、地月空间环境与月球表面环境的特征、月壤的形成与组成、地质构造的特征、岩石类型及其分布规律、月球内部结构与物理场、月球的演化历史与重大事件、月球的起源等，仍然存在着许多令人困惑未解的问题和争议。

目前，人类对月球比较关注的问题主要包括：

（1）月球大气层的成分及其来源、大气组分的运动规律及其演化过程

月球形成时可能存在稀薄的原始大气层，月球早期岩浆与火山活动必然释放出大量火山气体，但这些气体早已逃逸殆尽，没有留下可供追索的痕迹。现今月球极稀薄的大气层主要由氮、氢、氦、氩等气体组成，推测其可能主要来源于太阳风，但月球大气层中的气体分子的精确组成、月球白昼与黑夜期间气体分子被月壤吸附或释放的过程与机理、月球大气层的演化过程等问题，至今仍然模糊不清。

# (2）月球的电离层

现有的月球探测资料证明, 31亿年前月球曾经有过全球性偶极磁场, 月球早期可能存在过比现今更为稠密的大气层, 因而可能曾经有过电离层的发育; 但随着月球全球性偶极磁场的消失和大气分子的逃逸, 月球全球性电离层或局部区域电离层是否存在, 尚有待新的探测结果来证实。

# （3）月球地形地貌的类型与成因

月球的地形地貌类型主要包括月海、高地与撞击坑。为什么绝大多数月海分布在月球正面，而月球背面却以高地为主，月海分布很少？月球上存在着撞击坑、月球古代火山口及构造盆地三种不同成因的环形构造，但从遥感影像上难以区分与鉴别，因此只能统称为撞击坑。月球上撞击坑的大小与密度随月球演化的历史进程而减小，是否说明太阳系小天体的数量和小天体撞击行星的频率随时间而变小？

月球的地质构造与地体类型都是根据月球地形和地貌的遥感影像反演得到的，由于遥感影像的多解性，也就带来了月球地质构造解释的不确定性。

# (4) 地月空间环境与月表环境

太阳活动与银河宇宙线通量、能量的变化直接制约了地月空间环境的变化，由于探测数据不足，对地月空间环境难以取得规律性认识。而月表环境受太阳活动、宇宙射线、昼夜温差、月

球表面地形地貌、月壤和月球岩石的物理化学性质与分布、内部物理场等诸多因素的影响，不同区域的差异性极大，需要进行多种区域环境探测的资料积累。

# （5）月壤的物质来源、分布特征与形成过程

一般认为，月壤起源于就地岩石粉碎和外来溅射物的混合，月壤形成的主要动力包括：月球岩石受温差变化热胀冷缩破碎，受撞击粉碎溅射，以及宇宙射线、太阳辐射和太阳风粒子的长期注入。月壤中的各种组分具有不同的来源和成因，不同地区的月壤在物质组成和物理化学性质上差异性极大，这些需要进行系统的探测与积累。

月壤的成熟度与颗粒直径、物质成分和月壤厚度存在一定的相关性，月壤成熟度的判定指标尚存在争议。月壤吸附太阳风粒子的机制和效应，特别是不同矿物吸附太阳风粒子的能力差异的内在机制尚不清晰，太阳风组分在月壤颗粒中的赋存状态还缺乏精细研究。

# (6) 月球的主要岩石类型与分布

月球的主要岩石类型仅仅是根据月球正面9个采样点获得的样品和月球陨石的精细研究而确定的，各种岩石类型的分布是根据地形地貌、矿物、化学成分的遥感探测资料的反演而推测的，存在一定的多解性，月球上是否存在酸性岩浆岩仍存在疑问。

# （7）月球水冰之谜

1994年,克莱门汀号使用双基地雷达对月球进行了探测。根据雷达波反射的数据,估计在极区的撞击坑(处于太阳照射不到的永久阴影区)中可能存在水冰。1998年,月球勘探者号探测器使用中子谱仪,也对该地区进行了探测。为了进一步证实水冰是否存在,当月球勘探者号完成基本探测任务后,美国国家航空航天局发布指令让它撞击在可能含有水冰的地区。如果这个地区确实含有大量的水冰,则当月球勘探者号撞击之后的瞬间,将会产生大量的水蒸气;但是,通过在地面观测站和太空卫星观测,都没有观测到这种现象,因此,月球极区是否有水冰,目前还是一个谜。

如果月球上真的存在水冰，那么，水冰来自哪里，如何在极区的永久阴影区中富集并保存下来，月球上的水冰有没有开发利用前景，能否解决未来月球基地的水资源需求？这些问题的分歧和争议还很多，目前尚无定论。

# （8）月球的大地构造区划

目前，人类对月球地质构造的认识完全是基于对全月球的小比例尺(约1:2000000)遥感影像的解译，因而存在难以确定的多解性。关于月球地体单元的划分，也仅仅是依据月球某些化学元素的遥感探测并结合少量样品的精细研究而做出的推测。因此，关于月球的大地构造区划、地体单元的厘定、断裂与环形构造的分布等，仍然缺乏实地的验证。

至于月球的大地构造区划, 当地球的槽-台学说盛行时, 按槽-台学说的框架划分月球的大地构造单元; 当地球的板块构造学说盛行时, 又按地体的思路划分月球的大地构造单元。月球的演化历史与地球有较大的差异, 套用地球的大地构造框架显然不合理, 但月球自身的大地构造区划的理论框架至今仍未建立。

至于月球的区域地质构造研究，目前几乎还是空白。仅有月球正面的个别区域（如东海、雨海等），根据月表遥感影像进行过地质构造的初步分析。

# (9) 月球内部层圈结构与形成过程

根据月震波和小天体撞击产生的弹性波在月球内部传播速度的变化，月球的内部结构与地球和其他类地行星相似，可以划分出月亮、月幔与月核。对于月亮的物质成分、岩石类型与

物理性质的判断有比较充分的科学依据，而对于月幔与月核的矿物组成、化学成分、热状态与物理特征的描述，目前推测与地球的地幔和地核相似，但仍然缺少严密的科学论证。

至于月球内部层圈结构的成因，基本上是套用地球层圈结构成因的模型，带有较大的不确定性，尚缺乏月球内部层圈结构的成因模型。

# （10）月球内部物质分布的不均一性

月球内部物质分布不均一性的研究主要是依据月球勘探者号探测器轨道的精确测定数据和Apollo月球地震台站对月震产生的弹性波传播速度的记录数据，反演重力场模型和内部物质分布的不均一性，证明月球内部物质分布极不均一，发育着多个重力异常区——质量瘤。质量瘤的大小、分布、物质组成与成因仍然众说纷纭。

# （11）月球全球性偶极磁场的演化

根据月球岩石剩余磁场强度的测定，证明月球在31亿年前曾经存在过较强的全球性偶极磁场，以后逐渐消失。月球古磁场的消失过程与原因至今仍然难以解释，是否是月球内部随着能量的散失、内部物质冷凝固化所致？月球古磁场的消失是否表明类地行星随着内部物质的冷凝固化，其也具有由全球性偶极磁场向多极磁场转化，再由多极磁场到磁场消失的演化过程呢？

# （12）月球内部能源与月球物质的演化

月球和行星的内部能源以及随时间的衰减、传导和分配是制约月球和行星演化的原动力。根据组成月球的初始物质和放射性元素丰度的模型，可以推测出月球的热演化历史。据推测，月球形成初期曾发生过岩浆洋事件，形成广泛分布的斜长岩；相继发生了广泛的月海玄武岩喷发。由于月球内部能源衰减的速率较快，内部冷却固化，岩浆难以持续分异演化，因而自31亿年以来，月球的岩浆活动、火山活动和构造运动趋于平和与宁静。月球内部能源的衰减、热平衡与内部物质的演化至今还没有建立定量的演化模型。

# （13）月球演化的重大事件

人类对地球进行过全面、深入、系统的研究，但对于地球演化的重大事件至今仍然难以精细确定，对每一重大事件的起因和对地球演化进程的作用与影响，仍然争论不息。

月球演化重大事件的研究是建立月球演化历史的关键，而其与地月系的轨道演化、月球内部层圈结构的形成与内部物质的分异调整、内部物理场的形成与演变、月球空间环境变化、内部能量的积累、传输与平衡的内在联系，仍然很不清晰。

# （14）月球与地月系的起源

月球与地月系的起源是月球科学研究的主要目标。有关月球与地月系的起源, 有众多假说(主要有捕获说、共振潮汐分裂说、双星说与大碰撞分裂说), 有些月球起源假说持续有上百年的争论。

大碰撞分裂说认为，月球是因地球在45亿年前遭到一次大的碰撞而从地球分离出来的。这一学说逐渐得到了愈来愈多的探测数据、月球岩石的地面研究数据和计算机理论模型的支持，但毕竟还是比较粗糙的构想，大量的细节尚待进一步积累科学资料，提供进一步的证据。月球的起源不是孤立的事件，必须与地球的起源与演化相互联系、相互制约，这就需要对地球和月球的成分进行精确的对比研究。目前，有关地球的起源仍然争论不一，所以对月球和地月系的起源研究将是月球科学长期的研究任务。

人类自诞生以来，已经经历了大约300万～400万年的漫长历史，而现代人的诞生及文明

的发展,尤其是科学技术的进步与经济的发展,使人类对地球的认识日益广泛而深入,已经进入到全面、系统、深入地建立与完善“地球系统科学”的阶段。但是,对于地球科学中的一系列重大科学问题,人类的认识仍然比较肤浅。关于地球的起源与演化的认识,仍然是迷雾重重。人类还需要通过长期坚持不懈的努力探索,才能逐步加深对地球的理解与认识。地球的起源和演化将是人类永恒探索的重大主题。

人类对月球的科学认识比起地球来显得极其零散和浅薄。上述提出的月球科学中存在的主要问题仅仅是反映当代对月球科学认识的初步见解。几千年来人类对月球的长期观测与探索，望远镜发明400年以来科学家们对月球的孜孜不倦的观测与分析，特别是近50年来，100多个月球探测器和6次载人登月对月球的全面探测与精细研究，人类付出了高昂的代价和艰巨的努力。能够提出月球科学中存在的主要问题，是月球探测成果的结晶，体现了人类对月球科学认识的进展。人类对月球的探测和开展月球科学研究，只能说是刚刚拉开了序幕，是“万里长征”的第一步。

人类对月球科学的探索将是无止境的。随着世界各国重返月球计划的实施、月球基地建设方案的提出与付诸实施，人类对火星和太阳系探测逐步取得进展，这些努力必将推动月球科学跨越式发展并取得长足进步，也必将全面系统地深化人类对月球的科学认识。
