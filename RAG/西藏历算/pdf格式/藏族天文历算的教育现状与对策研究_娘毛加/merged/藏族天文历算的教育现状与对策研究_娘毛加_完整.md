# 藏族天文历算的教育现状与对策研究

娘毛加

（西南民族大学西南民族研究院,四川 成都 610041）

[关键词]藏族天文历算;教育现状；对策

[摘要]藏族天文历算独具民族特色,在我国天文历算学中占有重要地位。目前除民族高校开设有相关课程外,在高院教育中其仍占有较大比重。对藏族的天文历算进行有效、有序的保护和传承，有利于丰富我国传统文化的多元性。

藏族天文历算是世代居住、生活在青藏高原的藏族人民在结合自己原有的天文、历法、占星术和算学等的基础上，吸收、借鉴国内外其他民族的相关知识而形成的独具民族特色和富含科技知识的一门学科，在祖国天文历算学宝库中占有重要地位。它在产生和发展的过程中，相继经历了口耳相传、世袭家传、师徒相承、寺院教育以及现代学校教育等教育模式。如今除了学校教育外，寺院教育仍占有一定的地位。笔者通过访问、实地调研、问卷调查等方式，对安多寺院时轮学院和甘青川藏的几所民族高校开设、研习藏族天文历算的基本情况进行调查，旨在分析了解这一学科的教育现状，探讨其传承和发展所面临的问题及对策。

# 一、寺院时轮学院的天文历算研习现状

# （一）时轮学院的设立

藏族传统教育中寺院教育是其主要教育形式，它是以僧人为教育对象，以大小十明为教育内容。天文历算作为小五明中的一“明"①，曾在寺院得到广泛传播和发展。在藏传佛教各大教派中萨迦派最早开设天文历算课，到了萨班贡嘎坚赞时代，萨迦派的教育内容出现了一个新的特点，除了学习显密教法外，还要学习《俱舍论》、《量释论》等七因明论，以及声明、工巧明、天文历算、诗词、韵律、歌舞、修辞等大小五明。这些课程的设置，说明萨迦派的教育内容不仅仅局限于佛学知识，而且也包括了科技和文化艺术等。尔后，藏区各教派寺院都陆续开设了天文历算课,萨迦寺、楚布寺等寺院还编制颁布过《藏历历书》。自18世纪以后，寺院内开始建立了大小不等的、培养天文历算人才的教学基地——时轮学院,即丁科尔扎仓。以甘、青藏区为例,1752年，甘肃卓尼禅定寺创建了时轮学院；1763年，甘肃拉卜楞寺创建了时轮学院；1773年，青海隆务寺创建时轮学院；1802年，青海夏琼寺将1797年所建的医明学院改为时轮学院;1817年，青海塔尔寺创建了时轮学院。在此影响下，莲花台寺、佑宁寺、却藏寺、东麻昂寺、东科尔寺、石藏寺、赛力亥寺、阿粗乎寺、都兰寺、拉加寺、卧科寺、郎木寺、西仓寺等也先后设立了时轮学院。[2]

时轮学院的创立把研习天文历算之风推向了新的高峰,造就了一大批精通天文历算的学者，产生了大量历算典籍。同时,也促进了历算古籍等科学书籍的收录、校订、翻译等工作，极大地推动了藏族天文历算学的发展进步。上世纪80年代改革开放后,各地寺院先后被批准开放，有的寺院也恢复了时轮学院。调查研究拉卜楞寺和塔尔寺这两座寺院时轮学院的天文历算研习现状，可窥见安多其他寺院藏族天文历算的学习概况。

# (二)天文历算的研习现状

据说时轮学院是根据《时轮金刚》的理论及去香巴拉理想国的进程，安排学习时间、内容和科目。僧人除了要学会诵读经文、音韵手诀等寺院仪轨外，主要研习天文历算和梵文,兼学语法、辞藻等知识。各寺时轮学院的学级设置、各级所学的内容及要求不同，学习年限一般没有严格规定。“孜然巴”曾是时轮学院最高学位,相当于历算博士学位,每年录取名额仅有1名。

拉卜楞寺时轮学院设有初、中、高3个学级，初级班主要背诵《妙吉祥名号经》（QE）、《无上供养经》（）等经文和音韵手诀,通过背诵测试即可升人中级班。中级班学僧要学会《时轮金刚经》（5）、《普济经》（）等经文外，还要学会用彩色细沙绘制坛城,考试合格者才能晋升。只有高级班的学僧才系统学习梵文、语法、天文历算等学科,并研究时轮金刚、怖畏九首金刚的升起次第与圆满次第之道，广授灌顶,修行禅坐。① 也只有高级班毕业的学僧或学业有成的僧人才有资格招收门徒、讲授天文历算，但并不是升人高级班的学僧都必须学习天文历算，学僧可以根据自己的兴趣专学一门学科（即天文历算可学可不学）,学习时间由老师安排,学僧学完某一课程后要参加考试,一年有四次考试,依考试规模大小安排监考人员，采取背诵、阐释、笔试等考核方式。天文历算的教学和研究主要以《时轮历算精要 ${ \tt \ P } ^ { \textcircled { 2 } } ( \overbrace { \tt ^ { 2 } } ^ { \tt S } \overleftrightarrow { \tt { 1 } } ^ { \tt S } \overleftrightarrow { \tt { 2 } } ^ { \tt S } ) = \overleftrightarrow { \tt { 2 } } ^ { \tt S } \overleftrightarrow { \tt { 2 } } ^ { \tt S } \overline { { \tt { 2 } } } )$ 为基本教材。此外,必修科目还有《时轮根本续》$( \sqrt { 3 } ^ { \pi } ) ^ { 2 } \sqrt { 1 7 } + \sqrt { 3 } ^ { \pi } \sqrt { 7 } )$ 和《时轮摄略经》（525）、《历算综论》 $( 8 \sqrt { 2 } ) ( 3 \sqrt { 2 } + 1 ) ( 2 \sqrt { 2 } + 1 ) ( 2 \sqrt { 2 } + 1 )$ 等，另学僧还参阅当代著名历算学家桑珠嘉措大师撰写的《历算运算大全》（5)。该时轮学院最盛时学僧达400余人，今共有150余人，民族成分除藏族外，还有蒙古族。其中给学僧讲授辅导天文历算的老师有4人，平均年龄在50岁以上，初级班有80余人，中级班15人，高级班40人。学院每年组织人员编制历书(手抄本）后敬献给寺院总法台，并不公开颁布发行。

据文献记载，寺院时轮学院曾是培养历算人才和从事历算研究的重镇，是编制《藏历历书》的主要机构。曾有严谨的教学体制、固定的教材、独特的教学方法、严格的考试制度以及健全的学位制度，历史上天文历算人才辈出，著书立说的学风四起,出现过很多“孜然巴”。天文历算虽设为时轮学院的学习内容，但实际上研习者较少，其传承逐渐式微,很多僧人只懂得诵读经文、吹打法器、绘制坛城等，而精通天文历算、掌握其核心技术、培养门徒的僧人屈指可数。上述两座寺院曾编制的历书是安多藏区通用的，深受蒙藏地区农牧民的欢迎，现今拉卜楞寺时轮学院虽编制历书，但不刊印发行，塔尔寺时轮学院已无人编写历书。

# 二、民族高校的天文历算教学现状

随着学校教育体系的建立，以寺院为主体的藏族传统教育形式发生了实质性的变化，学校成为传播科技知识和传承、弘扬民族传统文化的主要场所。为了了解藏族天文历算在民族高校的教学情况，笔者调查、访问了以西北民族大学为主的7所民族高校，其中三所学校未开设天文历算课,其余学校的基本情况为：西北民族大学藏汉双语数学专业的选修课程中有藏族天文历算课,据说是从1986年伊始讲授此课程。一般在三年级下学期开课,周课时为4学时,共72课时,考核方式为考查。该校藏语言文化学院是从1993年起开设这门课程,在本科培养方案中设为选修课,周课时为2学时。对专升本学生而言，此课程为学位考试必修课,一周四节课,共64课时左右。教材方面,藏数专业使用夏吾才让教授所著的《藏族天文历算研究与新探》,藏语言文化专业的本、专科通用傅千吉教授所著的《藏族天文历算学理论与方法》,这两位教授也是该校担任历算课的老师，除了讲授藏族天文历算外，还分别担任高等数学、线性代数、数学分析、藏族历史等课程。西藏大学文学院开设天文历算课有十几年的历史，一般设在大二下学期,周课时为2学时,共36课时，属选修考查课,现有2名历算老师,但非专任教师,目前无固定的教材。西藏藏医学院是从1985年起开设天文历算课,此课程是藏医专业的必修科目，安排在四年级第一、二两个学期,考核方式为考试。天文历算基础部分是藏药专业的副课（即选修课),在大四上学期开课,考核方式为考查，两个专业的周课时为4学时。教材是"21世纪藏医本科教育规划教材”《天文历算学》，现有6名讲授天文历算的老师,但并非专任教师。该校利用现有师资资源外,还外聘天文历算专家参与教学工作,提高教学质量。青海大学藏医学院是从1995年伊始开设这门课,课程类别为必修课,周课时是4学时，藏医、藏药两个专业都在大四第一学期讲授天文历算课,考核方式为考试,教材同上,有2名历算老师,均不是专任教师。①

了解民族高校的天文历算传习概况后可知：1、天文历算的授课学期较短，课时少。天文历算作为藏族传统文化的重要组成部分，其内容涉及到日月食预报、气象预测、星宿运动、历书的编制、人体脉象的变化规律等等。按现代意义上的学科分类，其内容涉及天文学、气象学、气候学、地理学、天体物理等学科,有科学的内涵和价值。据说著名历算学家黄明信先生曾说过藏族天文历算学不应该划为“小五明”之一,它应归为“大五明”，是藏族文化宝库中一颗璀璨的明珠和最具科学性的大学问。然而在有的学校其设置为选修课,授课时间一般为一学期左右,课时又少，故授课内容停留在天文历算发展史等基础知识的讲解上，而具体的推算原理、公式、程序等内容略去不讲。2、师资力量不足，课程资源开发较显单薄。从总体来看，西藏藏医学院有数量可观的天文历算教师外，其他高校均存在天文历算老师紧缺问题,担任该课程的大多老师是非专任教师，所承担的教学、科研等工作任务繁重,这对此课程的正常、长期开设会有一定影响。有的原开设天文历算课的学校，后因无历算老师而取消这门课程。另外，教材开发和课程资源的挖掘方面较显单薄,缺少统一、规范的教学大纲,有的教材在其适用性和针对性方面需改进。

# 三、加强藏族天文历算传承和发展的对策建议

寺院作为讲经说法、修行禅坐等宗教活动的重要场所，继承、传播和弘扬佛教文化永远放在首位,天文历算在寺院只是佛学教育的附庸，在寺院里天文历算很难得到独立发展。因此,笔者认为在传播和发展藏族天文历算方面学校应发挥重要作用。

# (一)搞好师资队伍建设,加强天文历算人才培养

从近几十年的藏学研究现状和成果而言，学习、研究宗教、医药、艺术等学科的人越来越多，成绩斐然，呈现着新时期“百花齐放,百家争鸣”的喜人景象。但潜心研习天文历算学的相对较少，在这一领域现今公认的历算专家平均年龄在50岁以上,其传承方面呈现青黄不接的现象，研究面临人才短缺问题,而解决这一问题的关键在于培养天文历算人才。高校作为培养各类人才的主要教育基地，作为传播现代科技文化、传承和弘扬民族传统文化的阵地，有条件也有能力培养适应当今和未来社会发展，具有创新精神和实践能力的高水平、高素质的藏族天文历算人才。因此，要正确认识天文历算的实用性、科学性、民族性等内在价值，高度重视这门学科的传播和发扬。不但要将其设置为必修课程,还应增加课时，延长授课学期;采取较严格的、多样的考核方式，让学生从思想上重视这门学科。这样,天文历算学的讲授才不会停留在基础知识的介绍和讲解，学生也能较全面地认识这门学科的重要性。

当然，要培养优秀的天文历算人才，首先需要建设师资队伍，要有一批精通天文历算的优秀老师。如今，除天文历算研究机构外，有的寺院里也有个别精通天文历算的高僧，学校在解决历算老师紧缺问题时可以从研究机构或寺院聘请历算专家做客座教授。西北民族大学曾聘请过拉卜楞寺喜金刚学院的天文历算专家桑珠嘉措大师做客座教授，且效果极好，今在该校担任历算课的老师和在天文历算研究机构任职的有些历算专家就是大师的及门弟子。可见，聘请寺院历算专家到高校任教是解决师资力量不足的途径之一。另外，学校还可利用署寒假组织天文历算师资培训班,邀请精通历算的专家、教授讲授天文历算学。这样既可以老带新提高师资队伍的质量，壮大师资队伍,也可拓展这一学科的交流空间。

# (二)大力开发教材，优化教材内容及结构，增强其适用性和针对性

藏族天文历算本以历史悠久、文献丰富、独具特色著名于世,在世界历法之林中独树一帜，据说西藏天文历算研究所在前几年又整理了20多部很多老专家都没见过的天文历算古籍。可见，前人确实留下了蔚为可观的文献资料，不用为缺乏研究、学习资料而发愁。尤其是藏族天文历算被列入第二批国家级非物质文化遗产名录后，其传承和发展工作更受世人的关注和各界的支持，为整理与保护、传承与发展其工作带来了很大的方便。同样,为学校开发教材、挖掘课程资源,优化课程内容和结构等工作的开展提供了丰富的资源，创造了良好的条件。故高校相关院系应组织资深望重的历算专家和专业教师一同编写天文历算教材，针对藏语言专业、藏医、藏药、藏数（藏汉双语数学)等不同专业开发、编写侧重点不同的、适应这些专业的教材，内容应由浅入深,文字通俗易懂。教学大纲要做到规范化、系列化，以便各有侧重、相互衔接,克服教学内容上的相互脱节和重复现象。保证教材的针对性、符合性和适用性极为重要，这对于提高天文历算的教学质量，推进其传承和发展有重要意义。

# (三)加强天文历算研习机构的建设，推进天文历算传承和发展

藏族传统文化中某些学科已成为当今热门学科,在国内外学习和研究者很多，在中、高等学校设为攻读专业，设立硕士、博士点，甚至建有独立的教学单位和研究机构。天文历算作为藏族传统文化的重要组成部分和最具代表性的学科，较其他学科而言已成为冷门学科，但在高下不一，寒热各异，有十里不同天之谓的青藏高原，依据藏族天文历算原理编写的藏历仍在充分发挥其预报天气、指示时令节气、指导民俗生活等内在功能。比如：西藏天文历算研究所编制的藏历不仅是农牧民生产和生活指南针，“还成为西藏气象、地震、水利等部门的重要参考依据，西藏的新闻媒体更是每日同时播报气象部门和藏历的两种天气预报”[4]可见,培养从事藏族天文历算工作的后继之人，使这一传统文化继续发挥其功能极为重要。天文历算学的历理深奥，术语难懂，难以自学成才，须靠以师带徒的形式传承。然而在国内缺少设有此专业或者专门培养天文历算人才的学校，有的高校虽开设这门课程，但课程类别设置为选修，课时少，授课学期短，又无继续深造的机会，故难以培养深通天文历算的人才。因此,创办专门培养历算人才的学校或者在高校设立这个专业,招收此专业的本硕博学生是真正解决天文历算人才紧缺的有效途径。另外，在甘青川等藏区或高校设立天文历算研究机构也是有力推动天文历算传承和发展的重要举措。

总之,有效、有序地开展文化遗产的保护、传承和发展工作，队伍和机构不可缺少。有学习和研究此学科的机构,其研习者才会连绵不断，才能将这一文化遗产代代相承,丰富祖国的天文历算宝库,保持我国文化的多样性。
