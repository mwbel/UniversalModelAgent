# 二、 “三种日”理论所体现的藏族天文历算之天文观

藏族天文历算除了编排年月日外，还有更广泛的天文学内容，其中最主要的研究内容为太阳、月亮及五大行星的视运动及位置的推算及日食和月食的预报等。 “三种日”理论正是古人观察太阳和月亮在天空中的视运动而建立起来的一套理论，同时又成为太阳、月亮及五大行星的运动以及日食、月食等天象预报的数据基础。

# ( 一) 太阳日与太阳的周日视运动

在时轮历中，太阳日的概念为“从天明能见掌纹至次日能见掌纹之间的时间间隔”，实际上是昼夜交替一次所需的时间。 因此，时轮历中的太阳日实为天文学上的真太阳日， $\textcircled{1}$ 即人们日常生活中所见到的太阳位置的真实变化情况。 每个太阳日又均分成了 漏刻的时间间隔， 漏刻应为地球自转一周的时间周期。 倘若地球没有对太阳的公转，只在原来的位置自转，那么一个太阳日的时间长度就能反映地球自转周期。 但是，真实情况是地球在自转的同时又绕太阳公转，一个太阳日显然比地球自转一周所经历的时间要长。 由于存在地球的自转和公转两种运动形式，在地球上所观测的太阳运动情况不仅有周日的视运动，还有太阳运动速度不均匀的周年视运动，因而太阳日的长度应该是不断变化的，但时轮历中认为 个太阳日为 漏刻的长度是均匀不变的。 因此，太阳日的概念只能体现地球的自转运动，并没有考虑地球同时存在的绕太阳公转的运动，那么太阳月或太阳年的时间长度就不能精确的表示太阳的周年视运动或者地球绕太阳的公转运动，需要引出宫年的概念。

# ( 二) 宫年与太阳的周年视运动

地球的公转反映在天球上就是地面上的观测者看到太阳在一年内沿着黄道自西向东转过一周，即所谓的太阳周年视运动。 这与时轮历中的宫年定义，“太阳在黄道十二宫中运行一周的时间，同时也是地上四季循环一周的时间”是相似的。 只是“宫年”定义的前半句，它与现代天文学中恒星年的定义，即太阳圆面的中心两次经过选做参考点的同一颗恒星所经历的时间间隔的概念是相同的; 而后半句又与回归年即以地球上的春分点为起算点，太阳沿黄道运动一圈又回到春分点的时间间隔的概念相同。 因而“宫年”的概念既有恒星年的意义，又有回归年的意义。 在现代天文学中，恒星年为地球绕太阳的真实公转周期，其值为  平太阳日， $\textcircled{1}$ 回归年则是地球上的观测者以地球为参考点测出的地球绕太阳的公转周期，值为  平太阳日，恒星年与回归年的时间长度不同［］。 这是由于地球自转轴的指向绕黄道极缓慢旋转，所以使春分点相对于黄道同一点不断西移形成岁差。 因此，回归年的长度并不能精确表示太阳的周年视运动，而在时轮历中宫年的时间长度为    个太阳日( 体系派的值，见表 ) ，均长于恒星年和回归年的时间，且比恒星年长 分 秒。 近代虽有人意识到了岁差现象的存在，做了一定的处理，但是将春分点的移动错误的认为是各地经度不同之故［］。 在时轮历中也考虑了太阳周年视运动的不均匀性，②并且能算出太阳距每宫宫首弧长内实际行度比平均行度多行或少行的弧刻。 由于篇幅所限，本文不再详述计算思想。

# ( 三) 太阴月与月亮视运动

月球为地球的一颗卫星，本身不发光，是靠反射太阳光才能被地球人看见，由日、月、地三者之间相对位置的改变引起了月亮从 “朔”到“望”的月相变化。 时轮历中太阴月的周期平均为  个太阳日，又可以根据“三种日”之间的关系算出 个太阴日之内太阳和月亮共同的的平均行度为 弧刻分 息 。 由于每个太阴日内除了太阳带动月亮所行的相同弧长之外，月亮还多行 弧刻，因而个太阴日内的月亮所行弧度均等于 弧刻 分 息 (  弧刻 $\div 3 0 + 4$ 弧 刻 分 息) 。 由于月亮运动受近点月的影响，其速度是不均匀的，而月亮以不同的速度在运动时，每个太阴日内所走的弧长却要相等，因而每个太阴日的时间长度是不同的。 如上所述，月行速度慢的时候太阴日的时间长度达到 漏刻，快的时候只有 漏刻。 时轮历中考虑月亮运动不均匀后给出了月亮连续两次经过某宿的周期公式: $3 0 - 2 + 1 / 1 2 6 = 2 7 . 9 9 2 0 6$ 太阴日 $= 2 7 . 5 5 4 0 7$ 太阳日［ ］，这一数值与恒星月的数值(  个平太阳日) 是相近的。

# 四、 “三种日”理论体系所体现的藏族历法观

天象的规律变化与地球上自然现象更替交错的结合成为天然的时间单位。 在各种历法中确定年、月、日的方法，都是根据天然的时间单位———回归年、朔望月和真太阳日。 藏历则以宫年、太阴月和太阳日为时间周期来编制历书，并且规定每月的 日必须使月相处于 “望”，每月 日使月相处于 “朔” 。因此，在藏历中根据太阳日 太阴月和宫年之间的时间关系确定重缺日和闰月。

# ( 一) 重日和缺日的产生

如上文所述， 个太阴月的平均长度为   个太阳日，需要安排重缺日使历法中的每月有个太阳日，且使 日月相处于“望”， 日月相处于“朔” 。 由于 个太阴月为 个太阴日，因此在藏历中根据太阴日与太阳日之间的关系确定每月中的重日和缺日，具体规定为每个太阴日结束时刻所在的太阳日的日序应该与那个太阴日日序要相同［ ］。 而太阴日与太阳日之间存在两种不同的关系，即月行速度快的时候太阳日比太阴日长，月行速度慢的时候太阴日比太阳日长。

缺日。 《时轮精要》指出，“当太阳日比太阴日长的时候，一个太阳日之内会出现一整个太阴日和前后两个太阴日的一段时间。”( ) ［ ］则  个太阳日之内有两个太阴日结束，需要舍弃其中一个太阴日。 这时可以根据 《时轮精要》中有关缺日产生的规定，“某曜连续出现两次时舍弃漏刻较多的太阴日”( ) ［ ］。 文献中给出的并不是太阴日与太阳日之间的直接关系，而是比较了太阴日与 曜 $( 9 7 \sqrt { 7 } ) ^ { 2 } = ( 2 7 \sqrt { 7 } \sqrt { 7 } ) ^ { 2 }$ $\textcircled{1}$ 之间的关系。 这是由于 曜在 星宿中运动时每曜每天运动位置的改变其实为太阳日的变换情况，而根据 “三种日”之间的关系亦可算出 曜在 星宿中的位置以及每天的行度( 由于篇幅所限，本文不详述计算方法) 。 若算出具体位置及行度，可以根据上述口诀确定具体要舍弃哪个太阴日，如藏历水蛇年 月 日( 公元 年 月 日) 之后会出现一个缺日，即缺 日。 这是由于 日和 日均出现的是金曜，根据口诀要看下一行的时间，即在一个太阳日出现了两个太阴日结束的现象，而前一个太阴日结束时金曜走了 漏刻的时间，后一个结束时则走了漏刻，由此可以判断，要舍弃后面的太阴日［ ］，故在这月中就不会出现 号，即 号为缺日。

重日。 当太阴日比太阳日长的时候，如 《时轮精要》所述，“当太阴日比太阳日长的时候，一个太阴日会占据一整个太阳日和前后两个太阳日的一段时间，这时要选择前后两个太阳日中的其中一日为重日 。 ” ( ) ［15］可以根据 《时轮精要》中的规定“ 曜没有按顺序出现，即中间缺了某曜时，要将前后两个太阴日中漏刻较少的设为重日。”( ) ［ ］来设置重日。 如藏历水蛇年  月  日为重日，即公元  年  月  日和  日均为藏历的 日，这是由于藏历的 日出现了水曜，根据 曜顺序到 日本应出现木曜，却出现了金曜，因此，需要一个重日来安排木曜，根据口诀要看下一行的时间，发现 日水曜所走的时间为 漏刻，而 日金曜的时间为 漏刻，所以要选 日为重日［ ］。

根据重日、缺日的有无和多少决定该月是 天还是 天。 个月中既没有重日又没有缺日的月份，在藏历中称为“吉祥月” 。

# ( 二) 闰月的产生

综上所述，藏历中重日、缺日和闰月的产生，是根据太阳日、太阴月及宫年之间的关系，经过严格的运算规则计算得出的结果，是具有科学意义的，并不是如一些对藏族历法了解皮毛的人所言吉日重复、凶日缺掉那么简单。

“三种日”理论中太阳日、太阴月和宫日与现代天文学中的真太阳日、朔望月和回归年( 恒星年) 相对应。 人们根据观察得出一个太阳日为 漏刻与一个真太阳日分为 个小时的意义完全相同，只不过是使用的单位有所不同而已，两种日在天文上都不够精确，一般用于历法的制定。 太阴月与朔望月的值是相同的，考虑月亮的不均匀运动后又给出了月亮连续两次经过 星宿中某宿的周期为太阳日，这与恒星月的值也相似，由于时轮历中对月球运动的规律认识较透彻，因而其推算出的日月食的初亏、食甚和复原等数据精确，这也是时轮历中使用太阴日的一个很重要的优点。 宫年均长于恒星年和回归年的时间，且比恒星年长 分 秒，而且在时轮历中不管是体系派还是作用派，均把恒星年和回归年混为一谈。

根据“三种日”之间的关系，在历法中设置了重日、缺日和闰月。 太阳日与太阴日之间日序的相互配合产生了重日与缺日这种藏历中独有的结构，重日与缺日的积累便形成大、小月份。 置闰则是按宫月与太阴月的配合关系，闰章为 年 闰。
