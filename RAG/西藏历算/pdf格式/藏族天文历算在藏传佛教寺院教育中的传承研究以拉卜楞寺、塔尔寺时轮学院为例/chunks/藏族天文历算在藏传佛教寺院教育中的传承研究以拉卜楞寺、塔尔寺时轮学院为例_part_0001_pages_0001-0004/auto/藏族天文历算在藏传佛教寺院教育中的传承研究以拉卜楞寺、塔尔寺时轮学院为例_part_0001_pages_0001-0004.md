# 藏族天文历算在藏传佛教寺院教育中的传承研究

以拉卜楞寺、塔尔寺时轮学院为例

[摘要］藏族天文历算是藏族传统文化的重要组成部分。在藏区建立学校教育体系之前,藏族天文历算的主要传承机构是藏传佛教寺院中的时轮学院，即"丁科尔扎仓”,该学院在继承和发展藏族天文历算、培养历算人才方面做出了重大的贡献。其中拉卜楞寺和塔尔寺的时轮学院历史久远，堪称藏族科技教育史上的奇葩。以历史文献和实地调研相结合,概述藏族天文历算在寺院教育中的传承历史、现状以及原因等，旨在了解这一传统文化在寺院教育中的传承情况。

# 一、概述藏族天文历算在寺院教育中的传承历史

藏族天文历算是藏族传统文化的重要组成部分，归“大小五明”之“小五明”，它是世代居住、生活在青藏高原的藏族人民结合自己原有的天文、历法、占星术、算学等研究的基础上，吸收、借鉴国内外其他民族的相关知识而形成的独具民族特色并富含科技知识的一门学科，其内容涉及气象预测、星宿运动、宇宙学说、五行学说、数据速算、历书编制等等。从古至今,藏族天文历算不仅在藏族人民的农牧业生产和生活中起着重要的指导作用,而且在诊病治病、时令采药、季节养生等方面以及宗教节日确定、学法修行等宗教仪轨方面均起着至关重要的作用。这一学科的传承和发展与藏传佛教寺院教育有着极为密切的关系。据历史记载,藏传佛教各大教派中最早开设天文历算课程的是萨迦派寺院，该派到萨班贡嘎坚赞时代，除了学习显密教法外，还学习天文历算、因明等学科。之后，天文历算逐渐受到藏区各大寺院的重视,各教派寺院都陆续开设了天文历算课,萨迦寺、粗朴寺、敏珠林寺等寺院编制颁布了《藏历历书》，天文历算在藏区各大寺院得到了广泛传播和发展。

藏族天文历算是以时轮历为主的历算体系，18世纪以后，安多藏区一些较大的寺院中相继设立了专门学习、研究藏族天文历算以及培养历算人才的教育机构——时轮学院,即丁科尔扎仓(dus vkho hr-wa tshang)。以甘、青藏区为例,1763年甘肃拉卜楞寺创建了时轮学院;1773年青海隆务寺创建了时轮学院;1802年,青海夏琼寺将1797年所建的医明学院改为时轮学院;1817年青海塔尔寺创建了时轮学院。在此影响下,莲花台寺、佑宁寺、却藏寺、东麻昂寺、东科尔寺、石藏寺、赛力亥寺、阿粗乎寺、都兰寺、拉加寺、卧科寺、郎木寺、西仓寺等寺院也先后设立了时轮学院。①此后,藏族天文历算的教育中心也逐渐由卫藏转移到安多。

时轮学院的创立把研习天文历算之风推向了新的高峰,造就了一大批精通天文历算的学者，留下了许多极其丰富的天文历算研究专著，极大地推动了藏族天文历算学的发展进步。同时也促进了历算典籍等科学书籍的收录、校订、翻译等工作。据不完全统计，拉卜楞寺所收藏的天文历算书籍达 500余种。另外,拉卜楞寺、塔尔寺、隆务寺等一些大寺院编制了各种详本、略本《藏历历书》,对蒙藏地区的农牧业生产、生活等各个领域起到了举足轻重的作用。后来由于受“文革"的影响，天文历算的研习曾一度处于低谷,甚至停止了《藏历历书》的编制。“文革"结束后,藏区一些寺院先后恢复了时轮学院，并再次承担起传承和延续藏族天文历算学的任务。现今，拉卜楞寺和塔尔寺传承研习“十明"之风未减，属安多藏区规模最大、僧众最多、学习制度最完备、人才荟萃的两座著名寺院。调查研究这两大寺院时轮学院的天文历算研习现状，可管窥藏族天文历算在寺院教育中的传承和发展概况。

# 二、天文历算在寺院教育中的传承现状

如上所述，寺院时轮学院是集中培养天文历算人才和传播藏族天文历算的重要基地，如同藏传佛教寺院的其他学院一样，在教学内容、方法以及学位授予、考试制度等方面有一套完整的体制。据记载，该学院是根据《时轮金刚》的理论以及去香巴拉理想国的进程,安排学习时间、内容和科目等。僧人除了要学会诵读经文、音韵手诀等仪轨外，主要研习天文历算和梵文，兼学语法、辞藻等知识。各寺时轮学院在学级设置、选用教材、“曲妥(chos thog，即学期)”②安排方面有所差异。

# (一)拉卜楞寺时轮学院

拉卜楞寺时轮学院是由第二世嘉木样·久美昂吾遵照第六世班禅华丹益西的旨意创建的，设有初、中、高3个学级，但没有严格的学习年限。初级班主要学习《妙吉祥名号经》《无上供养经》《诸佛赞》《满足心愿经》等经文和音韵手诀,通过背诵等测试即可升入中级班。中级班学僧除了要学会《时轮金刚经》《普济经》时轮金刚升起次第经》等外,还要学会用彩色细沙绘制坛城,考试合格者才能升班。高级班的学僧除了系统学习天文历算、梵文等知识外，还要研究时轮金刚、怖畏九首金刚的升起次第与圆满次第之道,并广授灌顶，修行禅坐。目前，拉卜楞寺时轮学院共有150 余人，其中初级学僧 80余人，中级学僧15人，高级学僧40余人。

因时轮历源自《时轮根本经》时轮摄略经》，以及《胜乐金刚空行经》《无垢光释广严论》《韵律占星经》等经论，故拉卜楞寺时轮学院规定以上典籍为必修课程，但目前掌握其中一些核心要义的历算专家或年事已高,或已圆寂。现今该学院天文历算的教学和研究主要是以《时轮历算精要》③为主，以《历算综论》④和《历算运算大全》③等为参考辅助教材。

“曲妥"是寺院教育中定期的学习、修法时间，一般被汉译为学期。拉卜楞寺时轮学院共有9个学期,其中包括四个大学期(每学期为30天)、两个中学期(每学期为 20天)、三个小学期(每学期为15天)。学僧除了在这些定期的学习时间学佛修法、研习五明知识外，平时还要根据老师安排的教学内容学习天文历算及课后的反复练习演算。天文历算的教学程序同修习经文典籍一样，首先要背诵《时轮历算精要》等历算教材中的基础知识部分,计算原理等有些口诀须倒背如流。其次就是讲授,即讲解、阐释天体结构、星宿运动等基础知识和基本概念;七曜位置的推算、日月食的预算等等。该院每年有四次大型的考核，即每季度一次，采取背诵、阐释、笔试等考核方式。

时至今日，拉卜楞寺时轮学院虽然沿袭了学习天文历算的传统，但从调查得知，该院的一些教育制度已呈现式微趋势。按照旧时的传统，学僧到高级阶段就要系统学习天文历算知识，由于精心钻研天文历算的僧人较多，故拉卜楞寺时轮学院曾出现过许多知名的天文历算学家,如近现代闻名全藏区的历算学者智观巴·贡却乎丹巴饶杰、俄琪·嘉央勒谢、毛尔盖·三木旦、桑珠嘉措、喜饶群培、洛桑嘉措等等，其中有的历算学家名扬全国，乃至国外。但现今，天文历算在时轮学院已变成选修课,升入高级班后，学僧可以根据自己的兴趣研习天文历算，但投身于天文历算学习和研究的僧人较少，一般都是学院堪布和翁泽根据编制历书等需要，安排几个高级班的学僧跟着本学院的历算老师学习天文历算,对学僧的基本要求就是能够独立编制出一年的年历，在旧时这仅是考取“孜然巴”①学位的条件之一。按照传统惯例，该院虽然每年都组织僧人编制年历，但除了将其中一本奉送给寺主之外，并不公开颁布发行。目前，该学院有两名天文历算老师，五、六名学习天文历算的学僧。

# (二)塔尔寺时轮学院

塔尔寺时轮学院系第四十五任寺主却西·阿旺协智丹贝尼玛按照上师香撒班智达的意愿创建的，目前有80余名僧人，民族成分有藏、汉、蒙、土族等。

据记载,该学院曾设有五个学级：初级班、吉然班、佐然班、多杰罗宏班和尕然班。初级班(zur pavi vdzin grwa)的学僧主要背诵《妙吉祥名号经》等经文;升入吉然班(bskyed rimvdzin grwa)和佐然班 (rdzog rimvdzin grwa)后,学习音韵手诀、天文历算、梵文、书法、绘制坛城等，然后升到多杰罗宏班(rdo rje slob dpon yivdzin grwa),多杰罗宏其实是个职务,担任三年的多杰罗宏后即可升级;尕然班（bgv rams vdzingrwa,)学僧要通晓天文历算、辞藻、声明等各种知识。

关于天文历算的学习内容和教材，在《塔尔寺史话》和《塔尔寺志》中有记载,该院学僧除了参加寺院举行的佛事活动和本院进行的法会外,学习研究的内容主要有三个方面,其中之一就是“表科外修，主要讲解整个宇宙的生起、运行规律及毁灭。时轮乘认为宇宙轮转与时季的变化,犹如一个有规则的巨轮，年复一年,昼夜不息地运转,故称‘金刚时轮’,即表科外轮。这门学科详细论述了整个宇宙的生起形成与运转规律以及将来毁灭的道理，并解释了‘劫'的缘起。在修学过程中,以讲、听为主，传授推算仪和图谱,观测经纬和星座位置,亦修‘数理'推算法和占卜法、天文历算法、天干地支运用法等。"该学院将“《时轮经无垢光释》、班禅洛桑却吉坚赞的《时轮广释》、赤代嘉木样图旦尼玛的《时轮生起次第》、克珠杰的《时轮圆满次第笔录》等典籍设为必修课。"[2]

从上述记载以及实地调研收集的材料来看，塔尔寺时轮学院曾是青海地区传承和发展藏族天文历算的主要场所。据了解，以前该学院有成就的僧人将测算出的日月食的发生时刻、食延时间、入食方向、食分大小、颜色等写在纸上并贴在大经堂的墙上或门上,看谁推算的更准确。有的僧人还能测算出地震发生的时日，将其预算结果也贴到大经堂门上。每天下午学院还举办天文历算辩论会，师生或学僧之间相互交流、切磋。可见，以前塔尔寺时轮学院研习天文历算的氛围很浓厚。另外，该学院还结合藏区的气候冷暖、季节变化等特征编制《藏历》,这不仅为蒙藏地区的农牧民提供了纪年月日的历法功能，还能预报天文现象、气象趋势、季节变化和自然灾害等，为农牧民的生活和生产起到了指导作用，倍受蒙藏人民的欢迎。但现今，该院研究、学习天文历算之传统不尽人意，僧人除了参加佛事活动外，一般都以学习经文、绘制坛城等为主，已无人学习天文历算，也没有严格的学级设置，只有年迈的个别老僧人才说得出以上学级名称。虽然时轮学院的介绍栏中仍记载着天文历算是该扎仓的学习科目，成绩优秀者授予“孜然巴"的学位,但这一学位在期待人才辈出,填补空缺。该学院现设有8个学期,包括一个大学期（45天),四个中学期即春夏秋冬四季学期(每学期30天),三个小学期，分别为26天、15天、10天。这些学期的学习内容也是学习经文、练习跳法神舞、吹奏法器等有关宗教事项。

# 三、结论

寺院时轮学院曾是继承和发扬藏族天文历算的重要场所，是编制《藏历历书》的主要机构，为藏族科技文化的传承和发展做出了重大贡献。但现今，寺院时轮学院培养天文历算人才的教育功能已呈现式微趋势，其原因来自多方面，而主要原因可归为以下几点：

1.寺院教育体制的变化。时轮学院虽有严谨的教学体制、固定的教学内容、严格的考试制度以及健全的学位制度,但随着社会发展和历史的变革,有些教育制度执行的比较松弛，教学内容的侧重点发生了变化，学院对天文历算的学习和研究方面没有较高的要求。

2.师资力量的不足。天文历算是一门专业性很强的学科，为了便于背诵，其典籍是以藏族传统的书写形式,即格律撰写，由于每句音节数目须相同则必有缩写之处，且文中应用大量的藻语异名，所以若不经过老师讲解,即便是最基本的运算方法和理论，也让藏文水平较好者难以入门。从时轮学院目前的总体情况来看，精通天文历算的僧人较少，天文历算教师短缺问题较为明显。

3.传承目的的变化。时轮学院研习天文历算的主要目的有两点：一是编制历书；二是测算日月食预报。编制历书虽是时轮学院的惯例，但随着研习天文历算传统的衰落，会编制《藏历》的僧人也日益减少，又因其工作量较大，计算过程较复杂,编写一本历书最少需要三个多月的时间，故从事编制历书的僧人逐年减少。另外，寺院测算日月食是基于修行的需要,当今人们需了解日月食等天文现象的信息时可以通过广播、电视等媒体预知，故以修正圆满而研究天文历算的目的渐渐疏远。

总之，随着现代社会的发展，寺院教育在各方面都发生了实质性的变化。虽然藏族天文历算仍是时轮学院的学习内容，《藏历》也在充分发挥着其预报天气、指示时令节气、指导民俗生活等内在功能，但寺院时轮学院传承和发展藏族天文历算、编制《藏历历算》的传统已呈现势微的趋势。
