# 1．第一类奇点的性质与无奇点变量

$e = 0$ 对应的是圆轨道，此时，近地点已失去意义，相应的近地点经度 $\omega$ 以及从近地点起量的各种近点角 $M , E , f$ 也随之失去它们的几何意义．这当然要反应到摄动运动方程中去，即 $\dot { \pmb { \omega } }$ 和 $\dot { \mathcal { M } }$ 的右端出现1/e因子，见(4.80)一(4.82)式．因此，该方程的解在p=0时失效就不难理解了．i=0或180°对应的轨道与赤道面重合，既然如此，升交点变得无意义，因而升交点经度 $\varOmega$ 以及从升交点方向起量的近地点经度 $\omega$ 也随着变得无意义，相应的摄动运动方程和的右端出现1/sini因子，这就是前面给出的解在此情况下失效的原因．但是，上述e=0和i=0或180°对应的运动，并无特殊之处,仍然是正常的。 $e = 0$ 和 $i = 0$ 或 $1 8 0 ^ { \circ }$ 之所以是奇点，其原因在于这两种情况对应的轨道不能用 $\omega , M$ 和 $\varOmega , \omega$ 来表示．只要另选一些参数代替它们，奇点就会消失．例如，对于圆轨道，可选择

$$
a , i , \mathcal { Q } , \xi = e \cos \omega , \eta = - e \sin \omega , \lambda = M + \omega
$$

这六个量作为椭圆轨道的基本变量，问题就得以解决．这组变量对应的摄动运动方程(4.83）式和(4.84）式右端不再出现 $\mathbf { 1 } / e$ 因子，其解亦是如此．从几何上看，当 $e = 0$ 时，尽管 $\omega , M$ 无意义，但 $\xi = 0 , \eta = 0 , \lambda$ 和相应的 $\pmb { \mathscr { u } } = \pmb { f } + \pmb { \omega } , \tilde { \pmb { u } } = \pmb { E } + \pmb { \omega }$ 是确定的，它们都是从升交点方向起量的卫星“经度”，近地点不定无妨．由上述六个变量可以完全确定卫星的向径 $\pmb { r }$ 和速度 $\dot { \pmb { r } } _ { 3 }$ ，相应的关系式下面将要给出．因此， $e { = } 0$ 和 ${ \bf \nabla } { \bf \dot { \ell } } = 0$ 或 $1 8 0 ^ { \circ }$ 这类奇点只是对椭圆轨道的几何描述问题，并无任何力学内涵．既然如此，适当的选择变量来代替六个椭圆根数就可解决问题，这类变量常被称为无奇

点变量.

# 2．第二类奇点出现的原因

尽管通约奇点与上述第一类奇点都是摄动解中的小分母问题，但却有截然不同的性质，从几何上看，这类奇点并无反常之处，摄动运动方程在 $i = i _ { c }$ 和 $\scriptstyle { \overline { { n } } } = n _ { c }$ 时，右函数仍旧是正常的，相应的点是方程的常点，而不是奇点，显然，这类奇点与求解摄动运动方程的方法有关，具体地说，就是按小参数 $\pmb { \varepsilon }$ 的幂构造级数解时出现的问题．要说明这点也很容易，如按一般的摄动法构造 $\boldsymbol { J } _ { 2 } ,$ $J _ { 3 } , J _ { 4 }$ 项摄动解时，参考解不是平均根数 $\boldsymbol { \overline { { \sigma } } } ( t )$ ，而是无摄根数 $\pmb { \sigma } _ { \mathfrak { S } }$ $( t )$ ，那么与 $\pmb { \omega }$ 有关的长周期项 $\sigma _ { \textit { t } } ^ { ( 1 ) } ( \textit { t } ) , \sigma _ { \textit { t } } ^ { ( 2 ) } ( \textit { t } ) _ { : }$ …不会出现，从而就不存在临界角问题.即使引用平均法思想，只要按 ${ \pmb \varepsilon } ^ { 1 / 2 }$ 的幂来构造级数解，整个通约问题都不会出现（见下一章有关内容)．这说明，对卫星的运动及其对应的摄动运动方程而言，通约奇点并非实质性的，但是，相应的运动是否具有与一般运动不同的特征，或是某种“异常”现象，这是值得探讨的，下一节就专门介绍这一内容.

\Delta \omega _ { i } ^ { ( 1 ) } ( t ) + \Delta \varOmega _ { i } ^ { ( 1 ) } ( t ) \} .
$$

各式右端括号 $f \}$ 内的项都是原平均根数法的相应结果，其中被积函数 $\left\{ \frac { d \sigma } { d t } \right\} _ { \sigma } ,$ 还表示原结果 $\sigma _ { 2 } ( t - t _ { 0 } )$ 和 $\sigma _ { \textit { \textbf { i } } } ^ { ( 1 ) } ( t )$ 积分前的对应

形式．同样， $\left\{ { \frac { d \sigma } { d t } } \right\} .$ 不包括 $\sum _ { j } \frac { \partial f _ { \uparrow \circ } } { \partial \sigma _ { j } } ( \sigma _ { \iota } ^ { \left( 1 \right) } ) _ { j }$ 和 $\frac { \partial n } { \partial a } a _ { \imath } ^ { \left( 2 \right) }$ 这两部分.将相应的结果引进，按定义(6.6)式表示成无奇点变量的函数形式，积分后即得 $\sigma _ { 2 } ( t - t _ { 0 } ) + \Delta \sigma _ { \ell } ^ { \scriptscriptstyle ( 1 ) } ( t )$ 的具体表达式.

上述结果的精度同样可达到 $1 0 ^ { - 5 }$ ，在通讯卫星定轨工作中的应用已证实了这一点。其简化形式曾公开发表过，请见文[8].

这一章给出的方法,同样适用于其它各种摄动因素，将在后面几章逐一介绍。
