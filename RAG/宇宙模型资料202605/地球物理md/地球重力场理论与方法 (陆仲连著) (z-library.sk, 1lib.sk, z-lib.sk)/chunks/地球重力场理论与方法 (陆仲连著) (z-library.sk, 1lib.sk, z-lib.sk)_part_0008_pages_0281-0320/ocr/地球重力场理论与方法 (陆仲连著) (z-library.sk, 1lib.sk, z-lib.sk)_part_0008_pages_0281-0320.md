# 12.9天文重力水准的重力点布设

为了计算重力改正项，需要在天文点周围一定范围内进行重力测量。为了找到重力点的布设方案，先要分析用天文重力水准方法所求得的相对高程异常差包含那些误差，并概略估计它们的数值。

依据 $( 1 2 \cdot 5 1 )$ 式可以看出，所求得的相对高程异常差的误差来自以下三个因素。

1.天文水准项中所用的天文大地垂线偏差值有误差。

假设A、B两点的天文大地垂线偏差的中误差相同，都是 $\mathfrak { m } _ { \grave { v } } { } ^ { * }$ ，则运用误差传播定律可得，由它们引起的相对高程异常差的误差 $\mathfrak { m } _ { \varDelta \zeta | }$ 为

$$
\mathrm { m } _ { \Delta \xi _ { 1 } } = \sqrt { \mathrm { ~ 2 ~ } } \langle \mathrm { m } _ { \mathrm { 4 } } \cdot
$$

亦可写为

$$
\frac { \pi _ { 5 3 } } { 2 i \sin 1 ^ { \prime \prime } } = \frac { \pi _ { \theta } } { \sqrt { 2 } }
$$

其中 $\mathfrak { m } _ { \theta }$ ，以角秒为单位。因为

$$
\theta ^ { * } = ( \varphi \mathrm { - } \mathrm { B } \mathrm { - } 0 . 1 7 1 \mathrm { s i n } 2 \mathrm { B } \mathrm { H } ^ { \gamma } ) \mathrm { c o s } \mathrm { A } + ( \mathrm { \lambda \mathrm { - } \mathrm { L } } ) \mathrm { c o s } \varphi \mathrm { s i n } \mathrm { A }
$$

其中大地坐标(B，L)的偶然误差很小，可以不予顾及，因此可得

$$
\mathfrak { m } _ { \mathfrak { s } } \cdot = \sqrt { \mathfrak { m } _ { \mathfrak { s } } { } ^ { 2 } \mathfrak { c o s } ^ { 2 } \mathbb { \Lambda } \ \overset { ! } { \mathfrak { i } } \mathfrak { m } _ { \mathfrak { k } } { } ^ { 2 } \cos ^ { 2 } \mathfrak { c o s } ^ { 2 } \mathfrak { c o s i n A } }
$$

取 $\mathsf { A } = 4 5 ^ { \circ }$ 的情况来估算，考虑到

$$
\cos ^ { 2 } 4 5 ^ { \circ } = \sin ^ { 2 } 4 5 ^ { \circ } = { \frac { 1 } { 2 } }
$$

则得

$$
\mathbf { m } _ { 6 } \cdot \mathbf { \Omega } = \sqrt { \frac { 1 } { 2 } ( \mathbf { m } _ { 5 } \cdot \tan \frac { \cdot ^ { 2 } } { \alpha } \mathbf { c } \mathbf { o s } ^ { 2 } \phi ) }
$$

通常天文经纬度的测定精度为

$$
\begin{array} { r } { { \bf m } _ { \mathrm { e } } = \pm 0 . 3 ^ { \prime \prime } \qquad { \bf m } _ { \mathrm { { a } } } = \pm 0 . 0 3 ^ { \mathrm { s } } = \pm 0 . 4 5 ^ { \prime \prime } } \end{array}
$$

并且对于我国适中的纬度 $\varphi = 4 0 ^ { \circ }$ .则可算得

$$
\mathfrak { m } _ { \mathfrak { s } } \cdot = \pm 0 . 3 3 ^ { \prime \prime }
$$

代人 $( 1 2 \cdot 8 1 )$ 式得到

$$
\frac { \mathfrak { m } _ { \le z _ { \uparrow } } } { 2 l \sin 1 ^ { \prime \prime } } = \pm 0 . 2 4 ^ { \prime \prime }
$$

2．在计算重力改正项时未顾及模板以外区域重力异常的影响。

$( 1 2 \cdot 6 2 )$ 式给出了忽略远区域重力异常的影响可能引起的最大误差。而通常讨论误差时要的是中误差，据误差理论，中误差一般为最大误差的三分之一。所以，由此而引起的相对高程异常差的误差 $\mathfrak { m } _ { \Delta \zeta _ { \geq } }$ 是

$$
\cfrac { \mathrm { m } _ { { \Delta } \zeta _ { 2 } } } { 2 l \mathrm { s i n } 1 ^ { \prime \prime } } = \pm 0 . \cot ^ { \prime \prime } \cfrac { \widetilde { \Delta } \mathbf { g } _ { \mathrm { m } } } { \rho ^ { 2 } - 1 }
$$

3.在计算重力改正项时所用重力数据的不充分。

由 $( 1 2 \cdot 5 2 )$ 式,重力改正项为

$$
\Delta \zeta _ { * } = \zeta _ { * } ^ { \mathrm { B } } - \zeta _ { * } ^ { \mathrm { A } } + l ( \theta _ { \mathrm { e } } ^ { \mathrm { B } } + \theta _ { \mathrm { g } } ^ { \mathrm { A } } )
$$

假设右端前两项的误差总和与后一项的误差相等，且认为与有相同的误差m。，则可得到重力改正项的误差 $\ m _ { \Delta \xi _ { 3 } }$ 为

$$
\frac { { \mathfrak { m } } _ { { \mathfrak { s } } _ { 3 } } } { 2 l { \mathfrak { s i n } } 1 ^ { \prime \prime } } { = } { \mathfrak { m } } _ { \mathfrak { s } }
$$

·从上面的分析看出，后两项误差m和mx与所用重力数据的范围(积分半径R=pl)和在此围范内所有的重力点密度有关。为了求得较大的天文大地网上所有起始边处的相对高程异常值，可以分为两个等级布设天文重力水准路线。其中一等路线的天文点周围的重力点可按$\mathfrak { m } _ { \mathfrak { g } } = \frac { \bar { 5 } . 0 ^ { \prime \prime } } { \sqrt { 2 l } }$ 和R=4l进行布设；二等路线的天文点周围的重力点按me $\mathrm { m } _ { \mathfrak { g } } = \frac { 1 2 . 0 ^ { \prime \prime } } { \sqrt { 2 l } }$ 和 $\mathbf { R } = 3 l$ 进行，其中的2以公里为单位。从这些规定出发，并假定2l=100km，由（12·83）、（12·84)和(12·85)式算得如下的结果：

对于一等路线：

$$
\begin{array} { r } { \mathfrak { m } _ { \mathtt { a v } _ { 1 } } = \pm 0 . 1 1 6 \mathfrak { m } \qquad \mathfrak { m } _ { \mathtt { a v } _ { 2 } } = \pm 0 . 0 4 5 \mathfrak { m } \qquad \mathfrak { m } _ { \mathtt { a v } _ { 3 } } = \pm 0 . 2 4 2 \mathfrak { m } } \end{array}
$$

对于二等路线

$$
\begin{array} { r } { \mathfrak { m } _ { \mathtt { a } \mathtt { t } _ { 1 } } = \pm 0 . 1 1 6 \mathrm { m } \qquad \operatorname* { m } _ { \mathtt { a } \mathtt { t } _ { 2 } } = \pm 0 . 0 8 5 \mathrm { m } \qquad \operatorname* { m } _ { \mathtt { a } \mathtt { t } _ { 3 } } = \pm 0 . 5 8 2 \mathrm { m } } \end{array}
$$

计算时还采用了△gm=士35毫伽。比较这些结果看出,mz比m小得多，是可以忽略不计的；  
至于 m&x,对于二等路线也是可以忽略的,但对于一等路线,严格说来，还是应当顾及的。

根据规定的mq和R，按照&11.8所述的方法就可以进行满足天文重力水准需要的重力点布设。

# 第十三章 最小二乘配置的应用

# 13.1最小二乘配置的原理

最小二乘配置是综合平差、滤波和推估的一种广义推估方法。除了一组参数以外，它还包含两组随机变量：一组称为信号，它们之间通过协方差而相关;另一组是测量误差·亦称噪声。由于它可以将不同类型的观测数据作统一处理，特别适用于研究有关地球重力场的问题，但也可用于大地测量和摄影测量中的某些问题。

最小二乘配置采用的数学模型是

$$
l { = } \mathbf { A } \mathbf { X } { + } \mathbf { U } \mathbf { S } { + } \mathbf { n }
$$

式中 $\iota$ 为观测值向量：X为非随机的参数向量；A是一个已知的系数阵，它表明参数X对观测值l的影响；AX通常是由参数的原始非线性函数线性化后得到的;S为信号向量·它的形式为

$$
{ \mathsf { S } } = ( { \mathsf { t } } , { \mathsf { u } } ) ^ { \top }
$$

其中向量 $\mathfrak { t }$ 包括 $\mathfrak { q }$ 个观测点的信号，向量 $\mathfrak { u }$ 包含 $\mathbf { m } - \mathbf { q }$ 个待推估的信号；U是如下的一个系数阵

$$
\mathrm { U } = ( \mathrm { I } , \mathrm { O } )
$$

其中I是 ${ \mathfrak { q } } \times { \mathfrak { q } }$ 阶的单位阵，O为 $\mathsf { q } \times ( \mathsf { m } - \mathsf { q } )$ 阶的零矩阵； $\pmb { \mathrm { n } }$ 为随机的观测值误差向量。

把所有的随机量合并成一个向量 $\mathrm { v }$ ，则有

$$
{ \mathrm { V } } { = } ( \mathsf { \Omega } _ { { \mathrm { n } } } ^ { \mathrm { s } } ) { = } ( \mathsf { s } _ { 1 } , \mathsf { s } _ { 2 } , { \cdots } { \cdots } , \mathsf { s } _ { \mathrm { m } } , \mathsf { n } _ { 1 } , \mathsf { n } _ { 2 } , { \cdots } { \cdots } , \mathsf { n } _ { \mathrm { q } } ) ^ { \mathrm { T } }

# 813.6引力位球谐系数的求定

在第十章中讲到，地球重力场模型的低阶位系数，利用对卫星的观测所获得的数据来求定，可以有较高的精度。此时，最小二乘配置是一种有效的数据处理方法。。

对卫星的光学观测得到的观测值是卫星在观测时刻的赤经 $\pmb { \alpha }$ 和赤纬8,对卫星的激光观测得到的观测值是观测时刻由测站到卫星的距离 $\pmb { \mathrm { r } }$ 。这些观测值是测站坐标X、Y、Z的函数，也是观测时刻的卫星轨道根数的函数。而观测时刻的轨道根数又可表示为某一时刻 $\mathrm { \Delta T _ { 0 } }$ 的轨道根数ao、e。、i、Ω、ω、M及其摄动值。当认为已经消除了其他摄动因素的影响以后,剩余的摄动值就只和地球扰动位系数C和Sm有关了。因此，对每个观测值可列出如下的函数式：

的该两点的水平距离。 ${ \sf K } = { \bf K } ( { \bf P } , { \bf Q } )$ 作为 $\mathbf { P }$ 和 $\mathbf { Q }$ 点的位置函数，还必需是个调和函数，即满足

$$

# 13.8重力异常的局部协方差函数

扰动重力场中具有最多数据的信号是重力异常，因此对于局部地区，总是先找出重力异常的协方差函数，再由它导出其它信号的协方差函数。

如前面讨论扰动位的局部协方差函数那样，地面上重力异常的协方差函数也可以看成是距离 $\iota$ 的函数，即

$$

其中的l按(13·125)式计算。莫里兹提出[30],可以用三个基本参数来表征重力异常协方差函数的局部特征，那就是：方差C、相关长度S和曲率参数X(或梯度方差G)。它们的定义如下。

方差 $\mathbf { C } _ { \mathfrak { d } }$ 为协方差函数C(l)在 $\scriptstyle { l = 0 }$ 时的数值，即

它的大小体现了在本区域内重力异常的变化幅度。
