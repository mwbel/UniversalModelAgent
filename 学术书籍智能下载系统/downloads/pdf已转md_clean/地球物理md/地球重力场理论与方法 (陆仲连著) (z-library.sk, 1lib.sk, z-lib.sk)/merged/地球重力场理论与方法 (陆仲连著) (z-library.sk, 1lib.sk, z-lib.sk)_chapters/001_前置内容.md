# 内容简介

本书探讨论述了利用地面重力数据研究地球外部重力场的理论和方法。第一章介绍运用可倒摆和物体自由下落测定地面点的绝对重力值；第二章介绍进行重力差测定的仪器和数据处理过程；第三章对在海面上进行重力测量作简要论述；第四、五章分别论述位理论和调和函数及其边值问题；第六章讨论水准椭球的重力场。第七章初步论述地球重力场：第八章讨论研究地球重力场的传统理论—-司托克斯理论；第九章讨论研究地球重力场的现代理论；第十章介绍地球重力场模型的概念、建立方法、现有模型的状况及其应用；第十一、十二章分别论述利用地面重力数据推求垂线偏差和高程异常的理论和方法；第十三章阐述最小二乘配置方 十二章分别论还利用地面基力默据m
