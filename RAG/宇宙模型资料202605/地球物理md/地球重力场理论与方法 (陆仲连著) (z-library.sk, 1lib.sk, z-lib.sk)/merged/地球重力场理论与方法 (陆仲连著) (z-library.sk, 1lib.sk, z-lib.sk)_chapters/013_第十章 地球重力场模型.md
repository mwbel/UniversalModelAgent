(3)美国海面武器中心(NSWC)发展的系列

该系列的模型主要是为支持导航卫星系统而研制的。为此，在全球及美国本土布设有它们的卫星跟踪站网。曾给出了用于海军导航卫星系统(NNSS)产生精密星历的位系数模型NWL9D、10F、10E 和 NSWC9Z-2,用于广播星历的模型有NWL-10C和WGS-72EGM。用于全球定位系统(GPS)产生星历的模型是WGS-84EGM,它是以NSWC9Z-2的改进，完全到180 阶次。这些模型的有关资料大多未予公布。

（4)法国空间大地测量研究组(GRSS)和德国大地测量研究所(DGFI)合作发展的GRIM系列模型

已经见到的有GRIM-1、2、3、3B、4C2和4S2六个一般的模型和一个适用于Lageos 卫星定轨的GRIM3B-L1模型,它们的阶数也较低，其中的GRIM-4模型将用于TOPEX/POSEI-DOM卫星的定轨。此系列也在向高阶 $( \mathbf { n } ) 2 1 8 0 )$ 模型发展。

(5)美国俄亥俄州立大学(OSU)大地测量科学系发展的OSU 系列模型

这个系列的模型主要是为大地测量的应用而研制的，因此致力于高阶模型。有全球第--个完全到180阶次的模型RAPP78,以后又有RAPP81（180阶）OSU86C/D(250阶）、OSU86E/F（360 阶）OSU89A/B(360阶)和OSU91(360阶）。

除了上述的5·个系列之外，还有一些其他的模型。如：德国汉诺威(Hornovir)大学的GPM1和GPM2,完全阶次为200;美国德克萨斯(Taxes)大学的 TEG-1、TEG-1S和 TEG-2,都是完全到50阶次的位系数模型。

我国西安测绘研究所在1977年首次推出了我国建立的地球重力场位系数模型DQM77A和 DQM77B其最高阶数分别为22和20,在1984年又作了改进，推出了最高阶数为50的DQM84A、B、C、D四个略有差别的模型。1989年武汉测绘科技大学推出了完全到180阶次的WDM-89模型。这些模型都是以国外的低阶模型为基础，利用我国及周围地区的地面重力数据对高阶位系数值进行改进,因此在表征我国重力场特征上它们要略优于同类的国外模型。
