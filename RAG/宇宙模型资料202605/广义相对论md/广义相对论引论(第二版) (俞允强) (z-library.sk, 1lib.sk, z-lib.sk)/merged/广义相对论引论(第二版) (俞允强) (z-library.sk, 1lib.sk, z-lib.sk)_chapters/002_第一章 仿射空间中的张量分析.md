# 第一章 仿射空间中的张量分析

任何物理量都是用…组数表示的，这组数的值一般与坐标的选择有关.研究这种数组与坐标变换的关系导致了张量的概念.借助于张量，相对论把物理规律表达为张量方程，使它在任一种坐标下具有相同的形式.本章我们将在仿射空间中建立张量的定义和运算，并用它讨论空间的几何性质.

# 1.2张量的对称性

首先看二阶逆变张量 $T ^ { \mu \nu } .$ ，它的分量可用一矩阵表示

$$
T ^ { n } = \left[ \begin{array} { l l l l } { T ^ { \backslash 1 } } & { T ^ { 1 2 } } & { \cdots } & { T ^ { 1 n } } \\ { T ^ { \gamma _ { 1 } } } & { T ^ { 2 2 } } & { \cdots } & { T ^ { 2 n } } \\ { \cdots } & { \cdots } & { \cdots } & { \cdots } \\ { T ^ { n } } & { T ^ { n 2 } } & { \cdots } & { T ^ { n n } } \end{array} \right] .
$$

如果它是对称矩阵，即

$$
T ^ { \mu \nu } = T ^ { \nu \mu } ,
$$

那就叫张量 $T ^ { \mu \nu }$ 对它的指标 $\mu$ 和 $\nu$ 是对称的.若它是反对称矩阵，即

$$
T ^ { \mu \nu } = - \ T ^ { \nu \mu } ,
$$

则叫张量 $T ^ { \mu \nu }$ 对它的指标 $\mu$ 和 $\pmb { \nu }$ 反称.容易看出，当张量在某一坐标下是对称(或反称)的，那么在任一坐标下它都是对称(或反称)的.重要的正是张量的对称性与坐标无关.

上述对称性质的讨论对协变张量也适用.但是它对混合张量不适用.当混合张量在一个坐标系中有对称性，在变换到另一个坐标系后，它一般不能保持对称性.

一个不对称的二阶逆变(或协变)张量总可分解成两部分之和，其一部分是对称张量，另一部分是反称张量.即

$$
T ^ { \prime \prime \nu } = S ^ { \mu \nu } + A ^ { \mu \nu } ,
$$

其中

$$
\begin{array} { c } { { S ^ { \mu \nu } = S ^ { \nu , \nu } , } } \\ { { \ldots \left( \ O \right) ^ { \mu \nu } = \dots \dotsb } } \end{array}
$$

这种分解容易唯-地确定下来.我们有

$$
\begin{array} { r c l } { { S ^ { \prime } { } ^ { n } = } } & { { \frac { 1 } { 2 } ( T ^ { \prime \prime } + T ^ { \prime \prime } ) \equiv T ^ { \prime \prime \prime } { } ^ { n } , } } \\ { { } } & { { } } & { { } } \\ { { { \cal A } ^ { \prime } { } ^ { n } = } } & { { \frac { 1 } { 2 } ( T ^ { \prime \prime } - T ^ { \prime \prime } ) \equiv T ^ { \prime \prime \prime } . } } \end{array}
$$

人们常常用圆括号表示对称组合，用方括号表示反对称组合.

对高阶张量·完全可以在上述意义下说它对某两个上标(或下标)是对称的或反称的.仍须注意，对一个上标--个下标说对称性没有意义·因这种对称性在坐标变换下不能保持.当不加限制地说某逆变(或协变)张量是对称的或反称的，那意指它对任一对上标（或下标)都是对称或反称的.如三阶的对称逆变张量应满足

$$

T ^ { \prime \prime \prime \prime \lambda } \ = - \ - \ T ^ { \prime \prime \lambda \nu } \ = - \ T ^ { \prime \lambda \nu \mu } \ = - \ T ^ { \prime \prime \prime \lambda } ,

反称张量还具有如下常用的性质：

（a）当任意两个指标取同样值时，张量的该分量为零.

$( \mathrm { ~ b ~ } ) \_ n$ 维空间中最高阶的反称张量是 $_ n$ 阶的，这张量只有一个独立分量.如三维空间中的三阶反称张量 $T ^ { \cdots , \cdots , i }$ 只有 $T ^ { \textnormal { \scriptsize i 2 3 } }$ 是独立分量.其他不为零的分量均与它相等或等值异号.

# 1.3矢量的平移和仿射联络

前面曾指出，两个不同点上的张量相减后将失去张量的性质.可是张量场的微分却正需要这样的相减.为了使微分运算不破坏张量性质，必须引入--种新的操作，叫做张量的平移.它能把 $P$ 点（坐标为 $\boldsymbol { \mathcal { X } } ^ { \star ^ { \ell ^ { \ell } } }$ )的张量平移至邻近点 $\boldsymbol { \mathrm { Q } }$ （坐标为 $\boldsymbol { x } ^ { \mu } + \mathrm { d } \boldsymbol { x } ^ { \mu } )$ 而变成 $\boldsymbol { \mathrm { Q } }$ 点的张量.平移后再作减法，将仍能保持张量性质.为实现这平移所要的新概念就是仿射联络.

为具体起见，我们从协变矢量的平移来引入仿射联络.设有 $I ^ { \gamma }$ 点的协变矢量 $A _ { \mu } ( P )$ .它平移至 $Q$ 点后相应的矢量记作 $A _ { \mu } ( P \to$ $\textit { Q } )$ .作为线性的理论，平移引起的改变 $\delta A _ { \mu } ( P )$ 应正比于 $A _ { \mu } ( P )$ ，并正比于位移 ${ \mathrm { d } } { \boldsymbol { x } } ^ { \mu }$ .这样，

$$
\delta A _ { \mu } ( P ) = A _ { \mu } ( P \to Q ) - A _ { \mu } ( P ) = { \cal { T } } _ { \mu \nu } ^ { \lambda } ( P ) A _ { \lambda } ( P ) \mathrm { d } x ^ { \nu } .
$$

$$

$$
{ \left| \begin{array} { l } { { \underline { { \mathsf { \sigma } x } } ^ { \alpha } } } \\ { { \mathsf { \sigma } } { \widetilde { x } } ^ { \prime \prime } } \end{array} \right| } _ { \mathcal { Q } } = { \left| \begin{array} { l } { { \underline { { \mathsf { \sigma } x } } ^ { \alpha } } } \\ { { \mathsf { \sigma } } { \widetilde { x } } ^ { \ast } } \end{array} \right| } _ { P } + \left( { \frac { { \mathsf { \sigma } } ^ { 2 } x ^ { \alpha } } { \mathsf { \sigma } { \widetilde { x } } ^ { \prime } \mathsf { \sigma } \partial { \widetilde { x } } ^ { \nu } } } \right) _ { P } { \mathsf { d } } { \widetilde { x } } ^ { \nu }

$$

是 $\boldsymbol { Q }$ 点的逆变矢量.(1.3.7)就是用联络对逆变矢量作平移的公式.因更高阶张量的平移公式不需要用到，我们不再写出.值得指出，标量是坐标变换下的不变量，因此它的平移不需要专门的操作.

# 1.4张量的协变微商

在张量平移概念的基础上，我们可以对张量场定义一种新的微商，它叫协变微商.对张量场求协变微商后所得的量将仍具有张量的性质.

先看标量场的微商. $T ( x )$ 对 $x ^ { \mu }$ 的普通微商用 $T _ { \mathbf { \delta } \cdot \mu }$ 表示：

$$
T _ { \cdot \mu } \equiv \frac { \mathsf { \vec { o } } T } { \mathsf { \vec { o } } x ^ { \mu } } .
$$

坐标变换后 $, T , \mu ^ { - }$ 变为 $\tilde { T } _ { \mathbf { \Gamma } , \mu }$

$$
\tilde { T } _ { \cdot \mu } \equiv \frac { \ni \tilde { T } } { \ni \tilde { x } _ { \mu } } = \frac { \ni T } { \ni x ^ { \alpha } } \frac { \ni x ^ { \alpha } } { \ni \tilde { x } ^ { \mu } } = T _ { \cdot ^ { \alpha } } \frac { \ni x ^ { \alpha } } { \ni \tilde { x } ^ { \mu } } .
$$

这表明 $T _ { \mathbf { \theta } , \mu }$ 是协变矢量.因为标量场的普通微商自动具有张量性质，所以它就定义为标量场的协变微商.协变微商用 $T _ { : \mu }$ 表示，则有

$$
T _ { \mu } = T _ { , \mu } .
$$

然后看协变矢量场 $T _ { \mathbf { \mu } _ { \mu } } ( \mathbf { \mu } _ { \mathcal { X } } )$ 的微商.它对 $x ^ { \nu }$ 的普通微商

$$
T _ { \mu , \nu } ( x ) \equiv \frac { \ni T _ { \mu } ( x ) } { \ni x ^ { \nu } } = \operatorname* { l i m } _ { Q  P } \frac { T _ { \mu } ( Q ) - T _ { \mu } ( P ) } { \Delta x ^ { \nu } }
$$

显然不再是一个张量.为了使协变微商后的 $T _ { \mu ; \nu }$ 仍是张量，我们利用平移操作，把它定义为

$$
T _ { \mu ; \nu } = \operatorname * { l i m } _ { Q \to I } \frac { T _ { \mu } ( Q ) - T _ { \mu } ( P \to Q ) } { \Delta x ^ { \nu } } .
$$

这样 $T _ { \mu ; }$ 是一个(0,2)阶张量.由协变矢量的平移公式(1.3.1)，(1.4.4)式化为

$$
T _ { \mu : \nu } = T _ { \mu , \nu } - T _ { \mu \nu } ^ { \lambda } T _ { \lambda } ,
$$

这就是协变矢量的协变微商公式.注意，(1.4.5)式右边两项都不是张量，它们之差才是张量.

为了唯一地确定其他阶张量的协变微商，我们要求协变微商满足与普通微商一样的乘法规则，即

$$
( A { \because } B _ { \cdots } ^ { \cdots } ) _ { * } = ( A _ { \cdots , \lambda } ^ { \cdots } ) ( B _ { \cdots } ^ { \cdots } ) + ( A { \cdots } ) ( B _ { \cdots , \lambda } ^ { \cdots } ) .
$$

由这一要求及标量和协变矢量的协变微商公式(1.4.3)和（1.4.  
5),就可导出其他阶张量的协变微商公式.

看逆变矢量场 $A ^ { \mu } ( x )$ .它与任意协变矢量场 $B _ { \mu } ( x )$ 可构成标量 $A ^ { \prime \prime } B _ { \prime \prime }$ .利用标量的协变微商公式(1.4.3)，我们有

$$
( \triangledown _ { \lambda } \mathfrak { s } ^ { \varrho } \ B _ { \mu } ) \ O _ { : \lambda } = ( A ^ { \mu } B _ { \mu } ) \ O _ { , \lambda } .
$$

再用普通微商和协变微商的乘法规则，得

$$
A _ { ; \lambda } ^ { \mu } B _ { \mu } + A ^ { \mu } B _ { \mu ; \lambda } = A _ { , \lambda } ^ { \mu } B _ { \mu } + A ^ { \mu } B _ { \mu , \lambda } .
$$

对 $B _ { \mu ; \lambda }$ 用公式（1.4.5)，上式改写为

$$
{ \cal A } _ { ; \lambda } ^ { \mu } { \cal B } _ { ; \lambda } = { \cal A } _ { ; \lambda } ^ { \mu } { \cal B } _ { \mu } + { \cal T } _ { \alpha \lambda } ^ { \mu } { \cal A } ^ { \alpha } { \cal B } _ { \mu } .
$$

考虑到 $B _ { \mu }$ 是任意矢量，于是有

$$
{ \cal A } _ { ; \dot { \lambda } } ^ { \prime \prime } = { \cal A } _ { ; \dot { \lambda } } ^ { \mu } + { \cal T } _ { \alpha \lambda } ^ { \prime \prime } { \cal A } ^ { \alpha } .
$$

这就是逆变矢量的协变微商公式.注意，如果对逆变矢量的协变微商用类似(1.4.4)式的定义，再利用逆变矢量的平移公式(1.3.7）.所得的结果是一样的.

对高阶张量也可类似地做.如对二阶混合张量场 $T ^ { \pi } { } ^ { \mu } ( x )$ .它与两个任意矢量场 $A ^ { \mu }$ 和 $B$ ，可以构成标量 $T _ { \nu } ^ { \mu } A ^ { \nu } B _ { \mu }$ .因此有

$$
( T _ { \nu } ^ { \mu } A ^ { \nu } B _ { \mu } ) _ { ; \dot { x } } = ( T _ { \nu } ^ { \mu } A ^ { \nu } B _ { \mu } ) _ { , \dot { x } } .
$$

然后用乘法公式展开.利用 $A ^ { \mu }$ 和 $B _ { \mu }$ 的协变微商公式已经求出，我们就可得到 $T ^ { \mu }$ 的协变微商公式

$$
T _ { \nu _ { \lambda } \lambda } ^ { \prime \prime } = T _ { \nu , \lambda } ^ { \ast \prime } + { \cal T } _ { \rho \lambda } ^ { \mu } T _ { \nu } ^ { \rho } - { \cal T } _ { \nu \lambda } ^ { \rho } T _ { \rho \bullet } ^ { \prime \prime } \ :
$$

它的规律是对每一上标按逆变矢量的协变微商那样操作一次，而对每一下标按协变矢量的协变微商那样操作一次.按这办法写出二阶逆变张量和协变张量的协变微商公式如下：

$$
T _ { \ast \lambda } ^ { \ast \mu \nu } = T _ { \ast \lambda } ^ { \ast \prime \ast } + { \cal T } _ { \circ \lambda } ^ { \mu } T ^ { \ast \prime \ast } + { \cal T } _ { \rho \lambda } ^ { \nu } T ^ { \ast \rho } ,
$$

$$
T _ { _ { I ^ { \alpha } \nu ; \lambda } } = T _ { _ { I ^ { \alpha } \nu , \lambda } } - { \cal { F } } _ { \mu \lambda } ^ { \rho } T _ { _ { \rho \lambda } } - { \cal { F } } _ { \nu \lambda } ^ { \rho } T _ { _ { I ^ { \alpha } \rho } } .
$$

更高阶的情形不难类推.

作为应用，我们计算克龙涅克张量 $\delta _ { \nu } ^ { \prime \prime }$ 的协变微商.按(1.4.9)式有

$$
\delta _ { \nu _ { \tt t } \lambda } ^ { \mu } = \delta _ { \nu , \lambda } ^ { \mu } + I _ { \rho \lambda } ^ { \mu } \delta _ { \nu } ^ { \rho } \mathrm { ~ -- ~ } I _ { \nu \lambda } ^ { \rho } \delta _ { \rho } ^ { \mu } ,
$$

由于 $\delta _ { * } ^ { \prime \prime }$ 的分量是常数，所以上式右边第一项为零.后两项是对消的，因为

$$
\begin{array} { r } { \begin{array} { r } { \Gamma _ { \varkappa \doteq \dot { \mathcal { O } } _ { \nu } ^ { \star } } ^ { \prime } = \Gamma _ { \nu \dot { \mathcal { A } } } ^ { \mu } = F _ { \nu \lambda } ^ { \rho } \delta _ { \rho \bullet } ^ { \mu } } \end{array} } \end{array}
$$

这样得出结果：

$$
\delta _ { \nu _ { 3 } \lambda } ^ { \mu } = 0 .
$$

即克龙涅克张量 $\delta _ { \nu } ^ { \mu }$ 的普通微商和协变微商都是零.

另一例子是证明 $A _ { \mu , \nu }$ 的反称组合为张量.由(1.4.5)式知

$$
{ A } _ { [ \mu ; \nu ] } = { A } _ { [ \mu , \nu ] } = { \cal { I } } _ { [ \mu \nu ] } ^ { \lambda } { A } _ { \lambda } ,

$$
A _ { \left[ \mu _ { 3 } \nu \right] } = A _ { \left[ \mu , \nu \right] } .
$$

# 1.5测地线方程

普通空间中的直线可定义为线上任意相邻两点的切矢量都相互平行的曲线.把这概念推广到 $\pmb { n }$ 维仿射空间，相应的曲线就叫测地线.现在我们要导出测地线所满足的微分方程.

$_ { n }$ 维空间的曲线由 $\pmb { n }$ 个参量式描述，

$$
x ^ { \mu } = x ^ { \mu } ( \lambda ) , \quad \mu = 1 , 2 , \cdots , n ,
$$

设其中 $\lambda$ 是一个标量性的参量.曲线上任一点的切矢量定义为

$$
A ^ { \mu } = { \frac { \mathrm { d } x ^ { \mu } } { \mathrm { d } \lambda } } ,
$$

它是一个逆变矢量.令 $P$ 和 $\mathbf { Q }$ 是曲线上的两个相邻点，它们的坐标为 $x ^ { \mu }$ 和 $x ^ { \mu } + \mathrm { d } x ^ { \mu }$ .为了使 $P \cdot Q$ 点上的切矢量能够比较，我们先把 $P$ 点的切矢量 $A ^ { \mu } ( P )$ 平移至 $\mathbf { Q }$ 点，变成 $A ^ { \mu } ( P {  } Q )$ .然后定义：若曲线上任意两相邻点 $P$ 和 $\boldsymbol { Q }$ 的一切矢量满足

$$
A ^ { \mu } ( P  Q ) \ / / \ A ^ { \mu } ( Q ) ,
$$

则这曲线叫测地线.测地线条件(1.5.3)也可写作

$$
A ^ { \mu } ( Q ) = ( 1 + f ( \lambda ) \mathbf { d } \lambda ) A ^ { \mu } ( P \to Q ) .
$$

这里对比例因子按dλ作了展开，并保留至 $\mathbf { d } \lambda$ 的一级小量.

利用逆变矢量的平移公式(1.3.7)，我们有

$$
A ^ { \mu } ( P \to Q ) = { \frac { \mathrm { d } x ^ { \mu } } { \mathrm { d } \lambda } } - \Gamma _ { \alpha \beta } ^ { \mu } { \frac { \mathrm { d } x ^ { \alpha } } { \mathrm { d } \lambda } } { \frac { \mathrm { d } x ^ { \beta } } { \mathrm { d } \lambda } } \mathrm { d } \lambda .
$$

由切矢量的微分公式，又有

$$
A ^ { \mu } ( Q ) = A ^ { \mu } ( I ^ { \nu } ) + \mathrm { d } A ^ { \mu } ( I ^ { \nu } ) = \frac { \mathrm { d } x ^ { \mu } } { \mathrm { d } \lambda } + \frac { \mathrm { d } ^ { 2 } x ^ { \mu } } { \mathrm { d } \lambda ^ { 2 } } \mathrm { d } \lambda .
$$

把(1.5.5)式和(1.5.6)式代入(1.5.4)，保留至一级小量，并加整理,得出

$$
{ \frac { \mathrm { d } ^ { 2 } x ^ { \mu } } { \mathrm { d } \lambda ^ { 2 } } } + \Gamma _ { \alpha \beta } ^ { \mu } { \frac { \mathrm { d } x ^ { \alpha } } { \mathrm { d } \lambda } } { \frac { \mathrm { d } x ^ { \beta } } { \mathrm { d } \lambda } } = f ( \lambda ) { \frac { \mathrm { d } x ^ { \mu } } { \mathrm { d } \lambda } } .
$$

这就是测地线的微分方程.

如果我们采用一类特殊的标量性参量，测地线方程将得到简化.为看清这一点，考虑参量的变换

$$
\lambda = \lambda ( \sigma ) .
$$

这时相应有

$$
{ \frac { \operatorname { d } _ { \mathcal { X } } ^ { \mu } } { \operatorname { d } \lambda } } = { \frac { \operatorname { d } _ { \mathcal { X } } ^ { \mu } } { \operatorname { d } \sigma } } { \frac { \operatorname { d } \sigma } { \operatorname { d } \lambda } } ,
$$

$$
\frac { \mathrm { d } ^ { 2 } x ^ { \mu } } { \mathrm { d } \lambda ^ { 2 } } = \frac { \mathrm { d } ^ { 2 } x ^ { \mu } } { \mathrm { d } \sigma ^ { 2 } } \bigg ( \frac { \mathrm { d } \sigma } { \mathrm { d } \lambda } \bigg ) ^ { 2 } + \frac { \mathrm { d } x ^ { \mu } } { \mathrm { d } \sigma } \frac { \mathrm { d } ^ { 2 } \sigma } { \mathrm { d } \lambda ^ { 2 } } .
$$

于是，测地线方程(1.5.7)改变为

$$
\left( { \frac { \mathrm { d } ^ { 2 } x ^ { \mu } } { \mathrm { d } \sigma ^ { 2 } } } + T _ { a \beta } ^ { \mu } { \frac { \mathrm { d } x ^ { \alpha } } { \mathrm { d } \sigma } } { \frac { \mathrm { d } x ^ { \beta } } { \mathrm { d } \sigma } } \right) \left( { \frac { \mathrm { d } \sigma } { \mathrm { d } \lambda } } \right) ^ { 2 } = { \frac { \mathrm { d } x ^ { \mu } } { \mathrm { d } \sigma } } { \bigg ( } f ( \lambda ) { \frac { \mathrm { d } \sigma } { \mathrm { d } \lambda } } - { \frac { \mathrm { d } ^ { 2 } \sigma } { \mathrm { d } \lambda ^ { 2 } } } { \bigg ) } .
$$

从(1.5.8)式看出·若让参量变换满足

$$
{ \frac { \mathrm { d } ^ { 2 } \sigma } { \mathrm { d } \lambda ^ { 2 } } } = f ( \lambda ) \ : { \frac { \mathrm { d } \sigma } { \mathrm { d } \lambda } } ,
$$

测地线方程将简化为

$$
\frac { \mathrm { d } ^ { 2 } x ^ { \prime \prime } } { \mathrm { d } \sigma ^ { \prime \prime } } + \Gamma _ { \alpha \beta } ^ { \varkappa } \frac { \mathrm { d } x ^ { \alpha } } { \mathrm { d } \sigma } \frac { \mathrm { d } x ^ { \beta } } { \mathrm { d } \sigma } = 0 .
$$

这样的标量性参量叫仿射参量.

最后指出仿射参量的两点性质.

(a）对比(1.5.7)和(1.5.10)式看出,当采用仿射参量 $\pmb { \sigma }$ 时有

$f ( \pmb { \sigma } ) = = 0$ .再由（1.5.4)式知，这时的测地线条件是

$$
A ^ { \prime \prime } ( Q ) \stackrel { . . } { = } A ^ { \mu } ( P  Q ) ,
$$

这里 $A ^ { \prime } { \equiv } \frac { \mathrm { d } x ^ { \prime \prime } } { \mathrm { d } \sigma }$

（b）仿射参量并不唯--.若 $\sigma$ 与 $\widetilde { \pmb { \sigma } }$ 是两种不同的仿射参量，(1.5.9)式告诉我们，它们之间的变换满足

$$
\frac { \mathrm { d } ^ { 2 } \sigma } { \mathrm { d } \tilde { \sigma } ^ { 2 } } = 0 .
$$

即仿射参量的变换只能是线性变换.

# 1.6曲率张量

我们已经知道，联络是决定空间几何性质的重要量，但它不是张量.挠率是由联络构成的重要张量.下面将指出，由联络还能构造出另一个重要的张量，它叫曲率张量.

我们借助协变微商来引入曲率张量.看任一协变矢量场$A _ { \mathfrak { r } } ( x )$ 的二阶协变微商，

$$
\begin{array} { r l } & { A _ { \lambda ; \mu ; \nu } = A _ { \lambda ; \mu , \nu } - \Gamma _ { \lambda \nu } ^ { \rho } A _ { \rho ; \mu } - \Gamma _ { \mu \nu } ^ { \rho } A _ { \lambda ; \rho } } \\ & { \qquad = A _ { \lambda , \mu , \nu } - \Gamma _ { \lambda \mu , \nu } ^ { \rho } A _ { \rho } - \Gamma _ { \lambda \mu } ^ { \upsilon } A _ { \rho ; \nu } - \Gamma _ { \lambda \nu } ^ { \rho } A _ { \rho , \mu } } \\ & { \qquad + \Gamma _ { \lambda } ^ { \rho } \Gamma _ { \rho \mu } ^ { \sigma } A _ { \sigma } - \Gamma _ { \mu \nu } ^ { \rho } A _ { \lambda ; \rho } . } \end{array}
$$

取 $A _ { i , \mu _ { i } }$ 对 $\mu$ 和 $\nu$ 的反称组合，它可以整理成

$$
\begin{array} { l } { { A _ { \lambda ; [ \mu _ { \nu } \nu _ { \lambda } ^ { = } } \equiv \displaystyle \frac { 1 } { 2 } ( A _ { \lambda ; \mu ; \nu } - A _ { \lambda ; \nu ; \mu } ) } } \\ { { \mathrm { } } } \\ { { \mathrm { } = \displaystyle \frac { 1 } { 2 } ( R _ { \lambda \mu \nu } ^ { \rho } A _ { \rho } - 2 { \cal { r } } _ { \scriptscriptstyle - \rho \nu _ { \lambda } ^ { \rho } } ^ { \sigma } A _ { \lambda ; \rho } ) , } } \end{array}
$$

其中

$$
R _ { \lambda \mu \nu } ^ { \prime } \equiv \dots \ : P _ { \lambda \mu \textit { , } \nu } ^ { \prime } + \ : P _ { \lambda \nu \textit { , } \mu } ^ { \prime } - \ : P _ { \lambda \mu } ^ { \sigma } \Gamma _ { \sigma \nu } ^ { \rho } + \Gamma _ { \lambda \nu } ^ { \sigma } \Gamma _ { \sigma \mu } ^ { \rho } .
$$

(1.6.2)式左边是张量，右边第二项也是张量，因此 $R _ { \lambda \mu \nu } ^ { \rho }$ 必是张量.它就叫曲率张量.(1.6.2)式说明，当曲率和挠率都等于零(注意，因为它们都是张量，这条件与坐标的选取无关）， $A _ { \mu }$ 的两次协变微

商才可以变换次序.

由定义式(1.6.3)可看出曲率张量的两点性质.首先，它完全由联络和联络的一阶微商所决定.其次，它对下标 $\mu$ 和 $\pmb { \nu }$ 是反称的，即

$$
R _ { \lambda \mu \nu } ^ { \rho } = - \ R _ { \lambda \nu \mu } ^ { \rho } .
$$

曲率作为(1,3)阶张量，它可以缩并.独立的缩并方式有两种.一种得出

$$
A _ { \mu \nu } \equiv R _ { \lambda \mu \nu } ^ { \lambda } ,
$$

另一种得出

$$
R _ { \mu \nu } \equiv R _ { \mu \nu \lambda } ^ { \lambda } .
$$

由于有(1.6.4)的反称性质， $R _ { \mu \lambda \nu } ^ { \lambda }$ 不给出新的张量.

曲率与挠率一起，构成了刻画空间弯曲情况的基本张量.如果在空间某区域 $V$ 内曲率张量和挠率张量都恒等于零,那可以证明，总能找到一个适当的坐标变换 $x ^ { \mu } { \to } { \overset { \sim } { x } } ^ { \mu }$ ，使得

$$
\widetilde { \cal I } _ { \ s u } ^ { \lambda } = 0 , \quad \widetilde { \cal I } \widetilde { \pm } { \cal V } \not \equiv 0 .
$$

我们不证明这定理，仅仅指出它的含意.在 $V$ 内的联络为零，由矢量平移公式看出

$$
\delta A _ { \mu } = 0 .
$$

这表明在 $V$ 内矢量的平移不改变其分量.此外，测地线方程简化为

$$
\frac { \mathrm { d } ^ { 2 } x ^ { \mu } } { \mathrm { d } \sigma ^ { 2 } } = 0 ,
$$

即在 $V$ 内测地线是直线.由于这些性质，人们称 $V$ 内的空间是平坦的.这也就是说，上述定理告诉我们，曲率和挠率张量是否都为零是空间是否平坦的标志.

值得指出一点，由张量定义看出，当张量在某一坐标下为零，则在一切坐标下均为零.联络不是张量，从联络的变换公式看出·它在某一坐标下为零，在其他坐标下一般不为零.这正是人们要引用曲率张量和挠率张量来刻画空间几何性质的原因.

最后顺便说明，曲率张量还可以由别的途径来引入，为此我们考虑矢量 $A ^ { \mu }$ 沿闭合路径的平移.令 ${ \mathrm { d } } x ^ { \mu }$ 和 $\delta \boldsymbol { x } ^ { \mu }$ 是 $P$ 点的两个无穷小位移(图1.1).把 $P$ 点的矢量 $A ^ { \mu }$ 平移 $\mathbf { d } \boldsymbol { x } ^ { \mu }$ 至 $\mathbf { Q }$ ,再平移 $\delta \boldsymbol { x } ^ { \mu }$ 至$M$ ,再平移 $\mathrm { -- } \mathrm { d } \boldsymbol { x } ^ { \prime \prime }$ 至 $N$ ,最后平移一 $\delta { x ^ { \mu } }$ 回到 $P$ 点.返回 $P$ 点后， $A ^ { \mu }$ 变成了 $A ^ { \mu } - \vdash \delta A ^ { \mu }$ .由平移的定义决定， $\delta A ^ { \mu }$ 仍是 $P$ 点的矢量.用直截了当的计算算出这沿闭合路径平移后的增量为

\delta A ^ { \mu } = - \mathrm { \small { ~ \frac 1 2 { \cal R } ^ \lambda _ { \rho \mu \nu } } } A ^ { \rho } ( { \mathrm { d } } x ^ { \mu } \hat { \delta } x ^ { \nu } - { \mathrm { d } } x ^ { \nu } \hat { \delta } x ^ { \mu } ) ,
$$
