# 第六章 相对论宇宙学

# s61无限宁宙与牛顿理论两者之间的困难

爱因斯坦在1916年确立了他的广义相对论之后，紧接着就转向了对宇宙的考查，1917年，爱因斯坦发表了他的第一篇宇宙学论文：《根据广义相对论对宇宙学所作的考查》，这也是相对论宇宙学的第一篇论文.

爱因斯坦之所以很快将广义相对论应用于宇审学，是因为在那个时代，宇宙学问题是唯一可以充分显示广义相对论作用的领域，在当时已知的各种物理系统中，只有宇宙的 $\frac { ( I + I ) I } { I ^ { 2 } ( I ) }$ 值近于或等于1，也就是说牛顿理论对宇宙不再适用.

到20世纪初，牛顿理论的正确性和 $\frac { r ^ { 2 } } { q } \frac { z ^ { 2 } } { f ^ { 2 } } =$ 宙空间的无限性，好象在科学中已经有定论了，的确，无限宇宙作为一种自然哲学的观念，在冲破中世纪宗教宇宙观的精神束缚的斗争中，起过非凡的作用，以哥白尼、开普勒、伽利略为代表的革新者，应用科学的论证推翻了宇宙以地球为中心的教义，牛顿的理论则最终给这些新观念奠定了动力学基础，这一系列的成功，使人们默认了这样一幅宇宙结构图：宇宙的空间是无限的，无论向前向后，向上向下，向左向右都可以一直走下去，没有一个方向是有终点的，也就是说，宇宙空间是三维欧儿里得几何的无限空间，无限的天体则布满在无限的空问之中，然而，这样的结构图是不是真有科学的根据呢？没有，不仅没有，而且存在着内在的矛盾性。

每一次科学的重要进展之后，由于它的巨大声誉往往会使某些后来人不去认真思索或者不愿去思索由这些进展所带来的新观点中，哪一些是真正被证明了的真理，哪一些还只是猜测或假说，以致将各种观点不加分析地接受下来，象宇宙空间的三维欧几里得几何的无限性和牛顿理论在宇宙学上的适用性，就属于后一类观念，尽管许多人已经习惯于接受它们，然而它们的正确性并不是先验自明的，

爱因斯担的第一步工作，就是分析这个似乎不成问题的观念中存在的问题.

首先，如果物质布满在整个无限宇宙中，那么，无限远处的引力场就不会变为零，但是，另一方面，在讨论局部的天体运动时，我们总是假定可以选取这样一个参考系，以使引力场在无限远处变为零，这个无限远的条件，对于解决许多局部的天体运动问题，是不可缺少的，这样，我们就处在二者择一的境地：或者放弃牛顿力学作为关体运动的动力学基础；或者认为天体并不是均匀地分布在整个无限空间里，而是只占据一个有限的范围.

其次，即使假定天体只分布在有限的空间里，按照牛顿理论，这个体系也不是稳定的，它或者将发生坍缩，或者一部分将会飞散而导致整体的瓦解.

总之，只要旧牛顿理论来分析这个无限空间中星的体系，就会发现它们存在着内在的矛盾，根据牛顿力学不能建立(更谈不上精确)无限宇宙这一物理体系的动力学，换句话说，从牛顿理论及无限宇宙这两点出发，我们并没有一个自洽的宇宙模型，因此，要么修改牛顿的理论，要么修改无限空间的观念，或者两者都要加以修改.

这就是爱因斯坦在发展相对论字宙学时所面对的问题。

# $\beta = 2$ 均匀各向同性宇宙的时空

按照广义相对论的观点，宇宙间的时空是弯曲的，弯曲的尺度小到原子、核子，大到星系团，都是存在的，但是宇宙学所关心的是大尺度结构，在研究大尺度结构时，我们可以忽略掉小尺度结构，也就是说将小尺度的弯曲平均掉，进行平均的尺度约在 $I \theta ^ { \eta }$ 秒差距的量级，这种作法相当于把物质的分布均匀化，即把集中在星系、星系团等天体系统上的物质均匀地分布于空间中，这就是平均化的结果，这种方法在其它的物理学领域中，也是常用的，例如，当讨论非湍动的流体时，我们不必关心流体中每个原子团的运动特征，而只考虑平均的密度、及流体元的整体速度.

然而，也可能在原则上不允许进行这种平均，就象在湍动流体中的速度分布，是不能进行平均的，所谓宇宙的等级式模型，就是认为在进行宇宙学研究时，上述的平均在原则上是不对的，光学观测的确表示出宇宙有一定的等级的结构，但是，目前还没有足够的观测根据使我们不得不接受等级式模型，

因此，我们把宇宙中的物质当作一种流体，不过这种流体不是由普通的原子构成的，而是由星系等较小尺度的天体系统构成的：现在，我们在每一个流体元上放上一个钟，它与流体元一起运动，称为共动钟，共动钟指示的时间值称为共动坐标时，它也就是该流体元的原时，不同的流体元上有各自的共动钟，这些钟给出的坐标时之间有什么关系呢？

为了解决这个问题，提出了一种假说，称为宇宙学原理：在各流体元上的观察者将看到相同的历史，所谓历史就是在各时期所发生的物理现象，应当强调，“相同的历史”也是在大尺度平均的意义上说的，这就是说，各流体元上的历史在短时间上物理现象可以有区别，但从大的时间尺度上看，却是相同的，在这种假说下，我们就可以在所有的流体元之间进行对钟，例如，当它们各自看到自己的流体元的密度发展到某个确定值时，取相同的时间 $f$ 加以标志，这就把各流体元上的钟相互对准了，这样，我们就可以用一个时间坐标 $\ddagger$ 来描写宇宙在时间上的变化.

宇宙学原理要求各流体元有相同的历史，这相当于要求宇宙的大尺度特征是均匀的，在下面的讨论中，我们还要求宇宙是各向同性的，即每个随流体元运动的共动观察者将看到各个方向也都是同样的，观测结果相当好地支持各向同性的假说，例如，当角分辨率为1时，微波背景辐射的温度值的各向异性小于 $1 0 ^ { - 5 }$ ：又如，射电源的分布是近于各向同性的。
