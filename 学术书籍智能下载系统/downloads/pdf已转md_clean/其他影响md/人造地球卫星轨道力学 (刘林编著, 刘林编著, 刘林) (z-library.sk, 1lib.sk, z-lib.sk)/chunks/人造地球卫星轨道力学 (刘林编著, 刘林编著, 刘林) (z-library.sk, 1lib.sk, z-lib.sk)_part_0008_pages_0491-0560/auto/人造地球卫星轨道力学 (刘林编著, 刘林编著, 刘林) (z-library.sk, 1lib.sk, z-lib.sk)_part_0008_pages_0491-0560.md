显然,两类中间轨道与相应的瞬时椭圆轨道相差甚小,若取中间轨道根数为

+ - \frac { c ^ { 2 } } { a } \biggl ( 1 - \frac { 3 } { 2 } \sin ^ { 2 } i \biggr ) ( 1 - e ^ { 2 } ) ^ { - 3 / 2 } .

# 13.4 中间轨道摄动法

已经给出的各类中间轨道都是不完整地反映了地球引力位带谐项的一部分（即使AKceHOB型中间轨道也未能反映 $J _ { 4 }$ 项的全部)或主要田谐项 $A _ { 2 , 2 } , B _ { 2 , 2 } ^ { \Gamma 1 0 1 }$ ，因此对于卫星运动的精密定轨问题，还得在中间轨道的基础上进一步计算剩余摄动，包括地球引力位带谐项的一部分和田谐项，日、月，光压,大气阻力等摄动。

若用六个积分常数 $\alpha _ { 1 } , \alpha _ { 2 } , \alpha _ { 3 } , \beta _ { 1 } , \beta _ { 2 } , \beta _ { 3 }$ 作为基本变量(它们是正则共轭变量),则相应的摄动运动方程为．

$$
\dot { \alpha } _ { i } = \frac { \partial R } { \partial \beta _ { i } } , \qquad \dot { \beta } _ { i } = - \frac { \partial R } { \partial \alpha _ { i } } .
$$

其中 $\pmb { R }$ 是摄动函数，即

$$
\pmb { R } { = } V { - } V _ { 0 } ,
$$

这里的 $V _ { 0 }$ 就是中间轨道位函数．对于非保守力，可在方程(13.124)的右端以摄动加速度形式列入，形式如下：

$$
\dot { \alpha } _ { i } = F _ { e } \cdot \frac { \partial r } { \partial \beta _ { i } } , \qquad \dot { \beta } _ { i } = - F _ { e } \cdot \frac { \partial r } { \partial \alpha _ { i } } ,
$$

其中 $\pmb { F } ,$ ，是摄动加速度.

考虑到实际计算的需要，有时要引入一些类似于椭圆轨道的六个根数 $\sigma _ { j } ( j = 1 , 2 , \cdots , 6 )$ 代替 $\alpha _ { 1 } , \ldots , \beta _ { 3 }$ 作为基本变量,如果 $\sigma _ { j }$ 仍是一组正则共轭变量,那很简单，否则要写出以新变量 $\pmb { \sigma _ { j } }$ 表达的运动方程,将有一个相应的推导过程，假定有如下变换关系：

$$
\sigma _ { j } = \sigma _ { j } ( \alpha , \beta , t ) , \qquad j = 1 , 2 , \cdots , 6 .
$$

则

$$
\sigma _ { j } = \sum _ { i = 1 } ^ { 3 } \biggl ( \frac { \partial \sigma _ { j } } { \partial \alpha _ { i } } { \dot { \alpha } } _ { i } + \frac { \partial \sigma _ { j } } { \partial \beta _ { i } } { \dot { \beta } } _ { i } \biggr ) + \biggl ( \frac { \partial \sigma _ { i } } { \partial t } \biggr ) .
$$

将方程(13.124)或(13.126)代入，并将 $\pmb { R }$ 对 $\alpha _ { i } , \beta _ { i }$ 的偏导数化为对 $\sigma _ { j }$ 的偏导数，经整理后即得

$$
\left. \begin{array} { l } { { \displaystyle { \sigma _ { j } = \sum _ { k = 1 } ^ { 0 } \{ \sigma _ { j } , ~ \sigma _ { k } \} \frac { \partial R } { \partial \sigma _ { k } } + \left( \frac { \partial \sigma _ { j } } { \partial t } \right) } , } } \\ { { \displaystyle { \sigma _ { j } = \sum _ { k = 1 } ^ { 8 } \{ \sigma _ { j } , ~ \sigma _ { k } \} F _ { * } \cdot \frac { \partial r } { \partial \sigma _ { k } } + \left( \frac { \partial \sigma _ { j } } { \partial t } \right) } . } } \end{array} \right\}
$$

其中 $\{ \sigma _ { j } , \sigma _ { k } \}$ 是泊松括号，即

$$
\displaystyle { \{ \sigma _ { j } , \sigma _ { k } \} = \sum _ { i = 1 } ^ { 3 } \bigg ( \frac { \partial \sigma _ { j } } { \partial \alpha _ { i } } \frac { \partial \sigma _ { k } } { \partial \beta _ { i } } { - \frac { \partial \sigma _ { j } } { \partial \beta _ { i } } \frac { \partial \sigma _ { k } } { \partial \alpha _ { i } } \bigg ) } . }
$$

求解摄动运动方程(13.129)后，给出 $\sigma _ { i } { = } \sigma _ { j } ( t ) _ { i }$ ，或 $\alpha _ { i } ( t ) .$ $\beta _ { i } ( t )$ ，由此便可计算有摄星历表．下面将以地球引力位 $J _ { 2 } , J _ { 3 } , J _ { 4 }$ 三项摄动为背景，介绍以Vinti型和AKceHOB型中间轨道为基础的剩余摄动计算方法.

\left. \begin{array} { l } { { \rho = \rho _ { 1 } ( L , G , l ) + c ^ { 2 } \rho _ { 2 } ( L , G , H , l , g ) , } } \\ { { \eta = \sin i \sin ( f \div g ) + { \cal O } ( c ^ { 2 } ) . } } \end{array} \right\}
$$

其中 $\cos i = H / G , f$ 与 $\vec { E } , l$ 之间的关系不再列出。如果求一阶解，则 $\pmb R$ 只需取到 $O ( c ^ { 0 } )$ 项，问题很简单，如果求二阶解，则上述 $\pmb R$ 应取到 $O ( c ^ { 2 } )$ 项.

对于AKceHOB型中间轨道,相应的摄动函数为

$$
R { = } \frac { ( - J _ { 4 } ^ { \prime } ) } { r ^ { 5 } } P _ { 4 } ( \sin \varphi ) .
$$

由变换(13.26)式给出

$$
\begin{array} { l } { { r = \rho \displaystyle [ 1 + c ^ { 2 } ( \frac { 1 - \mu ^ { 2 } } { 2 \rho ^ { 2 } } + ( \frac { d } { c } ) \frac { \mu } { \rho } ) + O ( c ^ { 4 } ) ] } , } \\ { { \sin \varphi = \displaystyle \frac { 1 } { r } ( \rho \mu + c d ) . } } \end{array} )  ,

# 3．剩余摄动的计算方法

将用基本变量表达的摄动函数 $\pmb { R }$ 代入摄动运动方程（13.133),就构成一小参数方程:

$$
\scriptstyle \sigma = f ( \sigma , t , c ^ { 2 } ) ,
$$

其中 $\pmb { c } ^ { 2 } { = } \pmb { O } ( \pmb { J } _ { 2 } )$ 是小参数。因此，在中间轨道基础上如何计算剩余摄动，显然可采用前面第五章和第七章分别介绍的各种方法，与受摄二体问题中构造瞬时椭圆的摄动解并无实质性差别，文[20]和[17]曾分别按上述过程给出了建立在AKceHOB型中间轨道基础上的一阶解和高阶解。因为并无特殊之处，无需再给出构造 $\pmb { J _ { 3 } }$ 和 ${ \pmb J } _ { 4 }$ 项摄动解的具体过程。

# 4.中间轨道实用化的前景

上述两类中问轨道解与椭圆轨道相比，最大的优点在于包含了地球引力位展开式中最重要的 $\pmb { J _ { 2 } }$ 项．但对于精密定轨而言,往往需要二阶解甚至高阶解，那么必须在中间轨道基础上去求各种剩余摄动， ${ \pmb J } _ { 2 }$ 项引起的轨道“变化”则部分地通过中间轨道解转移到摄动函数(或摄动加速度)中，增加求剩余摄动的麻烦.尽管所采用的方法和基本运算都可模仿受摄二体问题以椭圆轨道为基础的各种做法，但并未使求高阶摄动（除 $\pmb { J _ { 2 } }$ 项外）的麻烦有明显的改观,至少目前给出的中间轨道是如此。关于 $\pmb { J _ { 2 } }$ 项本身，虽然无摄动问题,但通过中间轨道解使星历表计算复杂化．当然，作为中间轨道的探讨，本身确是一项基本研究工作，只是从实用角度来看，在人造卫星轨道理论中的应用前景并不乐观,到目前为止,在实际工作中,仍采用椭圆轨道作为基本解。

从前面讨论不难看出,中间轨道实用化的最大困难,在于很难（几乎不可能)找到一种理想的中间轨道，既能完整地包含更多的摄动因素,又不至于使其解本身过于复杂化，而这一点对于分析方法而言，正是问题的关键所在.

# 14.1 问题的提法

通过前面各章的讨论，读者已经清楚地看出，人造卫星绕地球运动所涉及的数学模型，是一个相当复杂的非线性动力系统，在地.心赤道坐标系中，相应的运动微分方程可写成下列形式：

$$
\left. \begin{array} { l } { { \ddot { r } = \dot { F } ( r , \dot { r } , t ) , } } \\ { { \dot { r } ( t _ { 0 } ) = r _ { 0 } , \dot { r } ( t _ { 0 } ) = \dot { r } _ { 0 } , } } \end{array} \right\}
$$

若以椭圆运动作为其参考解，即处理成受摄二体问题，则方程(14.1)又可写成

$$
\ddot { r } { = } F _ { 0 } ( r ) + F _ { \varepsilon } ( r , \dot { r } , t ; \varepsilon ) ,
$$

其中 $\pmb { F } _ { 0 }$ 是地球中心引力，即

$$
\scriptstyle { F _ { 0 } ^ { \therefore } = - { \frac { 1 } { r ^ { 2 } } } \left( { \frac { r } { r } } \right) , }
$$

$\pmb { F }$ ，为各种摄动力(加速度)．由于 $\pmb { F _ { 0 } }$ 对应于一可积系统，相应的解为不变椭圆，问题(14.1)式可经常数变易将其转化为以椭圆根数来描述的小参数方程，即

$$
\left. \begin{array} { c } { { \pmb { \sigma } = f ( \pmb { \sigma } , t ; \varepsilon ) , } } \\ { { . . . \qquad \pmb { \sigma } ( t _ { 0 } ) = \sigma _ { 0 } . } } \end{array} \right\}

这里 $\pmb { \sigma }$ 是 $\dot { \pmb 6 }$ 维向量，其元素即6个轨道根数， $f$ 是相应的6维向量函数.

如果问题(14.1)或(14.4)式的数学模型(即右函数 $F _ { c }$ 或 $f )$ 以及相应的初始条件是准确的,那么，可以积分上述方程给出人造地球卫星状态的演变,即

从解微分方程的角度来看，轨道改进就是将一个常微分方程初值问题转化为边值问题(观测值即“边值")，由边值条件反过来确定初值(轨道),此即“打靶法”,这将涉及到求代数方程的解．不过轨道改进过程中还要用到大量观测资料的统计特性，而不是简单的将一个初值问题转化为边值问题来求解，这又将涉及到最优估计问题，因此也叫统计定轨．在整个天体力学领域中，各种天体(包括人造卫星)的定轨问题，几乎都是采用最小二乘估计[．为了便于引用最优估计理论的一些结果，下面将用相应的语言和符·号[2]来阐明轨道改进问题的具体提法，有关概率统计的基本知识和常用符号，本书将不作过多说明，读者如有需要，可翻阅有关书藉.

# 14.2 最小二乘估计

# 1．无偏最小方差估计

假定有一测量序列 $Y _ { 1 } , Y _ { 2 } , \cdots , Y _ { k } ,$ ，它们包含一定的随机测量误差,由这一测量序列来估计某一状态量 $\pmb { X }$ ，给出估计值 $\hat { X }$ ，显然$\hat { X }$ 也是一随机量.

所谓无偏估计，是指随机量 $\pmb { \hat { X } }$ 的数学期望(即平均值)等于状态量 $X$ 的数学期望，即

$$
\pmb { \cal E } \hat { X } = \pmb { \cal E } \pmb { X } .
$$

当状态量 $\pmb { X }$ 为非随机量时,即要求

$$
{ \boldsymbol { \mathbf { \mathit { E } } } } { \hat { X } } = { \boldsymbol { \mathbf { \mathit { X } } } } .
$$

无偏性表明，当进行大量重复性测量，获得多个测量序列的独立样本时，由每个测量序列样本所求得的估计量 $\hat { X }$ 之平均近似地为$E X$ 或 $\pmb { X }$ ),这是相当重要的，但仅有无偏性还不够，要寻找最优的估计，还要求 $\hat { X }$ 是最小方差估计，即估计误差 ${ \hat { X } } { - } X$ 的方差达到极小：

$$
\operatorname { v a r } ( { \hat { X } } - X ) = E ( { \hat { X } } - X ) ( { \hat { X } } - X ) ^ { \tau } = \operatorname* { m i n }
$$

:min 是对一切可能的估计 $\hat { X }$ 而言.

无偏性将保证由不同的测量序列 $Y _ { 1 } , Y _ { 2 } , \cdots , Y _ { k }$ 的样本所获得的估计量 $\hat { X }$ 在状态 $\pmb { X }$ 的附近摆动，而最小方差将保证这种摆动的平均值达极小.

# 2．最小二乘估计

这里所说的最小二乘估计是一种简单的最小二乘估计，亦称古典最小二乘估计．考虑下列线性系统：

$$

B = \left( \begin{array} { c } { { \displaystyle \frac { \partial \alpha } { \partial X } } } \\ { { \displaystyle \frac { \partial \delta } { \partial X } } } \end{array} \right) = \left( \begin{array} { c } { { \displaystyle \frac { \partial \alpha } { \partial r } } } \\ { { \displaystyle \frac { \partial \delta } { \partial r } } } \end{array} \right) \left( \frac { \partial r } { \partial \sigma } \right) \left( \frac { \partial \sigma } { \partial X } \right) ,
$$

$$
B = \left( \begin{array} { l } { \medskip \displaystyle \frac { \partial A } { \partial X } } \\ { \displaystyle \frac { \partial h } { \partial X } } \end{array} \right) = \left( \begin{array} { l } { \medskip \displaystyle \frac { \partial A } { \partial r } } \\ { \displaystyle \frac { \partial h } { \partial r } } \end{array} \right) \left( \frac { \partial r } { \partial \sigma } \right) \left( \frac { \partial \sigma } { \partial X } \right) ,
$$

$$
\begin{array} { l } { { \displaystyle B = \left( \frac { \partial \rho } { \partial X } \right) = \left( \frac { \partial \rho } { \partial r } \right) \left( \frac { \partial r } { \partial \sigma } \right) \left( \frac { \partial \sigma } { \partial \bar { X } } \right) } , } \\ { { \displaystyle B = \left( \frac { \partial \rho } { \partial X } \right) = \left( \frac { \partial \rho } { \partial r } \frac { \partial \rho } { \partial \dot { r } } \right) \left( \frac { \partial \sigma } { \partial \dot { \sigma } } \right) \left( \frac { \partial \sigma } { \partial X } \right) } . } \end{array}
$$

如果测站向量 $\pmb R$ 也有待改进的内容,即增加待估参数，那么上述$\pmb { B }$ 矩阵也相应地增加

$$
{ \left( \frac { \partial \alpha } { \partial \mathbf { R } } \right) } { \left( \frac { \partial \mathbf { R } } { \partial X } \right) } , \ { \left( \frac { \partial \mathbf { A } } { \partial \mathbf { R } } \right) } { \left( \frac { \partial \mathbf { R } } { \partial X } \right) } , \ { \left( \frac { \partial \rho } { \partial \mathbf { R } } \right) } , \ { \left( \frac { \partial \rho } { \partial \mathbf { R } } \right) } { \left( \frac { \partial \mathbf { R } } { \partial X } \right) } , \ { \left( \frac { \partial \rho } { \partial R } \right) } { \left( \frac { \partial \rho } { \partial X } \right) } .
$$

如果 $\dot { R }$ 也有待改进的参数，处理方法与上类似。

最后说明一点，如果待估状态量中的卫星根数不用 $\overrightarrow { \pmb { \sigma } } _ { o } ,$ 而改用 $\sigma _ { o } ,$ 相应的矩阵 $\partial \sigma / \partial X$ 的形式不变，只是上述各式中出现的 $\scriptstyle { \overrightarrow { \pmb { \sigma } } } _ { \pmb { \sigma } }$ 全部改成 $\sigma _ { o }$ 即可.

# 14.5 轨道改进中的若干问题

# 1．简单轨道改进的基本过程

在观测资料不多，统计特性不明显，而精度要求也不高的情况下，可以简化轨道改进的过程，这里将在不加权的前提下,以改进上一节提出的七个状态量为例，介绍轨道改进的基本过程．其内容可用于目前的一般定轨工作，而且通过对这一基本过程的了解，就能清楚高精度要求下如何进行轨道改进以及需要解决的问，题.

已知一测量序列 $Y _ { 1 } , Y _ { 2 } , \cdots , Y _ { s }$ 和待估状态量 $X _ { o }$ 的初值 $\hat { \mathbf { X } } _ { \pmb { \mathscr { s } } } ,$ $X _ { \mathrm { { \mathfrak { o } } } }$ 对应 $\ell _ { \bullet }$ 时刻，一般有 $t _ { 1 } \leqslant t _ { 0 } \leqslant t _ { s } ,$ 且

$$
\begin{array} { r } { \mathbf { X } _ { o } = ( \bar { \boldsymbol { n } } _ { o } , \bar { \boldsymbol { i } } _ { o } , \overleftrightarrow { \mathcal { Q } } _ { o } , \xi _ { o } , \bar { \boldsymbol { \eta } } _ { o } , \overbar { \lambda } _ { o } , \overleftrightarrow { \boldsymbol { n } } _ { o } ) ^ { \boldsymbol { \tau } } . } \end{array}
$$

轨道改进的任务是：在上述条件下用简单最小二乘估计给出待估 状态量 $X _ { o }$ 的最优估计 $\hat { X } _ { 0 / 8 } .$

假定已迭代到数 $j$ 步，有 $\hat { X } _ { 0 / s } ^ { ( j ) } , \hat { X } _ { s }$ 即 $\hat { X } _ { 0 / 8 } ^ { ( 0 ) }$ ，首先给出观测量$Y _ { \iota } ( \iota = 1 , 2 , \cdots , s )$ 的计算值 $H ( \hat { X } _ { \sigma / s } ^ { ( j ) } , t _ { l } )$ ，这一步实为有摄星历表的计算，可用多种方法进行，大致分为三类：数值方法，相应的变量可用根数 ${ \pmb { \sigma } } _ { \pmb { \mathscr { s } } }$ 也可用坐标和速度 $\pmb { r }$ 和 $\dot { \pmb { r } } ;$ 分析方法，如果只需要一阶解，相应的公式在前面各章中均已给出；半分析半数值方法，如文[6]给出的一种方法就属这一类，当用根数外推时(状态转移),算出 $\pmb { t } _ { \pmb { \imath } }$ 时刻的 $\sigma _ { \downarrow }$ 后，即可计算 $r _ { l } , \dot { r } _ { l } ,$ 从而给出观测量 $\boldsymbol { Y } _ { \boldsymbol { \imath } }$ 的计算值.第二步即计算 $B$ 矩阵， $\pmb { { \cal B } } = { \pmb { { \cal B } } } ( \hat { X } _ { \sigma / \sigma } ^ { ( j } , t _ { l } )$ ，相应的公式在上一节已全：

部给出。于是构成线性化系统：

$\begin{array} { r } { Y _ { \iota } - H ( \hat { X } _ { o / s } ^ { ( j ) } , t _ { \iota } ) = B ( \hat { X } _ { o / s } ^ { ( j ) } , t _ { \iota } ) [ X _ { o } - \hat { X } _ { o / s } ^ { ( j ) } ] + V _ { \iota } , } \end{array}$ 这也叫条件方程.

第三步是根据最小二乘估计的最优准则得到下述法方程：

$$
[ \sum _ { l \ = 1 } ^ { s } B ^ { \tau } ( \hat { X } _ { \ o / s } ^ { ( j ) } , t _ { l } ) B ( \hat { X } _ { o / s } ^ { ( j ) } , t _ { l } ) ] [ X _ { o } - \hat { X } _ { 0 n / s } ] = \sum _ { l = 1 } ^ { s } B ^ { \tau }
$$

$$
( \hat { X } _ { o / s } ^ { ( j ) } , t _ { l } ) [ Y _ { l } - H ( \hat { X } _ { o / s } ^ { ( j ) } , t _ { l } ) ] ,
$$

其中 $\sum _ { l \ = 1 } ^ { \bullet } B ^ { \tau } B$ 即法化矩阵(方阵).

第四步求解法方程，一种常用的方法是通过法化矩阵求逆的方式进行，给出解如下：

$$
\hat { X } \stackrel { ( j + 1 ) } { \circ / s } - \hat { X } \stackrel { ( j ) } { \circ / s } = \sum _ { l \mathrm { ~ } = 1 } ^ { s } ( B ^ { T } B ) \AA ^ { - 1 } [ B ^ { T } ( Y _ { l } - H ) ] .
$$

相应的残差平方和为

$$
U ^ { ( j + 1 ) } = \sum _ { l = 1 } ^ { s } [ Y _ { l } - H ( \hat { X } _ { o / s } ^ { ( j + 1 ) } , t _ { l } ) ] ^ { \pi } [ Y _ { l } - H ( \hat { X } _ { o / s } ^ { ( j + 1 ) } , t _ { l } ) ] .
$$

下一步是比较两次残差平方和之差,如果达到下列要求:

$$
\left| U ^ { ( j ) } - U ^ { ( j + 1 ) } \right| \leqslant \mu ,
$$

测迭代中止， $\hat { X } ^ { ( j + 1 ) }$ 即所求状态量的最小二乘估计值，否则继续迭代下去。 $\mu { > } 0$ ，其具体值将根据实际需要和资料情况而定。有时用均方差 $\pmb { \sigma } ^ { * }$ 代替残差平方和 $U$ ，即

$$
{ \pmb { \sigma } } ^ { * } = \sqrt { \frac { U } { s \times m } } ,
$$

其中 $\pmb { m }$ 是观测量 $\pmb { Y }$ 的维数，而 $\pmb { \delta }$ 即观测次数．事实上， $\pmb { s X m }$ 即观：测数据的总数目，其中 $\alpha , \delta$ 和 $A , h$ 均算两个数据.

# 2.轨道改进中的若干问题

（1）从上一段的基本过程可以看出，实际上给出状态量的最优估计是一个多变元的迭代过程，对待估状态量 $\scriptstyle { \pmb { X _ { o } } }$ 的初值 $\hat { X } _ { \pmb { \mathscr { s } } }$ 是有一定要求的．不过，只要初值误差不太大，总是收敛的，但估计值 $\hat { X } _ { o / 8 }$ 的精度将取决于资料的测量精度和分布状况,每圈资料分布弧段要合理，理想情况是由全球布站所获得，对于局部地区布站，最好是升降段都有，如果初值误差大、资料也不理想，影响迭代收敛或估值精度，可以采用初始估计(如果有)或固定某个已知精度较高、变化较小的待估状态量(如轨道倾角 $\pmb { i }$ ）的办法来解决，具体处理方法要视具体情况而定.

（2）在资料不太多的情况下，错误资料将影响估值精度．因此有必要把误差较大的(包括“错误”的)资料剔除掉，这类资料又称野值．通常是根据统计特性（即随机差的分布规律）来进行剔除，凡在第 $j$ 次迭代过程中，残差大于 ${ 3 \sigma ^ { * } }$ 的资料都作为野值剔除．这一剔除原则具有“服从”大多数的特点，有时并不绝对合理，因此，如何剔除野值问题仍需进一步研究.

(3）既然资料本身具有不同的精度(除野值外)，那么像简单最小二乘估计那样，把所有资料都按等权处理显然是不合理的，因此就有一个加权问题．前面已从理论上证明了最优加权的方式，但在资料不充分多，统计特性不完全具备的情况下(事实上，实际:情况确实如此),按随机采样的方差来加权，有时效果并不好．因此,到目前为止,在加权问题上，基本上还是依赖于人们的经验，如何针对具体的样本给以合理的加权(包括理论上的解释)，是很值得研究的一个课题，它的解决对估值精度的提高将起重要作用。

（4）在资料和待估状态量较多的情况下，批处理方法将受到数据存贮量大和法化矩阵求逆困难的影响，如何合理的将序贯估算法与批处理方法结合起来也是一个值得探索的问题．当然，关于法化矩阵求逆,可以从数学方法上去寻求解决的途径，例如以别的方法代替矩阵求逆等。

（5）事实上，动力模型和数学方法都是在一定程度上的近似，这将给状态转移计算带来相应的误差，因此就存在一个模型噪声问题.关于状态方程加不加模型噪声，这本来是最小二乘估计与卡尔曼滤波的重要区别,尽管最小二乘估计中也可以加一模型噪声，但这比噪声更复杂，对于高精度的人造卫星定轨问题，很难给出相应的数学模型,有待进一步探讨。

总之，利用多资料进行人造卫星的定轨工作，是一个涉及面较广而又极其重要的问题[],它不仅为卫星跟踪、测量提供精确的轨道，还为进一步利用精确轨道(如导航等)或在精确定轨的同时测定某些物理参数[8](包括地球物理和高层大气等)提供一种不可代替的重要途径和手段.

# 附录【 小推力摄动问题

在卫星刚入轨执行“任务”期间，或入轨后由于某些原因，常需要继续点火喷气，以此来达到调节姿态或轨道的目的．不过，这一调节喷气过程，卫星本身的质量变化很小，对涉及到卫星质量变化的光压与大气阻力摄动的影响甚微，因此可归结为一个质量不变的小“冲力”摄动问题．这一冲力的方向将取决于喷气的状况，通常该方向与卫星运动方向并不相同(或相反)．但是，将这一冲力引起的加速度简单地处理成只有沿卫星飞行方向的分量来讨论它对卫星轨道的影响还是有意义的，其结果可以表明卫星运行过程中微小喷气对轨道影响的一些规律，至于小冲力在另两个方向引起的加速度分量(如果存在)对卫星轨道的影响，可用类似方法进行讨论．在上述处理下,冲力加速度只有切向分量 $\boldsymbol { U }$ ,因此就简称小推动力摄动.

根据第四章(4.81)式;相应的运动方程为

$$
\begin{array} { l } { \displaystyle \frac { d a } { d t } = \frac { 2 } { n \sqrt { 1 - e ^ { 2 } } } ( 1 + 2 e \cos f + e ^ { 2 } ) ^ { 1 / 2 } U , } \\ { \displaystyle \frac { d e } { d t } = \frac { 2 \sqrt { 1 - e ^ { 2 } } } { n a } ( 1 + 2 e \cos f + e ^ { 2 } ) ^ { - 1 / 2 } \left( c \cos f + e \right) U , } \end{array}
$$

$$
\frac { d \varOmega } { d t } { = } 0 ,
$$

$$
\left. \begin{array} { l } { \displaystyle \frac { d \omega } { d t } = \frac { 2 \sqrt { 1 - e ^ { 2 } } } { n a e } ( 1 + 2 e \cos f + e ^ { 2 } ) ^ { - 1 / 2 } \sin f U , } \\ { \displaystyle \frac { d M } { d t } = n - \frac { n \sqrt { 1 - e ^ { 2 } } } { n a e } ( 1 + 2 e \cos f + e ^ { 2 } ) ^ { - 1 / 2 } ( \sqrt { 1 - e ^ { 2 } } \sin f } \\ { \displaystyle \qquad + e ^ { 2 } \sin L ) U , } \end{array} \right\}
$$

其中n=a-3/2。从这一方程不难看出，当U为常量时,a,e两根数的变化有长期项,@的变化仅存在短周期项，而M的变化受到α变化的影响，也将出现长期项.

在喷气持续时间超过卫星运动周期的情况下，我们来讨论a,e两根数的变化规律．由于推力不大，丢掉方程中的O(eU）项，对问题的讨论并无影响，于是方程(1)可简化为

$$
\left. \begin{array} { l } { \displaystyle \frac { d a } { d t } = \frac { 2 } { n } \bigg [ ( 1 + e ^ { 2 } ) + e \cos f - \frac { e ^ { 2 } } { 2 } \cos ^ { 2 } f \bigg ] U , } \\ { \displaystyle \frac { d e } { d t } = \frac { 2 } { n a } \bigg [ \left( \cos f + e \right) - e \cos ^ { 2 } f + e ^ { 2 } \left( \frac { 3 } { 2 } \cos ^ { 3 } f - 2 \cos f \right) \bigg ] U . } \end{array} \right\}
$$

至于ω和M,前已有说明,不再写出。利用cosf，cos2f,cos3f的平均值，消除方程(2)中的短周期部分，得

$$
\left. \begin{array} { l } { { \overline { { { d } } } \overline { { { a } } } = \displaystyle \frac { 2 } { n } \bigg ( 1 - \frac { e ^ { 2 } } { 4 } \bigg ) U , } } \\ { { } } \\ { { \overline { { { d t } } } = \displaystyle - \frac { e } { n a } U . } } \end{array} \right\}
$$

由此可给出小推力摄动引起的a和e的长期变化,即

$$
\left. \begin{array} { l } { { \displaystyle \dot { \Delta } a = 2 a ^ { 3 } ( 1 - \frac { e ^ { 2 } } { 4 } ) U n ( t - t _ { 0 } ) , } } \\ { { \Delta e = - e a ^ { 2 } U n ( t - t _ { 0 } ) . } } \end{array} \right\}

这表明，在小推力摄动下，卫星轨道逐渐变大变圆;如果U为负值(即减速)，则轨道逐渐变小变扁．这一规律与大气阻力摄动显然不同，读者是不难理解的.

当喷气持续时间较短或有间断时，卫星轨道的变化不会这么简单。在实际工作中，还是一段段(有喷气时)分别计算,这将涉及到准确掌握喷气规律问题.

# 附录Ⅱ 数值方法简介

# 1．有关数值方法的基本知识

在实际问题中所遇到的微分方程往往很复杂，不可能给出解：的分析表达式，有时即使能在一定条件(或近似)下给出相应的表达式，但常常又因计算量太大而不适用，人造卫星的精密定轨问题就是这样。不过，对于某些问题，只需要求得解在若干点上的近似值(满足规定的精度)，人造卫星的有摄星历表计算正是如此、所以.解微分方程(这里都是指常微分方程，以下从略)的数值方法就显.得十分重要.

（1）数值方法的基本思想人造卫星有摄星历表的计算，是一个微分方程的初值问题．这里先以一阶方程

$$
\left. \begin{array} { l } { \displaystyle \frac { d x } { d t } { = } f ( x , t ) , \ ~ \right. } \\ { \displaystyle { x ( a ) = } \eta , \ ~ \left. a \leqslant t \leqslant b \ \right\} } \end{array}
$$

为例，说明建立数值方法的基本思想．其实，把问题(1)中的 $\pmb { \mathcal { X } }$ 看成向量(如坐标 $\pmb { r }$ 和速度 $\yen )$ 也无妨，相应的就是一阶方程组．求上述初值问题的数值解就是设法在区间 $[ a , b ]$ 上的一系列离散点$\pmb { t } _ { n } ( n { = } 1 , 2 , \cdots , m )$ 处，计算出解 $x ( t )$ 的近似值 $x ( t _ { n } )$ ，这里

$$
a = t _ { 0 } < t _ { 1 } < t _ { 2 } < \cdots < t _ { m } = b .
$$

一般把 $\pmb { t _ { n } }$ 取成等间隔的，即

$$
t _ { n } = a + n h , n = 0 , 1 , 2 , \cdots , m .
$$

其中 $\hbar$ 就称为步长．由初值 $\pmb { x } _ { 0 } = \pmb { x } ( t _ { 0 } ) = \pmb { \eta }$ 求出 $\pmb { x } _ { 1 } ,$ 再由 $\pmb { x } _ { 1 }$ 求出$\yen 2$ 依次求出 ${ x } _ { m } ,$ 得到 $m + 1$ 个离散点(包括 $t _ { 0 } )$ 上的 ${ \pmb x } _ { \pmb n }$ 值，就是用

一种离散化方法来处理连续性问题(1).

当然，要由 ${ \pmb x _ { n } }$ 求出 ${ \pmb x } _ { { \pmb n } + 1 } ,$ 一个最直接的方法是利用泰勒(Taylor)展式,即

$$
x _ { n + 1 } { \approx } x _ { n } + h { \bigg ( } { \frac { d x } { d t } } { \bigg ) } _ { t _ { n } } + { \frac { h ^ { 2 } \cdot { \bigg ( } { \frac { d ^ { 2 } x } { d t ^ { 2 } } } { \bigg ) } _ { t _ { n } } } { 2 ! } } + \cdots + { \frac { h ^ { p } \cdot { \bigg ( } { \frac { d ^ { p } x } { d t ^ { p } } } { \bigg ) } _ { t _ { n } } . } { p ! } }
$$

其中 $\pmb { p }$ 为一正整数,通过方程(1)可以逐步把各阶导数求出，有

$$
\scriptstyle \left. { \begin{array} { l } { \left( { \frac { d x } { d t } } \right) _ { t _ { n } } = f ( x _ { n } , t _ { n } ) , } \\ { \left( { \frac { d ^ { 2 } x } { d t ^ { 2 } } } \right) _ { t _ { n } } = \left( { \frac { \partial f } { \partial t } } + { \frac { \partial f } { \partial x } } { \frac { d x } { d t } } \right) _ { t _ { n } } , } \\ { \ldots \ldots \ldots } \end{array} } \right\}
$$

这显然也是一种离散化的方法，但必须计算高阶导数，当方程（1）式的右函数 $f ( x , t )$ 比较复杂时，计算高阶导数就变得相当麻烦，而人卫运动方程的右函数就属于这种情况．因此，在一般情况下，泰勒多项式不宜于直接应用，各种数值解法往往是采用若干个右函数值来代替高阶导数值，使计算得以简化，这就是微分方程数值解法的基本思想。

（2）一些基本概念 将泰勒展式从 $\pmb { p = 2 }$ 截断，即取多项式（4）中的 ${ \pmb p } = { \bf 1 }$ ，得欧拉(Euler)公式：

$$
\left. \begin{array} { l } { { x _ { n + 1 } = x _ { n } + h f ( x _ { n } , t _ { n } ) , } } \\ { { x _ { 0 } = \eta _ { \ast } } } \end{array} \right\}
$$

下面借用这一离散化的差分方程来介绍数值方法的一些基本概念。

从欧拉公式可以看出，每前进一步，计算相应步点 $\smash { t _ { \perp + 1 } }$ 上的${ \pmb x } _ { \pmb u + 1 }$ 值时，只要知道前一步点 $t _ { n }$ 上的 ${ \pmb x } _ { \pmb n }$ 值，这种方法称为单步法．若每前进一步时，需要知道前面多个步点上的 $\pmb { x }$ 值，这样的数值方法称为多步法.

从另一个角度来看，欧拉公式每前进一步计算 ${ \pmb x } _ { { \pmb n } + 1 }$ 时，只需知道前一步的 $x _ { n }$ 值，这样的计算公式称为显式公式，如果改为后退的欧拉公式：

$$
\left. { \begin{array} { l } { x _ { n + 1 } = x _ { n } + h f ( x _ { n + 1 } , t _ { n + 1 } ) , } \\ { } \\ { x _ { 0 } = \eta . } \end{array} } \right\}
$$

则所要计算的 ${ \pmb x } _ { { \pmb n } + 1 }$ 包含在右函数 $f$ 中，此公式为隐式公式．对于隐式，往往要采用迭代求解办法,而且还要提供 ${ \pmb x } _ { n + 1 }$ 的初值 ${ \pmb x } _ { n + 1 } ^ { ( 0 ) }$

既然数值方法给出的是解 $x ( t )$ 在离散步点 $\pmb { t _ { n } }$ 上的近似值${ \pmb x } _ { { \pmb n } ; }$ ，因此就有一个误差问题，除初值误差外，还有截断误差和舍入误差．关于截断误差，仍用欧拉公式 (6)来说明．相应的精确解$x ( t _ { n + 1 } )$ 应满足带余项的泰勒展式：

$$
\scriptstyle { \boldsymbol { x } } ( t _ { n + 1 } ) = { \boldsymbol { x } } ( t _ { n } ) + h { \dot { \boldsymbol { x } } } ( t _ { n } ) + { \frac { h ^ { 2 } } { 2 } } { \boldsymbol { x } } ( \xi _ { n } ) ,
$$

其中 $\xi _ { n }$ 为区间 $( t _ { n } , t _ { n + 1 } )$ 中的某一点,因为

$$
\dot { x } ( t _ { n } ) = f ( x ( t _ { n } ) , t ) ,
$$

上式可改写为

$$
\pmb { x } ( t _ { n + 1 } ) = \pmb { x } ( t _ { n } ) + \hbar \pmb { f } ( \pmb { x } ( t _ { n } ) , t ) + \frac { \hbar ^ { 2 } } { 2 } \pmb { x } ( \xi _ { n } ) .
$$

由此可见，欧拉公式(6)就是用精确解满足的表达式(8)截去其中的 $\frac { h ^ { 2 } } { 2 } \pmb { \mathscr { x } } ( \xi _ { n } )$ 所得到的近似公式.而 $\frac { h ^ { 2 } } { 2 } \dot { x } ( \xi _ { n } )$ 就称为欧拉方法的局部截断误差(简称截断误差),或称局部离散误差,它与 $h ^ { 2 }$ 同阶，记为$O ( h ^ { 2 } )$ ．当然，其实际大小还取决于导数 ${ \pmb x } ( t _ { n } )$ 的值，这涉及到解$x ( t )$ 的性质。由于丢掉的是二阶导数，所以欧拉方法称为一阶方法，后面将会看到,对于一些常用的单步法，方法的阶数正好比方法的截断误差中 $\pmb { h }$ 的幂次低1,即若截断误差为 $O ( h ^ { p + 1 } )$ ，则方法为 $\pmb { p }$ 阶的。因此，方法的阶数可作为衡量方法精度的一个重要标志．关于舍入误差，比较复杂，它的产生涉及到多种因素，大致有这几个方面：计算数据字长不可能无限,使用任何计算工具都是如此；具有有限字长的数与数之间的运算；在计算机上计算右函数f(x,t)所使用的程序库中子程序的“精度”等。尽管很复杂，但还是可以设法估计它的界,例如把舍入误差当作随机量,采用统计方法可得到较好的结果。

在讨论截断误差时，不免要提出这样的问题，即当h→0时，是否有xn→x(tn)，这里x(tn)和xn各表示方程(1)的精确解和步点t"上的数值解(近似解)，这就是数值方法的收敛性问题．另一个问题是，某一步产生的误差在以后的计算中以什么样的方式传播？又以什么规律累积(即全局误差累积问题)？只有当误差不是恶性膨胀时，相应的数值方法才是适用的，这一问题就是数值方法的数值稳定性问题，它与数值方法的阶数以及步长都有关系．关于收敛性、稳定性以及误差估计问题，是数值方法中必须研究的几个重要理论课题.

有关上述内容，读者如有需要进一步深入了解,可翻阅两本教材11,[2]和有关文献。下面将分别介绍几种常用的数值方法，包括单步法和多步法.

# 2.常用的单步法—RK方法和RKF方法

一种常用的单步法就是众所周知的Runge-Kutta方法(以下简称RK方法),其基本思想就是间接引用泰勒展式，即用积分区间[tm,tn+]上若干个点的右函数值f的线性组合来代替f的导数，然后用泰勒展式确定相应的系数，这样既能避免计算f的各阶导数,又能保证精度。

以常用的四阶RK方法为例,令

$$
x _ { n + 1 } = x _ { n } + \sum _ { i = 1 } ^ { 4 } c _ { i } k _ { i } ,
$$

其中c;为待定的权因子,k;满足下列方程：

$$
k _ { i } = \hbar f { \left( t _ { n } + \alpha _ { i } \hbar , \ x _ { n } \div \sum _ { j = 1 } ^ { i - 1 } \beta _ { i j } k _ { j } \right) }
$$

$$
\alpha _ { 1 } = 0 , \beta _ { 1 } { \mathrm { \Omega } } _ { j } = 0 , i = 1 , 2 , 3 , 4 ,
$$

御

$$
\left. \begin{array} { l } { { \displaystyle { \cal k } _ { 1 } = h f ( t _ { n } , x _ { n } ) , } } \\ { { \displaystyle { \cal k } _ { 2 } = h f ( t _ { n } + \alpha _ { 2 } h , x _ { n } + \beta _ { 2 1 } k _ { 1 } ) , } } \\ { { \displaystyle { \cal k } _ { 3 } = h f ( t _ { n } + \alpha _ { 3 } h , x _ { n } + \beta _ { 3 1 } k _ { 1 } + \beta _ { 3 2 } k _ { 2 } ) , } } \\ { { \displaystyle { \cal k } _ { 4 } = h f ( t _ { n } + \alpha _ { 4 } n , x _ { n } + \beta _ { 4 1 } k _ { 1 } + \beta _ { 4 2 } k _ { 2 } + \beta _ { 4 3 } k _ { 3 } ) . } } \end{array} \right\}
$$

系数 $\mathfrak { a } _ { i } , \beta _ { i j }$ 的确定方法如下：将 $k _ { i }$ 右端的 $f$ 在点 $( t _ { n } , x _ { n } )$ 处展开，有

$$
\begin{array} { l } { { k _ { 1 } { = } h f _ { n } , } } \\ { { \ } } \\ { { k _ { 2 } { = } h f _ { n } { + } h ^ { 2 } ( \alpha _ { 2 } f _ { t } ^ { \prime } { } _ { n } { + } \beta _ { 2 1 } f _ { x _ { n } } ^ { \prime } f _ { n } ) { + } { \cdots } , } } \\ { { \ } } \\ { { \ldots } { \ldots } { \ldots } , } \end{array}
$$

以此代人(9)式并与 $x ( t _ { n } + h )$ 在点 $t _ { n }$ 处的泰勒展式

$$
x ( t _ { n } + h ) = x ( t _ { n } ) + h f ( t _ { n } , x ( t _ { n } ) ) + { \frac { h ^ { 2 } } { 2 } } { \big ( } f _ { t _ { n } } ^ { \prime } + f _ { x _ { n } } ^ { \prime } f _ { n } { \big ) } + \cdots
$$

逐项进行比较，令 $\bar { h } , \bar { h } ^ { 2 } , \bar { h } ^ { 3 } , \bar { h } ^ { 4 }$ 项的系数相等，便得到确定 $c _ { i }$ 和 $\beta _ { i j }$ 的关系式，即

$$
\begin{array} { r } { \sigma _ { 1 } + c _ { 2 } = 1 , c _ { 2 } \alpha _ { 2 } = 1 / 2 , c _ { 2 } \beta _ { 2 1 } = 1 / 2 , \cdots , } \end{array}
$$

显然，上述参数 $c _ { i } , \alpha _ { i } , \beta _ { i j }$ 的选择不是唯一的，有自由参数，它的不同选择就确定了不同的RK公式。下面给出三种形式的四阶RK公式:

（1）古典形式

$$
x _ { n + 1 } = x _ { n } + \frac { 1 } { 6 } [ k _ { 1 } + 2 k _ { 2 } + 2 k _ { 3 } + k _ { 4 } ] ,
$$

$$
\begin{array} { r l } & { { k _ { 1 } } = h f ( t _ { n } , x _ { n } ) , } \\ & { { k _ { 2 } } = k f \bigg ( t _ { n } + \frac { 1 } { 2 } h , x _ { n } + \frac { 1 } { 2 } k _ { 1 } \bigg ) , \Bigg \{ }  \\ & { { k _ { 3 } } = h f \bigg ( t _ { n } + \frac { 1 } { 2 } h , x _ { n } + \frac { 1 } { 2 } k _ { 2 } \bigg ) , \Bigg \} } \\ & { { k _ { 4 } } = h f ( t _ { n } + h , x _ { n } + k _ { 3 } ) . } \end{array}
$$

（2）Kutta 公式

$$
\left. \begin{array} { l } { { x _ { n + 1 } = x _ { n } + \displaystyle \frac { 1 } { 8 } [ k _ { 1 } + 3 k _ { 2 } + 3 k _ { 3 } + k _ { 4 } ] , } } \\ { { } } \\ { { k _ { 1 } = h f ( t _ { n } , x _ { n } ) , } } \\ { { } } \\ { { k _ { 2 } = h f \bigg ( t _ { n } + \displaystyle \frac { 1 } { 3 } h , x _ { n } + \displaystyle \frac { 1 } { 3 } k _ { 1 } \bigg ) , \qquad \Bigg \} , } } \\ { { } } \\ { { k _ { 3 } = h f \bigg ( t _ { n } + \displaystyle \frac { 2 } { 3 } h , x _ { n } + \displaystyle \frac { 1 } { 3 } k _ { 1 } + k _ { 2 } \bigg ) , \Bigg \} } } \\ { { { } } } \\ { { k _ { 4 } = h f ( t _ { n } + h , x _ { n } + k _ { 1 } - k _ { 2 } + k _ { 3 } ) . } } \end{array} \right\}
$$

(3）Gill 公式(它有减少舍人误差的优点)

$$
x _ { n + 1 } = x _ { n } + { \frac { 1 } { 6 } } [ k _ { 1 } + ( 2 - \sqrt { 2 } ) k _ { 2 } + ( 2 + \sqrt { 2 } ) k _ { 3 } + k _ { 4 } ] ,
$$

$$
\left. \begin{array} { l } { \displaystyle k _ { 1 } = h f ( t _ { n } , x _ { n } ) , } \\ { \displaystyle k _ { 2 } = h f \bigg ( t _ { n } + \frac { 1 } { 2 } h , x _ { n } + \frac { 1 } { 2 } k _ { 1 } \bigg ) , } \\ { \displaystyle k _ { 3 } = h f \bigg ( t _ { n } + \frac { 1 } { 2 } h , x _ { n } + \frac { \sqrt { 2 } - 1 } { 2 } k _ { 1 } + \bigg ( 1 - \frac { \sqrt { 2 } } { 2 } \bigg ) k _ { 2 } \bigg ) , } \\ { \displaystyle k _ { 4 } = h f \bigg ( t _ { n } + h , x _ { n } - \frac { \sqrt { 2 } } { 2 } k _ { 2 } + \bigg ( 1 + \frac { \sqrt { 2 } } { 2 } \bigg ) k _ { 3 } . \bigg ) } \end{array} \right\}
$$

根据四阶公式可以看出，其阶数与每前进一步所需计算的函数值f的次数是一致的，但更高阶的RK公式所需要计算的f值的次数要比阶数多，若用N(v）表示只需计算v次f值所获得的方法的最高阶数，则有

$$
N ( v ) = v , 1 { \leqslant } v { \leqslant } 4 ,
$$

而

$$
\left. \begin{array} { l } { { N ( 5 ) = 4 , N ( 6 ) = 5 , N ( 7 ) = 6 , N ( 8 ) = 6 , } } \\ { { N ( 9 ) = 7 , \cdots . } } \end{array} \right\}
$$

上述四阶RK公式是显式，即 $k _ { i }$ 的算法是递推过程，由 $\pmb { k } _ { 1 }$ 算$k _ { 2 } ,$ 由 $k _ { 1 } , k _ { 2 }$ 算 $k _ { 3 } , \cdots$ ，如果把(9)式改为

$$
{ \boldsymbol { x } } _ { n + 1 } { = } { \boldsymbol { x } } _ { n } + \sum _ { i = 1 } ^ { \upsilon } { \boldsymbol { c } } _ { i } { \boldsymbol { k } } _ { i } ,
$$

$$
k _ { i } = h f { \Biggl ( } t _ { n } + \alpha _ { i } h , x _ { n } + \sum _ { j = 1 } ^ { \nu } \beta _ { i j } k _ { j } { \Biggr ) } , i = 1 , 2 , \cdots v .
$$

这种形式的RK公式称为隐式的，因计算每个 $\pmb { k } _ { i }$ 时，右函数 $f$ 的计算又总是涉及到 $\vec { k } _ { i }$

四阶RK方法是应用较广的一种单步法，对于天体力学(包括人造卫星）中的某些精度要求不太高的问题也是适用的．一般，RK方法估计局部截断误差比较麻烦,为此,Fehlberg[3]提出了一种使用嵌套技术的RK方法,同时给出 $\pmb { \mathcal { P } }$ 阶和 $p + 1$ 阶的两组RK公式，用两组公式算出的 ${ \pmb x } _ { { \pmb n } + 1 }$ 之差来估计局部截断误差，由此可给出下一步的步长，这就起了自动控制步长的作用，这种方法称为Runge-Kutta-Fehlberg方法,简称RKF方法，看上去要同时计算两组公式，但事实上，所谓嵌套，即两组公式是配套的， $\pmb { p }$ 阶公式与 $p + 1$ 阶公式相差甚少，见后面(23)一(31)式，只不过多算很少几次右函数值，例如7阶和8阶配套，只增多两次，却能同时给出局部截断误差，正由于RKF方法具备这种优点，因此它已成为目前被广泛采用的单步法.

Fehlberg已给出5(6),6(7),7(8)和8(9)四个嵌套公式[],相应地称为RKF5(6)，RKF6(7)，RKF7(8)和RKF8(9)方法.至于更高阶的嵌套，已不再能提高效率，我们在人造卫星和天体力学的一些定量和定性研究工作中采用了RKF方法，效果很好。方法简单，容易在计算机上实现，变步长方便，能保持所需要的精度,而且稳定性好．下面给出RKF5(6),RKF6(7)和RKF7(8)三种方法的具体计算公式，以便读者直接引用.
