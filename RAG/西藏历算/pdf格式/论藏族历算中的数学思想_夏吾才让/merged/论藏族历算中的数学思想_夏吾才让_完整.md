# 论藏族历算中的数学思想

夏吾才让

（西北民族大学数学与计算机科学学院 甘肃兰州 730030）

摘要 数学思想在藏族文化中无处不在，特别是作为藏族“小五明”文化之一的历算学中蕴含着丰富的现代数学思想和方法。文章通过分析藏历纪年法和我国农历干支纪年法之间的关系，指出藏历纪年法与现代数学的关系，又通过分析藏历中3种不同的五曜推算法，即太阳日推算法、太阴日推算法和宫日推算法，指出了藏族天文历算领域中数理运算的科学观点、定位和特色，以及其中呈现的现代数学的逻辑思维、基本思想、推算性质和应用方法等。

关键词 藏族历算；现代数学；藏历纪年；五曜 DOI:10.16249/j.cnki.2096-4617.2018.03.016

# 引 言

藏族传统历算历史悠久、文献丰富，经过千百年的演变已自成体系，在藏族人民的社会生活和传统数学中占有非同寻常的地位。藏族人民通过对大自然的观察，把一些规律性较强的东西总结了出来，比如西藏古谚语所述：“观察禽鸟和植物是珞巴、门巴法，观察星和风雪是藏北法，观察日、月运行是苯象法，观察山、湖、牧畜是岗卓法”。早在公元7、8世纪，吐蕃地区已经有了较粗疏的历算，之后由于唐文成公主和金城公主先后进藏，带来了卜笼占星之术，融合了汉族历算的成分。11世纪之后，藏族又从印度引进了时轮历体系，其中有经典的《时轮根本经》《时轮历法秘诀》《时轮摄略》《十二月目经》《摩登伽经》《金刚空行经》《日藏经》等，藏族传统历算进一步发扬壮大，随后出现的藏族学者论著计数百卷，内容非常丰富。如，布顿大师的《智者先悦篇》、克朱杰的《时轮经无垢先释大疏》、克朱·诺桑嘉措的《时轮经无垢先释庄严篇》、普巴·天成海的《白莲教亲论》、达摩师利的《日光论》、普巴·伦珠嘉措的《晶教》、第巴·桑结嘉措的《白琉璃》和桑卓特·桑热的《时轮法散心要》等。至18世纪中期，由于清朝初期颁行于汉族地区的时宪历能够准确预报日月食发生的时刻，引进到西藏地区，称为“藏传时宪历”，马杨寺汉历心要成为“藏传时宪历”的祖本。至此，藏族历算学内容由藏族固有的物候历、印度引进的时轮历和汉族地区引进的时宪历三部分构成。

远古时期，由计数产生了自然数，由分物产生了分数，由刻画相反意义的量产生负数，由线段间不可公度引进无理数，虽然当时在书写标记中没有现代意义上的分数、负数和无理数，但是人们已经知道，对于两个自然数的差与商未必是一个自然数的道理。同样，运算是藏族传统天文历算中的一个最基本概念。本文主要分析藏历纪年法和藏历中三种不同的五曜推算法，指出藏族天文历算领域中数理运算的科学观点、定位和特色，以及其中呈现的现代数学的逻辑思维、基本思想、推算性质和应用方法等。

# 1藏历纪年法中的数学思想

藏历纪年是藏族历算学的主要内容之一，是藏汉文化交流的主要环节，这对藏族历史文化的传承与保护及与汉文化的友好交流、共进作出贡献[1]。在藏历的纪年法中蕴含着现代数学中的四则运算等数理运算思想。如，藏历作为阴阳历，藏历纪年和我国农历干支纪年具有很多相似之处，吐蕃时期藏历曾先后采用过周边汉族地区12生肖纪年法以及12生肖配以五行和阴阳纪年，再融合了孟仲季月份和日期纪年。在藏历纪年法中，利用公元和公元前的个位数可知天干（用10去除公元或公元前的数目，其余数为天干），用12年去除公元或公元前年份，余数为十二生肖（地支）（见表1和表2）。因此，从藏历纪年法中可以知道，除法是乘法的逆运算，在自然数集、整数集中，除法是不封闭的。

甲子纪年法和藏历天干、地支的关系可为现代数学二元关系给出定义和解析。定义：设 $A , B$ 为任意两个集合，集合 $\{ ( a , b ) a \in A , b \in B \}$ 称为 $A$ 和 $B$ 的笛卡儿积，记做 $A \times B$ ,读做 $A$ 乘 $B$ ，其中 $( a , b )$ 为有序的元素偶[4] ， $a$ 称为 $( a , b )$ 的第一个坐标， $b$ 称为 $( a , b )$ 的第二个坐标。特别地， $A \times A$ 记做 $A ^ { 2 }$ 。

案例1： $A = \{ ( a , b ) \} , B = \{ ( c , d ) \} ,$ 求 $A \times B \ , \ B \times A$ .

解： $A \times B = \{ ( a , c ) , ( a , d ) , ( b , c ) , ( b , d ) \}$ , $B \times A = \{ ( c , a ) , ( c , b ) , ( d , a ) , ( d , b ) \}$ .A 表示天干所形成的集合，B 表示地支所形成的集合，在这种法则下产生的就是笛卡尔二元关系。从案例 中可以看出交换律不成立。

传统藏族历算中的读写风格与现代数学有所不同，藏族历算中加、减、乘、除的书写风格都在传统沙盘中进行，算法步骤及口诀同步实施，计算和验证逻辑严密，语言表达流畅。在藏族传统历算中不但给出了加、减、乘、除的基本概念和性质，且有加法交换律、加法结合律、非零整数、相反数、乘法交换律、乘法结合律、乘法对加法的分配律等性质[5]。如：

$\textcircled{1}$ 在藏历中进行除法运算时，具有有理数集 $Q$ 中有唯一数 $0 = \frac { 0 } { \mathrm { k } }$ ， $\mathrm { k }$ 是一非零整数，使得 $a + 0 = a$ 的数学思想。

$\textcircled{2}$ 在藏历中隐引正负概念时，具有 $\forall a \in Q$ ,存在 $( - a ) \in Q$ 使 $a + ( - a ) = 0$ .称 $( - a )$ 是 $a$ 的相反数的数学思想。

$\textcircled{3}$ 在藏历中1乘任何数都是自身时，具有有理数集 $Q$ 中存在数 $1 = \frac { \mathrm { m } } { m }$ ，使 $\forall a \in Q$ 有 $a \times 1 = a$ 的数学思想。

$\textcircled{4}$ 在藏历中逆运算时，具有 $\forall a \in Q , a \neq 0$ ,存在 $a ^ { - 1 } = \frac { 1 } { a } \in Q .$ , 使 $\boldsymbol { a } \times \boldsymbol { a } ^ { - 1 } = 1$ 的数学思想。

以上算律性质的思想及其方法在藏族天文历算中不仅存在，也给出了被除数、除数、小数（循环小数）、分数等概念。有理数集 $Q$ 与循环小数集是一一对应的，从代数的结构来看，循环小数与有理数只是形式不同，没有本质区别。藏族天文历算中的数理运算在实数集 $R$ 内进行的[4]。

# 2藏族传统历算推算法中的数学思想

在藏族传统历算学中曜、日期、星宿、会合、作用被称为五支，或称为五括、五要素。五支中的曜是“定曜”；日期则指日期和喜、满、胜、善、空五种名称相互配合；星宿指太阳日月宿；会合是由（定）日和月（宿）和合而成的；作用指作用的前后分。《藏历运算大全》中“五曜”的推算主要包括3种方法：太阳日推算法、太阴日推算法和宫日推算法[5]。现以藏历第 饶迥水龙年藏历 月（公历 年 月，农历 月）即角宿月的五曜算法设计与结果如下：

$\textcircled{1}$ 公积日：时轮历以角宿月为岁首，翼宿月为前一年的第十二个月，按已过十一个月又三十日计算。 积月 $\times 3 0$ 得公积太阳日 $L$ ， $L - { \frac { L } { 6 4 } } = L ( 1 - { \frac { 1 } { 6 4 } } ) = \frac { 6 3 } { 6 4 } L =$ 公积太阳日，其中， $\frac { 6 3 } { 6 4 }$ 近似于 $\frac { 1 1 1 3 5 } { 1 1 3 1 2 }$

$\textcircled{2}$ 殊日：（公积日 $^ +$ 历元常数）/ 恒星周期 $\mid =$ 商数（整周），余数（殊日），三武曜（火曜，木曜，土曜）迟行中数，两文曜（水曜，金曜）检步[6]。

$\textcircled{3}$ 中日又法：先以积月乘以30（不加入日月数）。置五曜的“殊日”，分别乘以27（宿），除以各自的周期，即火687，木4332，土10766，取其商余，依次乘以60，60，6，再除以各自的周期，（退至息位）记其商数，（息位之）商余，分别乘以各自子位之分母，即：火229，木361，土5383，再除以各自的周期，应除尽无余数。求两文曜（内行星）“检步”，水曜的殊日乘以27，60，60，6，除以8797（依次退位），记其（各位之）商数及（息位之）商余，即得水曜之“引检步数”。金曜的殊日乘以 ， ， ，，除以 ，（息位之）商余乘以749，再除以2247，除尽无余数。记各级商数，得金曜之“引检步数”，运算时仍应分别标出。三武曜之子位最好在这一步中就化出第六位，以免记入得数表后又要更改，两文曜之子位则可待算出迟行定数后再化。商余 $\times$ 圆周长 $\div$ 周年 $=$ 该日太阳的行度。又法：置未加（入当月日期）之公积日，重张五位，乘以$0 , 4 ^ { q } , 2 6 , 0 ^ { - } , 9 3 1 5 6 / 1 4 9 2 0 9$ （ 即 每 一 太 阳 日 中 太 阳 所 行 弧 度 ），恒 加 25,0,45,62370，自 下 而 上 除 以149209,6,60,60,27依次进位，取各位的余数，名为太阳日的太阳基数。

$\textcircled{4}$ 太阳的平黄经：(积日 $\mathrm { D } + \mathrm { A } ) \div$ 太阳恒星周期 $=$ 商 $^ +$ 余数，这里的 $A = \frac { 6 2 2 0 1 5 5 } { 1 8 3 8 2 }$ ，它乘以太阳的日行度，即为历元时太阳的平黄经。太阳所行弧度表、五曜迟行步度表和疾行盈缩步度表略。

利用现代数学的推算方法和数学工具也可以实现藏族天文历算中“五曜”的计算，是始终运用四则运算来实现的。另外，高等数学中的最大公因式（数）和最小公倍数的计算方法在藏族历算中也广泛使用，但其推算方法的全序性尚待研究[7]。

# 结 语

数学思想在藏族传统历算中占有独特的地位，藏族传统历算中的数学语言简洁、逻辑思维缜密，具有较强的概括性。在对藏族传统历算的研究过程中，可以利用挖掘、创造、借用的科学研究态度和严谨的数学逻辑思维来提升对藏族传统历算的认知层面，反过来也可以从藏族传统历算著作中寻找数学思想、方法和性质，普及藏族传统的数学文化知识。

# Discussion about the mathematical thought in the field of Tibetan calendar calculation

Xiawu-Cairang

(School of Mathematics and Computer Science, Northwest Minzu University, Lanzhou 730030,China)

Abstract: Mathematical thoughts exist everywhere in Tibetan culture, especially in Tibetan calendar (astrologi⁃ cal) calculation, which is regarded as one of Tibetan "small five ming" cultures, contains rich modern mathemati⁃ cal thoughts and methods. Through analyzing the relationship between Tibetan calendar calculation and Chinese lunar Ganzhi method, the relationship between the Tibetan calendar calculation method and the numbers in mod⁃ ern mathematics was revealed in this paper. And then through analyzing three different five-pronged algorithms, namely solar day algorithm, Lunar day algorithm, and Gongri algorithm, the scientific viewpoint and characteris⁃ tics of mathematical operations in the Tibetan astronomical calendar field were shown, as well as the logical thinking, basic ideas, the nature of calculation, and application method of modern mathematics presented therein. Keywords: Tibetan calendar calculation; modern mathematics; Tibetan year; Five Yao
