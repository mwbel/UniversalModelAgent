# A.3 蒙古历

蒙古历是藏历的一个版本。1911 年，当中国革命推翻最后一位皇帝后，蒙古宣布从中国独立，成为蒙古的官方日历；然而，在20世纪20年代（在共产党统治下）被公历（1948年正式生效）取代，当局甚至试图废除传统的蒙古新年庆祝活动。20世纪90年代共产党统治结束后，公历与其他传统一起复兴，尽管公历仍然是官方日历，用于日常民用；特别是蒙古新年（Tsagaan Sar）再次被视为一个重要的国家节日。参见[11]，[27]，[35，4.1.3]。

# A.11 黄历

元素会合.在印度体系中，一周中的每一天都有一个集合{地、火、水、风}中的相关元素，见表5。此外，每个星宿也与同一集合中的一个元素相关联，有关列表，请参见亨宁（Henning）[7，附录I]。因此，每个日历日都有两个元素的组合，分别是星期几（曜）和星宿（在（9.1）和（10.3）中计算）；这两个元素的顺序无关紧要，组合被视为无序对。因此，有10种可能的不同组合（会合），每种都有一个名字，见[7，p.204]。
