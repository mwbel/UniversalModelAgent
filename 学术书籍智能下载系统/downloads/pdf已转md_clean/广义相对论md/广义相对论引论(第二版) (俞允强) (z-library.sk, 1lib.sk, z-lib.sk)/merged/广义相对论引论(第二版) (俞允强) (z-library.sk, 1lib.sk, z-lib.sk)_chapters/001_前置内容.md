# 引 言

爱因斯坦(Einstein)在建立狭义相对论后遗留了两个问题没有回答.第一，自然界中什么参考系才是惯性系，为什么惯性系在描述物理规律中居于特殊的地位.第二，牛顿(Newton)引力理论是非协变的，即不满足相对论要求的.怎么才能建立一个相对论性的引力理论.粗略一看，这两个问题似乎没有联系.爱因斯坦对解决第一个疑问的想法是：一切参考系一—惯性系和非惯性系，在描述物理规律上都应该是平等的.为了使这想法能够成立，他发现必须推广引力的概念.于是这两个问题就联系起来了.沿着这思路发展下去，他所建立的新理论就是广义相对论.这理论对上述两个问题作了统一的协调的解决.因此，广义相对论既是狭义相对论的发展，也是牛顿引力理论的发展.我们在引言中先对这条线索作一简略的介绍.

# 1）相对性原理

相对性原理的观念来自牛顿力学.牛顿力学的基本规律 $F =$ ma 只是对惯性系才成立.原来人们认为两个相对作匀速运动的参考系之间的时空关系可表达为

$$
\begin{array} { l } { { { \boldsymbol x } ^ { \prime } = x - v t } } \\ { { { \boldsymbol y } ^ { \prime } = y , } } \\ { { { \boldsymbol z } ^ { \prime } = { \boldsymbol z } , } } \\ { { { \boldsymbol t } ^ { \prime } = { \boldsymbol t } . } } \end{array}
$$

这叫伽利略(Galileo)时空变换.容易验证,若牛顿力学规律对其中一个参考系成立，那么对另一个参考系也成立.这称为牛顿力学规律对伽利略时空变换的协变性，也称为力学的相对性原理.它告诉我们，惯性系虽有无穷多个，但不同的惯性系对于力学问题是完全等价的.非惯性系与惯性系则并不等价，因为在非惯性系中讨论力学问题时必须引入惯性力的概念.

19 世纪末确立了电磁学的基本规律，即麦克斯韦(Maxwell)方程.这方程对伽利略时空变换是不协变的.这个事实暗示了两种可能.一是相对性原理对电磁学不适用.从电磁规律看，只有一个真正的惯性系.二是相对性原理对电磁学仍适用，然而伽利略时空变换或麦克斯韦方程不是精确的规律，它需要修正.爱因斯坦的狭义相对论为这个疑问找到了答案.他从光速不变原理导出了一个新的时空关系

$$
\begin{array} { l } { { x ^ { \prime } = \displaystyle \frac { x - v t } { \sqrt { 1 - v ^ { 2 } / c ^ { 2 } } } , } } \\ { { \displaystyle y ^ { \prime } = y , } } \\ { { \displaystyle z ^ { \prime } = z , } } \\ { { \displaystyle t ^ { \prime } = \displaystyle \frac { t - x v / c ^ { 2 } } { \sqrt { 1 - v ^ { 2 } / c ^ { 2 } } } . } } \end{array}
$$

这就是洛伦兹(Lorentz)变换.在 ${ \boldsymbol { v } } \ll { \boldsymbol { c } }$ 时，它还原为伽利略变换.他证明了电磁规律对洛伦兹变换是协变的.此后他又修正了牛顿力学，使它对洛伦兹变换也协变.修正的结果后来被实验所证实.这说明相对性原理对力学和电磁学都是适用的.在这基础上，爱因斯坦把它推广为一条普遍原理：所有的基本物理规律都应在任一惯性系中具有相同的形式.这就叫狭义相对性原理.

按照狭义相对性原理，各惯性系完全等价.这样就原则上不可能用物理实验来判断哪一个惯性系是真正静止的.在一大群相对作匀速运动的惯性系中，哪些静止，哪些有速度，速度是多大，完全只具有相对的意义.因此狭义相对性原理说明速度的相对性.

# 2）惯性系疑难

从实践中总结，提升而得出的简洁的基本物理规律原则上仅对惯性系才严格成立.那就引出一个问题，自然界中哪一个或哪一些参考系是惯性系.

实践表明，地面上的实验室参考系就很像是一个惯性系.可是某些精确的测量表明它不是严格的惯性系.其原因可归结为地球在太阳参考系中的转动.地面参考系偏离惯性系的程度可用它自转引起的加速度来标志，它的大小为

$$
a = \omega ^ { 2 } R = 3 . 4 \times 1 0 ^ { - 2 } \mathrm { ~ m ~ } ^ { \bullet } \mathrm { ~ s ~ } ^ { - 2 } ,
$$

与地面上的重力加速度相比，这数值不太大.

如果因地球在太阳参考系中转动，使它不能作为严格的惯性系，那太阳又在银河系中转动，它是严格的惯性系吗？太阳在银河系中公转的加速度估算为

$$
a = \omega ^ { 2 } R = 3 \times 1 0 ^ { - 1 0 } \mathrm { ~ m ~ } \cdot \mathrm { s ~ } ^ { - 2 } .
$$

可见它比地面参考系是好得多的惯性系.但它终究也不是严格的惯性系.

那么银河中心是完全不加速的吗？银河系是宇宙中一个中等大小的普通星系.由于各星系间有引力，它必然也在作加速运动.或许这加速度更小了，但它终究也不会是严格的惯性系.

这样的分析使我们得到一个概念：由于引力作用的普遍存在，任一物质的参考系总有加速度，因而总不会是真正的惯性系.只不过尺度越大，物质越稀疏，相应的引力越弱，因此能找到更好的近似惯性系.从实用上这是一个很好的结论.从理论上，它却给狭义相对性原理提出了-个使人困惑的问题：在表述物理规律时惯性系占有特殊的优越地位，但自然界却不存在一个真正的惯性系.

现实的参考系都不是惯性系，使爱因斯坦产生了一个看法，任一参考系在表达物理规律上都应该是等价的.这就是说，他想把狭义相对性原理推广为广义相对性原理.

# 3）相对论性的引力

那么惯性系与非惯性系能等价吗？按原来的看法回答应是否定的.从牛顿力学看，运动物体在惯性系中的加速度取决于外界对它的作用力，可是在非惯性系中的加速度除取决于上述作用力外还取决于惯性力.惯性力是非惯性系自身的加速运动在质点上的反映，而不是一种物质间的相互作用.因此，要把惯性系与非惯性系看成等价，首先遇到的问题是怎样重新看待惯性力.

我们知道引力有一个特点，它的强度与受力物体的质量成正比，因而它引起的加速度与受力物体的固有性质无关.惯性力恰好具有同样的性质.这一点暗示人们：惯性力可能是引力的一种表现.这结论如果正确，惯性力与引力对一切物理现象的影响都应该是不可区分的.否则，它们本质上还是两种不同的力.爱因斯坦把这由事实所暗示的可能的结论作为原理接受了.这个结论称作等效原理.这样，相对论性的引力作为牛顿引力的推广，有了崭新的含意.

因为引力引起的加速度与运动体的固有性质无关，它仅依赖于该处引力场的情况，所以引力场的效果可以用空间的几何结构来描述.借助等效原理能论证，有引力场存在时的四维物理时空应当是弯曲的黎曼空间.刻画黎曼空间几何结构的度规张量起着引力势的作用.广义相对论所采用的正是这样的观点.接着的任务就是寻找度规张量(即引力势)对物质分布(即引力源)的依赖关系.爱因斯坦找到了这关系，它就是相对论性的引力场方程.就这样，广义相对论的基本框架被确定了.

从上面的分析看出，广义相对论本质上是一种新的引力理论.牛顿引力理论所刻画的仅仅是静止源的引力场，即静态的引力场.而广义相对论所刻画的却是一般的作任意运动的引力源所产生的变化引力场.在这意义上，广义相对论是在有引力时对狭义相对论的推广；而狭义相对论是广义相对论在没有引力时的特殊结果.

# 4）广义相对论的应用领域

广义相对论作为新的引力理论，它必定在特殊条件下以牛顿理论作为自己的近似.而使人感兴趣的却是这新旧理论的差别，以及这差别是否能得到实践的支持.

广义相对论建立后，爱因斯坦本人首先用它解释了用牛顿引力理论不能完全解释的水星近日点进动问题.这是广义相对论的第一个成功.其次他计算了太阳引力场对星光的弯曲，所得结果比牛顿理论的相应结果大一倍，而不久后，天文观测的结果与用广义相对论计算的结果很接近.这样，爱因斯坦在完全缺乏实践知识的基础上几乎全凭他深邃的洞察力而建立起来的新引力理论，很快得到了人们的承认.

尽管广义相对论很快得到了人们的承认和赞赏，可是大约有半个世纪，它却并没有受到人们充分的重视，因而也没有得到迅速的发展.因为在人们当时所研究的绝大部分引力现象中，牛顿引力理论都已是足够好的近似，广义相对论只提供了一个极微小的修正，所以人们在实用上不需要它.为说明这一点，让我们看一个球状物体的引力场.牛顿理论对它给出著名的公式

$$
F = { \frac { G M m } { r ^ { 2 } } } .
$$

广义相对论与它的差别可用一个无量纲参量 $R _ { g } / R$ 来标志，这里$R$ 是球半径， $R _ { \varepsilon } \equiv 2 G m ^ { \prime } c ^ { 3 }$ 叫引力半径，它取决于球的质量.仅当$R _ { \varepsilon } / R$ 接近于1，广义相对论的结果才会与相应的牛顿引力理论结果有实质的区别.否则前者只为后者提供一个量级为 $R _ { g } / R$ 的修正.具体看地球的引力场，它有

$$
( R _ { g } / R ) _ { \# \# } \approx 1 0 ^ { \cdot 9 } .
$$

对太阳的引力场，它有

$$
{ } ( R _ { g } / R ) _ { \parallel } \approx 1 0 ^ { - 6 } .
$$

这都是一些很小的量.即使对当时知道的最稠密的星体—-白矮星，它的密度高达 $1 0 ^ { 9 } - 1 0 ^ { 1 2 } \mathbf { k g } \cdot \mathbf { m } ^ { \mathrm { ~ \tiny ~ \cdot ~ 3 ~ } }$ ，可是它也只有

$$
( R _ { \varepsilon } / R ) _ { \stackrel { \prime } { \scriptscriptstyle \mathrm { d } \mathscr { H } \mathscr { H } } } \approx 1 0 ^ { - 3 } \sim 1 0 ^ { - 4 }
$$

人们不知道哪些实际天体的引力场需要用广义相对论来研究.

有 $\twoheadrightarrow$ 种引力问题，即宇宙的结构问题，用牛顿理论不能处理，必须用广义相对论来研究.可是因为当时宇宙学的观测资料很少，虽然人们建立过若干种宇宙学理论模型，这方面也没有取得巨大的进展.

情况的变化发生在60年代，即广义相对论建立后过了半个世纪.一个有重要影响的事件是中子星的发现.中子星的密度是$1 0 ^ { 1 8 } \mathbf { k g } \cdot \mathbf { m } ^ { - 3 }$ ，它有 $R _ { g } / R \approx 1 / 3 .$ 因此这是第一种必须用广义相对论来研究的强引力天体.另一个重要的发现是3K宇宙背景辐射，它使大爆炸宇宙模型得到了普遍的接受，从而成了广义相对论的又一重要应用领域.

近二十年来，广义相对论得到了蓬勃的发展.中子星的形成和结构，黑洞物理和黑洞探测，引力辐射理论和引力波探测，大爆炸宇宙学…成了它的广阔的应用领域.在基础理论方面，量子引力，大尺度时空的拓扑结构等都正在深入地研究中.在这种背景下，广义相对论也自然地成了物理学中一门重要的基础学科.
