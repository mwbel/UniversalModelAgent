# 第九章 研究地球重力场的现代理论

# 9.1莫洛琴斯基问题

所有的大地测量工作，包括水准测量、三角测量、天文测量和重力测量等都是在地球表面上进行的。而在随后的数据处理时，经典的方法是光归算到比地球表面平滑得多的大地水准面上。如果测量数据分布在较大的范围内，则还需进行第二次归算，即由大地水准面归算到更便于数学运算的椭球面上。此时，求定大地水准面就是一项必不可少的工作。然而,在利用司托克斯理论推求大地水准面时，从理论上说必需知道存在于大地水准面外的地壳物质的密度。但人们掌握的有关信息还不能满足这个要求，而只能采用一些假说来弥补。虽然这些假说的不真实性所造成的影响不是很大，但至少在理论上是不严格的，而且也使精度难于提高。原则上，地面上的大地测量数据可以直接归算到椭球面上，不一定要采用大地水准面作为过渡面。当然，必需知道地球表面与椭球面之间的相对关系。

为此，原苏联的学者莫洛琴斯基(M.C.MoJIogeHkun)在1945年提出了一种直接研究地球表面及其外部重力场的理论，它不需要使用任何有关地壳密度的信息。这一理论具有十分重要的意义，这一发展开创了研究地球形状及其外部重力场的新阶段。因此，在此处将他和此后的一些学者的有关这方面的研究成果称之为研究地球重力场的现代理论。

为了说明利用地面测量数据可以求定地球表面及其外部重力场，需要先证明一个定理：如果已知物体的外表面S、总质量 $\mathbf { M }$ 、物体绕某一固定轴作匀速自转的角速度 $\pmb { \omega }$ 和S面上任一点相对于面上某一固定点的重力位差 $\mathbf { \bar { w } _ { s } } - \mathbf { \bar { w } _ { \rho } }$ ，则S面外和面上的重力位W及其导数可以唯一地确定，而无需已知其内部质量的分布情况。

证明采用反证法。先假定在两种质量分布的情况下存在两个重力位，分别用 $\mathbf { W } ^ { \prime }$ 和W"表示之，然后证明它们之差

$$
\delta \mathbf { W } = \mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime \prime }
$$

在物体表面S上和面外处处为零。

依据已知条件，在S面上任意点相对某一固定点的重力位差 ${ \bf W _ { s } ^ { \prime } } - { \bf W _ { \theta } ^ { \prime } }$ 和 $\mathbf { W _ { s } ^ { ' } } - \mathbf { W _ { \eta } ^ { ' } }$ 都应与已知值相等。因此,在S面上的8W之值由 $( 9 \cdot 1 )$ 式可得

$$
\mathbf { \Delta } \widetilde { \mathbf { d } } \mathbf { W } \vert _ { \mathbf { s } } = \mathbf { W } _ { \mathbf { s } } ^ { \prime } - \mathbf { W } _ { \mathbf { s } } ^ { * } = \mathbf { W } _ { \mathbf { \eta } _ { 0 } } ^ { \prime } - \mathbf { W } _ { \mathbf { \eta } _ { 0 } } ^ { * }
$$

因为 $\mathbf { W } _ { \mathrm { ~ \scriptsize ~ \mathfrak ~ { ~ \mathfrak { ~ . ~ } ~ } ~ } } ^ { \prime }$ 和 $\mathbf { W } _ { \mathrm { ~ 0 ~ } } ^ { \eta }$ 分别为S面上固定点的两种重力位值，其差值为一定值。若用C表示这一定值，则得

$$
\delta \mathbf { W } \mid _ { \mathbf { s } } = \mathbf { C }
$$

该式表明，在S面上任一点的δW值都为同一数值。另一方面，由于质量分布不同只使引力位存在差异，而离心力位不会发生变化。因此，8W只是引力位差，因而它在S面外是个调和函数。在格林第一恒等式(5·7)中，取 $\mathbf { U } = \mathbf { V } = \delta \mathbf { W }$ ，考虑到 $\Delta \delta \mathbf { W } = 0$ ，则可写出

$$
\int _ { \Omega } ( \frac { \partial \mathbb { W } } { \partial \mathbf { x } } ) ^ { 2 } + ( \frac { \partial \mathbb { W } } { \partial \mathbf { y } } ) ^ { 2 } + ( \frac { \partial \mathbb { W } } { \partial \mathbf { z } } ) ^ { 2 } ] \mathrm { d } \Omega = - \int _ { \Omega } \mathbf { \{ } \frac { \partial \mathbb { B } \mathbb { W } } { \partial \Omega } \mathrm { d } \mathrm { S } 
$$

式中Ω为S面以外的无限空间,n为S面的相对于Ω的外法线，与(5·7)式中的法线方向相反，因而加了一个负号。再利用 $( 9 \cdot 1 )$ 和 $( 9 \cdot 2 )$ 式变换其右端，则得

$$
\int _ { \Omega } ( c \frac { \partial \bar { \bf W } } { \partial \bf x } ) ^ { 2 } + ( \frac { \partial \bar { \bf W } } { \partial \bf y } ) ^ { 2 } + ( \frac { \partial \bar { \bf W } } { \partial \mathrm { z } } ) ^ { 2 } ) \mathrm { d } \Omega = - \mathrm { C } ( \int \frac { \partial \bar { \bf W } ^ { \prime } } { \partial \mathrm { n } } \mathrm { d } \mathrm { S } - \int \frac { \partial \bar { \bf W } ^ { \prime \prime } } { \partial \mathrm { n } } \mathrm { d } \mathrm { S } )
$$

由 $( 5 \cdot 8 0 )$ 和 $( 4 \cdot 8 8 )$ 式，对两个重力位 $\mathbf { w } ^ { \prime }$ 和 $\mathbf { \mathbf { \mathbf { W } } } ^ { \prime \prime }$ 可以分别写出

$$

$$

在第五章论述调和函数的性质时已指出，当Ω为界面外的无限空间时，满足这一条件的调和函数只能是零。因此， $\delta \overrightarrow { W } = 0$ 。这样一来，由 $( 9 \cdot 1 )$ 式可得 $\mathbb { W } ^ { \prime } = \mathbb { W } ^ { \prime \prime }$ 。这就证明了在S面外不存在两个重力位，同时也就唯一地确定了重力位的导数。

上面所证明的定理表明，地球外部重力位及其导数是地球总质量、旋转角速度、地球表面的形状和地面上任一点相对于某一固定点的重力位差的函数。而所谓的莫洛琴斯基问题.则是在已知地球总质量、地球自转角速度、并在地面上所有点给定了相对某一点的重力位差和重力向量确定地球表面的形状及其外部重力位。它就是上述定理的反问题，因此从理论上说，利用地面上的测量数据完全可以求定地球表面的形状及其外部重力场。
