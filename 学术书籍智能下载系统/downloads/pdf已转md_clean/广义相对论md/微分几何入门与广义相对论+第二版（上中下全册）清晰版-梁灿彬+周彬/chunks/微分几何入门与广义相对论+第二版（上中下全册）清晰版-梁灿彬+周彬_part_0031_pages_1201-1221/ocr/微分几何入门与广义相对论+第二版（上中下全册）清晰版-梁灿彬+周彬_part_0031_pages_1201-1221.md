de Sitter 度规在静态坐标系的线元(J-2-19)使人联想到施瓦西线元(J-2-20),并猜想r=l是事件视界.事实上,正如Penrose图J-11所表明的，r=l的确是竖直线r=0所代表的静态观者的事件视界,也可说是一族静态观者(含r=0)的共同事件视界,这些观者的世界线是以竖直线r=0为一条积分曲线的那个静态Killing矢量场(a/at)a的积分曲线.这与施瓦西时空的事件视界[图9-13(a)的Ni]类似,因为N+是A区中所有静态观者(世界线为双曲线的观者)的共同事件视界.然而两者也有重要区别.N+是施瓦西时空的黑洞区B的边界(的一部分),是由时空的内禀几何完全决定的，(给定施瓦西时空后就有这一与众不同的边界,无须人为指定.）而de Sitter时空的视界r=l却取决于静态参考系(观者族)的选择.关键在于de Sitter时空的最高对称性保证存在无限多个静态观者族——任选一条类时测地线作为原点建球坐标系{t,r,0,φ}，该系的第零坐标基矢场(a/at)°的积分曲线(在I区内)就是一个静态观者族.可见,与施瓦西时空存在绝对的事件视界不同，de Sitter 时空中静态观者族的事件视界是相对的.也常把de Sitter 时空中的这种事件视界称为宇宙事件视界(cosmological event horizon),以区别于黑洞的事件视界.

# SJ.5 施瓦西-德西特时空

de Sitter度规是 $A > 0$ 的真空爱因斯坦方程的最高对称解.把对称性减弱为只有静态球对称性并允许 $\varLambda$ 取任意实数值(包括 $\varLambda > 0 , \varLambda = 0 , \varLambda < 0 \ v$ 将给出另一重要解.对静态球对称度规总可选择适当坐标系 $\{ t , r , \theta , \varphi \}$ 把线元表为式(8-3-2)那样的简单形式：

$$
\begin{array} { r } { \begin{array} { r c l } { \displaystyle \mathtt { d } s ^ { 2 } = - \mathtt { e } ^ { 2 A ( r ) } \mathtt { d } t ^ { 2 } + \mathtt { e } ^ { 2 B ( r ) } \mathtt { d } r ^ { 2 } + r ^ { 2 } ( \mathtt { d } \theta ^ { 2 } + \sin ^ { 2 } \theta \mathtt { d } \varphi ^ { 2 } ) . } \end{array} } \end{array}
$$

仿照小节8.3.2对无 $\varLambda$ 项的真空爱因斯坦方程 $G _ { a b } = 0$ （等价于 $R _ { a b } = 0$ 的求解过程(其解为施瓦西线元)求解 $A \neq 0$ 的真空爱因斯坦方程 $G _ { a b } + \varLambda g _ { a b } = 0$ （等价于$R _ { a b } = A g _ { a b } )$ ，易得

$$
{ \mathrm e } ^ { 2 A ( r ) } = { \mathrm e } ^ { - 2 B ( r ) } = 1 - ( 2 M / r ) - A r ^ { 2 } / 3 ,
$$

其中 $M$ 为积分常数.代入式(J-5-1)便得

$$
\mathbf { d } s ^ { 2 } = - \left( 1 - { \frac { 2 M } { r } } - { \frac { A r ^ { 2 } } { 3 } } \right) \mathbf { d } t ^ { 2 } + \left( 1 - { \frac { 2 M } { r } } - { \frac { A r ^ { 2 } } { 3 } } \right) ^ { - 1 } \mathbf { d } r ^ { 2 } + r ^ { 2 } ( \mathbf { d } \theta ^ { 2 } + \sin ^ { 2 } \theta \mathbf { d } \varphi ^ { 2 } ) ,
$$

$$
0 < r < \infty .
$$

在 $A > 0$ 时令 $l \equiv \left( 3 / A \right) ^ { 1 / 2 }$ ，则上式可改写为

$$
\textnormal d s ^ { 2 } = - ( 1 - \frac { 2 M } { r } - \frac { r ^ { 2 } } { l ^ { 2 } } ) \textnormal d t ^ { 2 } + ( 1 - \frac { 2 M } { r } - \frac { r ^ { 2 } } { l ^ { 2 } } ) ^ { - 1 } \textnormal d r ^ { 2 } + r ^ { 2 } ( \textnormal d \theta ^ { 2 } + \sin ^ { 2 } \theta \textnormal d \varphi ^ { 2 } ) .
$$

虽然 $M$ (作为积分常数)也可为负,但我们(出于物理考虑)只讨论 $M \geqslant 0$ 的情形.deSitter线元(J-2-19)是上式在M=0时的特例.上式在r→o时趋于deSitter线元,所以是渐近 de Sitter 解而非渐近平直解,称为施瓦西-德西特度规(Schwarzschild-deSitter metric).这一线元在

$$
r = 0 \quad \mathrm { ~ \mathcal { R } ~ } \quad 1 - \frac { 2 M } { r } - \frac { r ^ { 2 } } { l ^ { 2 } } = 0
$$

处呈现奇性,前者是时空奇性(真奇性),后者只是坐标奇性,而且事实上还是事件视界所在处(见稍后).为了确定视界位置,注意到r的范围是0<r<∞，应该寻求与方程 $1 - 2 M r ^ { - 1 } - l ^ { - 2 } r ^ { 2 } = 0$ 对应的3次代数方程

$$
r ^ { 3 } - l ^ { 2 } r + 2 M l ^ { 2 } = 0
$$

的所有正(实)根.令f(r)=r³-l²r+2Ml²，由df/dr=3r²-l²可知f(r)在r>0的区间内只有一个极值点,即r =l/√3>0,而d²f/dr²lr=n=6r >0则表明

$$
f ( r _ { 1 } ) = 2 l ^ { 2 } ( M - l / 3 \sqrt { 3 } )
$$

是极小值.函数 $f ( r )$ 的曲线随 $M$ 的增加而向上平移,当 $M = l / 3 \sqrt { 3 }$ (临界情况)时$f ( r _ { 1 } ) = 0$ ，曲线与 $r$ 轴正半轴相切, $f ( r )$ 有一个正根；当 $M > l / 3 \sqrt { 3 }$ 时曲线与 $r$ 轴正半轴无交，f(r)无正根,当0<M<l/3√3时曲线与r轴正半轴交于两点，f(r)有两个正根,两者之差随M的增加而减小,当M增至l/3√3时合而为一.由卡尔丹公式(见数学手册)并用适当技巧(见选读 J-5-1)可求得这两个正根为

$$
r _ { + } = ( 2 l / \sqrt { 3 } ) \cos ( \alpha / 3 ) , r _ { + + } = ( 2 l / \sqrt { 3 } ) \cos [ ( \alpha + 4 \pi ) / 3 ] , 0 < r _ { + } < r _ { + + } ,
$$

其中 $\alpha$ 由下式定义：

$$
\cos \alpha = - 3 \sqrt { 3 } \ l ^ { - 1 } M , \alpha \in ( \pi , 3 \pi / 2 ) \ . ^ { \mathbb { O } }
$$

为了证明线元(J-5-3)在 $r _ { + + }$ 和 $r _ { + }$ 的奇性只是坐标奇性(而且是事件视界),可以引人类Kruskal坐标并对时空进行延拓,最大延拓的 Schwarzschild-de Sitter 时空的Penrose图由一个冗长的计算(略)给出(图J-16).图中包括由 $r = 0$ 代表的时空奇性和由 $r = \infty$ 代表的(类空的)无限远 $( \mathcal { I } ^ { \pm } )$ 所构成的无限序列.相应地就有由I,II,III,，IV标明的四个区的无限序列.Killing 矢量场 $( \partial / \partial t ) ^ { a }$ 在I,II区中类时(在I区指向未来而在IⅢII区指向过去),在II,IV区中类空.无限多个IⅡ区又分为 $\mathrm { { I I } _ { B } }$ 和 $\mathrm { { I I } _ { C } }$ 两类，每个 $\mathrm { { I I } _ { B } }$ 区可解释为一个黑洞区,其类光边界(两条标明 $r = r _ { + }$ 的 $4 5 ^ { \circ }$ 斜直线代表的类光超曲面)可解释为相应的I(或II)区中所有静态观者的共同未来事件视界,称为黑洞事件视界；每个IIc区是这样一个时空区,其类光边界(r=r++)也可解释为相应的I(或IV)区中所有静态观者的共同未来事件视界,称为宇宙事件视界.通常说施瓦西解代表渐近平直时空中的一个黑洞,类似地也可说施瓦西-德西特解代表渐近德西特时空中的一个黑洞.这一时空中两类事件视界的共同存在给黑洞热力学和弯曲时空量子场论带来若干有挑战性的问题.

虽然近年来的天文观测表明我们的宇宙有一个小的、正的宇宙常数 $\pmb { \cal A }$ ，但超引力理论和弦理论的数学进展倾向于相信在超引力中的自然“基态”是AdS时空（其 $A < 0$ )，因而关于渐近AdS时空的研究一直是热门课题.这里首先出现的问题是如何定义渐近AdS时空.早期的文献通过对度规的坐标分量提出条件来定义，这些条件虽然直观看来是自然的,但却不够准确.选择坐标系时总要涉及(虽然有时只是暗中)一个充当背景的AdS度规,并认为物理度规应趋于这一度规.不幸,给定一个将要被称为渐近AdS时空的时空 $( M , g _ { a b } )$ 后，在选择一个可供 $g _ { a b }$ 趋于的AdS 度规时却存在若干含糊性.作为结果,渐近AdS时空的守恒量的定义自然也有含糊之处.为了克服用坐标语言下定义的困难，Ashtekar等又在渐近平直时空的几何定义(见 $\ S 1 2 . 4 )$ 的基础上提出了渐近AdS 时空的如下几何定义：

定义14维时空 $( M , g _ { a b } )$ 称为渐近AdS时空，如果存在“非物理时空”$( \tilde { M } , \tilde { g } _ { a b } )$ 使 $M \subset { \tilde { M } }$ ，而且

（a) $\tilde { M }$ 上有 $\mathbf { C } ^ { \infty }$ 函数 $\pmb { \mathscr { Q } }$ ，在 $M$ 上满足 $\tilde { g } _ { a b } = \varOmega ^ { 2 } g _ { a b }$ ·

(b) $\tilde { M }$ 是带边流形,满足：(b1) $\tilde { M }$ 的边界 $\mathcal { I }$ 与 $\mathbb { S } ^ { 2 } \times \mathbb { R }$ 同胚； $( \mathtt { b } 2 ) \varOmega$ 的梯度 $\tilde { \nabla } _ { a } \varOmega$ 在 $\mathcal { I }$ 上处处非零.

(c）时空 $( M , g _ { a b } )$ 的能动张量 $T _ { a b }$ 满足：(c1) $R _ { a b } - R g _ { a b } / 2 + A g _ { a b } = 8 \pi T _ { a b }$ ，其中$A < 0$ 是常数； $( \mathbf { c } 2 ) \varOmega ^ { - 2 } T _ { a b }$ 在趋于 $\mathcal { I }$ 时有 $\mathbf { C } ^ { \infty }$ 极限(记 $\overline { { T } } _ { a b } \equiv \operatorname * { l i m } _ {  \mathcal { I } } \varOmega ^ { - 2 } T _ { a b } ^ { } )$

不难看出上述定义与渐近平直时空的定义有不少相似之处,但还应注意两者的若干重要不同.例如,渐近平直时空的 $\mathcal { I }$ 是两个类光超曲面 ${ \mathcal { I } } ^ { + }$ 和 $\mathcal { I } ^ { - }$ 的并集，而渐近AdS时空的 $\mathcal { I }$ 却是类时的,在某些意义上它更类似于渐近平直时空的 $i ^ { 0 }$ 的切空间 $V _ { _ { i } \mathrm { { ^ { \circ } } } }$ 的3维子集 $K$ （见选读12-5-2).事实上,渐近AdS时空的“守恒”量的定义方式非常类似于借助 $i ^ { 0 }$ 及 $K$ 给渐近平直时空定义守恒量的方式.

闵氏时空与AdS时空都有最高对称性，因此其对称性群(等度规群)都是10维李群，前者就是Poincaré 群,后者称为AdS群.正如渐近平直时空的渐近对称群SPI是无限维李群那样，从定义1出发的研究表明渐近AdS时空的渐近对称群也是无限维李群.为了给渐近平直时空定义全部(10个)守恒量，应先对物理时空提附加要求来把渐近对称群从无限维的SPI群约化为10维的Poincaré群(见选读12-5-2)；类似地,为了给渐近AdS时空定义10个“守恒”量,也应先对物理时空提附加要求以使其无限多维渐近对称群约化为10维AdS群.以 $C$ 代表 $\mathcal { I }$ 的任一截面(是个拓扑2球面,代表一个“时刻”）， $\xi ^ { a }$ 代表约化后的渐近对称群的任一生成元(独立的 $\xi ^ { a }$ 共10个)， $E _ { a b }$ 代表渐近外尔张量的“电”部分,则与 $\xi ^ { a }$ 相应的那个“守恒”量在“时刻” $C$ 的数值 $\varrho _ { \xi } [ C ]$ 由下式定义：

$$
\mathcal { Q } _ { \xi } [ C ] : = - \frac { 1 } { 8 \pi } \sqrt { \frac { - 3 } { \varLambda } } \int _ { C } E _ { a b } \xi ^ { a } \mathrm { d } S ^ { b } . ( \mathrm { d } S ^ { b } \ \not \to \not \Xi \ \not \equiv \bar { \pi } \ \bar { \pi } )
$$

由爱因斯坦方程可以证明 $Q _ { \xi }$ 满足一个平衡方程：给定 $\mathcal { I }$ 的两个截面 $C _ { 1 }$ 和 $C _ { 2 }$ ，把$\mathcal { I }$ 介于它们之间的3维域记作 $\Delta \mathcal { F }$ ，有

$$
Q _ { \xi } [ C _ { 2 } ] - Q _ { \xi } [ C _ { 1 } ] = \int _ { \Delta \mathcal { T } } \overline { { T } } _ { a b } \xi ^ { a } \mathrm { d } V ^ { b } . ( \mathrm { d } V ^ { b } / \mathrm { \sharp K } \mathrm { \sharp K } / \mathrm { \sharp K } \bar { \pi } )
$$

上式表明,当右边非零时量 $Q _ { \xi }$ 并不守恒(所以守恒一词加引号).然而,上式积分号内的 $\overline { { T } } _ { a b } { \xi } ^ { a }$ 加负号后类似于式(12-6-26)的 $L _ { b }$ ,可解释为在“时刻” $C _ { 1 }$ 与 $C _ { 2 }$ 之间的“时间”内“流走”的该量,可见在这个意义上该量是守恒的.详见Ashtekar andDas(2000). [选读J-6-1完]
