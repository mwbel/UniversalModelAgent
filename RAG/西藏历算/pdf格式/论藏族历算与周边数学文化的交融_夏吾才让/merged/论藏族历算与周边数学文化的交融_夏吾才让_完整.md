# 论藏族历算与周边数学文化的交融

夏吾才让

# （西北民族大学计算机科学与信息工程学院，甘肃兰州 730030）

[摘要]藏族天文历算学研究具有悠久的历史和丰富的文献资料及实践经验.它是藏族天文学家根据藏区的实际，在长期的生产、生活实践中创造出来的.它以时轮历为主，以汉历为辅，形成了自己独有的特色.文章从阿拉伯数字、印度关于数的计算和藏族关于数的计算、日食和月食三个相关问题出发，探讨藏历的思想方法与周边文化的关系.

[关键词]藏族历算；周边文化；交融

藏历是藏族的传统历法，藏历的形成和流传与周边地区有密切的关系.吐蕃兴盛的二百多年间，以观测天象测算日、月、年周期的历算学在西藏已经形成，藏族历算家在积累的观测经验基础上，广泛吸取中原和周边民族先进历算理论、测算方法，创造了当地的历法体系.并且对月亮的朔望、四季和日月运行等天文现象做了较为科学的解释.因而，这种历法用于月球运行周期、回归年和日月食的推算，都是比较准确的.西藏天文学分为四科：“星历算”即印度“时轮”派数理天文学；“音韵占”是印度的一种占法；“黑算”是汉族的阴阳五行占；“汉历算”即汉族的《时宪历》.下面对相关问题作一些探讨.

# 1阿拉伯数字和藏族数学的关系

数的概念和计数方法远在有文字记载以前就有了.因而对其发展方式大都只能揣测、想象.我们有理由说，人类在最原始的时代就有了关于数的意识，至少在为数不多的一些东西中增加几个或从中取出几个时，能够辨认其多寡.因为研究表明：有些动物也具有这种意识.随着社会的发展，简单的计算成为必不可少的了.--个部落必须知道它有多少成员，有多少敌人；一个人也感到需要知道他羊群里的羊是否少了.或许最早的计数方法是使用简单算筹以---对应的原则来进行的.例如，当数羊的只数时，每有-只羊就扳--个手指头.还能够用下述方法计数：集攒小石子或小木棍；在土坏或石头上刻道道；在木头上刻槽；在绳上打结.后来，逐渐产生了各种各样的语言，作为对应于为数不多的东西的数目的语言符号.后来，随着书写方式的改变，逐渐形成代表这些数目的书写符号.人类学家关于原始人的研究报告表明上述想象中的发展过程是合理的.

人类的祖先，为了记述实际生产和生活中的数字，采取各式各样的方法，先是用手指计算，手指不够用了就用石子或木棍，这样就出现了计算工具.但是，这种计算工具如念珠、算盘等，还不方便，所以就发明了笔算.藏族沙盘的计算方法，就是-一种笔算形式.

要笔算就要有某种表示数字的符号，这就出现了各种记数符号.据有关资料显示，世界上著名的记数符号有如下几种：巴比伦记数法、古埃及记数法、中国古代记数法、古希腊记数法、占罗马记数法、古印度记数法.

虽然在上述各种记数法中没有藏族数字的地位，但它的形态、结构，引起笔者的思索：藏族数字和阿拉伯数字之间，存在着直接的联系.从具体形态上看，藏族数字弯钩1变成了直立而又稍带钩的1；带脚的2和3变成了无脚的2、3；4上面弯捺变成一竖的4；5的脚变成-个钩，上面第二竖变成-横；6去掉一撇；7去掉尾巴；8弯钩上连，9调转方向；0完全相同.从实际效果看，从复杂走向了简化，从原始走向了美观.所以，人们没有理由否定印度一—阿拉伯数字和藏族数字之间存在的某中亲缘关系.

# 2印度数的计算和藏族数的计算

罗马帝国统治时期是东西方之间学术的交流时期，并且从很早起，印度就和西方及远东互派使节.2000 年以前罗马人的书写用品是涂上薄薄一层蜡的小木板和一支硬笔.在罗马帝国之前和罗马帝国时代，常用沙盘进行简单的计算和画几何图形.显然，人们很早就用石头和粘土做书写记录了.

我们对印度的计算方法作一-说明.理解这种费力的算法的关键在于：弄清当时计算家们用的书写材料.根据德国历史学家汉克尔（H.Hankel）写的：用一杆笔蘸上稀薄(容易擦去的)白涂料写在-一块小黑板上，或用一根小棍写在小于1平方英尺，撒有红粉的白写字板上.无论在哪种情况下，写的地方都很小，而且为了做到清楚易读，要求把字写得相当大，但是，这样很容易被擦去和更正.因此，把计算程序改成：在一个数起到了它的作用后就擦掉，以保留书写空地.以上论述也就是藏族沙盘的特征.

# 3日食和月食

太阳月亮是天象，它们直接关系到人们的日常生活，因而人们对它们特别关注，研究和探讨它们的运动规律，准确计算出它们的运动方位，不仅是制定历法的基础，而且也是预报日月食的关键.

在时宪历传入西藏之前，藏族计算太阳月亮的运动方位大都依据时轮历，这种历法的基础来源于1000 年以前，其计算较为简略，误差也较为显著.

日食和月食都是自然界的一种现象.随着科技的进步，人类不仅对日食和月食有了正确的认识，而且还进一步掌握了日食和月食发生的规律，并能对日食和月食发生的时间以及见食的地点做出精确的预报.

日食的发生与太阳、地球、月球的运动有着密切的关系.地球围绕太阳运转，月亮围绕地球运转.这样，它们的位置也就时刻在变化着.地球和月亮都是自身不发光而又不透明的球体，地球和月球在太阳光的照射下，都会拖着一条长长的影子.当月亮运行到太阳和地球之间时，从地球上看去，日、月在同--方向(朔)，这时，如果日、月、地球三者恰好或几乎在一条直线上，月影就会扫到地球表面，被月影扫过的地区，人们就会看到日食.显然，日食必定发生在朔，即农历的初一.但不是每次朔都发生日食，因为在朔日，月亮虽然位于太阳和地球之间，但日、月、地三者不在同一条直线上，这是因为月球绕地球转动的轨道平面(白道面)和地球绕太阳公转的轨道平面(黄道面)并不重合，而是有一个平均为 $5 ^ { \circ } \ 0 9 ^ { \prime }$ 的交角.所以，只有当月亮运行到黄道和白道的升交点和降交点附近时，才有可能产生日食.如图1.

月食的基本原理和日食类似(图1).当月球运行到和太阳方向相反的位置，从地球上看去，日、月方向相差 $1 8 0 ^ { \circ }$ (望)，这时，如果日、地、月三者恰好在一条直线上，那么，月球就会走进地球的影子里，使得部分月面或整个月面照射不到太阳光，这就是月食.显然，月食必定发生在望，即农历的十五或十六.但并不是每次望日都发生月食，只有在望日时月亮运行到白道和黄道的交点附近才会发生月食.

图2日全食图

日食有三种类型：日全食、日环食、日偏食.日全食时月球将整个太阳圆面遮住；日环食时丹亮只遮住太阳圆面的中心部分，而在太阳圆面的边缘还露出像光环似的一圈；日偏食则是月球只遮住了太阳圆面的-一部分.为什么会有三种日食现象发生呢？前面已经进过，月球在太阳光的照射下会在背着太阳的方向上产生一个影子.月球的影子可以分为本影、半影和伪本影三个部分.如果月球本影锥扫过地球表面，那么在本影区内，太阳射向月亮的光线全部被遮住，因此在这--区域的人们就会看到太阳的整个圆面被月亮遮住的日全食现象(图2).如果是半影扫过地面，则在半影区内，太阳射向月球的光线部分被月亮遮住，在这一区域的人们就会看到日偏食现象.如果是伪本影扫过地面，那么在伪本影区内，太阳中心部分射来的光线被月亮挡住，但太阳边缘部分射来的光线却未被遮挡，因此在伪本影区内的人们就会看到太阳中央部分被遮，而太阳四周只留下一窄圈光环的日环食.
