1974年Hawking解决了这个矛盾，他证明黑洞的确存在由温度(4.22)所标志的辐射，证明的关键是在于考虑了量子效应.我们知道，按照量子理论，在真空态中不断地有所谓虚粒子的产生和湮没过程，形象地称为真空涨落，这种性质已由量子电动力学的实验精确地证实了，这样，在黑洞之外，邻近视界的地方，自然地有这种涨落产生的虚粒子，相对于远离黑洞的观测者来说，有些虚粒于具有负能，通过隧道效应，这种负能粒子可以穿过视界进入黑洞，使黑洞的质量减少，这种过程就相当于向无限远处发射一个正能的粒子，当然，相反的过程也是存在的，所以，从局部来看这种涨落也不会引起黑洞的质量变化，但是，如果考虑到黑洞是由普通的天休坍缩而的，那么，从引力坍缩到形成黑洞这整个过程并不是时间反演不变的，正是由于这种时间方向上的不对称性，才决定了真空涨落的净影响是使黑洞有发射.

黑洞的发射是稳定的，原则上它可以发射任何类型的物体，只要黑洞能吸收这种物体，发射率具有热谱，即通常在热平衡态下的发射谱，当温度为零时，只发射静质量为零的粒子（象光子、中微子等)，随着温度的升高，按照质量从小到大的顺序，各种粒子逐渐开始有显著的发射，黑洞的热平衡温度 $I ^ { J }$ 即由（4.22）规定.不管什么物体，强子、轻子、光子、…，它们的发射率并不与强作用、弱作用或者电磁作用有关，而仅仅决定于引力(见(4.22)）.这表明，在黑洞附近，引力作用占据决定注作用，尽管考虑了量子效应，仍然是如此.

这些讨论表明，黑洞观念与引力论、量子论、热力学统计物理学等几方面都有深刻的联系，它处在这些基本概念的联接点上，通过黑洞问题进一步揭示出这些基本理论之间的关系，无疑是很有意义的.

# $f ( 1 , 5 )$ 黑洞的证认

脉冲星发现并被确认为中子星之后，从观测上证认黑洞的存在就成了当务之急.

尽管有许多相似，在中子星和黑洞之间还是存在着一个重要的区别，这就是质量，中子星的质量都应当小于绝对临界质量(3.2$( 1 1 , \frac { 3 } { 2 } )$ ，虽然在原则上不排除质量小于临界质量的恒星也可以坍缩成黑洞，但是其可能性是极小的，因为必须要求恒星有足够的动能以克服中子星平衡态所构成的势垒，所以，一般地说，由恒星坍缩而形成的黑洞，质量应大于 $\frac { \pi } { 2 } = \frac { \pi } { 2 } [ k \pi _ { 1 } ]$ ，

中子星与黑洞的另一个基本的区别是它们的电磁结构，在中子星中，磁矩的方向可以偏离转动轴，我们正是用这种倾斜磁矩结构来解释脉冲星的周期发射，另方面，对于定态的黑洞来说，只能具有轴向的磁矩，所以，当有物质落到黑洞周围时，它的发射虽然会具有短时标的结构，其时间尺度约为质点绕黑洞的转动周期，但是，我们不能期望这种辐射会有周期性的时间结构.

可见，为了进行理论与观测的对证，我们需要测量致密星的质量、角速度以及它的电磁性质(即它们的磁层结构)，在中子星的情况我们还力图知道星的内部情况，显然，为了能测出这些量，用脉冲星是不行的，理由是：(1)脉冲星的发射过程是极稳定的，除了极少的脉冲周期的突然变化之外，看不到什么变化，这也就不便于进行工作；(2)射电脉冲星都是单星(PSR1913+16除外)，不可能直接测定它们的质量.

企图寻找孤立的黑洞，更没有希望，因为没有光线从黑洞发射出来，至于黑洞对于遥远星体的影响，只有当黑洞及其遥远的星体都在同一视线方向时，才是显著的，显然，这个位形是极不容易找到的.

由于这些原因，促使人们着重讨论可能存在于双星体系中的黑洞，如果一个黑洞与一个正常星体构成一个双星体系，则可能有较多的可观测效应。
