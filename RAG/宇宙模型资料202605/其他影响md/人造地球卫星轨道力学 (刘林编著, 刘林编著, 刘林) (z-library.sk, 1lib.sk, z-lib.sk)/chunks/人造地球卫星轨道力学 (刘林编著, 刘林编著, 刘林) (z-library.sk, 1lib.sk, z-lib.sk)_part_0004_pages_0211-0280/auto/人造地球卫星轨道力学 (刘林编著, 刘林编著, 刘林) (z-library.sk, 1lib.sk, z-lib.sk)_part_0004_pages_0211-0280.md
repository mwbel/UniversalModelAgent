# 6.5同时消除两类奇点的摄动计算方法

$2 4 ^ { \tt h }$ 同步卫星的轨道就是 $e \approx 0$ 和 $\begin{array} { r } { \pmb { i } \approx 0 , } \end{array}$ 而且地球形状摄动中田谐项的影响又导致通约问题，故同时存在两类奇点．对于这种情况，在 $\ S \ r _ { \mathrm { ~ \bf ~ 1 ~ } }$ 中已指出，可采用(6.6)式定义的，即

$$
a , \xi = e \cos \left( \omega + \Omega \right) , \eta = - e \sin \left( \omega + \Omega \right) ,
$$

$$
h = \sin i \cos \Omega , k { = } { - \sin i \sin \varOmega } , \lambda { = } M + \omega + \Omega .

# 第七章 变换理论在轨道力学中的应用

前面三章都是直接从一般摄动运动方程着手，采用摄动法和各种平均根数法来构造该小参数方程的幂级数解，原理很直观.方法也比较简单，但从构造一阶解的过程来看，给出具体的摄动表达式，还是比较麻烦的。若要给出高阶解，例如二阶解，原则上虽无困难，可其麻烦的程度已达到几乎无法实现的境地．另外，有些环节，如用平均根数法构造短周期摄动项的处理，数学上还显得不够清晰．而分析力学方法，尽管原理上显得有些抽象，但其解决问题的过程却简单得多，同样可用于求解人卫轨道变化问题.这一章将着重介绍哈密顿系统的正则变换方法及其应用，并将变换思想推广到非哈密顿系统，给出一般的变换方法.

有关哈密顿力学中的一些基本量的引入，将删去详细过程，这样做对即使没有系统学过分析力学的读者，也无关紧要．但是有关正则变换的内容，将采用一种简法的数学描述和推导(证明),以使读者能够系统地掌握这一内容，便于具体引用.

当变换(7.5)式不显含 $\pmb { \mathscr { t } }$ 时,有

$$
\begin{array} { r } { T = T _ { 2 } , \qquad T _ { 1 } = T _ { 0 } = 0 . } \end{array}
$$

此时

$$
H { = } T { - } U { = } H ( p , q ) .
$$

显然， $\pmb { H }$ 就是系统的总能量(动能与位能之和),而在一般情况下，就称 $\pmb { H }$ 为广义能量。这种 $\pmb { H }$ 不显含 $\pmb { \ t }$ 对应的哈密顿系统称为定常的，或自治的．该系统有

$$
\frac { d H } { d t } { = } \displaystyle \sum _ { j = 1 } ^ { 3 n } \bigg ( \frac { \partial H } { \partial p _ { j } } \dot { p } _ { j } + \frac { \partial H } { \partial q _ { z } } \dot { q } _ { j } \bigg ) .
$$

将满足方程(7.14)式的 $\dot { q } _ { j }$ 和 $q _ { j }$ 代人上式，即得

$$
\frac { d H } { d t } = 0 .
$$

这表示定常的哈密顿系统 ${ \pmb { H } } = { \pmb { H } } \left( { \pmb { p } } , { \pmb q } \right)$ 存在一积分

$$
H = h .
$$

此即能量积分， $\hbar$ 就是能量常数.

如果就取直角坐标 $x _ { i } , y _ { i } , z _ { i } ( i { = } 1 , 2 , \cdots , n )$ 作为广义坐标；那么 $m _ { i } { \dot { x } } _ { i } , m _ { i } { \dot { y } } _ { i } , m _ { i } { \dot { z } } _ { i }$ 即为广义动量．相应的动能 $\pmb { T } = \pmb { T _ { 2 } }$ ，即(7.1)式,于是

$$
H { = } T { - } U { = } H ( x , y , z , { \dot { x } } , { \dot { y } } , { \dot { z } } )
$$

不显含 $\pmb { \ell }$ ，因此，在惯性直角坐标系中， $\pmb { \mathscr { n } }$ 体引力问题存在能量积分 $H = h$ ·

哈密顿正则运动方程具有一种对称性，它有一种特殊解法，即寻找循环坐标来降阶，例如，当 $\pmb { H }$ 不含某个 $q _ { i }$ 时，有

$$
\dot { p } _ { i } = - \frac { \partial H } { \partial q _ { i } } { = 0 } .
$$

存在一积分

$$
{ \mathrm { ~ . ~ . ~ } } { \mathrm { ~ } } { \mathrm { ~ } } p _ { \mathrm { { i } } } { = } C .
$$

而因 $\pmb { H }$ 不含 $q _ { i }$ ,又可将其分离开，于是 $\pmb { \mathscr { n } }$ 体问题的正则运动方程组(7.14)式可以降低两阶．上述 $q _ { i }$ 即称为循环坐标，如果不含某个 ${ \pmb p } _ { i }$ ，效果也一样，它同样被称为循环坐标．用分析力学解决人造卫星运动问题，就是寻找一种特殊变换，使变换后的哈密顿函数具有足够的循环坐标，从而达到获得摄动解的目的，这是本章的主要内容.

# 7.4显函数定义的正则变换—-Lie 变换

在具体应用中，人们希望能由显函数来定义正则变换，堀源一郎(Hori)[2]和德普里(Deprit)[a]分别于1966年和1969年以不同的方式把李(Lie)级数引进正则变换理论，解决了上述问题，他们都是用李级数来定义正则变换，因此又称这种变换为Lie变换，或称Hori-Lie变换．由于两人的工作在原理上是一致的，结果也是等价的，而德普里的方法用起来显得方便些，又与前面介绍的内容有一定的连贯性，因此，关于这一问题，就以德普里的方法和结果，作为基础介绍给读者.

# 3.摄动运动方程的建立

对于受摄运动，系统的哈密顿函数为

$$

$$

相应的运动方程即

$$

这里的 $\mathcal { R }$ 是摄动函数，它是受摄系统力函数（去掉卫星的质量因子)的一部分，对于卫星绕地球运动而言，即地球引力场位函数的非球形部分．当 $R = 0$ 时，前面已用哈密顿-雅可比方法给出对应的解，即

$$

$$

$$
H _ { 0 } + S _ { \ell } ( q , \alpha , t ) = 0 ,
$$

$$
\pmb { \imath } = \frac { \partial S } { \partial q } , \qquad \pmb { \beta } = \frac { \partial S } { \partial \alpha } .
$$

$$

$$

当 $R { \neq } 0$ 时，用常数变易法来建立相应的解，即认为(7.140)式的形式仍满足受摄运动方程(7.139)，只是 ${ \pmb { \alpha } } { = } { \pmb { \alpha } } ( t )$ ， $\pmb { \beta } { = } \pmb { \beta } ( t )$ ，而不再是常向量．相应的 $\pmb { \mathcal { S } }$ 的形式当然也不变，有 $\pmb { S } = \ o { S } ( \pmb { q } , \pmb { \alpha } ( t ) , t ) ,$ 问题是需要建立 $\alpha ( t )$ 和 $\beta ( t )$ 所满足的微分方程．作 $\mu = 1$ 对应的伐夫式的第四种形式，即(7.62)式的第四式 $( u = \alpha , v = \beta )$ ：

$$

因 $\pmb { p } { = } \partial \pmb { S } / \partial q , \pmb { \beta } { = } \partial \pmb { S } / \partial \alpha ,$ 若取函数 ${ \pmb R } ^ { \prime } = \pmb { \partial } { \pmb S } / \pmb { \partial { t } } .$ ，则上式变为
