# v1 vs v2 版本对比

## 概述

本文档对比旧版本（frontend-v1）和新版本（frontend）的差异。

## 运行方式

| 版本 | 命令 | 端口 | 状态 |
|------|------|------|------|
| v1（旧版本） | `npm run dev:v1` | 5173 | 完整保留，可独立运行 |
| v2（新版本） | `npm run dev:v2` | 5174 | 基于原型页重构中 |

## 核心功能对比

| 功能 | v1 | v2 | 说明 |
|------|----|----|------|
| A2UI 引擎 | ✅ | ✅ | 完全复用 v1 代码 |
| 插件系统 | ✅ | ✅ | 完全复用 v1 代码 |
| SSE 流式对话 | ✅ | ✅ | 完全复用 v1 代码 |
| 状态管理 | ✅ | ✅ | 完全复用 v1 代码 |
| Gallery 模式 | ✅ | 🚧 | 待实现 |

## UI 设计对比

### 布局

| 方面 | v1 | v2 |
|------|----|----|
| 左侧栏 | 对话列表 | 项目卡片 + 知识树 + 对话列表 |
| 中央区域 | 对话内容 | 欢迎屏 / 对话内容 |
| 右侧栏 | 启发时刻 + 基石 | 能力进度 + 系统状态 + 技能标签 |
| 顶部导航 | 简单标题栏 | Logo + 导航链接 + 会话标题 + 用户头像 |
| 输入框 | 简单输入框 | Pill 形状 + 模式切换（Start/Plan/Ask） |

### 设计语言

| 方面 | v1 | v2 |
|------|----|----|
| 字体 | Newsreader (serif) | Newsreader (标题) + Manrope (UI) |
| 色彩 | Material Design 3 | Material Design 3（完整应用） |
| 圆角 | 标准 Tailwind | 极小 (0.125rem) + 大 (24-28px) |
| 玻璃态 | 有限使用 | 广泛使用（backdrop-blur） |
| 阴影 | 标准 Tailwind | 细微（0.06 opacity） |

### 新增功能（v2）

1. **欢迎屏**：首次访问显示项目建议卡片
2. **模式切换**：输入框支持 Start/Plan/Ask 三种模式
3. **项目卡片**：左侧栏显示上个项目信息
4. **知识树导航**：左侧栏显示知识节点（占位）
5. **能力进度**：右侧栏显示学习进度
6. **背景装饰**：SVG 曲线装饰

## 兼容性

### 后端 API

两个版本使用相同的后端 API，完全兼容：

- `POST /api/v1/chat` - SSE 流式对话
- `GET /api/v1/conversations` - 对话列表
- `GET /api/v1/plugins` - 插件列表
- `POST /api/v1/plugins/{id}/toggle` - 启用/禁用插件

### 数据库

两个版本共享同一个数据库，对话历史完全同步。

### 插件

两个版本共享插件系统，插件在两个版本中都可用。

## 测试建议

### 并行测试

1. 启动两个版本：
   ```bash
   # 终端 1
   npm run dev:v1
   
   # 终端 2
   npm run dev:v2
   ```

2. 在浏览器中打开两个标签页：
   - v1: http://localhost:5173
   - v2: http://localhost:5174

3. 对比测试相同功能的表现
