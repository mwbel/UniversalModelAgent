# 第二章 字处理软件

# 第一节字处理软件简介

# 一、什么是字处理软件

在人们的日常工作、学习和生活中，会经常接触到文字处理工作，例如，书写信函、修订文稿、印发文件、保管资料等。在计算机得到广泛应用之前，这类工作基本上要靠手工来完成，往往需要花费大量的时间和人力，工作效率比较低。

随着经济与科学技术的发展，计算机的应用已经扩展到文字处理领域。人们可以用计算机编写、保存、印发公文和信件；记者可以用计算机撰写消息、报导并通过电话线把它们传送到远方的编辑部；作家可以用计算机写小说，把几十万乃至几百万字的书稿存放在磁盘里，随时可以调出来加工润色而不必浪费一张稿纸；……。

用计算机进行文字处理时，需要给它们配上字处理软件。字处理软件是一种能够让人用计算机输入、输出、存储、编辑文稿，从而提高工作效率的软件。目前，为微型计算机设计的字处理软件有许多种，常用的有WORDSTAR、CCED、WPS、WORDPERFECT等。

# 二、字处理软件的主要功能

字处理软件一般具有输入、编辑、存储、输出等几方面的功能。例如，可以用计算机键盘输人汉字、英文或其他文字信息；可以非常容易地找到文章或书稿中的某一个位置，方便地进行插入、删除、修改、替换等编辑处理，还可以把文章或书稿中的句子、段落等部分复制或搬移到所需的位置上，然后重新整理，使版面整齐、美观；可以把已经编辑好的文稿或文稿的某些部分存人磁盘长期保存，需要时再把它们调出来；可以根据需要设定打印格式、打印字体，计算机自动地按要求把文稿打印在纸上。

此外，功能较强的英文字处理软件还备有一部存储了数万条词汇及各种语法规则的电子词典，用来自动地检测、修改文稿中的拚写或语法错误。

# 三、中文字处理软件

早期的字处理软件只能处理英文文稿。70年代中期，我国的计算机科技工作者开始研究汉字信息处理技术。目前，已经研制开发出了许多种适合不同机型的中文字处理软件。这些中文字处理软件除具有处理英文文稿的功能之外，还可以处理中文文稿。经过多年来的改进，中文信息处理的水平不断提高，用键盘输人汉字的速度已经大大超过英文输入的速度。中文字处理软件的功能也越来越强，输出的文字有各种漂亮的字体、不同大小的字号，可以满足办公、出版、印刷等众多方面的要求。因此，中文字处理软件应用也越来越广泛，逐渐改变了中国人进行文字处理的工作方式，从而大幅度地提高了中文文稿处理的工作效率。许多政府机关、工矿企业都用中文字处理软件管理各种文件资料。越来越多的公司在招聘职员时，把是否会使用字处理软件当作录取与否的重要条件之一。

# 第二节中文字处理软件CWS 简介

CWS 软件是可以在Apple ⅡI机上运行的中文字处理软件，它具有PC机上的通用字处理软件的大部分功能，操作方法也基本相同。利用CWS软件，可以了解中文字处理软件的基本思想和操作方法，为以后使用其他字处理软件打下基础。

# 一、CWS 软件所需要的运行环境

$\textcircled{1}$ 中华学习机或者具有64K内存的Apple 机及兼容机;  
$\textcircled{2}$ 一个或两个软磁盘驱动器;  
$\textcircled{3}$ 一片存有CWS 软件的系统盘;  
$\textcircled{4}$ 一片字库盘;  
$\textcircled{5}$ 一片用于存储文件的工作盘。

# 二、进入和退出CWS字处理软件

# 1． 进入 CWS

用冷启动方式或热启动方式都可以进入CWS。

# (1）冷启动方式

把CWS系统盘平稳地插入一号磁盘驱动器，关好驱动器的小门，然后开机。这时，磁盘驱动器上的红色指示灯会亮起来，计算机把CWS系统读入内存储器。略等片刻，驱动器指示灯熄灭，显示器屏幕上会显示主菜单，表明计算机已进入CWS系统。

# (2）热启动方式

在开机状态下，把CWS系统盘平稳地插入一号磁盘驱动器，关好驱动器的小门，键入命令PR#6并按回车键，磁盘驱动器上的红色指示灯亮，计算机读人CWS系统，等到驱动器指示灯熄灭后，显示器屏幕上显示主菜单，计算机进入CWS系统。

# 三、进入编辑

# 1.进入编辑

在主菜单显示状态下，键人英文字母D,按下列步骤操作即可进人文稿编辑。

(1）屏幕上提示：“请插入系统盘于第一驱动器，按任一键”由于系统盘已在驱动器内，所以，只要按下任一键即可。  
(2）屏幕上又提示：“请插入工作盘于第一驱动器，按任一键”这时，把系统盘从驱动器中取出，插入工作盘，再按任一键。  
(3）计算机询问要编辑的文件名，并等待用户输人一个文件名。

文件名可以由 $1 \sim 1 0$ 个字符组成，字符中间不能有逗号，且第一个字符必须是英文字母。键入文件名并按下回车键之后，计算机自动检查工作盘中是否已存有该文件，若磁盘中没有以该文件名命名的文件，则屏幕上显示出“新文件”；若该文件是从前已建立过的文件，则计算机把已建立好的文件从磁盘中调入内存。

(4）屏幕上提示：“请插入字库盘于第一驱动器，按任一键”这时，把工作盘从驱动器中取出，插入字库盘，再按下任一键，屏幕上会显示出文件名.XXXXXXXXXX 行 $= 0 0 1$ 列 $= 0 0 0 1$

# 2.CWS字处理软件的屏幕结构

进人编辑状态后，屏幕被分成两部分：状态区和编辑区。状态区在屏幕的第一行和最后一行，第一行共有三个信息，指出 $\textcircled{1}$ 被编辑文稿的文件名称（这里用一串×表示，实际操作时会显示出前面输入的文件名）、 $\textcircled{2}$ 当前光标所在的行号（这里的001表示第一行）和 $\textcircled{3}$ 当前光标所在的列号 (这里的0001表示第一列)。这个状态行下面的一条短横线叫做光标，光标的作用是指出将要输入的字符在屏幕上显示的位置。在编辑过程中，状态区中相应的行、列值会自动随着光标的移动而改变。如果所用的计算机能输入汉字，屏幕的最下面一行是中文提示行。

编辑区实际上是由若干张空白编辑页组成的，操作时可以在编辑区的任何一个位置上输入文字。

# 3．光标及其移动方式

编辑区的范围很大，不可能在屏幕上完全显示出来，计算机的屏幕相当于一个活动的窗口，可以在编辑区内上、下、左、右自由地移动，随着窗口的移动，就可以看到编辑区内的所有信息了。在编辑状态下，光标相当于作者手中的笔，它可以在操作者的控制下移到编辑区中的任意地方，光标移到哪里，就可以在那里输入或修改。

# 四、文稿编辑

文稿编辑是字处理软件的核心，它的主要功能包括：输入、插入、删除、修改文字。下面依

次加以介绍。

# 1．输入一段文字

从键盘键入的每一个汉字或字符都将出现在光标所在处，每输入一个西文字符，光标向后移动一个字符位，每输人一个汉字，光标向后移动两个字符位。当输人位置超过屏幕显示范围时，光标会自动跳到下一行的行首，不需要按回车键。只有输入完一个自然段之后，才应按下回车键。

# 2．插入一个或一段新文字

进入编辑状态时，系统已自动设置成插入方式。在文稿输入过程中，若发现遗漏了一些文字，可利用插人操作把它们补上。具体做法是：先把光标移到要插入文字的位置，再键入要插入的文字。这时，新文字会出现在光标所在处，而光标所在处原有的文字以及后面的文字会自动向后顺序移动。

# 3．删除一个文字

要删除文稿中的某一个文字时，先把光标移到要删除文字的后边，按下Esc键，则光标前面的文字会被删除。该文字被删除后，它后面的其余文字会自动跟上，删除处不留空格。

# 4．删除光标所在行的所有文字

为了加快删除操作的速度，CWS 软件提供了删除一行文字的操作命令，具体作法是：把光标移到要删除的那一行的任意位置，按下 $\mathrm { C t r l } + \mathbf { Y }$ 键，光标所在行的全部文字会被删除，下一行的信息会自动跟上，删除处不留空行。

# 5．大小写字母的转换

用CWS 软件编辑英文稿件时，要注意英文大小写字母的转换，转换的方法是：在主菜单状态下，键人英文字母G可以选择大写字母状态，键人字母S可以选择小写字母状态。如果不设定字母的大小写状态，CWS系统会自动选择大写字母状态。

# 6．临时存盘

为防止由于外界原因（如断电、机器故障等）造成数据的丢失，在文稿的编辑过程中，最好养成经常进行临时存盘的习惯。具体做法是：

(1)按下 $\mathbf { \boldsymbol { C } } \mathbf { \boldsymbol { t } } \mathbf { \boldsymbol { r } } 1 + \mathbf { \boldsymbol { K } } + \mathbf { \boldsymbol { D } }$ 键；

(2)当计算机屏幕上显示“请插入工作盘于第一驱动器，按任一键"时，按照提示信息的

要求把字库盘从驱动器中取出，插入工作盘，再按下任一键。

计算机会自动把已输入的文稿存人磁盘，存储完毕后光标回到临时存盘操作前所在的位置

# 7.退出编辑

要从正在编辑的文稿中退出来，有两种方法：

(1)在编辑状态下，按下 $\mathrm { C t r l } + \mathbf { W }$ 键，按照屏幕提示把当前正在编辑或输入的文件存入磁盘，存储完毕后返回到CWS主菜单状态。

(2)如果文件不需要存盘，则在编辑状态下按下 $\mathbf { C _ { } } \mathbf { f r l } + \mathbf { K } + \mathbf { Q _ { } }$ 键，放弃当前正在编辑或输入的文件，返回CWS主菜单。

# 第三节字块操作

在上一节中，我们介绍了文稿的编辑命令。利用这些命令，可以对文稿中的文字进行修改和调整。用CWS软件编辑文稿时，还可以很方便地处理一些更复杂的问题，例如，把文稿中的某一部分内容向前或向后移动、把某一部分内容复制到文章的某个位置上、把一篇文章中的一部分内容复制到另一篇文章中去、把文章中的某一部分内容删除，等等。处理这类问题时，需要使用CWS 软件的字块操作功能。顾名思义，字块就是文稿中成块的文字。字块可以是一句话、可以是一段文字、也可以是整篇文章。字块由操作人员来确定，确定好之后就可以对它进行移动、复制、删除、存储、读取等操作。

习惯上把对字块进行的操作叫做块操作。块操作是应用字处理软件与手工文字处理的最显著区别之一。利用块操作功能，可以避免重复输人，简化编辑过程，极大地提高用计算机进行文字处理的效率。

# 一、定义字块

要进行块操作，首先要在文稿中定义字块，即告诉计算机，哪里是字块的开始，哪里是字块的结尾。定义字块时，先把光标移到要定义成字块的那些文字的第一个字上，然后按$\mathbf { C } \mathbf { t r l } + \mathbf { K } + \mathbf { B }$ 键。这时，在光标前会出现一个 ${ \bf { \bar { \tau } } } < { \bf { B } } >$ 。此后，再把光标移到要定义成字块的那些文字最后的那个字后面，按下 $\mathbf { C _ { [ r ] } } + \mathbf { K } + \mathbf { K }$ 键。这时，光标前会出现一个 ${ \bf { < K > } }$ 。这样就完成了定义字块的操作。屏幕上显示的 ${ \bf { \bar { \tau } } } < { \bf { B } } >$ 是字块的块首标志，简称首标； $\mathbf { < K > }$ 是字块的结尾标志，简称尾标。首标与尾标之间的文字就是字块。在CWS 软件中，每次只能定义一个字块。

# 二、块复制

用块复制操作可以把整块文字复制到从光标处开始的一片区域里。实际操作时，在定义好字块以后，把光标移到要复制字块内容的起始位置上，按下 $\mathbf { C } \mathbf { t r l + K + C }$ 键，定义好的字块就会被复制到当前光标处，光标所在处以及后面的文字会自动向后顺序移动，定义好的原始字块不变。

# 三、块删除

在编辑过程中，对于文稿中大段无用的文字，可以用块删除操作把它们删掉。实际操作时，按下 $\mathbf { C _ { [ r ] } } + \mathbf { K } + \mathbf { Y }$ 键，定义好的字块就会被从屏幕上删除，但字块的首尾标志还留在屏幕上，这时，再用Esc键把字块的首尾标从屏幕上删掉。

要把定义好的字块移到某个位置上，可以先把字块复制到指定的位置上，然后再用块删除操作删去原来的字块。

# 四、块写文件

编辑文稿时，可以用块写文件操作把字块中的文字存人磁盘备用。具体操作步骤如下。

$\textcircled{1}$ 把要存盘的文字定义成字块;

$\textcircled{2}$ 按下 $\mathbf { \Lambda } _ { \mathbf { C } \mathbf { t } \mathbf { r } 1 + \mathbf { K } + \mathbf { W } }$ 键；

$\textcircled{3}$ 计算机询问存人磁盘时给文件取个什么名。当屏幕上会显示出“文件名：_”时，输入一个新的文件名，并按回车键；

$\textcircled{4}$ 屏幕上又提示“请插入工作盘于第一驱动器，按任一键”时，按照提示信息的要求把字库盘从驱动器中取出，插人工作盘，按下任一键。

这时，计算机会把字块中的文字存入磁盘中指定的文件中。

# 五、取消块定义

CWS 软件中每次只能定义一个字块。要定义新字块就必须先使旧字块的定义无效。在这种情况下，需要清除旧字块的首、尾标。具体做法是：

$\textcircled{1}$ 把光标分别移到首标和尾标的右侧，用 Esc 键删除屏幕上的字块的首标<B>和尾标$\mathbf { < K > }$ ；

$\textcircled{2}$ 按 $\mathbf { C _ { [ r ] } } + \mathbf { K } + \mathbf { H }$ 键。这时，字块的首尾标会从内存中消失，原字块定义被取消了，可以再定义新的字块。

# 六、读取文件

编辑文稿时，还可以利用读文件操作，从磁盘上读入一个文件，把这个文件的内容加到文稿中光标指定的位置上。操作步骤如下。

$\textcircled{1}$ 把光标移到要添加内容的起始位置;

$\textcircled{2}$ 按下 $\mathbf { C _ { i r l } + K + R _ { \alpha } }$ 键；

$\textcircled{3}$ 屏幕上显示“请插入工作盘于第一驱动器，按任一键”时，依提示插入工作盘，按下任一键;

$\textcircled{4}$ 屏幕上显示“文件名：”时，输入要读入的文件名。按回车键后，计算机会把该文件读入

内存并在光标处显示出来，然后就可以继续进行编辑。

# 第四节 文件管理

在主菜单状态下，可以用相应的命令对软磁盘上的文件进行管理。CWS软件的文件管理功能有目录查询、删除文件、更改文件名等。

# 1．目录查询

在主菜单状态下，键入英文字母C,计算机会把1号驱动器中磁盘上的文件目录显示在屏幕上。目录显示完毕后，又返回主菜单。

# 2．删除文件

在主菜单状态下，键人英文字母Y可以删除磁盘上的文件。操作步骤如下。

$\textcircled{1}$ 在主菜单状态下，键人英文字母Y;  
$\textcircled{2}$ 计算机询问要删除的文件名，输入文件名并按下回车键;  
$\textcircled{3}$ 计算机又询问“正确吗？ $( \mathbf { Y } \mathbf { \alpha } / \mathbf { \beta } \mathbf { N } ) \mathbf { \beta } ^ { \prime }$ ，用 $\mathbf { Y }$ 键确认后，计算机会把指定的文件从磁盘中删除。

注意：文件一旦被删除，就很难恢复。因此，在执行删除命令前，一定要确认是否真的要删除该文件。

# 3.更改文件名

在主莱单状态下，键入英文字母E可以把旧文件名换成新文件名。操作步骤如下。

$\textcircled{1}$ 在主菜单状态下，键入英文字母E;  
$\textcircled{2}$ 计算机询问“原文件名？”时，输入原文件名;  
$\textcircled{3}$ 计算机又询问“新文件名？”时，键入新的文件名并按回车键;  
$\textcircled{4}$ 屏幕上显示“请插入工作盘于第一驱动器，按任一键”时，插入工作盘，按下任一键。这时，计算机会把磁盘上的旧文件名更换为新文件名。  
注意：改名的操作只更改文件名，不改变文件的类型和内容。

# 第五节其他命令

# 一、参数设置

参数设置是CWS 软件为满足编辑及打印文件时不同的需要而设计的一项功能，在打印文件

之前，最好先进行参数设置。

在主菜单状态下，键入英文字母L可以设置5个参数。

当屏幕上显示出

字库盘在？号驱动器（1-2)_

时，若所用的计算机只联接了一个软盘驱动器，则输入1；若有两个驱动器可以输入2并且把系统盘和字库盘分别放入一号和二号驱动器。这项参数对打印和编辑都有效。配置两个驱动器，可以提高输入数据时读取字库的速度，并可以减少调换磁盘盘片的次数。

当屏幕上显示出

打印时：每一行字符数？(1-255)_

时，计算机询问打印时每行的字符个数。由于一个汉字要占两个英文字符位，所以每行的汉字字数乘以2才是每行的字符数。开机时，系统自动设定的参数为60。可根据需要重新设置这个参数，选择的范围在1至255之间。

屏幕上显示出

左边空格数？（ $<$ 行宽）_

时，可以输入一个最小为0、并且不超过一行宽度的数。

屏幕上显示出

行距(8-63)？_

时，可以用8到63之间的数来控制两行文字之间的距离。其中，8表示两行之间无间隔，计算间隔的公式是 $\left( \mathbf { N } - 8 \right) / 7 2$ (英寸)。开机时，系统自动设置行距为12。

屏幕上显示出

字型(1-16)？_

时，可以选择16种不同的字型。系统自动设置的字型代码为5。

# 二、文件打印

在主菜单状态下，键人英文字母P,可以把用CWS编好并存放在磁盘上的文件打印在纸上。具体操作方法是：

$\textcircled{1}$ 接通打印机的电源开关，把打印纸上好;  
$\textcircled{2}$ 在主菜单状态下，键入英文字母 P;  
$\textcircled{3}$ 屏幕显示“文件名：_”时，输入要打印的文件名;

$\textcircled{4}$ 按屏幕提示要求插入相应的磁盘，按任一键。这时，计算机会按照参数设置的要求打印文件。打印完成后，计算机会询问是否还要继续打印，用Y来回答，则继续打印，否则返回主菜单。
