### Q3: 算法的核心是什么？

**答**: 切比雪夫多项式插值

**位置计算**:
```
r(t) = Σ c_k * T_k(t_normalized)
```

**速度计算**:
```
v(t) = Σ c_k * dT_k/dt
```

其中 T_k 是切比雪夫多项式，通过递推关系高效计算。
