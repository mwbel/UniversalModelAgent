# 10.6联合摄动问题

# 1.问题的提出

前面几节都是将大气阻力摄动当作二阶小量来考虑的，但是面质比较大的卫星，在(或大部分时间在）离地面 $2 0 0 { - } 3 0 0 ~ \mathrm { k m }$ 高的近地空间飞行时，大气阻力有可能上升到与 $\pmb { J _ { 2 } }$ 项的影响相当.在此情况下，若仍需给出完整的一阶解，就要产生如下问题，即

(1）平均根数 $\pmb { \sigma }$ 中同时包含地球形状 $( J _ { 2 }$ 项）和大气阻力摄

动的一阶长期项，

（2）在求二阶长期项和一阶长周期项时，间接部分

$$
\sum _ { j } \frac { \partial f _ { 1 } } { \partial \sigma _ { j } } \big ( \sigma _ { s } ^ { ( 1 ) } \big ) _ { j }
$$

中的右函数 $f _ { 1 }$ 和 $\sigma _ { \mathfrak { s } } ^ { ( 1 ) }$ 同时包含 ${ { J } _ { 2 } }$ 和大气阻力两个部分。

上述两点表明,必须同时考虑地球形状和大气阻力两种摄动，不再像前面那样可以分开考虑，这就是联合摄动问题.

# 2．解决联合摄动问题的困难

从形式上看，要构造一阶解意义下的联合摄动解,在推导上有两点不同于引力摄动：一是大气阻力摄动的短周期项较复杂，见(10.73)和(10.74)式，即使展成 $M$ 的三角级数，一般取项也较多，另一点，由于大气阻力是耗散力，不能直接引用正则变化方法[16],[17]。但更重要的是,在大气阻力这种耗散因素影响下,卫星轨道半长径有长期变化，当大气阻力摄动与 $\pmb { J } _ { 2 }$ 项摄动为同阶小量（记作e）时，用一般方法构造的小参数幂级数解，在间隔△t～内也几乎是发散的[16]，这从平近总角 $M$ 的变化是很容易看出的.只有当大气阻力摄动量级低于 ${ { J } _ { 2 } }$ 项摄动时（例如为一阶半小量，即 $\varepsilon ^ { 3 / 2 } .$ )，可以构造一种满足一定精度要求的联合摄动解[18]．否则，要解决联合摄动问题，必须进一步简化大气密度模式，这种单纯为了数学上的需要而进行的简化，又往往使得到的结果丧失其实用价值.

事实上，根据已有的工作[16-18]可以看出，在大气与 $J _ { 2 }$ 项联合摄动情况下，卫星轨道的变化并已实质性改变。那么，从了解卫星轨道变化规律和精密定轨的角度来看，当上述情况发生时(即面质比较大的低轨卫星，或卫星在回收、陨落前的一段飞行过程)、就用数值方法直接给出相应的离散解。而不必作过多的近似去寻求分析解。当然，对这种情况，要特别注意沿迹误差的思想问题。

# 11.1 坐标系摄动问题的提出

# 4.两点说明

（1）从上述两种附加摄动的结果不难看出，早期为什么提出轨道坐标系，其原因已很清楚．在用分析方法建立卫星轨道摄动解时，显然采用轨道坐标要简单得多，尤其对于一般精度要求的一阶解，无需考虑岁差、章动和极移引起的附加摄动。

（2）在高精度要求下，往往是采用数值方法建立卫星的有摄星历表，在历元地心惯性系中，上述附加摄动是通过坐标转换解决的，即前面的(11.3)式．若采用轨道坐标系，是否会简单些？不妨作些具体分析．在此情况下，需解决两个问题，一是在以卫星位置矢量 $\pmb { r }$ 和速度矢量 $\dot { \pmb { r } }$ 作为基本变量建立的运动方程中，如何给出上述附加项;另一个问题是如何简单地将历元地心惯性系中的日、月位置转换到轨道坐标系中来。

# 第十二章 后牛顿效应

# \$12.1动力模型

牛顿引力理论是天体力学的基础，从天体力学诞生以来，人们一直用牛顿引力理论去研究天体的运动，前面各章也是在这种框架下讨论人造卫星运动的．本世纪六十年代以来，随着测量技术的发展,观测精度大大提高，以牛顿力学为基础的天体力学理论所达到的精度已不能满足需要，必须以新的引力理论对其加以修正.这里只是说修正，其原因是：到目前为止，在天体力学(或与其有关的领域)问题中所涉及的新引力理论的内容，主要是如何给出相应的引力场中，一个实验质点的运动微分方程，它虽然不同于牛顿引力场中的运动方程，但只是增加一些微小的修正项而已．在此情况下讨论天体力学问题，无论从定量角度还是从定性角度来看，研究方法都无实质性改变．因修正项较小，所以仅仅是在牛顿引力场对应的运动方程中增加一些摄动项，相当于增加一个摄动源.

尽管人卫是在地球附近运动，可以选取局部惯性参考系，但又与日、月运动有关，这就不可避免地涉及到地心参考系和日心参考系以及两者之间的“坐标”(时空坐标)转换问题．因此，讨论人卫运动时，有时不得不采用非惯性地心参考系(这里所说的非惯性是指非牛顿引力效应引起的)，相应的运动方程中的修正项将包含地球史瓦西(Schwarzschild)场的广义相对论效应，地球自转和扁率的广义相对论效应，以及测地岁差等[1-3]．但就修正项的大小而言，最主要的是作为一体广义相对论效应的地球史瓦西场修正项，通常把这些修正项称为后牛顿修正项.

# 第十三章 中间轨道论

从前面几章对各种摄动的研究可以看出，椭圆轨道只是卫星真实轨道的一个粗略的近似，对于精度要求较高的人卫工作，需要在椭圆轨道的基础上求出极其复杂的摄动项．因此，和天体力学中对自然天体的研究一样，为了解除分析方法面临的困境，人们总想以更接近真实运动的某种轨道来代替椭圆轨道，这种轨道常被称为中间轨道(事实上，椭圆轨道也可以说是一种最简单的中间轨道).在人卫工作中，对中间轨道的要求基本上有两条：一是中间轨道解要尽量接近卫星的真实轨道，即相应的力学系统除考虑地球的中心引力外，还应包含主要摄动力；二是尽量与椭圆轨道一样，使相应的中间轨道解具有封闭形式（避免级数展开形式引起一系列的麻烦),而且要简单，便于进一步求剩余摄动.

中间轨道仍是一种近似，它是将卫星绕地球运动同时受各种因素影响的复杂力学系统简化成一个近似的可积系统，根据各种摄动力的相对大小来看，在简化系统中，除考虑地球中心引力外，要尽量反映地球带谐项的影响，特别是 $J _ { 2 } , J _ { 3 } , J _ { 4 }$ 三项 $( J _ { 2 }$ 项尤其重要)的影响。已经给出的一些中间轨道，就是这样考虑的，在不同程度上反映了地球为旋转对称体(即考虑了带谐项)这一明显特征．最好的一类中间轨道包含了全部 ${ \pmb J } _ { 2 }$ 和 ${ \pmb J } _ { 3 }$ 项以及部分 ${ \pmb J } _ { 4 }$ 项的影响，本章将对此作详细介绍。

# 13.1非球形地球引力场中卫星运动的积分问题

根据天体力学的一些基本结论可知,到目前为止,只有二体问题和某些特殊的三体问题可以彻底解决。在卫星绕地球运动中，即使把地球看成质点,考虑日、月的影响时，将构成一个限制性四体问题，无法解决;若考虑大气、光压的影响，作用力的形式更复杂．如果只考虑地球的引力作用，而且将地球处理成旋转对称体，情况又如何？这相当于一个特殊的限制性三体问题，两有限天体中一个即对应球形引力场(中心引力部分)，另一个就是非球形附加部分(带谐项摄动部分)．这又类似于两个固定中心问题．亦称为Fatou 问题[],尽管这个问题比较特殊，也只有个别情况可以获得其严密解，下面具体讨论.
