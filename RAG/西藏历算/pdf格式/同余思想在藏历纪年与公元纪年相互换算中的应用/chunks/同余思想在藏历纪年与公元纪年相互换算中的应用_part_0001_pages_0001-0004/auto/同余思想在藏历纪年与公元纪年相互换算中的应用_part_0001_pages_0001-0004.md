藏历纪年是藏族天文历算的重要组成部分，具有悠久的历史、丰富的文献资料和长期的实践经验，有着鲜明的民族特色。古籍研究中，往往需要将藏历纪年与公元纪年进行换算。本文通过探索藏历纪年与农历干支纪年中的组合对应关系和循环规律，发现藏历纪年、农历干支纪年与公元纪年之间存在一定的内在规律性。藏历纪年和干支纪年都是以60年为一个循环周期，因此，将公元纪年数除以60的余数把所有公元纪年数分成60个同余类，制成同余类换算表。再通过查同余换算表，就能速算公历任何年份的藏历纪年，也能推算某藏历年出现在公历纪年中的规律。

# 一、藏历纪年与农历干支纪年的对应关系

藏历纪年类似于我国农历干支纪年。农历纪年是我国古代采用的一种干支纪年法，天干地支简称为“干支”。它们是一个互相依存、互相配合的整体。天干有10个字，依次是：甲、乙、丙、丁、戊、已、庚、辛、壬、癸。地支有12个字，依次是：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。用天干地支计算年,就是把每一个天干和地支按照一定的顺序，不重复地搭配起来，用来作为纪年的代号。把天干中的一个字摆在前面，地支中的一个字摆在后面，就构成了一对干支。按照这样的顺序，每年用一对干支表示,60 年一循环,这就是干支纪年法。

藏历纪年也一样,它采用五行：木、火、土、铁、水与十二生肖鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪组合而成。其搭配方法是：“五行"在前且每“行”用两次，“地支”在后且阴阳交错,第一个“行”木配第一个“地支”鼠为木鼠年,再用第一个“行”木配第二个“地支”牛为木牛年,第二个“行”火配第三个“地支”虎为火虎年,依次类推。当"行”用完后，再从第一个“行”开始配第十一个“地支"为木狗年,当十二个“地支”用完后,再从第一个“地支"开始为火鼠年。这样,两两相配,正好配出60种,其排列顺序见表1：

上面排列中奇数位为“阳”,偶数位为“阴”。每年用一对五行与生肖表示,60 年一循环,它以12 年为一小循环,60 年为一大循环,称为一个“饶迥”,第一个“饶迥”是从1027年(宋仁宗天圣五年)丁卯年开始的，这便是延续使用至今的藏历纪年法。

可以看出,藏历纪年的组合配对方式及循环次序与干支纪年相同,记住对应循环的序数是换算的基础。下面着重介绍藏历纪年与公元纪年的同余类换算法,同余换算法也同样适用于干支纪年与公元纪年的互相换算。

# 二、公历纪年法与藏历纪年的同余类换算法

# (一)公元年余数换算法

我们在历史研究中，有时要把公元纪年换算成藏历纪年,有时要查出藏历纪年的公元纪年,传统的换算方法较多，利用同余表能非常便捷地查出公元任何年份的藏历纪年，也能推算某藏历年出现在公元纪年中的规律,计算很简单。公元纪年法与藏历纪年法换算中最重要的年份是公元1年，因为这是公元纪年的起点。由于已知条件是公元年份，因此公元1年可以作为计算的起点。于是,可按传统方法查出公元1年是铁鸡年,对应干支辛酉年,公元4年是甲子年，所以,公元年数要比藏历年的顺序数大3。那么，表1中的排列方式就不符合要求了，需要做一下变换。然后再列一张重新排列的五行生肖年份循环表,即公元纪年和藏历纪年同余换算表(见表2）。

由于藏历纪年与干支纪年都是以60年为一个循环周期，也就是说藏历纪年与干支纪年共有60种。将公元年份除以60得到 $0 , 1 , 2 , 3 , \cdots 5 9$ ,这60 个余数,每一个余数对应一组公元年数，叫做一个同余类,即60个不同的藏历年所对应的公元年数的集合。由于已把公元1年设为1,于是就可以直接用公元年份N除以60求得余数n,然后，查表2中n对应的行列就是所要求的藏历年的五行及生肖了。换算的数学模型是 $\mathbf { N } = \mathbf { K } \cdot 6 0 + \mathbf { n }$ ,其中 $\mathbf { N }$ 为公历纪年数， $\mathbf { k }$ 为从零开始的正整数商， $\mathbf { n }$ 为余数。用余数在表中查出的五行及生肖即为公元N年的藏历纪年。当N小于60时， 按N的数值直接查表。

# (三)藏历纪年推算公元纪年

在算式中 $\mathbf { N } = \mathbf { k } \cdot 6 0 + \mathbf { n }$ 中,N 为公元纪年； $\mathbf { k }$ 为从零开始的正整数; $\mathbf { n }$ 为某藏历年在同余类数表中行列交叉处所对应的数。取定$\mathbf { n }$ ，让 $\mathbf { k } = 0 , 1 , 2 ,$ …等,可算得 $\mathbf { n }$ 对应的藏历年在公元年中的纪年数。例如,求藏历水马年在公元纪年中出现的规律,查表：水、马在表2中所交叉对应的余数为22，根据公式， $\mathbf { k }$ 取从零开始的任意正整数，可得 $6 0 \times 0 + 2 2$ $= 2 2$ ； $6 0 \times 1 + 2 2 = 8 2$ ； $6 0 \times 2 + 2 2 = 1 4 2$ ；$6 0 \times 3 + 2 2 = 2 0 2 \cdots$ 等等。藏历水马年出现在公元纪年中的规律是公元22年、公元82年、公元142、公元202年等每隔60 年的公历纪年中。由于藏历年实际只有60个，而公元年却有无穷多个。因此，由公元年推算藏历纪年往往只有一个结果，而由藏历纪年推算公元年却有多种结果，至于究竟是它们中的哪一年,有时要根据相关的提示来确定。例如,要知道藏历第16饶迥水马年是公元多少年，我们可先算出藏历第16饶迥在公元年中的大致范围，即 $6 0 \times 1 6 + 1 0 2 6$ $= 1 9 8 6$ （第16饶迥下限）， $1 9 8 6 \mathrm { ~ - ~ } 6 0 \mathrm { ~ = ~ }$ 1926（第16饶迥上限），在这个年限范围中 $6 0 \times 3 2 + 2 2 = 1 9 4 2$ ,因此正整数 $\mathbf { k }$ 只能取确定的一个 $\mathbf { k } = 3 2$ 。这样,有了“第16饶迥"这个重要提示，就可确定藏历第16饶迥水马年是公元1942年。当然，还可以根据藏历纪年的其他提示信息解决换算中的更多的问题。
