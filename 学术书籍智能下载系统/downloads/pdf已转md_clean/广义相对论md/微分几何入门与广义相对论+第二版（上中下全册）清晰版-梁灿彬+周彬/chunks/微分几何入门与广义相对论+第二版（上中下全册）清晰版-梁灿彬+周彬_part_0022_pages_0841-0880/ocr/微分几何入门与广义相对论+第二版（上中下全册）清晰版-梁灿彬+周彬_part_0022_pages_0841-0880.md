# 15.2.1 有限自由度正规系统的哈氏理论

拉氏理论的基本变量是广义坐标 $q ^ { i }$ 和广义速度 $\dot { q } ^ { i }$ ,它们的一组值代表系统的一个状态.哈氏理论的基本变量则是广义坐标(位形变量) $q ^ { i }$ 和广义动量(动量变量 $p _ { i }$ .给定拉氏函数 $L ( q ^ { i } , \dot { q } ^ { i } )$ 后,动量变量由下式定义：

$$
p _ { i } : = \frac { \partial L ( q , \dot { q } ) } { \partial \dot { q } ^ { i } } , ~ i = 1 , \cdots , N .
$$

位形变量和动量变量称为互相共轭的一对正则变量(canonical variables）,它们的一组值 $( q ^ { i } , p _ { i } )$ 代表系统的一个状态.量 $p _ { i } \dot { q } ^ { i } - L ( q , \dot { q } )$ 在哈氏理论中有重要地位,其微分为

$$
\mathrm { d } [ p _ { i } \dot { q } ^ { i } - L ( q , \dot { q } ) ] = \dot { q } ^ { i } \mathrm { d } p _ { i } + p _ { i } \mathrm { d } \dot { q } ^ { i } - \frac { \partial L } { \partial q ^ { i } } \mathrm { d } q ^ { i } - \frac { \partial L } { \partial \dot { q } ^ { i } } \mathrm { d } \dot { q } ^ { i } = \dot { q } ^ { i } \mathrm { d } p _ { i } - \frac { \partial L } { \partial q ^ { i } } \mathrm { d } q ^ { i } .
$$

上式右边不含 $\mathrm { d } \dot { q } ^ { i }$ ,这意味着 $p _ { i } \dot { q } ^ { i } - L ( q , \dot { q } )$ 可用 $q ^ { i }$ 和 $p _ { i }$ 表出.量 $p _ { i } \dot { q } ^ { i } - L ( q , \dot { q } )$ 表为$q ^ { i } , p _ { i }$ 的函数后称为系统的哈氏量(Hamiltonian），记作 $H ( q , p )$ ，即

$$
H ( q , p ) : = p _ { i } \dot { q } ^ { i } - L ( q , \dot { q } ) .
$$

以下的讨论取决于 $N$ 个 $\dot { q } ^ { i }$ 可否从式(15-2-1)中被反解出,而这又取决于$L ( q ^ { i } , \dot { q } ^ { i } )$ 的函数形式.原始的哈氏理论假设全部 $\dot { q } ^ { i }$ 都可反解出,即每个 $\dot { \boldsymbol { q } } ^ { i }$ 都可表为$q ^ { j }$ 和 $p _ { j }$ 的函数：

$$
\dot { q } ^ { i } = \dot { q } ^ { i } ( q , p ) , \qquad i = 1 , \cdots , N .
$$

在现代理论中,满足这一要求的拉氏函数 $L ( q ^ { i } , \dot { q } ^ { i } )$ 称为正规(regular)的,相应的哈氏理论称为有正规拉氏量的哈氏理论.这时式(15-2-3)可以表为

$$
H ( q , p ) = p _ { i } \dot { q } ^ { i } ( q , p ) - L ( q , \dot { q } ( q , p ) ) ,
$$

故

$$
\frac { \partial H } { \partial p _ { i } } = \frac { \partial p _ { j } } { \partial p _ { i } } \dot { q } ^ { j } + p _ { j } \frac { \partial \dot { q } ^ { j } } { \partial p _ { i } } - \frac { \partial L } { \partial \dot { q } ^ { j } } \frac { \partial \dot { q } ^ { j } } { \partial p _ { i } } = \delta ^ { i } { } _ { j } \dot { q } ^ { j } = \dot { q } ^ { i } ,
$$

$$
\frac { \partial H } { \partial q ^ { i } } = p _ { j } \frac { \partial \dot { q } ^ { j } } { \partial q ^ { i } } - \frac { \partial L } { \partial q ^ { i } } - \frac { \partial L } { \partial \dot { q } ^ { j } } \frac { \partial \dot { q } ^ { j } } { \partial q ^ { i } } = - \frac { \partial L } { \partial q ^ { i } } ,
$$

其中第一式的第二步用到式(15-2-1).利用拉氏方程(15-1-5)可把上式的第二式改写为 $\partial H / \partial q ^ { i } = - \dot { p } _ { i }$ ，因此有

$$
\dot { q } ^ { i } = \frac { \partial H } { \partial p _ { i } } , \dot { p } _ { i } = - \frac { \partial H } { \partial q ^ { i } } , i = 1 , \cdots , N .
$$

请注意各式中偏导数的含义.例如，δH/aq1 是指函数H(q,,qN；pi,,PN)对 $q ^ { 1 }$ 的偏导数,求导时保持 $\boldsymbol q ^ { 2 } , \cdots , \boldsymbol q ^ { N }$ 和 $\boldsymbol { p } _ { 1 } , \cdots , \boldsymbol { p } _ { N }$ 不变；而 $\partial L / \partial q ^ { 1 }$ 则是指函数$L ( q ^ { 1 } , \cdots , q ^ { N } ; \dot { q } ^ { 1 } , \cdots , \dot { q } ^ { N } )$ 对 $q ^ { 1 }$ 的偏导数,求导时保持 $\boldsymbol q ^ { 2 } , \cdots , \boldsymbol q ^ { N }$ 和 $\dot { q } ^ { 1 } , \cdots , \dot { q } ^ { N }$ 不变.

\phi _ { m } \left( q , p \right) = 0 , m = 1 , \cdots , M \left( \equiv N - Z \right) ,

其中 $\phi _ { m } ( q , p )$ 是 $q$ 和 $p$ 的某种函数.这意味着 $2 N$ 个 $q$ 和 $p$ 受到 $M$ 个方程的限制，因此说系统存在约束(constraint). $\phi _ { m } ( q , p )$ 称为约束函数， $\phi _ { m } ( q , p ) = 0$ 称为约束方程(简称约束).读者熟悉的是用方程 $f ( q ^ { 1 } , \cdots , q ^ { N } ) = 0$ 或 $f ( q ^ { 1 } , \cdots , q ^ { N } , t ) = 0$ 对位形变量 $\boldsymbol q ^ { 1 } , \cdots , \boldsymbol q ^ { N }$ 加以限制的约束(不涉及速度 $\dot { \boldsymbol { q } } ^ { i }$ 或动量 $p _ { i }$ ），称为完整约束（holonomicconstraint）,例如，牛顿力学中被限制于某曲面上的质点所受的约束以及能把质点结合为刚体的约束,其特点是(各)质点的笛卡儿坐标要满足若干方程,可通过选择适当的(数目较少的)非笛卡儿坐标(广义坐标)来消除约束.Dirac 等人从20 世纪40年代开始为研究引力量子化而提出一套涉及动量 $p _ { i }$ 的非完整约束(动力学约束)理论[后称Dirac-Bergmann理论,见Dirac(1964)],理论中出现约束的前提是拉氏函数的形式使 $\dot { q } ^ { i }$ 不能全部被“反解出”(奇异拉氏函数).①本书只讨论这种约束.有约束时的哈氏量 $H$ 仍由式(15-2-3)定义，而且，由于式(15-2-2)仍成立， $H$ 仍只是 $q$ 和 $p$ 的函数(重要的是不含任一 $\dot { \pmb q }$ ).②

约束的几何意义是系统在相空间 $\boldsymbol { \Gamma }$ 中的代表点必须位于由约束方程(15-2-11)决定的约束面(constraint surface） ${ \cal T } _ { 1 } \subset { \cal T }$ 上.如果只有一个独立约束 $\phi ( q , p ) = 0$ ，则约束面 $T _ { 1 }$ 是 $\boldsymbol { \Gamma }$ 中的 $2 N - 1$ 维子流形(超曲面).如果有 $M$ 个独立约束, $\varGamma _ { 1 }$ 便是 $M$ 个超曲面之交,是 $2 N - M$ 维子流形.系统状态的代表点必须落在 $\varGamma _ { 1 }$ 上,描述演化的曲线 $\gamma ( t )$ 只能躺在 $T _ { 1 }$ 上,它服从的演化方程不再像式(15-2-6)这么简单.由于 $H$ 表达式(15-2-3)中的 $\dot { q } ^ { i }$ 不再能表为 $\dot { \boldsymbol { q } } ^ { i } ( \boldsymbol { q } , \boldsymbol { p } )$ ,式(15-2-6)的推导不再成立.第二种推导,即从式(15-2-2')和(15-2-7)直接读出式(15-2-6)的做法也由于 $( \mathrm { d } q ^ { i } , \mathrm { d } p _ { i } )$ 不完全独立而失效:由式(15-2-2')和(15-2-7)只能得到

$$
\left( \dot { q } ^ { i } - \frac { \partial H } { \partial p _ { i } } \right) { \mathrm d } p _ { i } - \left( \dot { p } _ { i } + \frac { \partial H } { \partial q ^ { i } } \right) { \mathrm d } q ^ { i } = 0 ,
$$

而不能进一步宣称两个括号内的表达式分别为零,因为状态代表点必须落在 $\varGamma _ { 1 }$ 上，而 $\varGamma _ { 1 }$ 的独立坐标[因而 $( \mathsf { d } q ^ { i } , \mathsf { d } p _ { i } )$ 中的独立微分]只有 $2 N - M$ 个.为了克服这一困难,可改而采用 $\varGamma _ { 1 }$ 的独立坐标.先讨论只有一个约束 $\phi ( q , p ) = 0$ 的情况,这时约束面 $\varGamma _ { 1 }$ 是 $2 N - 1$ 维超曲面.设 $\{ r ^ { \alpha } ( \alpha = 1 , \cdots , 2 N - 1 ) \}$ 是 $T _ { 1 }$ 上的一个坐标系,则 $\varGamma _ { 1 }$ 上的点的 $q ^ { i }$ 和 $p _ { i }$ 都可看作 $r ^ { \alpha }$ 的函数,由式(15-2-12)可知对 $\gamma ( t )$ 上任一点有

$$
\Bigg [ \Bigg ( \dot { q } ^ { i } - \frac { \partial H } { \partial p _ { i } } \Bigg ) \frac { \partial p _ { i } } { \partial r ^ { \alpha } } - \Bigg ( \dot { p } _ { i } + \frac { \partial H } { \partial q ^ { i } } \Bigg ) \frac { \partial q ^ { i } } { \partial r ^ { \alpha } } \Bigg ] \mathbf { d } r ^ { \alpha } = 0 .
$$

$r ^ { \alpha }$ 是 $\varGamma _ { 1 }$ 上的独立坐标保证 $2 N - 1$ 个 $\mathrm { d } \boldsymbol { r } ^ { \alpha }$ 互相独立,因而上式给出 $2 N - 1$ 个方程：

$$
\left( \dot { q } ^ { i } - \frac { \partial H } { \partial p _ { i } } \right) \frac { \partial p _ { i } } { \partial r ^ { \alpha } } - \left( \dot { p } _ { i } + \frac { \partial H } { \partial q ^ { i } } \right) \frac { \partial q ^ { i } } { \partial r ^ { \alpha } } = 0 , \alpha = 1 , \cdots , 2 N - 1 .
$$

用大写拉丁字母代表 $\boldsymbol { \Gamma }$ 上张量场的抽象指标. $\varGamma _ { 1 }$ 上任一点的坐标基矢 $( \partial / \partial r ^ { \alpha } ) ^ { A }$ 可用坐标系 $\{ q ^ { i } , p _ { i } \}$ 的坐标基矢 $( \partial / \partial q ^ { i } ) ^ { A }$ 和 $( \partial / \partial p _ { i } ) ^ { A }$ 表出:

$$
\left( \frac { \partial } { \partial r ^ { \alpha } } \right) ^ { A } = \frac { \partial q ^ { i } } { \partial r ^ { \alpha } } \Bigg ( \frac { \partial } { \partial q ^ { i } } \Bigg ) ^ { A } + \frac { \partial p _ { i } } { \partial r ^ { \alpha } } \Bigg ( \frac { \partial } { \partial p _ { i } } \Bigg ) ^ { A } .
$$

（对 $i$ 从1至 $N$ 求和; $\alpha = 1 , \cdots , 2 N - 1 . )$ 定义 $\gamma ( t ) \subset T _ { 1 }$ 上的对偶矢量场

$$
\xi _ { \cal A } : = - \Biggl ( { \dot { p } } _ { i } + \frac { \partial H } { \partial q ^ { i } } \Biggr ) ( { \mathrm d } q ^ { i } ) _ { \cal A } + \Biggl ( { \dot { q } } ^ { i } - \frac { \partial H } { \partial p _ { i } } \Biggr ) ( { \mathrm d } p _ { i } ) _ { \cal A } ,
$$

则由式 $( 1 5 - 2 \ – 1 2 ^ { \prime \prime } )$ 和(15-2-13)可知对 $\gamma ( t )$ 的任一点有

$$
\xi _ { \cal A } \left( \frac { \partial } { \partial r ^ { \alpha } } \right) ^ { A } = 0 , \qquad \alpha = 1 , \cdots , 2 N - 1 .
$$

$\{ ( \partial / \partial r ^ { \alpha } ) ^ { A } \}$ 是超曲面 $\varGamma _ { 1 }$ 上的坐标基底场,所以 $\xi _ { A }$ 是定义在 $\gamma ( t )$ 上的一个法于 $\varGamma _ { 1 }$ 的余矢场(对偶矢量场).另一方面，Γ由约束方程Φ(q,p)=0定义又表明VAΦ是Γi上的法余矢场,因而 $\gamma ( t )$ 上必有实函数 $\lambda ( t )$ 使

$$
\xi _ { \mathcal { A } } = \lambda \nabla _ { \mathcal { A } } \phi = \lambda \frac { \partial \phi } { \partial q ^ { i } } ( \mathrm { d } q ^ { i } ) _ { \mathcal { A } } + \lambda \frac { \partial \phi } { \partial p _ { i } } ( \mathrm { d } p _ { i } ) _ { \mathcal { A } } .
$$

与式(15-2-14)对比得

$$
\dot { q } ^ { i } = \frac { \partial H } { \partial p _ { i } } + \lambda \frac { \partial \phi } { \partial p _ { i } } , \dot { p } _ { i } = - \frac { \partial H } { \partial q ^ { i } } - \lambda \frac { \partial \phi } { \partial q ^ { i } } , i = 1 , \cdots , N .
$$

这就是有一个约束时的哈氏正则方程.以上手法不难推广到有M个约束的情况,不同之处只在于约束面 $\varGamma _ { 1 }$ 变为 $\boldsymbol { \Gamma }$ 的一个 $2 N - M$ 维子流形.这时 $\varGamma _ { 1 }$ 上每点有 $M$ 个线性独立的法余矢 $\nabla _ { A } \phi _ { 1 } , \cdots , \nabla _ { A } \phi _ { M }$ ，而 $\xi _ { A }$ 作为曲线 $\gamma ( t ) \subset T _ { 1 }$ 上与 $\varGamma _ { 1 }$ 正交的余矢场,[见式(15-2-15),只需把 $\alpha = 1 , \cdots , 2 N - 1$ 改为 $\alpha { = } 1 , \cdots , 2 N { - } M . 1$ 在线上每点总可由 $M$ 个 $\nabla _ { A } \phi _ { m }$ 线性表出：

$$
\boldsymbol { \xi } _ { \scriptscriptstyle A } = \lambda ^ { m } \nabla _ { \scriptscriptstyle A } \phi _ { { m } } = \lambda ^ { m } \frac { \partial \phi _ { { m } } } { \partial \boldsymbol { q } ^ { i } } ( \mathrm { d } \boldsymbol { q } ^ { i } ) _ { \scriptscriptstyle A } + \lambda ^ { m } \frac { \partial \phi _ { { m } } } { \partial p _ { i } } ( \mathrm { d } p _ { i } ) _ { \scriptscriptstyle A } .
$$

[对 $i$ 从1至 $N$ 求和,对 $m$ 从1至 $M$ 求和,下同.]于是哈氏正则方程(15-2-17)现在成为

$$
\dot { q } ^ { i } = \frac { \hat { \partial } H } { \hat { \partial } p _ { i } } + \lambda ^ { m } ( t ) \frac { \hat { \partial } \phi _ { m } } { \hat { \partial } p _ { i } } , \mathrm { ~ } \dot { p } _ { i } = - \frac { \hat { \partial } H } { \hat { \partial } q ^ { i } } - \lambda ^ { m } ( t ) \frac { \hat { \partial } \phi _ { m } } { \hat { \partial } q ^ { i } } , \mathrm { ~ } i = 1 , \cdots , N ,
$$

其中a"(t)是M个待定函数,称为拉氏待定乘子(Lagrange undetermined multiplier),与 $2 N$ 个 $q ^ { i } ( t ) , p _ { i } ( t )$ 一起合共 $2 N + M$ 个待定函数，原则上可由 $2 N$ 个演化方程(15-2-19)和 $M$ 个约束方程(15-2-11)联立解出.下一小节讨论与解有关的一系列问题

# 15.2.3 初级约束和次级约束

$\boldsymbol { \Gamma }$ 上的标量场 $f : \Gamma \to \mathbb { R }$ 可看作 $q ^ { i } , p _ { i }$ 的函数.设 $\gamma ( t )$ 是 $\boldsymbol { \Gamma }$ 中的一条演化曲线,则 $f$ 通过 $\gamma ( t )$ 诱导出一元函数 $f ( t ) = f ( q ^ { i } ( t ) , p _ { i } ( t ) )$ ,其时间导数为

$$
\dot { f } \equiv \frac { \mathrm { d } f } { \mathrm { d } t } = \frac { \partial f } { \partial q ^ { i } } \dot { q } ^ { i } + \frac { \partial f } { \partial p _ { i } } \dot { p } _ { i } = \frac { \partial f } { \partial q ^ { i } } \left( \frac { \partial H } { \partial p _ { i } } + \lambda ^ { m } \frac { \partial \phi _ { m } } { \partial p _ { i } } \right) - \frac { \partial f } { \partial p _ { i } } \left( \frac { \partial H } { \partial q ^ { i } } + \lambda ^ { m } \frac { \partial \phi _ { m } } { \partial q ^ { i } } \right) ,
$$

其中第三步用到式(15-2-19).为便于简洁地表达 $\dot { \boldsymbol { f } }$ ,可定义泊松括号. $\boldsymbol { \Gamma }$ 上两个函数 $f ( q , p )$ 和 $g ( q , p )$ 的泊松括号(Poisson bracket) $\left\{ f , g \right\}$ 定义为

$$
\{ f , g \} : = \frac { \partial f } { \partial q ^ { i } } \frac { \partial g } { \partial p _ { i } } - \frac { \partial f } { \partial p _ { i } } \frac { \partial g } { \partial q ^ { i } } .
$$

请注意 $\boldsymbol { \Gamma }$ 上任意两个函数的泊松括号仍是 $\boldsymbol { \Gamma }$ 上的函数.不难证明泊松括号有以下性质：

(a)反称性 $\{ f , g \} = - \dot { \{ g , f \} }$ ；

(b)对每个槽都有线性性,例如

$$
\{ f _ { 1 } + f _ { 2 } , g \} = \{ f _ { 1 } , g \} + \{ f _ { 2 } , g \} \ ;
$$

(c)服从莱布尼兹律,例如

$$
\{ f _ { 1 } f _ { 2 } , g \} = f _ { 1 } \{ f _ { 2 } , g \} + f _ { 2 } \{ f _ { 1 } , g \} ;
$$

(d)满足雅可比恒等式,即

$$
\{ f , \{ g , h \} \} + \{ h , \{ f , g \} \} + \{ g , \{ h , f \} \} = 0 .
$$

利用泊松括号可把式(15-2-20)简洁地表为

$$
\dot { f } = \{ f , H \} + \{ f , \phi _ { m } \} \lambda ^ { m } . ^ { \mathrm { ~ \textregistered ~ } }
$$

请注意 $\phi _ { m }$ 在 $\varGamma _ { 1 }$ 上为零不导致 $\{ f , \phi _ { m } \}$ 在 $\varGamma _ { 1 }$ 上为零,因为 $\{ f , \phi _ { m } \}$ 涉及导数 $\partial \phi _ { m } / \partial p _ { i }$

和 $\partial \phi _ { m } / \partial q ^ { i }$ .为了防止以为 $\phi _ { m } \left( q , p \right) = 0$ 会导致 $\{ f , \phi _ { m } \} = 0$ 的这种可能误解，仿照Dirac(1964),我们把 $\phi _ { m } ( q , p ) = 0$ 改写为 $\phi _ { m } ( q , p ) \approx 0$ （“弱等于”号 $\approx$ 代表“在约束面上等于”），①并强调只有算出泊松括号后才能用约束方程 $\phi _ { m } ( { \boldsymbol { q } } , { \boldsymbol { p } } ) \approx 0$ .

式(15-2-23)是 $\boldsymbol { \Gamma }$ 上任一函数 $f : T \to \mathbb { R }$ 同演化曲线 $\gamma ( t )$ 结合而得的一元函数 $f ( t )$ 的时间导数表达式.令 $f$ 分别为 $q ^ { i }$ 和 $p _ { i }$ ,则式(15-2-23)给出

$$
\dot { q } ^ { i } = \{ q ^ { i } , H \} + \{ q ^ { i } , \phi _ { m } \} \lambda ^ { m } , \dot { p } _ { i } = \{ p _ { i } , H \} + \{ p _ { i } , \phi _ { m } \} \lambda ^ { m } , i = 1 , \cdots , N .
$$

这无非是约束系统正则方程(15-2-19)用泊松括号的重新表述.再令 $f$ 为约束函数$\phi _ { m } ( q , p )$ ，则式(15-2-23)给出

$$
\dot { \phi } _ { m } = \{ \phi _ { m } , H \} + \{ \phi _ { m } , \phi _ { n } \} \lambda ^ { n } , \qquad m = 1 , \cdots , M .
$$

演化的自洽性要求演化曲线 $\gamma ( t )$ 躺在约束面 $\varGamma _ { 1 }$ 上,故 $\gamma ( t )$ 上任一点的 $\phi _ { m }$ 应为零,因而 $\dot { \phi } _ { m } \approx 0$ ,相当于要求

$$
\{ \phi _ { m } , H \} + \{ \phi _ { m } , \phi _ { n } \} \lambda ^ { n } \approx 0 , \qquad m = 1 , \cdots , M .
$$

这称为自洽性条件(consistency condition）.对此有必要详加讨论.首先，自洽性条件能否满足取决于拉氏函数 $L ( q ^ { i } , \dot { q } ^ { i } )$ 是否选得恰当.例如,如果选 $L = q ^ { 1 } + ( \dot { q } ^ { 2 } ) ^ { 2 }$ ，则 $\partial L / \partial q ^ { 1 } = 1$ 而 $\partial { \cal L } / \partial \dot { q } ^ { 1 } = 0$ ,于是拉氏方程给出 $1 = 0$ 的不自洽结果.这一不自洽性在哈氏形式中的表现就是自洽性条件(15-2-26)无法满足[详见式(15-2-54)后的一段].可见拉氏函数 $L ( q ^ { i } , \dot { q } ^ { i } )$ 的选择并不完全任意.下面讨论拉氏函数选择得当(系统不存在不自洽性)时自洽性条件(15-2-26)的后果[包括对拉氏待定乘子 $\lambda ^ { m } ( t )$ 以及对演化曲线经过的范围所提出的可能限制].令 $\boldsymbol { \varPhi } _ { m n } \equiv \{ \boldsymbol { \phi } _ { m } , \boldsymbol { \phi } _ { n } \}$ ，则 $\Phi _ { m n }$ 组成$M \times M$ 反称矩阵 $( \Phi _ { m n } )$ .再令 $h _ { m } \equiv - \left\{ \phi _ { m } , H \right\}$ ，则式(15-2-26)可表为

$$
\varPhi _ { m n } \lambda ^ { n } \approx h _ { m } , \qquad m = 1 , \cdots , M .
$$

这是关于 $M$ 个未知量 $\lambda ^ { n }$ 的 $M$ 个非齐次线性方程组,其表现因 $\mathtt { d e t } ( \varPhi _ { m n } )$ 是否为零而有质的区别.当 $\varGamma _ { 1 }$ 上任一点的 $\operatorname* { d e t } ( \varPhi _ { m n } )$ 都非零时,每点的方程组(15-2-26)有唯一解.设 $( \varPhi ^ { - 1 } ) ^ { m n }$ 是 $\Phi _ { m n }$ 的逆矩阵,则以 $( \varPhi ^ { - 1 } ) ^ { m n }$ 作用于 $\varPhi _ { n p } \lambda ^ { p } \approx h _ { n }$ 两边便可解得

$$
\lambda ^ { m } \approx ( \phi ^ { - 1 } ) ^ { m n } h _ { n } , \qquad m = 1 , \cdots , M .
$$

注意到泊松括号是 $q , p$ 的函数,可知上式把 $\lambda ^ { m }$ 确定为 $T _ { 1 }$ 上的函数.代入式(15-2-23)便有

$$
\dot { f } = \{ f , H \} + \{ f , \phi _ { m } \} ( \varPhi ^ { - 1 } ) ^ { m n } h _ { n } .
$$

把 $f ( t )$ 分别取作 $q ^ { i } ( t )$ 和 $p _ { i } ( t )$ 便可得到不含待定乘子的演化方程:

$$
\left. \begin{array} { l } { { \dot { q } ^ { i } = \{ q ^ { i } , H \} + \{ q ^ { i } , \phi _ { m } \} ( \Phi ^ { - 1 } ) ^ { m n } h _ { n } } } \\ { { \dot { p } _ { i } = \{ p _ { i } , H \} + \{ p _ { i } , \phi _ { m } \} ( \Phi ^ { - 1 } ) ^ { m n } h _ { n } } } \end{array} \right\} \qquad i = 1 , \cdots , N .
$$

上两式右边都是 $q ^ { i }$ 和 $p _ { i }$ 的函数,把函数关系分别记作 $F ^ { i }$ 和 $G ^ { i }$ ,便可改写为

$$
\begin{array} { l l } { { \dot { q } ^ { i } ( t ) = F ^ { i } ( q ( t ) , p ( t ) ) \bigg \} } } & { { \quad i = 1 , \cdots , N . } } \\ { { \dot { p } _ { i } ( t ) = G ^ { i } ( q ( t ) , p ( t ) ) \bigg \} } } & { { \quad } } \end{array}
$$

这是关于 $2 N$ 个待求函数 $q ^ { i } ( t )$ 和 $p _ { i } ( t )$ 的 $2 N$ 个一阶常微分方程,只要给定 ${ \varGamma } _ { 1 }$ 上任一点作为初态,便可求得唯一解 $( q ^ { i } ( t ) , p _ { i } ( t ) )$ ,对应于唯一的、躺在 ${ \varGamma } _ { 1 }$ 上的演化曲线 $\gamma ( t )$ ， $M$ 个 $\lambda ^ { m }$ （作为 ${ \varGamma } _ { 1 }$ 上的确定函数)限制在 $\gamma ( t )$ 上便得 $M$ 个一元函数 $\lambda ^ { m } ( t )$ ：在这种情况下，自洽性条件对 $M$ 个拉氏待定乘子的限制作用是把它们完全确定，而对 $T _ { 1 }$ 上初态点(因而演化所经历的范围)则毫无限制.这是一个极端情况.

物理上更常见的情况是 $\mathrm { d e t } ( \boldsymbol { \varPhi } _ { m n } ) \approx 0$ .一个最简单(且常见)的例子是所有泊松括号 $\{ \phi _ { m } , \phi _ { n } \}$ 都弱为零，即

$$
\phi _ { _ { m n } } \approx 0 , \qquad m , n = 1 , \cdots , M .
$$

这时自洽性条件(15-2-26)简化为

$$
h _ { m } \approx 0 , m = 1 , \cdots , M .
$$

如果 $M$ 个函数 $h _ { m } ( q , p )$ 在 $\varGamma _ { 1 }$ 上点点为零,则自洽性条件自动满足,拉氏乘子 $\lambda ^ { m }$ 因而不受任何限制,求解也就变得简单:对任一初态 $( q ^ { i } ( 0 ) , p _ { i } ( 0 ) ) \in T _ { 1 }$ ，任意指定 $M$ 个一元函数 $\lambda ^ { m } ( t )$ 代入演化方程(15-2-19)所得唯一解 $( q ^ { i } ( t ) , p _ { i } ( t ) )$ 必自动满足约束方程(15-2-11),所以待定乘子 $\lambda ^ { m } ( t )$ 在此情况下完全自由.这里出现一个问题：给定初态 $( q ^ { i } ( 0 ) , p _ { i } ( 0 ) ) \in T _ { 1 }$ 而任选两组拉氏乘子 $\{ \lambda ^ { m } ( t ) \}$ 和 $\{ \lambda ^ { \prime m } ( t ) \}$ ,便得到两条演化曲线 $\gamma ( t ) \equiv ( q ^ { i } ( t ) , p _ { i } ( t ) )$ 和 $\gamma ^ { \prime } ( t ) \equiv ( q ^ { \prime i } ( t ) , \ p _ { i } ^ { \prime } ( t ) )$ .系统到底沿哪条曲线演化？答案是都可以.初态不能决定唯一演化线的这一现象表明 $\varGamma _ { 1 }$ 上的点同物理状态的关系不是一一对应而是多一对应.这其实是物理系统存在某种规范自由性的反映.为了有利于心存疑虑的读者理解这一结论,这里举一个人为例子.设系统的 $N = 2$ ，拉氏函数 $L = ( \dot { q } ^ { 2 } ) ^ { 2 } - q ^ { 2 }$ (不含 $q ^ { 1 }$ 和 $\dot { \boldsymbol { q } } ^ { 1 }$ )，则 $p _ { 1 } = 0$ ， $p _ { 2 } = 2 \dot { q } ^ { 2 }$ ， $H = { p _ { 2 } } ^ { 2 } / 4 + q ^ { 2 }$ ,其中$p _ { 1 } = 0$ 可看作约束方程,约束函数 $\phi = p _ { 1 }$ ，代入式(15-2-17)得4个哈氏方程：

$$
\dot { q } ^ { 1 } = \lambda ( t ) , \quad \dot { q } ^ { 2 } = \frac { 1 } { 2 } p _ { 2 } , \quad \dot { p } _ { 1 } = 0 , \quad \dot { p } _ { 2 } = - 1 .
$$

# 15.2.4 $L$ 不含 $\dot { \pmb q } ^ { 1 }$ 的特别情况
