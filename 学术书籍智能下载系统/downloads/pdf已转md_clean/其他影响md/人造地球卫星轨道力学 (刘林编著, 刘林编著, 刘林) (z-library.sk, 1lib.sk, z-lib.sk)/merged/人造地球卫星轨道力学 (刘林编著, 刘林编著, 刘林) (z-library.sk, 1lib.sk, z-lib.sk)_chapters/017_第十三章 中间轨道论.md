# 第十三章 中间轨道论

从前面几章对各种摄动的研究可以看出，椭圆轨道只是卫星真实轨道的一个粗略的近似，对于精度要求较高的人卫工作，需要在椭圆轨道的基础上求出极其复杂的摄动项．因此，和天体力学中对自然天体的研究一样，为了解除分析方法面临的困境，人们总想以更接近真实运动的某种轨道来代替椭圆轨道，这种轨道常被称为中间轨道(事实上，椭圆轨道也可以说是一种最简单的中间轨道).在人卫工作中，对中间轨道的要求基本上有两条：一是中间轨道解要尽量接近卫星的真实轨道，即相应的力学系统除考虑地球的中心引力外，还应包含主要摄动力；二是尽量与椭圆轨道一样，使相应的中间轨道解具有封闭形式（避免级数展开形式引起一系列的麻烦),而且要简单，便于进一步求剩余摄动.

中间轨道仍是一种近似，它是将卫星绕地球运动同时受各种因素影响的复杂力学系统简化成一个近似的可积系统，根据各种摄动力的相对大小来看，在简化系统中，除考虑地球中心引力外，要尽量反映地球带谐项的影响，特别是 $J _ { 2 } , J _ { 3 } , J _ { 4 }$ 三项 $( J _ { 2 }$ 项尤其重要)的影响。已经给出的一些中间轨道，就是这样考虑的，在不同程度上反映了地球为旋转对称体(即考虑了带谐项)这一明显特征．最好的一类中间轨道包含了全部 ${ \pmb J } _ { 2 }$ 和 ${ \pmb J } _ { 3 }$ 项以及部分 ${ \pmb J } _ { 4 }$ 项的影响，本章将对此作详细介绍。

# 13.1非球形地球引力场中卫星运动的积分问题

根据天体力学的一些基本结论可知,到目前为止,只有二体问题和某些特殊的三体问题可以彻底解决。在卫星绕地球运动中，即使把地球看成质点,考虑日、月的影响时，将构成一个限制性四体问题，无法解决;若考虑大气、光压的影响，作用力的形式更复杂．如果只考虑地球的引力作用，而且将地球处理成旋转对称体，情况又如何？这相当于一个特殊的限制性三体问题，两有限天体中一个即对应球形引力场(中心引力部分)，另一个就是非球形附加部分(带谐项摄动部分)．这又类似于两个固定中心问题．亦称为Fatou 问题[],尽管这个问题比较特殊，也只有个别情况可以获得其严密解，下面具体讨论.

根据第五章\$5.1给出的结果,在地心赤道球坐标系 $( r , \lambda , \varphi )$ 中,若地球为旋转对称体,相应的位函数为

$$
V { = } \frac { 1 } { r } \bigg [ 1 { - } \sum _ { n \geqslant 2 } \frac { J _ { n } } { r ^ { n } } P _ { n } \left( \sin \varphi \right) \bigg ] ,
$$

卫星运动的动能是

$$
T = { \frac { 1 } { 2 } } ( \dot { r } ^ { 2 } + r ^ { 2 } \cos ^ { 2 } \varphi \dot { \lambda } ^ { 2 } + r ^ { 2 } \phi ^ { 2 } ) .
$$

对于卫星而言, $V$ 即力函数,它与 $\pmb { T }$ 都丢掉了质量因子，相应的哈密顿函数为

$$
{ \cal H } { = } \frac { 1 } { 2 } \biggl ( p _ { 1 } ^ { 2 } + \frac { 1 } { r ^ { 2 } \cos ^ { 2 } \varphi } p _ { 2 } ^ { 2 } + \frac { 1 } { r ^ { 2 } } p _ { 3 } ^ { 2 } \biggr ) { - } V ( r , \varphi ) ,
$$

其中 $r , \lambda , \varphi$ 即广义坐标(相当于 $q _ { 1 } , q _ { 2 } , q _ { 3 } )$ ， ${ \mathfrak { p } } _ { 1 } , { \mathfrak { p } } _ { 2 } , { \mathfrak { p } } _ { 3 }$ 就是广义动量，即

$$
\widetilde { p } _ { 1 } = \frac { \partial T } { \partial \dot { r } } = \dot { r } , p _ { 2 } = \frac { \partial T } { \partial \check { \lambda } } = r ^ { 2 } \cos ^ { 2 } \varphi \dot { \lambda } , \ p _ { 3 } = \frac { \partial T } { \partial \varphi } = r ^ { 2 } \dot { \varphi } .
$$

于是卫星运动的正则运动方程可写成

$$
\left. \begin{array} { c } { { \dot { r } = \displaystyle \frac { \partial H } { \partial p _ { 1 } } , \ \dot { p } _ { 1 } = - \displaystyle \frac { \partial H } { \partial r } , } } \\ { { \dot { \lambda } = \displaystyle \frac { \partial H } { \partial p _ { 2 } } , \ \dot { p } _ { 2 } = - \displaystyle \frac { \partial H } { \partial \lambda } , } } \\ { { \dot { \phi } = \displaystyle \frac { \partial H } { \partial p _ { 3 } } , \ \dot { p } _ { 3 } = - \displaystyle \frac { \partial H } { \partial \varphi } . } } \end{array} \right\}
$$

由于哈密顿函数H不显含t又不含坐标λ，因此存在两个积分：

$$
H { = } \alpha _ { 1 } , ~ p _ { 2 } { = } \alpha _ { 2 } ,
$$

α和α为积分常数，有了两个积分就可将六阶方程（13.5）降为二阶方程，首先由积分 $x _ { 2 } = x _ { 2 }$ 可得

$$
\lambda { = } \frac { { \partial } H } { { \partial } p _ { 2 } } { = } \frac { { \alpha } _ { 2 } } { r ^ { 2 } \cos ^ { 2 } \varphi } ,
$$

即

$$
\lambda { = } \int _ { } \frac { \alpha _ { 2 } } { r ^ { 2 } \cos ^ { 2 } \phi } d t + \alpha _ { 3 } ,
$$

$\pmb { \alpha } _ { 3 }$ 又是一个积分常数;由 $\pmb { H } = \pmb { \alpha _ { 1 } }$ 可解出 $\pmb { p } _ { 1 }$ 为

$$
p _ { 1 } = \pm \biggl [ 2 ( \alpha _ { 1 } + V ) - \frac { \alpha _ { 2 } ^ { 2 } } { r ^ { 2 } \cos ^ { 2 } \phi } - \frac { p _ { 3 } ^ { 2 } } { r ^ { 2 } } \biggr ] ^ { 1 / 2 } ,
$$

利用 $H$ 不显含 $\pmb { \mathscr { t } }$ 的特点，作自变量变换： $t  r _ { i }$ 有如下关系：

$$
\frac { d } { d t } { = } \frac { d } { d r } \frac { d r } { d t } { = } p _ { 1 } \frac { d } { d r } ,
$$

面 $\pmb { r }$ 与 $\pmb { t }$ 的关系可由

$$
\dot { r } = \frac { \partial H } { \partial p _ { 1 } } = p _ { 1 }
$$

给出

$$
\int \limits _ { \log _ { 1 } } ^ { \infty } = t + \alpha _ { 4 } ,
$$

$x _ { 4 }$ 为第四个积分常数，其中 ${ \pmb p _ { 1 } }$ 由（13.8）式给出。于是方程组(13.5)剩下 $\pmb { \varphi }$ 和 ${ \pmb { \eta } } _ { 3 }$ 两个方程,化为对 $\pmb { r }$ 的形式即

$$
\scriptstyle  \left. { \begin{array} { l } { \displaystyle { \frac { d \varphi } { d r } } = { \frac { 1 } { p _ { 1 } } } \left( { \frac { p _ { 3 } } { r ^ { 2 } } } \right) , } \\ { \displaystyle { \frac { d p _ { 3 } } { d r } } = - { \frac { 1 } { p _ { 1 } } } \left( { \frac { \alpha _ { 2 } ^ { 2 } } { r ^ { 2 } } } { \frac { \sin \varphi } { \cos ^ { 3 } \varphi } } - { \frac { \partial V } { \partial \varphi } } \right) . } \end{array} } \right\}
$$

因 $p _ { 1 } = p _ { 1 } ( r , \varphi , p _ { 3 } , \alpha _ { 1 } , \ \alpha _ { 2 } )$ ，所以在这样一个特殊的非球形引力场、中，卫星运动问题就归结为一个二阶方程的求解问题．如果这一

方程可解,即给出

$$
\left. \begin{array} { l } { { \pmb { \mathcal { P } } = \varphi \left( r , \alpha _ { 1 } , \alpha _ { 2 } , \alpha _ { 3 } , \alpha _ { 5 } , \alpha _ { 6 } \right) , } } \\ { { \pmb { \mathcal { P } } _ { 3 } = p _ { 3 } \left( r , \alpha _ { 1 } , \alpha _ { 2 } , \alpha _ { 3 } , \alpha _ { 5 } , \alpha _ { 6 } \right) , } } \end{array} \right\}
$$

$\pmb { \alpha _ { 5 } }$ 和 $\alpha _ { \beta }$ 是最后两个积分常数。从而由(13.10)式得到

$$
r = r ( t ; \alpha _ { 1 } , \alpha _ { 2 } , \alpha _ { 3 } , \alpha _ { 4 } , \alpha _ { 5 } , \alpha _ { 6 } ) ,
$$

再由(13.7)和(13.12)式给出

$$
\begin{array} { r l } & { \lambda { = } \lambda ( t ; \alpha _ { 1 } , \alpha _ { 2 } , \alpha _ { 3 } , \alpha _ { 4 } , \alpha _ { 5 } , \alpha _ { 6 } ) , } \\ & { } \\ & { \varphi { = } \varphi ( t ; \alpha _ { 1 } , \alpha _ { 2 } , \alpha _ { 3 } , \alpha _ { 4 } , \alpha _ { 5 } , \alpha _ { 6 } ) . } \end{array}
$$

$\pmb { p } _ { 1 }$ 和 $\pmb { p _ { 3 } }$ 有类似的表达式,于是问题完全解决．但是，还无法给出积分(13.12),因此,和一般三体问题一样，问题仍不能解决.

根据以上讨论可知，只有对地球模型作进一步的简化,才有可能使相应的问题成为可积的，例如，假定地球不仅是旋转对称体，又是南北对称，此时非球形部分只有偶次带谐项．在这种引力场作用下，卫星在赤道面内运动是可能的，对于这种运动，有

$$
\phi = 0 , \phi = 0 , p _ { 3 } = 0 .
$$

相应的问题彻底解决，由前面四个积分（13.6)，（13.7）和(13.10）表达。不过,对于这种特殊运动状态，并没有什么实用价值，不再讨论下去。

正因为卫星在非球形引力场中运动，相应的运动方程无法给出严密解，加上其它一些力学因素的影响，问题更复杂，所以在目前的人卫工作中基本上都采用前面几章介绍的以椭圆运动为基础的各种解法．但这并不影响人们对中间轨道以及建立在此基础上的中间轨道理论进行探讨，即寻找一种比较理想的中间轨道，使其尽可能满足前面提出的两点要求，尽管目前有关这一领域的工作并不算成功，但还是有必要仔细介绍一下这类问题的基本知识和进展情况.

# 13.2 几类中间轨道

针对人造卫星的运动，已找出的中间轨道都是考虑地球非球形引力场的，大致可以分成三种类型，下面作一简单介绍。

# 1.斯特恩(Sterne)等人提出的中间轨道[2-6]

这类中间轨道采用的是球坐标 $( r , \varphi , \lambda )$ 形式,其位函数为

$$
V _ { 1 } { = } \frac { 1 } { r } { - } 3 J _ { 2 } [ C _ { 1 } ( \sin ^ { 2 } \varphi { - } C _ { 2 } ) / 2 r ^ { 2 } { + } C _ { 3 } / r { + } C _ { 4 } / r ^ { 3 } ] ,
$$

这里 $\smash { C _ { 1 3 } C _ { 2 3 } C _ { 3 } , C _ { 4 } }$ 是待定常数，根据需要接近真实位函数(13.1)式的程度而定．注意，从这里开始，为了表达方便，坐标的序号已不按正常的 $r , \lambda , \varphi$ 排列，改为 $r , \varphi , \lambda$ 的顺序.

# 2.文梯(Vinti)等人提出的中间轨道[7-10]

这类中间轨道采用的是旋转椭球坐标 $( \rho , \eta , \lambda )$ 形式,即

$$
\left. \begin{array} { l } { x = r \cos \varphi \cos \lambda { = } [ ( \rho ^ { 2 } + c ^ { 2 } ) ( 1 { - } \eta ^ { 2 } ) ] ^ { 1 , 2 } \cos \lambda , } \\ { y = r \cos \varphi \sin \lambda { = } [ ( \rho ^ { 2 } + c ^ { 2 } ) ( 1 { - } \eta ^ { 2 } ) ] ^ { 1 / 2 } \sin \lambda , } \\ { z = r \sin \varphi { = } \rho \eta , ( - 1 { \leqslant } \eta \leqslant 1 ) . } \end{array} \right\}
$$

相应的位函数形式如下：

$$
V _ { 2 } = \frac { \rho } { \rho ^ { 2 } + c ^ { 2 } \eta ^ { 2 } } .
$$

在该坐标系中,卫星运动的动能为

$$
\mathcal { T } { = } \frac { 1 } { 2 } \Bigg [ \frac { \rho ^ { 2 } + c ^ { 2 } \eta ^ { 2 } } { \rho ^ { 2 } + c ^ { 2 } } \dot { \rho } ^ { 2 } + \frac { \rho ^ { 2 } + c ^ { 2 } \eta ^ { 2 } } { 1 - \eta ^ { 2 } } \eta ^ { 2 } + ( \rho ^ { 2 } + c ^ { 2 } ) ( 1 - \eta ^ { 2 } ) \dot { \lambda } ^ { 2 } ~ \Bigg ]
$$

相应的广义动量为

$$
p _ { 1 } = \frac { \rho ^ { 2 } + c ^ { 2 } \eta ^ { 2 } } { \rho ^ { 2 } + c ^ { 2 } } \rho , p _ { 2 } = \frac { \rho ^ { 2 } + c ^ { 2 } \eta ^ { 2 } } { 1 - \eta ^ { 2 } } \eta , p _ { 3 } = \left( \rho ^ { 2 } + c ^ { 2 } \right) ( 1 - \eta ^ { 2 } ) \dot { \lambda } .
$$

于是哈密顿函数可写成

$$
\begin{array} { c } { { H = \displaystyle \frac 1 2 \biggl [ \frac { \rho ^ { 2 } + c ^ { 2 } } { \rho ^ { 2 } + c ^ { 2 } \eta ^ { 2 } } p _ { 1 } ^ { 2 } + \frac { 1 - \eta ^ { 2 } } { \rho ^ { 2 } + c ^ { 2 } \eta ^ { 2 } } p _ { 2 } ^ { 2 } + \frac 1 { ( \rho ^ { 2 } + c ^ { 2 } ) \left( 1 - \eta ^ { 2 } \right) } p _ { 3 } ^ { 2 } \biggr ] } } \\ { { - \displaystyle \frac \rho { \rho ^ { 2 } + c ^ { 2 } \eta ^ { 2 } } . } } \end{array}
$$

-按照要求,根据(13.1)式,取

$$
c ^ { 2 } = J _ { 2 } ,
$$

此种选择将表明该中间轨道的特点是：位函数 $\pmb { V _ { 0 } }$ 包含地球引力场：的主要修正项一 ${ \pmb J } _ { 2 }$ 项．后来，文梯又改进了 $\pmb { V _ { 0 } }$ 的形式[]，它类似于下面要介绍的中间轨道.

# 3．阿克谢诺夫(AKCeHOB)等人提出的中间轨道 $[ 1 2 - 1 5 ]$

这种中间轨道是建立在两个不动中心问题[基础上的，其位函数为

$$
V _ { 3 } = \frac { 1 } { 2 } \bigg ( \frac { 1 + i d } { r _ { 1 } } { + \frac { 1 - i d } { r _ { 2 } } } \bigg ) ,

$$
\left. \begin{array} { l } { { r _ { 1 } = \left\{ x ^ { 2 } + y ^ { 2 } + \left[ z - c \left( d + i \right) \right] ^ { 2 } \right\} ^ { 1 / 2 } , } } \\ { { r _ { 2 } = \left\{ x ^ { 2 } + y ^ { 2 } + \left[ z - c \left( d - i \right) \right] ^ { 2 } \right\} ^ { 1 / 2 } . } } \end{array} \right\}

这里的 $\pmb { i }$ 不是轨道倾角,而是虚单位，即 $i = \sqrt { - 1 }$ $\pmb { c }$ 和 $\pmb { d }$ 是待定的实常数，且 $\pmb { c }$ 为距离量， $\pmb { d }$ 为无量纲量．显然， $\pmb { V _ { \textrm { \pmb { \imath } } } }$ 是用共轭复数之和来表达的，仍为 $x , \ y , \ z$ 的实函数．同样采用旋转椭球坐标$( \rho , \mu , \lambda )$ ,即引进变换：

$$
\left. \begin{array} { l } { { x = r \cos \varphi \cos \lambda = \lbrack ( \rho ^ { 2 } + c ^ { 2 } ) ( 1 - \mu ^ { 2 } ) ] ^ { 1 / 2 } \cos \lambda , } } \\ { { y = r \cos \varphi \sin \lambda = \lbrack ( \rho ^ { 2 } + c ^ { 2 } ) ( 1 - \mu ^ { 2 } ) ] ^ { 1 / 2 } \sin \lambda , } } \\ { { z = r \sin \varphi = \rho \mu + c d , \qquad ( - 1 \leqslant \mu \leqslant 1 ) . } } \end{array} \right\}

则 $V _ { 3 }$ 变为下列形式：

$$
V _ { 3 } = \frac { \rho - c d \mu } { \rho ^ { 2 } + c ^ { 2 } \mu ^ { 2 } } .

卫星运动的动能和相应的广义动量为

$$
\ . \mathcal { T } { = } \frac { 1 } { 2 } \biggl [ \frac { \rho ^ { 2 } { + } c ^ { 2 } \mu ^ { 2 } } { \rho ^ { 2 } { + } c ^ { 2 } } \hat { \rho } ^ { 2 } { + } \frac { \rho ^ { 2 } { + } c ^ { 2 } \mu ^ { 2 } } { 1 { - } \mu ^ { 2 } } \hat { \mu } ^ { 2 } { + } \left( \rho ^ { 2 } { + } c ^ { 2 } \right) \left( 1 { - } \mu ^ { 2 } \right) \dot { \lambda } ^ { 2 } \biggr ] ,

p _ { 1 } = \frac { \rho ^ { 2 } + c ^ { 2 } \mu ^ { 2 } } { \rho ^ { 2 } + c ^ { 2 } } \rho , p _ { 2 } = \frac { \rho ^ { 2 } + c ^ { 2 } \mu ^ { 2 } } { 1 - \mu ^ { 2 } } , p _ { 3 } = \left( \rho ^ { 2 } + c ^ { 2 } \right) ( 1 - \mu ^ { 2 } ) \dot { \lambda } .

相应的哈密顿函数可写成

$$
\begin{array} { c } { { H = \displaystyle \frac 1 2 \biggl [ \frac { \rho ^ { 2 } + c ^ { 2 } } { \rho ^ { 2 } + c ^ { 2 } \mu ^ { 2 } } p _ { 1 } ^ { 2 } + \frac { 1 - \mu ^ { 2 } } { \rho ^ { 2 } + c ^ { 2 } \mu ^ { 2 } } \varphi _ { 2 } ^ { 2 } + \frac 1 { ( \rho ^ { 2 } + c ^ { 2 } ) ( 1 - \mu ^ { 2 } ) } p _ { 3 } ^ { 2 } \biggr ] } } \\ { { - \displaystyle \frac { \rho - c d \mu } { \rho ^ { 2 } + c ^ { 2 } \mu ^ { 2 } } . } } \end{array}

以上三种类型的中间轨道有一个共同的特点，即 $\pmb { H }$ 不显含 $\pmb { \ell }$ ，还不含一个角坐标(经度 $\lambda$ ，这正是一个旋转对称体引力位的特征．于是存在两个积分，一个是能量积分：

$$
H = \alpha _ { 1 } ,
$$

另一个为

$$
{ \pmb \operatorname { / } } _ { 3 } = { \pmb \alpha } _ { 3 } .
$$

不难看出,对于阿克谢诺夫型,当 $\pmb { d } = 0$ 时，即退化为文梯型，相应地有

$$
V _ { 2 } { = } \frac { 1 } { 2 } \Big ( \frac { 1 } { r _ { 1 } } { + } \frac { 1 } { r _ { 2 } } \Big ) ,
$$

$$
\left. \begin{array} { l } { { r _ { 1 } { = } \lbrack x ^ { 2 } + y ^ { 2 } + ( z - c i ) ^ { 2 } ] ^ { 1 / 2 } , } } \\ { { r _ { 2 } { = } \lbrack x ^ { 2 } + y ^ { 2 } + ( z + c i ) ^ { 2 } ] ^ { 1 / 2 } . } } \end{array} \right\}
$$

由此可知，上述两类中间轨道都是建立在两个不动中心问题基础上的,把一个旋转对称的中心天体“分成”两个不动体，均在对称轴(z轴)上．对于文梯型,两个不动体的质量相等，各为总质量的一半,分别位于 $\pmb { z }$ 轴上的± ${ \pmb { c } } _ { } { \pmb { i } } ^ { \ 6 }$ 两点"处;而对于阿克谢诺夫型，相应的两个不动体则分别位于 $\pmb { z }$ 轴上的 $c d \neq c i ^ { \omega }$ 两点"处，两个质量也.

分别用共轭复数来表示，后者之所以这样选择，是因为有两个待定常数 $\pmb { \mathscr { o } }$ 和 $\pmb { d }$ 可供调节，有可能使其包含地球引力场位函数展开式(13.1)中的两项,从而更接近真实情况.

由于要求 $\pmb { V _ { \textrm { \pmb { \mathfrak { s } } } } }$ 尽量“逼近”真实地球， $\pmb { c }$ 和 $\pmb { d }$ 显然应为小量，于是可由

$$
\begin{array} { r l } & { r _ { 1 } = r \bigg [ 1 - 2 \Big ( \frac { z } { r } \Big ) \frac { c ( d + i ) } { r } + \cdot \frac { c ^ { 2 } ( d + i ) ^ { 2 } } { r ^ { 2 } } \bigg ] ^ { 1 / 2 } , } \\ & { r _ { 2 } = r \bigg [ 1 - 2 \Big ( \frac { z } { r } \Big ) \frac { c ( d - i ) } { r } + \frac { c ^ { 2 } ( d - i ) ^ { 2 } } { r ^ { 2 } } \bigg ] ^ { 1 / 2 } , } \end{array}
$$

给出下列展开式：

$$
\left. \begin{array} { l } { \displaystyle \frac { 1 } { r _ { 1 } } { = } \frac { 1 } { r } \sum _ { n \geqslant 0 } \frac { c ^ { n } ( d + i ) ^ { n } } { r ^ { n } } P _ { n } \bigg ( \frac { z } { r } \bigg ) , } \\ { \displaystyle \frac { 1 } { r _ { 2 } } { = } \frac { 1 } { r } \sum _ { n \geqslant 0 } \frac { c ^ { n } ( d - i ) ^ { n } } { r ^ { n } } P _ { n } \bigg ( \frac { z } { r } \bigg ) . } \end{array} \right\}
$$

·将此展开式代入(13.24)式得

$$
V _ { 3 } = \frac { 1 } { r } \sum _ { n \geqslant 0 } \frac { c ^ { n } } { 2 } [ ( 1 + i d ) ( d + i ) ^ { n } + ( 1 - i d ) ( d - i ) ^ { n } ] \frac { 1 } { r ^ { n } }

其中 $\sin \varphi = z / r$ ，上式求和中的 $\pmb { n } = 0$ 和 $\pmb { n = 1 }$ 两项，分别为1和0,于是有

$$
V _ { \vartheta } = \frac { 1 } { r } \bigg [ 1 + \sum _ { n \geqslant 2 } \frac { K _ { n } } { r ^ { n } } P _ { n } \left( \sin \varphi \right) \bigg ] ,
$$

其中

$$
K _ { n } { = } \frac { c _ { n } } { 2 } [ ( 1 + i d ) ( d + i ) ^ { n } { + } ( 1 { - } i d ) ( d { - } i ) ^ { n } ] .
$$

# 4．中间轨道根数及其含义

显然,两类中间轨道与相应的瞬时椭圆轨道相差甚小,若取中间轨道根数为

$$
\left. \begin{array} { l l l } { { a = ( - 2 \alpha _ { 1 } ) ^ { - 1 } , } } & { { e ^ { 2 } = 1 + 2 \alpha _ { 1 } \alpha _ { 2 } ^ { 2 } , } } & { { i = \cos ^ { - 1 } ( \alpha _ { 3 } / \alpha _ { 2 } ) , } } \\ { { \Omega = \beta _ { 3 } , } } & { { \omega = \beta _ { 2 } , } } & { { M = ( - 2 \alpha _ { 1 } ) ^ { 3 / 2 } ( t + \beta _ { 1 } ) , } } \end{array} \right\}
$$

它们与瞬时椭圆轨道根数有什么联系？略去推导，在一阶意义下，可以证明这些中间轨道根数(除M外)就是瞬时椭圆的初始平均根数 $\overline { { \sigma } } ( t _ { 0 } )$ ，而 $M$ 则为 $\pmb { t }$ 时刻的平均根数 $\pmb { M } ( t ) ^ { \{ 1 8 \} }$ ，即

$$
\begin{array} { r } { \left( \sigma \right) = \sigma + \sigma _ { \ast } ^ { \left( 1 \right) } \left( t \right) + \delta \left( \bar { n } + \sigma _ { \ast } \right) \left( t - t _ { 0 } \right) . } \end{array}
$$

其中 $( \sigma )$ 是瞬时椭圆根数 $\left( a \right) , \left( e \right) , \left( i \right) , \left( \varOmega \right) , \left( \omega \right) , \left( M \right) , \sigma _ { s } ^ { \left( 1 \right) } \left( t \right)$ 是一阶短周期项， $\delta = 1$ （对 $\pmb { M }$ )或0(对另五个根数)， $\sigma _ { 1 }$ 即一阶长期项系数， $\pmb { \bar { n } } = \pmb { \bar { a } } ^ { - 3 / 2 }$ ，前面所说的一阶意义就是指考虑到一阶长期和一阶短周期摄动项．下面给出一例，即 $\pmb { \alpha }$ 与 $( a )$ 的关系：

$$
\begin{array} { c } { { ( a ) = a + \displaystyle \frac { 3 c ^ { 2 } } { 2 \alpha } \left\{ \frac { 3 } { 2 } \biggl ( 1 - \frac { 3 } { 2 } \sin ^ { 2 } i \biggr ) \left[ \left( \frac { a } { r } \right) ^ { 3 } - ( 1 - e ^ { 2 } ) ^ { - 3 / 2 } \right] \right. } } \\ { { \left. + \biggl ( \frac { a } { r } \biggr ) ^ { 3 } \sin ^ { 2 } i \cos 2 \left( f + \omega \right) \right\} } } \end{array}
$$

$$
+ - \frac { c ^ { 2 } } { a } \biggl ( 1 - \frac { 3 } { 2 } \sin ^ { 2 } i \biggr ) ( 1 - e ^ { 2 } ) ^ { - 3 / 2 } .
$$

若取 $c ^ { 2 } = J _ { 2 }$ ，则与第五章中给出的 ${ { J } _ { 2 } }$ 项摄动解一致，仅差一常数项,这对解而言无妨.

# 13.4 中间轨道摄动法

已经给出的各类中间轨道都是不完整地反映了地球引力位带谐项的一部分（即使AKceHOB型中间轨道也未能反映 $J _ { 4 }$ 项的全部)或主要田谐项 $A _ { 2 , 2 } , B _ { 2 , 2 } ^ { \Gamma 1 0 1 }$ ，因此对于卫星运动的精密定轨问题，还得在中间轨道的基础上进一步计算剩余摄动，包括地球引力位带谐项的一部分和田谐项，日、月，光压,大气阻力等摄动。

若用六个积分常数 $\alpha _ { 1 } , \alpha _ { 2 } , \alpha _ { 3 } , \beta _ { 1 } , \beta _ { 2 } , \beta _ { 3 }$ 作为基本变量(它们是正则共轭变量),则相应的摄动运动方程为．

$$
\dot { \alpha } _ { i } = \frac { \partial R } { \partial \beta _ { i } } , \qquad \dot { \beta } _ { i } = - \frac { \partial R } { \partial \alpha _ { i } } .
$$

其中 $\pmb { R }$ 是摄动函数，即

$$
\pmb { R } { = } V { - } V _ { 0 } ,
$$

这里的 $V _ { 0 }$ 就是中间轨道位函数．对于非保守力，可在方程(13.124)的右端以摄动加速度形式列入，形式如下：

$$
\dot { \alpha } _ { i } = F _ { e } \cdot \frac { \partial r } { \partial \beta _ { i } } , \qquad \dot { \beta } _ { i } = - F _ { e } \cdot \frac { \partial r } { \partial \alpha _ { i } } ,
$$

其中 $\pmb { F } ,$ ，是摄动加速度.

考虑到实际计算的需要，有时要引入一些类似于椭圆轨道的六个根数 $\sigma _ { j } ( j = 1 , 2 , \cdots , 6 )$ 代替 $\alpha _ { 1 } , \ldots , \beta _ { 3 }$ 作为基本变量,如果 $\sigma _ { j }$ 仍是一组正则共轭变量,那很简单，否则要写出以新变量 $\pmb { \sigma _ { j } }$ 表达的运动方程,将有一个相应的推导过程，假定有如下变换关系：

$$
\sigma _ { j } = \sigma _ { j } ( \alpha , \beta , t ) , \qquad j = 1 , 2 , \cdots , 6 .
$$

则

$$
\sigma _ { j } = \sum _ { i = 1 } ^ { 3 } \biggl ( \frac { \partial \sigma _ { j } } { \partial \alpha _ { i } } { \dot { \alpha } } _ { i } + \frac { \partial \sigma _ { j } } { \partial \beta _ { i } } { \dot { \beta } } _ { i } \biggr ) + \biggl ( \frac { \partial \sigma _ { i } } { \partial t } \biggr ) .
$$

将方程(13.124)或(13.126)代入，并将 $\pmb { R }$ 对 $\alpha _ { i } , \beta _ { i }$ 的偏导数化为对 $\sigma _ { j }$ 的偏导数，经整理后即得

$$
\left. \begin{array} { l } { { \displaystyle { \sigma _ { j } = \sum _ { k = 1 } ^ { 0 } \{ \sigma _ { j } , ~ \sigma _ { k } \} \frac { \partial R } { \partial \sigma _ { k } } + \left( \frac { \partial \sigma _ { j } } { \partial t } \right) } , } } \\ { { \displaystyle { \sigma _ { j } = \sum _ { k = 1 } ^ { 8 } \{ \sigma _ { j } , ~ \sigma _ { k } \} F _ { * } \cdot \frac { \partial r } { \partial \sigma _ { k } } + \left( \frac { \partial \sigma _ { j } } { \partial t } \right) } . } } \end{array} \right\}
$$

其中 $\{ \sigma _ { j } , \sigma _ { k } \}$ 是泊松括号，即

$$
\displaystyle { \{ \sigma _ { j } , \sigma _ { k } \} = \sum _ { i = 1 } ^ { 3 } \bigg ( \frac { \partial \sigma _ { j } } { \partial \alpha _ { i } } \frac { \partial \sigma _ { k } } { \partial \beta _ { i } } { - \frac { \partial \sigma _ { j } } { \partial \beta _ { i } } \frac { \partial \sigma _ { k } } { \partial \alpha _ { i } } \bigg ) } . }
$$

求解摄动运动方程(13.129)后，给出 $\sigma _ { i } { = } \sigma _ { j } ( t ) _ { i }$ ，或 $\alpha _ { i } ( t ) .$ $\beta _ { i } ( t )$ ，由此便可计算有摄星历表．下面将以地球引力位 $J _ { 2 } , J _ { 3 } , J _ { 4 }$ 三项摄动为背景，介绍以Vinti型和AKceHOB型中间轨道为基础的剩余摄动计算方法.

# 1.基本变量与方程

若仿照受摄二体问题中所采用的德洛纳变量,可引人

$$
\left. \begin{array} { c c } { { L = ( - 2 \alpha _ { 1 } ) ^ { - 1 / 2 } , ~ l = ( - 2 \alpha _ { 1 } ) ^ { 3 / 2 } ( t + \beta _ { 1 } ) , } } \\ { { } } & { { } } \\ { { G = \alpha _ { 2 } , ~ g = \beta _ { 2 } , } } \\ { { } } & { { } } \\ { { H = \alpha _ { 3 } , ~ h = \beta _ { 3 } . } } \end{array} \right\}
$$

这组变量 $( L , G , H , l , g , h )$ 仍是一组正则共轭变量，相应的哈密顿函数为

$$
{ \pmb F } = \frac { \mathbf { 1 } } { 2 L ^ { 2 } } + \cdot { \pmb R } ,
$$

其中 $\pmb { R }$ 即由(13.125)式表达，相应的中间轨道摄动运动方程可直接写出，即

$$
\left. \begin{array} { l l } { \frac { d L } { d t } { = } \frac { \partial F } { d \mathit { l } } , } & { \frac { d l } { d \mathit { t } } { = } - \frac { \partial F } { \partial \mathit { L } } , } \\ { \frac { d G } { d \mathit { t } } { = } \frac { \partial F } { \partial \mathit { g } } , } & { \frac { d g } { d \mathit { t } } { = } - \frac { \partial F } { \partial \mathit { G } } , } \\ { \frac { d H } { d \mathit { t } } { = } \frac { \partial F } { \partial \mathit { h } } , } & { \frac { d h } { d \mathit { t } } { = } \cdots { \frac { \partial F } { \partial \mathit { H } } } . } \end{array} \right\}
$$

要具体写出右函数，必须将 $\pmb { R }$ 表达成基本变量的函数，下面将用实例来说明这一点.

# 2.摄动函数 $\pmb R$ 的表达式

对于Vinti型中间轨道,根据(13.42)式有

$$
R { = } \sum _ { n \geqslant 3 } { \frac { ( - J _ { n } ^ { \prime } ) } { r ^ { n + 1 } } } P _ { n } ( \sin \varphi ) ,
$$

其中

$$
\left. \begin{array} { l } { { J _ { 2 k + 1 } ^ { \prime } = J _ { 2 k + 1 9 } , } } \\ { { J _ { 2 ( k + 1 ) } ^ { \prime } = J _ { 2 ( k + 1 ) } + ( - 1 ) ^ { k + 1 } c ^ { 2 ( k + 1 ) } . } } \end{array} \right. \quad ( k = 1 , 2 , \cdots ) \Bigr \}
$$

对于AKceHOB型中间轨道，则根据(13.36)式有

$$
R { = } \sum _ { n \geqslant 4 } { \frac { ( { - } J _ { n } ^ { \prime } ) } { r ^ { n + 1 } } } P _ { n } ( \sin \varphi ) ,
$$

其中

$$
{ J _ { n } ^ { \prime } } = { J _ { n } } + { K _ { n } } .
$$

由于 $c ^ { 2 } = J _ { 2 } , c ^ { 4 } = J _ { 2 } ^ { 2 } , \cdots , K _ { 2 } = - J _ { 2 } , K _ { 3 } = - J _ { 3 } , K _ { 4 } = O ( 1 0 ^ { - 6 } )$ ，$K _ { 5 } = O ( 1 0 ^ { - 8 } ) , K _ { 6 } { = } O ( 1 0 ^ { - 9 } ) , { \cdots } ,$ 故对于Vinti型中间轨道，摄动函数 $\pmb { R }$ 的展开式(13.134)与真实地球引力位展开式(指带谐项部分）几乎无差别，只是求和从 $\pmb { \mathcal { R } } = 3$ 开始。而对于AKCeHOB型中间轨道，也有类似情况，但求和从 $\pmb { \mathscr { R } } = \pmb { 4 }$ 开始.

若以 ${ J _ { \tilde { \mathbf { z } } } } , { J _ { \mathbf { s } } } , { J _ { \mathbf { s } } }$ 三项为背景,那么,对于Vinti型中间轨道,相应的摄动函数为

$$
R = \frac { ( - J _ { 3 } ^ { \prime } ) } { r ^ { 4 } } P _ { 3 } ( \sin \varphi ) + \frac { ( - J _ { 4 } ^ { \prime } ) } { r ^ { 5 } } P _ { 4 } ( \sin \varphi ) .
$$

由变换(13.18)可知

$$
\left. \begin{array} { c } { { r = \rho \displaystyle \left[ 1 + c ^ { 2 } \left( \frac { 1 - \eta ^ { 2 } } { 2 \rho ^ { 2 } } \right) + O ( c ^ { 4 } ) \ \right] , } } \\ { { \mathrm { s i n } \varphi = \displaystyle \left( \frac { \rho } { r } \right) \eta . } } \\ { { . } } \end{array} \right\}
$$

而由Vinti型中间轨道解可给出 $\rho , \eta$ 与基本变量 $L , \ G , \ \cdots$ 的关系,即

$$
\left. \begin{array} { l } { { \rho = \rho _ { 1 } ( L , G , l ) + c ^ { 2 } \rho _ { 2 } ( L , G , H , l , g ) , } } \\ { { \eta = \sin i \sin ( f \div g ) + { \cal O } ( c ^ { 2 } ) . } } \end{array} \right\}
$$

其中 $\cos i = H / G , f$ 与 $\vec { E } , l$ 之间的关系不再列出。如果求一阶解，则 $\pmb R$ 只需取到 $O ( c ^ { 0 } )$ 项，问题很简单，如果求二阶解，则上述 $\pmb R$ 应取到 $O ( c ^ { 2 } )$ 项.

对于AKceHOB型中间轨道,相应的摄动函数为

$$
R { = } \frac { ( - J _ { 4 } ^ { \prime } ) } { r ^ { 5 } } P _ { 4 } ( \sin \varphi ) .
$$

由变换(13.26)式给出

$$
\begin{array} { l } { { r = \rho \displaystyle [ 1 + c ^ { 2 } ( \frac { 1 - \mu ^ { 2 } } { 2 \rho ^ { 2 } } + ( \frac { d } { c } ) \frac { \mu } { \rho } ) + O ( c ^ { 4 } ) ] } , } \\ { { \sin \varphi = \displaystyle \frac { 1 } { r } ( \rho \mu + c d ) . } } \end{array} )  ,
$$

同样可由AKCeHOB型中问轨道解给出 $\rho , \mu$ 与基本变量的关系，但比Vinti型中间轨道对应的关系(13.140)式复杂.

由于AKceHOB型中间轨道包含了地球引力位中的 $\pmb { J _ { 3 } }$ 项（即$d { \neq } 0 )$ ，故在摄动函数 $\pmb { R }$ 的表达式(13.141)中将要引进相应的$( d / c )$ 项，但对一阶解而言， $\pmb { R }$ 中无需考虑这一项，确实比Vinti型中间轨道好，不过，一阶解意义不大，不采用中间轨道也很简单.对二阶和高阶解，相应的 $\pmb { R }$ 比Vinti型复杂，尽管少计算一项 $( \sigma _ { 8 }$ 项）

豹剩余摄动，可相应的麻烦却通过中间轨道解转移到计算其它剩余摄动的摄动函数 $\pmb { R }$ 中去了，从总的效果来看，还不如采用Vinti型中间轨道.

# 3．剩余摄动的计算方法

将用基本变量表达的摄动函数 $\pmb { R }$ 代入摄动运动方程（13.133),就构成一小参数方程:

$$
\scriptstyle \sigma = f ( \sigma , t , c ^ { 2 } ) ,
$$

其中 $\pmb { c } ^ { 2 } { = } \pmb { O } ( \pmb { J } _ { 2 } )$ 是小参数。因此，在中间轨道基础上如何计算剩余摄动，显然可采用前面第五章和第七章分别介绍的各种方法，与受摄二体问题中构造瞬时椭圆的摄动解并无实质性差别，文[20]和[17]曾分别按上述过程给出了建立在AKceHOB型中间轨道基础上的一阶解和高阶解。因为并无特殊之处，无需再给出构造 $\pmb { J _ { 3 } }$ 和 ${ \pmb J } _ { 4 }$ 项摄动解的具体过程。

# 4.中间轨道实用化的前景

上述两类中问轨道解与椭圆轨道相比，最大的优点在于包含了地球引力位展开式中最重要的 $\pmb { J _ { 2 } }$ 项．但对于精密定轨而言,往往需要二阶解甚至高阶解，那么必须在中间轨道基础上去求各种剩余摄动， ${ \pmb J } _ { 2 }$ 项引起的轨道“变化”则部分地通过中间轨道解转移到摄动函数(或摄动加速度)中，增加求剩余摄动的麻烦.尽管所采用的方法和基本运算都可模仿受摄二体问题以椭圆轨道为基础的各种做法，但并未使求高阶摄动（除 $\pmb { J _ { 2 } }$ 项外）的麻烦有明显的改观,至少目前给出的中间轨道是如此。关于 $\pmb { J _ { 2 } }$ 项本身，虽然无摄动问题,但通过中间轨道解使星历表计算复杂化．当然，作为中间轨道的探讨，本身确是一项基本研究工作，只是从实用角度来看，在人造卫星轨道理论中的应用前景并不乐观,到目前为止,在实际工作中,仍采用椭圆轨道作为基本解。

从前面讨论不难看出,中间轨道实用化的最大困难,在于很难（几乎不可能)找到一种理想的中间轨道，既能完整地包含更多的摄动因素,又不至于使其解本身过于复杂化，而这一点对于分析方法而言，正是问题的关键所在.
