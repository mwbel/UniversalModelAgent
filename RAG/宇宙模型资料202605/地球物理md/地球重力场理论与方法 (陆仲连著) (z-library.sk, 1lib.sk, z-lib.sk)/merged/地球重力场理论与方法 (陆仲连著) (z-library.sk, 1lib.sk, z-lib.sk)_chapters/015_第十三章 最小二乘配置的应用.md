# 13.1最小二乘配置的原理

最小二乘配置是综合平差、滤波和推估的一种广义推估方法。除了一组参数以外，它还包含两组随机变量：一组称为信号，它们之间通过协方差而相关;另一组是测量误差·亦称噪声。由于它可以将不同类型的观测数据作统一处理，特别适用于研究有关地球重力场的问题，但也可用于大地测量和摄影测量中的某些问题。

最小二乘配置采用的数学模型是

$$
l { = } \mathbf { A } \mathbf { X } { + } \mathbf { U } \mathbf { S } { + } \mathbf { n }
$$

式中 $\iota$ 为观测值向量：X为非随机的参数向量；A是一个已知的系数阵，它表明参数X对观测值l的影响；AX通常是由参数的原始非线性函数线性化后得到的;S为信号向量·它的形式为

$$
{ \mathsf { S } } = ( { \mathsf { t } } , { \mathsf { u } } ) ^ { \top }
$$

$$

$$

其中I是 ${ \mathfrak { q } } \times { \mathfrak { q } }$ 阶的单位阵，O为 $\mathsf { q } \times ( \mathsf { m } - \mathsf { q } )$ 阶的零矩阵； $\pmb { \mathrm { n } }$ 为随机的观测值误差向量。

# 13.5全球协方差函数模型

扰动位的协方差函数 $\operatorname { K } ( \mathbf { P } , \mathbf { Q } )$ 被作为基本的协方差函数，利用协方差传播公式可以由它求得其它信号的协方差。但在实践中，人类掌握的却是地面上大量的重力异常数据，扰动位由重力异常求定。因此，实际上总是先求定重力异常的协方差函数。下面为了叙述方便，用C(P，$\mathbf { Q }$ )表示重力异常协方差函数。当以 $( 1 3 \cdot 5 4 )$ 式作为扰动位的协方差函数时，由 $( 1 3 \cdot 6 1 )$ 的第四式可以很容易地得到重力异常的全球协方差函数C(P,Q)具有如下的形式：

$$
\displaystyle \mathbf { C } ( \mathbf { P } , \mathbf { Q } ) = \sum _ { n = 2 } ^ { \infty } \mathbf { C } _ { \mathfrak { n } } ( \frac { \mathbf { R } ^ { 2 } } { \rho \rho ^ { \prime } } ) ^ { \mathfrak { n } + 2 } \mathbf { P } _ { \mathfrak { n } } ( \cos \psi )
$$

式中 $\mathbf { C _ { \mathfrak { n } } }$ 是常数，它与 $\mathtt { K } _ { n }$ 的关系为

$$
{ \bf C } _ { \mathfrak { n } } \equiv ( \frac { \mathfrak { n } - 1 } { \textsf { R } } ) ^ { 2 } { \bf K } _ { \mathfrak { n } }
$$

很容易证明,Cn就是&10.6中所讲的重力异常阶方差。将重力异常的阶方差模型(10·47)式代入，则得重力异常的全球协方差模型(二阶项另加）

$$
\mathbf { C } ( \mathbf { P } , \mathbf { Q } ) = \sum _ { \mathbf { n } = 3 } ^ { \infty } \frac { \mathbf { A } ( \mathbf { n } - 1 ) } { ( \mathbf { \bar { n } } - 2 ) ( \mathbf { n } + \mathbf { B } ) } \pmb { \sigma } ^ { \mathbf { n } + 2 } \mathbf { P } _ { \mathbf { \bar { n } } } ( \cos \psi )

其中

$$

$$

$$
{ \bf K } ( { \bf P } , { \bf Q } ) = { \bf R } ^ { 2 } \sum _ { { \bf n } = 3 } ^ { \infty } \frac { \bf A } { ( { \bf n } - 1 ) ( { \bf n } - 2 ) ( { \bf n } + { \bf B } ) } { \sigma } ^ { { \bf n } + 1 } { \bf P } _ { \bf n } ( \cos \psi )

# 13.8重力异常的局部协方差函数

扰动重力场中具有最多数据的信号是重力异常，因此对于局部地区，总是先找出重力异常的协方差函数，再由它导出其它信号的协方差函数。

如前面讨论扰动位的局部协方差函数那样，地面上重力异常的协方差函数也可以看成是距离 $\iota$ 的函数，即

$$
\mathsf { C } ( \mathsf { P } , \mathsf { Q } ) { = } \mathsf { C } ( l )

其中的l按(13·125)式计算。莫里兹提出[30],可以用三个基本参数来表征重力异常协方差函数的局部特征，那就是：方差C、相关长度S和曲率参数X(或梯度方差G)。它们的定义如下。

方差 $\mathbf { C } _ { \mathfrak { d } }$ 为协方差函数C(l)在 $\scriptstyle { l = 0 }$ 时的数值，即

$$
\mathtt { C _ { 0 } } \mathtt { = C } \mathtt { ( 0 ) }

它的大小体现了在本区域内重力异常的变化幅度。

相关长度S为协方差值为 $\frac { 1 } { 2 } \mathsf { C } _ { \circ }$ 时的 $\iota$ 值,参见图13.2,因此有

$$
\mathsf C ( \mathsf S ) = \frac { 1 } { 2 } \mathsf C _ { 0 }
$$

相关长度大说明本区域内重力异常的相关性强，反之相关性弱。通常，在0<l≤S的情况下进行推估才能取得较好的结果。

在实际操作时，逐步配置可以采用每次只增加一个新的观测值，即 $\iota _ { z }$ 中只包含一个观测值。只要连续地反复这个过程，所有的新观测值就可以都用进去了。这被称为序列配置。

举一个例子说明之。若共有50个观测值 $l _ { \mathrm { i } } ( _ { \mathrm { i } } { = } 1 , 2 , \cdots , 5 0 )$ ，参数向量X包含8个参数。第一步，取10个(必需大于8个)观测值作为第一组观测向量 $l _ { 1 }$ ，计算出 $\smash { \widetilde { \mathbf { X } } _ { \perp } }$ 和所需信号S的第一次估值;第二步，把第11个观测值作为一个分向量 $l _ { 2 }$ ,应用前面所讲的公式，对结果进行一次改正；第三步，再把第12个观测值，为 $l _ { 2 }$ ,再对结果进行修正；这样一直做下去，直到第50 个观测值用进去为止。当用这种方法计算时，只在求估值的第一次近似值时要进行矩阵的求逆。由于这个矩阵很小，很容易完成计算。以后的 $\overline { { \mathbf { C } } } _ { 2 2 }$ 只是一个量，它的逆就是这个量的倒数。而$\mathbf { P } ^ { - 1 }$ 可连续地由 $( 1 3 \cdot 1 9 6 )$ 式算出，因为前一步骤的 $\mathbf { E } _ { \mathbf { X } \mathbf { X } \cdot \mathbf { \boldsymbol { \iota } } _ { 1 } }$ 就是下一步骤的 $\mathbf { P } _ { 1 } ^ { - 1 }$ 。显然，虽然不做矩阵求逆，但却增加了很多的矩阵相乘，因此总的计算量并没有减少。事实上，逐步配置可以比作按分块矩阵求逆，而序列配置则相当于加边矩阵求逆。
