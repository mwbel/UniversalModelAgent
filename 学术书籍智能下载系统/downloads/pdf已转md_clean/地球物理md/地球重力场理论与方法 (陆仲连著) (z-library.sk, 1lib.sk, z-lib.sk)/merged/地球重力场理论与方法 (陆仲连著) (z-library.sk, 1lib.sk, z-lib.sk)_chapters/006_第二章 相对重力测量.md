# $\ S 2 . 1$ 用摆仪进行相对重力测量

用摆仪测定一点的绝对重力值时，其主要困难在于精确测定摆的归化摆长。若用摆仪测定两点的重力差值，则无需精确知道摆的归化摆长。现对原理论述于下。

由于两点的重力差是用观测周期差计算的，在两次观测中引起相同摆长变化的影响就可以免于校正。为了判断摆长是否变化，要求从起始点出发，在一个或几个待求点上完成观测以后，再回到起始点重新进行一次观测。在起始点上所得到的两次观测值的不符值的大小可以作为摆长是否变化的检查。为了减小摆长偶然变化的影响，在仪器中常常备有几个（三个或四个)摆，这些摆在每点上的周期差是否一致，也可以用来判断摆长是否发生变化。

自1881年奥匈帝国的测量家斯蒂尔涅克(R.Y.Sterneck)设计了一种较轻便的仪器之后，在欧洲很多国家都用摆仪进行了相对重力测量，并由许多技师对它进行了一系列的改良。至20世纪中叶，有过30多种不同类型的摆仪，测量精度由几个毫伽逐步提高到±0.2毫伽[24]

# 2.2进行相对重力测量的静力法

用摆仪进行重力测量时，观测的是运动着的摆，故被称为动力法。由于仪器构造复杂和笨重，观测的环境条件要求高，而且观测项目多和花费的时间长,数据处理也较繁琐，很难达到高精度和适应在广阔领土上的重力普查工作。

进行相对重力测量的静力法是利用一种力来平衡物体所受的重力，重力的变化将引起平衡位置的变化，只要用相应的方法测出平衡位置的变化，就可算出重力的变化，也就是测定两地的重力差。

早在20世纪初，德国地球物理学家海凯尔（O．Hecker）就曾用水银气压计和沸点温度计一起来测量重力差。虽然测量精度低于士0.02伽，但在当时却解决了海上重力测量的手段，也为用静重力仪测定重力开创了先例。

1913年德国的玛德逊（V．Madsen）在设计以气体的弹力平衡水银所受的重力为基础的气体重力仪时，选择了一种方法，它使温度变化得到补偿，以此代替测量气体的气温。在30年代，据此设计了一批仪器。它们的观测和计算只需几分钟，且能达到毫伽级的精度。图2.1为德国和哈尔克（H.Haalck）所制造的新型气体重力仪的比较详细的略图。它有两个水银柱AA和BB；五个容器V、V、V、V和V，其中V和V;连成一个；各容器内的气压分别为：V，内近于零，V和V内近于一个大气压，V4、V约两个大气压。V和V向上伸出的两个管子，在上部互相连通，但附有一个栓塞L。同时，L的两边弯下一个连通支管，分别有两个栓塞L和L；管内装有一滴液体M；管上刻有分划尺，用于读出液体滴M来回游动的位置。实际测量时，在已知点上将栓塞L、L2和L；都打开，使液滴M约居管的中央，然后将所有栓塞关闭。将仪器移至待测点，设该点的重力比已知点的小，则A,A和BB,两管水银的重量减小，故V内的水银面下降，V3内的水银面上升，由V来的气体压力大于Vz来的气体压力。此时打开栓塞L2和L，液滴M将往左移动。按照其位移量即可推算出两点的重力差。

以弹簧的弹力来平衡重力所制造的重力仪为弹簧重力仪，其理论是由别罗特(A.Berreth)于1932年建立的。野外仪器中最著名的是德国梯逊(St.Y.Thyssen）所创造的。美国的哈特利(K:Hartley）也制造了一个弹簧重力仪，其结构如图2.2所示。从仪器上盖垂直悬挂一个螺旋形弹簧R，它被称为主弹簧。在主弹簧的下端联着一个带有旋转点O的水平杠杆L，重锤P则安在杠杆下方。在杠杆左端有一个辅助弹簧T，它用来平衡主弹簧，其上端和测微螺旋V相联。S是反光镜，它依附在两个相互靠近的刀口上，这两个刀口中有一个是不活动的，与框F.联接;另一个处在杠杆L的一端。一束细细的光线射到镜子上，而后又射到分划尺M上。显然，重锤P和杠杆的重量变化都会引起镜子的转动，并使光线沿分划尺作相应的位移。旋转测微螺旋V，它改变弹簧T的张力，并使光线返回零分划线。这样，螺旋V上的读数差就是重力变化的大小。

# 2.4ZSM型重力仪的检验和调整

在将每台重力仪用于测量工作之前，必需对仪器作全面的检验和必要的调整，使其各个部件处于正确的位置和相应的工作状态，以确保测量工作的正常进行，取得好的测量结果。对于ZSM型重力仪，必需完成的检验和调整的项目及其方法如下。

# 一、面板位置的检验和调整

面板的正确位置应该是使水准器和脚螺旋之间保持如图2.12所示的状态。此时，在将横水准器调平后，再旋转纵向那个脚螺旋时，横水准器的气泡保持不动。如果发生移动，则说明没有处于正确的位置，应予调整。调整应由两人合作进行：一人用双手分握仪器外壳的两个提手，使仪器立稳不动；另一人用双手手指将仪器面板周圈的橡皮圈掀起，并抓住护圈缓慢地转动面板。但要注意，转动时切勿将仪器上提。操作完成后需再次作检验，直至达到转动纵向脚螺旋时，横水准器气泡居中不动为止。

# 三、横水准器的检验与调整

重力仪在工作时，弹性系统的石英扭丝 ${ \bf O } _ { 1 } { \bf O } _ { 1 } ^ { \prime }$ 必需处于水平位置。此时，摆杆所受的力才是重力 $\pmb { \mathrm { g } }$ 。如果倾斜一个 $\beta$ 角，则其所受的作用力是 $\tt { \tt g c o s } \beta$ ，它使测量结果带有误差。为此，仪器装有横水准器，当水准器轴与石英扭丝平行时,只要把横水准器调平，石英扭丝就处于水平位置。所以，在每次作业前，都必须对横水准器轴与石英扭丝是否平行作检验。如果不平行，则需进行调整，使之平行。

，检验所依据的原理是：扭丝倾斜时受力小，计数器读数小;扭丝水平时受力最大，计数器读数应该最大。为此，用一个横向脚螺旋改变扭丝的倾斜，找出其中读数最大的位置，这个位置就是扭丝处于水平的位置。此时，看一看横水准器气泡是否居中。若不居中，调节横水准器的调整螺旋，使气泡居中。这样，横水准器轴就与水平石英扭丝平行了。检验和调整的具体步骤如下：

1．将仪器整置水平，旋转测微螺旋使亮线与零线重合。

2．旋转横水准器方向的一个脚螺旋,使横水准器气泡分别向左和向右移动一格，并观察亮线运动的方向。注意：当横水准器气泡偏离中央时，纵水准器气泡应保持居中。

3.当横水准器气泡向左和向右偏离时，若亮线均由零线往左移动(这表示读数减小)，则横水准器不必调整。若横水准器气泡向左和向右偏离时，亮线向相反方向移动，则横水准器需要调整。

4.调整的操作方法是：旋转横向脚螺旋，使亮线向右移至最远；取下面板上的横水准器调节孔塞，插入调节改刀；旋转横水准器调整螺旋，使气泡居中。

5.重覆1一4步，反覆进行检验和调整，直至横水准器气泡向左和向右偏离时，亮线均由零线向左移动时为止。

# 四、纵水准器的检验和调整

纵水准器轴与摆杆都处于垂直于水平扭丝的平面内。当纵水准器气泡居中，用计数器测微螺旋使亮线与零线重合，则摆杆处于零位。在 $\ S \ 2 . 2$ 中已经指出，零位最好是水平位置。因此，纵水准器调整的最终目的是：当纵水准器气泡居中时，零位在水平位置。检验和调整的原理、方法和步骤与上述的横水准器检验和调整相类似。因为如果零位不在水平位置，则读数时摆杆不水平，此时摆杆受力小于重力，故读数变小。所以，用旋转纵向脚螺旋看亮线的走向就可检验出摆杆是否水平。在找到摆杆水平位置后，再用纵水准器调节螺旋把气泡调至中央，就完成了纵水准器的调整。

# 五、灵敏度的检验和调整

在 $\ S \ 2 . 2$ 已经讲到,对于ZSM型这一类的重力仪，灵敏度表示重力变化一个单位时摆杆产生的角位移。若灵敏度太大，由于摆杆不稳定，会造成观测困难;而灵敏度太小,则由于摆杆的角位移小而使测量精度变低。为此，仪器在出厂时都给定了一个仪器灵敏度的标准。ZSM 型重力仪的标准是指亮线由分划板的零线移到相邻的“1"线时,计数器上相应的读数差,这个差数应保持在16一20格。因为亮线移动表示摆杆的角位移，计数器上的读数与重力变化成正比，因此所给的标准实际上是 $\frac { \mathrm { d } \mathbf { g } } { \mathrm { d } \mathbf { a } }$ 乘一个常数，即与灵敏度的倒数成正比。所以这个读数差值越大则灵敏度越低；值越小，反而灵敏度越高。由此可见，灵敏度检验的方法是：将仪器整置水平，在检查纵、横水准器无误后，分别读取亮线与零线和“1"线重合时的计数器读数，并算出其差值，看是否符合标准值。只要大于20或小于16，都应进行灵敏度的调整。

# $\{ 2 . 5$ 重力仪的零点飘移

把一台重力仪放在固定的一点上，每隔一定的时间进行一次读数，结果是数值在不断变化，且时间间隔愈长，读数相差愈大，就像重力仪的零位置在不断的变化，所以称这种现象为零点飘移，简称零飘或掉格。出现零飘的根本原因是重力仪的弹性体长期受力的作用而产生弹性疲乏，造成弹性体永久形变的结果。零飘的大小及其稳定性与构成弹性体的材料有关，因此在设计制造重力仪时，应选用弹性疲乏较小且较规则的材料做弹性体。

现代重力仪的零飘一般都是与时间成正比的，但也有的会产生不规则的变化，如读数随时间的增加而减小，这种现象称为升格。这是因为弹性体本身永久形变的不规则，再加上整个弹性系统的结构以及外界不均匀的震动等因素的影响。

为了控制零飘，并通过计算零飘改正减弱它对观测结果的影响，在用重力仪进行相对重力测量时,要求在短时期内(数小时，最多一天)便要返回起始点或闭合至另一控制点。同时为了减小零飘的不规则变化，无论在测站或运输途中都应极力避免震动和温度变化。下面讨论零飘改正的计算方法。

先在已知重力值的A点进行观测,然后依次在未知点1、2、观测，最后闭合到另一已知重力值的B点观测,这一观测过程被称之为一测线。若以SA、S、S、、SB分别表示在各个点上的计数器读数，以tA、ti、t、、tB分别表示它们的观测时刻。如果没有零飘现象，则SB一SA乘以重力仪格值C就应该等于B点的重力值g减去A点的重力值g。由于零飘的存在,SB一SA与(gb一gA)/C不等，其差值就是在整个观测时间段tB一tA内总的零飘量。若视零飘量与时间成比例，并以K表示单位时间内的零飘量，则有

$$
\mathbf { K } { = } \frac { ( \mathbf { g _ { \mathrm { B } } } { - } \mathbf { g _ { \mathrm { A } } } ) \big / \mathrm { C } { - } ( \mathsf { S _ { \mathrm { B } } } { - } \mathsf { S _ { \mathrm { A } } } ) } { \mathbf { t _ { \mathrm { B } } } { - } \mathbf { t _ { A } } }
$$

它也被称为掉格率。有了掉格率就可算得该测线中任一观测值的零飘改正K(ti一tA),因此各点的正确读数应为：

$$
\left. \begin{array} { l } { { \bf S } _ { \mathrm { i } } ^ { \prime } = { \bf S } _ { \mathrm { 1 } } + \mathrm { K } \left( { \sf t } _ { \mathrm { 1 } } - { \bf t } _ { \mathrm { A } } \right) } \\ { { \bf S } _ { \mathrm { 2 } } ^ { \prime } = { \bf S } _ { \mathrm { 2 } } + \mathrm { K } \left( { \sf t } _ { \mathrm { 2 } } - { \bf t } _ { \mathrm { A } } \right) } \\ { . . . . . . . . . . . . . . . . . . . . . . . . . . } \end{array} \right\}
$$

如果一测线是由已知点A出发，最后又返回已知点观测，则掉格率的计算公式为：

$$
\mathbf { K } = - { \frac { \mathbf { S } _ { A 2 } - \mathbf { S } _ { A 1 } } { \mathbf { t } _ { A 2 } - \mathbf { t } _ { A 1 } } } \ .
$$

式中tAi和SA为在A点进行第一次观测的时刻和取得的读数,tA2和SAz为第二次在A点进行观测的时刻和取得的读数。各未知点上经零飘改正后的正确读数仍按(2.15)式计算，但其中的tA应取为tA1。

重力仪的零点飘移在运输条件下和在静止情况下是不一样的。为区别起见，前者称之为动掉格，后者称之为静掉格。由于两者有时差别较大，不能混在一起处理。为此，要求凡是在一测线中出现仪器静放1小时以上，应测定静掉格。即在重力仪动态结束和下一次动态开始时要分别进行观测，这两次观测读数之差即为该时间段内的静掉格。在计算整条测线的掉格率时，应将这段时间的静掉格去掉。具体计算方法可参见82.7的算例。

掉格率的大小及其数值的稳定性是判定重力仪质量的一个很重要的标准。掉格率小且数值变化不大的重力仪可认为是比较好的重力仪，用它测定重力差可以得到较好精度的结果。若仪器的掉格率大且数值变化大，这样的重力仪一般不能用于测量工作，尤其是不能用于长测线的测量。

# $\ S 2 . 6$ 重力仪格值的测定

所谓格值就是重力仪计数器的计数单位“格"所相应的毫伽数。当在两点上测得计数器读数差 $\Delta \mathsf { S }$ ，乘以格值C，即得两点的重力差 $\Delta \mathbf { g }$

$$
\Delta { \bf g } { = } { \bf C } \Delta { \bf S }
$$

如果格值有误差 $\mathfrak { m } _ { \mathtt { c } }$ ，依据 $( 2 \cdot 1 7 )$ 式，它使重力差产生的误差 $\mathbf { m } _ { \Delta \mathbf { g } }$ 为：

$$
\mathrm { \ m } _ { \Delta \mathrm { g } } { = } \Delta \mathrm { S } \mathrm { m } _ { c }
$$

将它与 $( 2 \cdot 1 7 )$ 式相除，则得

$$
\frac { \mathfrak { m } _ { \Delta g } } { \Delta g } = \frac { \mathfrak { m } _ { c } } { \mathrm { C } }
$$

(2.18)式表明，所测得的重力差的相对误差等同于格值的相对误差。反过来，可以根据所要联测的重力差的大小和对它的精度要求对格值的测定精度提出要求。如：要测的重力差可能达到1500毫伽,测量精度要求 $\mathrm { m } _ { \Delta \mathrm { g } } { \leqslant } \pm 0 . 1 5$ 毫伽，则要求格值的测定精度 $\frac { \mathrm { m _ { c } } } { \mathrm { C } } { \leqslant } \pm 1 \times 1 0 ^ { - 4 }$ ;若要联测重力差只有 500毫伽，测定精度要求 $\mathrm { m } _ { \Delta g } { \leqslant } \pm 0 . 5$ 毫伽，则格值的测定精度 $\frac { \mathrm { m } _ { \mathrm { c } } } { \mathrm { C } } { \leqslant } \pm 1 { \times } 1 0 ^ { - 3 }$ 就可以了。

由(2.17)式可得

$$
\mathrm { C } = { \frac { \Delta \mathbf { g } } { \Delta \mathbf { S } } }
$$

这个式子表明，如果两点的重力差 $\Delta \alpha$ 已知，用重力仪在这两点间测得计数器读数差△S，就可求得重力仪格值C。当然，已知的重力差值要有更高的精度才能保证所测定的重力仪格值的精度满足需要。这种事先用高精度相对重力测量求定的已知重力差被称为重力基线，在重力基线上测定重力仪格值的方法被称为基线法。

下面研究按基线法测得的格值的精度估算方法。从(2.19)式出发，考虑到 $\pmb { \Delta \mathbb { S } }$ 和 $\Delta \pmb { \mathrm { g } }$ 都有误差，根据误差传播定律可得：

$$
\frac { m _ { C } } { C } = \sqrt { ( \frac { m _ { \Delta s } } { \Delta S } ) ^ { 2 } + ( \frac { m _ { \Delta g } } { \Delta g } ) ^ { 2 } }
$$

# 2.7运用ZSM型重力仪进行相对重力测量

用 ZSM型重力仪测定的重力差值，其中误差一般在±(0.1-0.3)毫伽之间。因此，此类仪器只能用于大地重力测量中的二等控制点和加密点的测量。

在正式开始重力测量作业之前，需按2.4所述完成各项检验和调整，并测定仪器的格值常数。在进行测量的过程中，每到一测站的操作程序为：

1.选择一处质地坚硬且远离大树和高压线的平坦地方放置仪器；

2.接上电源,打开照明开关，调整脚螺旋使纵横水准器气泡居中，此时记薄员将手薄上端应填的项目写好；

3.转动计数器螺旋，以旋进方向使亮线与零线重合，读取计数器读数，至0.1格；

4．反覆上一步骤，直至读取三组互差不超过0.3格的读数为止,然后取出中数；

5.检查纵横水准器是否居中，允许偏离不大于0.5格，读取观测时间，然后切断电源；

6．由另一人对观测记录进行全面检查，如发现错误则应按上述步骤进行重测。

在一测站完成观测后就可移至下一测站进行观测。为了控制和计算零飘改正，要求从一已知点出发，在一个或几个待定点上观测后必需到另一已知点或返回原已知点观测，这构成一条

副线。完成一测线的时间应该尽可能的短。此外，在一测线的工作过程中还应注意下列几点：

1.若在一测线中,仪器静放时间超过1小时，应预动仪器，并测定静态零飘；

2.在测线观测全部完成之前，如发现记错、读错、仪器受到碰撞或有明显的读数奇变时，立返回前一站重测，并将重测测站上两次读数之差作为静态零飘处理；

3.测线中观测员不应更换，仪器除了测量范围调整外，不得进行其他项目的调整；

4.重力仪一般应置于点位中心,多台仪器同时观测时，亦可在标志东西两侧的对称位置；重复观测或检查观测时，应置于同一位置；

5．测线起闭点上的仪器高应保持同高。

全测线的观测结束后，再对全部记录进行一次全面检查，切实做到没有错误。然后，进行重力差计算。表2.2显示了一测线的观测记录及其重力差计算。计算项目有：

2.所有测站的读数的零飘改正；

3.求出每一测站的读数与已知点读数之差，再乘以格值即得相对于已知点的重力差。

·为了保证观测结果的准确和估算所得重力差的精度，在测定1等重力点时要求用两台仪器同时作两测线的观测，而对于加密点也至少用两台仪器测出两个结果。这样联测中误差就可以按贝塞尔公式计算，即

$$
\scriptstyle \mathbf { m } = { \sqrt { \frac { [ \mathbf { V } \mathbf { V } ] } { \mathbf { n } ( \mathbf { n } - 1 ) } } }
$$

式中m为一段重力差的联测中误差,n为该段重力差的观测结果的个数,V为该段重力差的最或是值与每个观测值的差数。如果m的数值超过规定的限差，则需进行重测。

当按支线布设时，所求点K的重力值gk可按下式计算：

$$
{ \bf g } _ { \bf \ l \ l } = { \bf g } _ { \bf A \ l } + \sum _ { i = 1 } ^ { \aleph } \Delta { \bf g } _ { i }
$$

式中gA为起始点的重力值,△g:为每测段重力差的最或是值。按误差传播定律可得，计算gk的中误差 $\mathbf { M } _ { \mathbf { K } }$ 的公式为：

$$
\bf { M } _ { \bf { k } } = \sqrt { M _ { A } ^ { 2 } + \sum _ { i = 1 } ^ { \bf { k } } { m _ { i } ^ { 2 } } }
$$

式中 $\mathsf { M } _ { \mathsf { A } }$ 为A点重力值的中误差， $\mathbf { m } _ { \mathrm { + } }$ 为按(2.24)式算得的第i段重力值的联测中误差。

若重力点的联测构成了闭合环，那末就产生了闭合条件。如：由一已知点出发，测了几段重力差后又回到原已知点，则所有各段重力差之和应为零；若由一已知点出发，测了几段重力差之后闭合至另一已知点，则所有各段重力差之和应等于两已知点重力值之差。但是，测量总是带有误差的，这就产生一个差值，这个差值称为闭合差。显然，闭合差的大小反映了重力差测量误差的大小。为了确保测量误差不超过规定的大小，必需对闭合差的数值规定一个限制。设各段重力差的联测中误差的限差为 $\mathbf { m }$ ，那末这些重力差之和的中误差一般不应超过 $\widehat { \mathbf { m } } \sqrt { \mathbf { \Lambda } _ { \mathbf { n } } }$ .其中$\pmb { \Omega }$ 为测段数。通常取两倍中误差为最大限差，因而规定闭合差不应超过 $2 \mathfrak { m } \sqrt { \mathfrak { n } }$ 。

既然存在闭合差，就要用平差方法予以消除。依据最小二乘法中的条件平差方法，在一个条件且认为各段重力差等精度的情况下，各段重力差的改正值 $\mathrm { \nabla V _ { i } }$ 是相等的，故有

$$
\mathbf { V } _ { 1 } = \mathbf { V } _ { 1 } = \cdots \cdots = \mathbf { V } _ { \mathfrak { n } } = - { \frac { \mathfrak { o } } { \mathfrak { n } } }
$$

式中 $\boldsymbol { \mathfrak { n } }$ 为测段数， $\omega$ 为闭合差。当由一已知点闭合至另一已知点B时，计算公式为

$$
{ \mathfrak { w } } = \sum _ { \mathrm { i } = 1 } ^ { \mathrm { n } } \Delta { \mathrm { g } } _ { \mathrm { i } } - ( { \mathrm { g } } _ { \mathrm { i } } - { \mathrm { g } } _ { \mathrm { a } } )
$$

如果由已知点A闭合回A点，则闭合差的计算式为

$$
\omega = \sum _ { \mathrm { i } = 1 } ^ { n } \Delta \mathbf { g } _ { \mathrm { i } }
$$

式中 $\Delta \mathbf { g } _ { \mathrm { i } }$ 为第i段的重力差， $\pmb { \ 8 } \mathbf { A }$ 和 $\mathbf { g _ { B } }$ 分别为A和 $\mathbf { B }$ 点的已知重力值。

$$
g _ { \mathrm { { K } } } \dot { = } \mathbf { g } _ { \mathrm { { A } } } + \sum _ { \mathrm { i } = 1 } ^ { \mathrm { K } } \Delta \mathbf { g } _ { \mathrm { { i } } } - \frac { \omega } { \mathrm { n } }
$$

K点与A点重力差的中误差 $\mathbf { m } _ { \mathbf { K } }$ 的计算公式为：

$$
\mathrm { m } _ { \mathrm { K } } = { \frac { \mu } { \sqrt { \mathrm { P } _ { \mathrm { K } } } \ . } }
$$

式中 $\pmb { \mu }$ 为单位权中误差，即

$$
\mu { = } \frac { \omega } { \sqrt { \textbf { n } } }
$$

$\mathbf { P _ { K } }$ 为K点与A点重力差的权，其计算式为

$$
\mathbf { P } _ { \mathbf { K } } { = } \frac { \mathbf { n } } { \mathbf { K } ( \mathbf { n } { - } \mathbf { K } ) }
$$

$$
\mathbf { m } _ { \mathbf { K } } { = } \frac { \sqrt { \mathbf { K } \left( \mathbf { n } { - } \mathbf { K } \right) } } { \mathbf { n } } \omega

最后， $\mathtt { g } _ { \mathtt { K } }$ 的中误差 $\mathbf { m } _ { \mathbf { K } }$ 的计算式为

$$

$$

# S2.yLCR型重力仪的检验和调整

运用LCR型重力仪进行相对重力测量之前，也必需对仪器进行检验和调整，使各个部件处于正确位置，以保证观测结果有好的精度。此项工作内容如下。

# 一、光学位移灵敏度的测定和调整

在上一节中讲到，灵敏度就是dβ/dg。计数器的读数变化就表示dg，而亮线在目镜分划尺上的移动量可以看成是dβ。仪器出厂时规定，当灵敏度接近于1个计数器单位相应于9到11个目镜分划尺分划，则是合格的。前面已经讲过，灵敏度太高或太低都不好。因此，必需对仪器的灵敏度进行测定，看它是否符合要求。测定的方法是：在整平仪器后，用亮线切准已知读数线左、右各一小分划线,读定相应的计数器与刻度盘的读数。例如：读数线为2.3，先将亮线切准分划线2.4,得读数8=2654.45；再将亮线切准分划线2.2,得读数δ=2654.26；又使亮线切准分划线2.4,得读数8'，=2654.45。由此可以算得位移灵敏度

当灵敏度偏低或偏高时，都需进行调整。因灵敏度与零位有关，故可通过调节纵水准器来达到目的。当 $\mathsf { q } < \mathsf { 9 }$ 时，降低仪器右端，使纵水准器气泡左移(离开目镜简方向)0.2格左右；而当 $\pmb { \ q } > 1 1$ 时，升高仪器右端，使纵水准器气泡右移0.2格左右，然后重新测定q值，直到数值合适为止。此时，松开纵水准器左侧压住盖板的螺丝，旋开盖板，露出调节孔;内有六方槽螺丝；再用配备的六方头螺丝刀来调节这个螺丝，使纵水准器气泡居中；完后将盖板复原：注意：在这过程中应始终保持横水准器气泡居中。

# 二、找出正确的读数线

如果纵水准器进行过调整，就可能会影响读数线的位置，需要找出正确的读数线。方法如下。

把仪器整平，将亮线移至仪器原规定的读数线。保持横水准器居中，用右侧脚螺旋使纵水准器气泡左移0.5格，读定视场中亮线所切准的分划线的数值 $\ S _ { 1 }$ ；再用同一脚螺旋使气泡右移1格，读定亮线切准的分划线的数值 $\ S _ { z }$ 。若 $\delta _ { 1 } { \approx } \delta _ { 2 }$ ，则表示读数线是正确位置；若不等，则说明读数线不正确。当 $\ S _ { 1 } { > } \Phi _ { 2 }$ 时，读数线偏大； $\delta _ { 1 } < \delta _ { 2 }$ 时读数线偏小。应据实际情况，在其旁重新选择一个分划线作为读数线，重复检查，直至找出正确的读数线为止。

# 三、横水准器的检查和调整

横水准器的正确安置可以使由于仪器微小倾斜所引起的影响无足轻重。检查的方法是：转动读数轮，使亮线切准读数线，读取刻度盘的读数 $\scriptstyle \mathbf { R } _ { 0 }$ ；转动左侧后脚螺旋，使横气泡分别向前和向后移动1格，读定亮线对准读数线时的刻度盘读数 $\mathbf { R _ { 1 } }$ 和 $\mathrm { R } _ { 2 }$ 。若 $\bar { \mathsf { R } } _ { 1 } - \mathsf { R } _ { 0 } = \mathsf { R } _ { 2 } - \mathsf { R } _ { 0 }$ 或者其中的小值与大值之比大于三分之二，则可认为横水准器位置正确。否则应进行调整，其步骤是：

1.转动左侧后脚螺旋，使横气泡偏移0.2格，其偏移方向应使亮线偏向于 $\scriptstyle \mathbf { R } _ { ! }$ 与 $\mathbb { R } _ { 2 }$ 之比值大的方向，然后重作检验，直至符合要求为止；

2.利用仪器配备的水准器改刀，调整横水准器使气泡居中，并重覆上面所述的检查过程。

# 四、电子读数零位和检流计零位的检验和调整

电子读数零位和检流计零位是采用电子读数时的基准，它应与光学读数的基准一—读数线相应。因此，当在视场中用亮线切准正确读数线时，输出电压在数字电压表上应显示为 $0 \pm$ $0 . 0 1 \mathbf { m } \mathbf { v }$ ，检流计指针应指 $^ { \prime \prime } 0 ^ { \prime \prime }$ 。如超出上述限差，则用小改锥调节"zero"孔内螺丝，直至正确位置。旋转方向顺时针为指针右移，反时针为左移。

# 五、电子灵敏度的测定与调整

电子灵敏度是电压表读数变化与计数器读数变化之比。当滤波盒的衰减系数为20倍时，在计数器读数变化10个单位时电压表读数变化应在15—20mv为宜。电子灵敏度的检查方法是：在亮线切准读数线的计数器读数两侧接近对称处，选取计数器读数差为10个单位的两个位置，分别读取电压表读数，看其读数差是否合乎要求。若不合要求，则需调整。调整的方法是：将小改锥插入面板上的“sens”孔内，旋转电位器调节螺丝。若需减小灵敏度，则反时针旋转；若需增大灵敏度，则顺时针旋转。调整后，应再次作检查，直至合乎要求为止。

摆杆是否在零位，通过一套光电装置来判断。这套装置的流程是：由元源34发出的光线经狭缝31以及物镜29射到与摆杆相联的反射镜25上；反射后，到达一个固定的反射镜28，再反射回25上；然后经过29射在一个微差光电池上。这个微差光电池的电路和一个灵敏的电流计32相联的，它的性能是：当微差光电池的两片感光体的表面上受到的光亮完全相等时，则电流计32中无电流通过；若受到的光亮不等,就会有电流通过电流计。电流计与一个小平面镜相连，有电流将使平面镜旋转。为反映出它的旋转,由另一光源发出的光线穿过小孔35,经两次反射到达这个平面镜，由它反射到电流计分划盘36上，在分划盘上呈现一个光点。当摆杆处于零位时，微差光电池不产生电流，此时电流计的反光镜在零位，电流计分划盘上的光点指在零。重力变化引起摆杆倾斜，反射镜25随之偏转，这使光电池因受光情况改变而产生电流。这股电流经导线传到电流计32的线圈中使小平面镜发生偏转，因而使得经小平面镜反射到电流计分划盘上的光点发生位移。旋转测微螺旋3改变读数弹簧7的张力调整摆杆的位置，使分划盘上的光点回到零，此时测微器4上的读数，再加上标尺上的读数，就是原始的观测数据。

# 2.12超导重力仪

在1911年荷兰物理学家卡麦林·翁纳斯(Kamerlingh.Onns）发现,水银冷却到稍低于4.2K(≈一269℃)时，其电阻急剧地下降到零。他把这种以零电阻为持征的电属状态叫做超导态,具有超导性能的物质称为超导体。以后人们发现，锡、铝等元素和锯钛、铅银等合金也都有上述的超导特性。

超导体有许多独特的物理特性，其中的完全导电性和完全抗磁性对制造重力仪最有价值。所谓完全导电性指的是：当物体处于超导状态时，可以承载很大的电流，只要这个电流不超过一定的限度，则因导体电阻为零而无能量消耗。因此，若将超导体作成回路，则电流能无衰减地永久流通下去，并产生一个极其稳定的永久磁场。所谓完全抗磁性指的是：对处于超导态的物体施加一个外磁场，只要不超过一定的强度，磁力线不能穿过这个物体。这是因为超导体表面可形成超导屏蔽电流而产生一种磁矩，起着排斥外磁场的作用。

1968年美国研制成超导重力仪，其结构如图  
2.22所示。有一个用铝制成的空心球，其外表镀

铝，是一个超导体。在其周围和下方分别安装有用超导锯丝绕成的线圈。依据超导体的完全导电性，当线圈接通电流后，马上再切断电流，线圈之间形成一个极其稳定的永久磁场。而由于超导空心球具有完全抗磁性，因而悬浮在磁场中，当悬浮力同作用在小球上的重力平衡时，超导球就静止在一个位置上。此时，超导线圈产生的磁场相当于其他类型重力仪中的弹簧，超导球相当摆杆。当重力发生变化时，超导球随之下降或上浮，且呈线性关系。它采用电容传感板测量反馈电压的方法测出超导球的位移量，从而求出重力的变化量。

整个系统放置在真空容器中，以避免空气对超导球的作用。然后将此容器浸在盛满液氢的杜瓦瓶内。杜瓦瓶用附加压缩系统进行冷冻，使重力仪温度保持4.2K，以保证锯丝线圈和小铝球处于超导状态。整个仪器又放在超导磁屏蔽的室内，以防止外界磁场的干扰。

超导重力仪具有很高的灵敏度，它可以反映出微伽级甚至更小的重力变化。它的零点飘移很小,约为每年5微伽左右，而且呈线性。因此，用它测量重力变化的精度可达微伽级。但因仪器体积和重量大，要求的环境条件高,维持超导状态的费用也很昂贵，目前只能用于重力台站记录重力随时间的变化。如果在常温下超导体的开发获得成功，可能会给制造小型的超导重力仪提供新的材料。
