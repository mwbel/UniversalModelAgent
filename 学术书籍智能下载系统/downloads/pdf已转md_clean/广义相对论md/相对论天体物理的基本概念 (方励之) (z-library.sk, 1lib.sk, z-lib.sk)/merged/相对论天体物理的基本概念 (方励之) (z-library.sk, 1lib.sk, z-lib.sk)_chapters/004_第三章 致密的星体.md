# $\beta _ { 3 } = 1$ 历史的记录

在很长的一个厉史时期中，亚里士多德学说，在欧洲占据着统治地位，亚里士多德在天文学上的基本观点是：宇宙是不变的.他认为，宇宙山一系列的有如水晶那样透明的天球组成，地球位于球心，各个球以不同的角速度绕着地球转动，行星及恒星就分别镶嵌在不同的水晶球体上.

在这种观念的支配下，不会认为天空图景会有任何反常的现象，更不会去留心天上的变化，甚至当天空中出现异常的现象时，仍然视而不见，在欧洲的历史文献中，缺乏有关天象变化的记录，其原因就在于此，科学史家Snrton曾说过，中世纪的欧洲人和阿拉伯人之所以未能认识变化的现象，不是因为它们难于进行观测，而是他们育目相信天体完美无缺的成见，以及思想上的惰性造成的.

因此，直到文艺复兴之后，欧洲才有这类的记录，1572年第谷观测到的超新星，在欧洲的科学思想的发展上起过很大的作用。第谷写道：“总之，我的结论是说这是一颗恒星（指1572超新星），不是任何可能发生的彗星，也不是某种可能发生的低于或高于月球的火流星，它正是自守宙开始以来，在以前的无穷岁月中出现于天穹的发光的星.”

中国的传统则不相同，中国古代的许多天文学家不排斥天象可以变化的观点，而且主张天体是常会有变化的，意大利教士利马窦（MatteoRicci）来到中国后，对中国的这科观念感到十分惊奇，当时他在一封信中写到："(中国人认为)只有一个天，而不是十重天，天上空虚无物，而不是固态的，星辰在太空中运动而不是固着于天穹之上.”
