# 8.6重力的层间改正和局部地形改正

地球质量的引力是地面点重力值的主要组成部分。因此，点周围的地形质量的去除或增加，都将使重力值发生变化。在这一节中，将讨论两种质量挪动所引起的重力变化。

# 一、重力的层间改正

如图8.5所示，P为研究点，dm为物质层中的单元质量,dm至P点的距离为p。依据万有引力定律,dm对$\mathbf { P }$ 点引力dF指向 $\mathbf { d m }$ ,大小为

二、重力的局部地形改正

所谓局部地形改正是过研究点P作一水平面,去除水平面以上的质量m,m等和补上水平面以下缺少的质量m2,m4等所引起的P点重力值的变化，见图8.6。由于在水平面上的质量(m,,m,)对P点的引力在重力方向上的分量为负值，去掉这些质量后将使P.点的重力值增加;在水平面下填补质量(m2,m,),也会使P点的重力值增加。因此，局部地形改正总是为正值。

局部地形改正8gn可以利用推求单元质量的引力在重力方向的分量(8·64)式导出,但在给定积分限时， $\pmb { z }$ 的积分上限应该是相对于研究点 $\mathbf { P }$ 的高差。显然，这个高差是另两个变量(r和A)的函数。由于这种函数关系不可能给出，故积分运算不可能严格完成。实践中，通常用一些近似计算方法来完成，以便获得所需精度的局部地形改正值。

：最普遍的近似计算方法是以有限求和代替积分，即将需去除和填补的地形质量分割成很多的扇形柱体，先计算出每块扇形柱体引起的改正8g，然后再总和起来。由于每个扇形代表较小的面积，在此面积内认为高程为常数(扇形的平均高)不会给计算结果带来很大的误差，而计算公式都很简单。由（8·64)式可以写出

$$
\hat { \boldsymbol { \Phi } } \mathbf { g } _ { \mathrm { { D } } } ^ { \mathrm { i , k } } = \mathrm { f D } \intop _ { \hat { \mathbf { A } } _ { \mathbf { k } } } \intop _ { \mathrm { ~ r _ i ~ } } ^ { \mathrm { ~ r _ i + 1 ~ } } \mathbf { r } ( \intop _ { 0 } ^ { \mathrm { ~ h _ { i , h } ~ } } \frac { z \mathrm { d } z } { ( \mathrm { r } ^ { 2 } + \mathrm { \mathbf { z } } ^ { 2 } ) ^ { 3 / 2 } ) } \mathrm { d } \mathbf { r }
$$

式中r;和ri十1分别为组成扇形柱体的内、外两个圆住的半径,Ax和Ak+是该扇形柱体两侧截面的方位角,hi.为该扇形较柱相对p点的平均高差，见图8.7。通常取

$$
\mathbf { A } _ { \mathbf { k } + 1 } - \mathbf { A } _ { \mathbf { k } } = \frac { 2 \pi } { \mathbf { n } _ { \mathrm { i } } }
$$

其中 $\mathfrak { n } _ { \mathfrak { i } }$ 为第i环带的扇形数。完成积分后得到

$$
\frac { 2 \pi \mathrm { f D } } { \bar { \bf n } _ { \mathrm { i } } } \mathbb { C } \mathrm { r } _ { \mathrm { i + 1 } } - \mathrm { r } _ { \mathrm { i } } - \sqrt { \mathrm { r } _ { \mathrm { i + 1 } } ^ { 2 } + \mathrm { h } _ { \mathrm { i , k } } ^ { 2 } } + \sqrt { \mathrm { r } _ { \mathrm { i } } ^ { 2 } + \mathrm { h } _ { \mathrm { i , k } } ^ { 2 } } \le \mathrm { r } _ { \mathrm { i } } + \mathrm { r } _ { \mathrm { i } } + \mathrm { r } _ { \mathrm { i } } \le 1
$$

而总的局部地形改正为

$$
\delta \mathbf { g } _ { \mathrm { D } } = \sum _ { \mathrm { i } } \sum _ { \mathbf { k } } \delta \mathbf { g } _ { \mathrm { D } } ^ { \mathrm { i } , \mathbf { k } } \left( 8 \cdot 7 1 \right)
$$

划分扇形柱体所采用的圆柱内、外半径的大小和n值的大小都可以任意选定。显然，扇形小些，把扇形的高程看作常数所引起的误差小，计算结果精度高，但计算工作量大；反之，扇形大，结果精度低，计算工作量小。一般说来，离P点附近的区域扇形应小些，越远则可越大些。近百年来，常用的内、外半径值和相应的扇形数如表8.2所示。

由此人们想到，若在去除大地水准面以外的质量的同时，在大地水准面内部填补一些质量，以减小大地水准面上P,点的重力位的变化。今以Wc表示填补质量对P。点的引力位，则间接影响的计算式变为

$$
\delta \mathsf { N } { = } \frac { \mathbb { W } _ { \Gamma } { - } \mathbb { W } _ { \Gamma } } { \mathbb { \gamma } _ { \mathfrak { g } } }
$$

在1905年鲁茨基(RudzkiM.P.)提出-一种逆点法，即对去除的大地水准面以外的任--个单元质量dm，在大地水准面的内部填补上一个单元质量dm'，它们对P点的引力位相等，而两个单元质量在同一向径上。这样一来，间接影响就为零了。然而，虽然调整大地水准面和真正的大地水准面是重合了，但大地水准面外部的重力场却变化了。因此，也不是一种可行的方法。

如果只对地面观测重力值g加空间改正0.3086H，减去正常椭球面上相应点的正常重力值 $\gamma _ { _ { \scriptscriptstyle ( j ) } }$ ，所得的数值称为空间异常。若以 $\Delta \mathbf { g }$ 表示空间异常，则有

当用空间异常代入司代克斯公式(8·33)推求大地水准面高时，间接影响只有1米左右。这是因为：对地面观测重力值加空间改正相当于将大地水准面以上的物质垂直下移H这一距离，如图8.12所示。由于在大地水准面下部填补了质量，因而使间接影响大为缩小。但由于空间异常与地形的关系明显，用内插方法推求未知点的空间异常值要达到好的精度比较困难。

人们倾向于根据地壳均衡理论来填补大地水准面以下的质量，由此所得的异常称为均衡异常，用它来计算大地水准面高较为适当。认为这种模型能最好地适应地质的实际情况；异常值很小，变化平缓，和地形无关，既适合于内插也可以外推；间接影响适中而不太大，约10m的数量级。因此，在下两节中将对地壳均衡理论进行论述。

# $\$ 8.8$ 地壳均衡假说

完全布格异常在山区是系统的负值，而且数值相当大，平均每1000m的高度绝对值增大约100毫伽。唯一的可能解释是山底下的质量有所欠缺，高出海平面的地形质量以某种方式被补偿。

垂线偏差的大小也有类似的现象。实际的垂线偏差值比可见的地形质量所造成的影响要小得多。这也只能用内部质量有某种补偿来解释。

几乎是在同一时间，有两种不同的理论产生，被用于说明这种补偿现象。这就是普拉特(PrattJ.H)在1854年和爱黎(AiryG.B)在1855年各自提出的地壳均衡理论。现分别叙述于下。

# 一、普拉特-—海福特系统

这种补偿系统的概念是由普拉特提出的，后来由海福特引进数学公式，并系统地用于大地测量。

根据普拉特的假说，地壳的下界面在平均海水面下的深度在各处都是-一样的,通常用D表示这个深度。在地壳下有岩浆，要使它达到静力平衡，就必需保持地壳对岩浆的压力在地壳下界面的各点上都是相等的。也就是地壳的下界面应是一个均衡面，因而D就是均衡面的深度，简称均衡深度。起伏的地形表明，地壳的厚度不是一个常数。若以H表示地表在海水面上的高程，则地壳厚度为D十H。为了保证在下界面上的压力处处相等，必需使一定切面的各个垂直柱体中的质量为一常数。

![](images/05a033d01364c44490c1713ff84e0a2682d48cc01f365cd37800abb438ef5459.jpg)  
图8.13

因此，海底地壳密度的增量为

如图8.13所示，设厚度为D的柱体的物质密度为P，而厚度为D+H的柱体的质量密度为p，依据普拉特假说，有如下方程成立：

$$
( \mathrm { D } + \mathbf { H } ) \varrho = \mathrm { D } \varrho _ { \mathrm { \scriptscriptstyle t } }
$$

因此，对于 $\mathsf { H }$ 为正的大陆地区，地壳的实际密度ρ一定小于标准密度p。由(8·85)式可得，密度差为

$$
\Delta \rho { = } \rho _ { \mathrm {  i } } - \rho _ { \mathrm { 6 } } { = } \frac { \mathrm { H } } { \mathrm { D } { + } \mathrm { H } } \rho _ { \mathrm { i } }
$$

而在海洋地区，满足平衡的条件为

$$
( \mathrm { D } { - } \mathrm { H } ^ { \prime } ) \rho { + } \mathrm { H } ^ { \prime } \rho _ { \mathrm { w } } { = } \mathrm { D } \rho _ { \mathrm { 0 } } \ .
$$

式中 $\mathbf { H ^ { \prime } }$ 是海深， $\rho _ { \mathbb { W } }$ 是海水的密度 $( 1 . 0 2 7 g { \tt c m } ^ { - 3 } )$ 。

$$
\rho - \rho _ { \mathrm { 0 } } = \frac { \mathrm { H } ^ { \prime } } { \mathrm { D } - \mathrm { H } ^ { \prime } } \rho _ { \mathrm { 0 } } - \rho _ { \mathrm { w } }
$$

根据这种假说，山岳不是某种不相干的附属物需要消除，而是地壳的组成部份，是由地壳下层膨胀而成的。那些高出海水面的物质是由深处升起的，上部质量的多余正好等于下部质量的不足。只有保持这个条件地壳才均衡，单纯消除外部质量就会破坏这种均衡性。

20世纪初期，海福特在将普拉特假说运用于美国重力测量和大地测量的数据处理时，对其稍作了改变。他把高出海水面的那部份地壳的密度都取为标准密度值p。此时，山岳的质量仅仅被地下部份密度的减少所抵偿，基本方程 $( 8 \cdot 8 5 )$ 变为

$$
\mathrm { D } \boldsymbol { \rho } \boldsymbol { + } \mathrm { H } \boldsymbol { \rho } _ { \mathrm { 0 } } \boldsymbol { = } \mathrm { D } \boldsymbol { \rho } _ { \mathrm { 0 } }
$$

下部密度的差值为

$$
\Delta \rho { = } \frac { \mathbf { H } } { \mathbf { D } } \rho _ { \mathrm { 0 } }
$$

这给实际计算带来了方便。在实际计算时他采用p=2.67gcm-3，而D曾用过122km、113.7km和 $1 0 2 \ k m$ 等不同的数值。

# 二、艾黎—-海斯卡宁系统

这个系统的假说是由艾黎提出的，而由海斯卡宁给出实际应用于大地测量的计算公式。

在艾黎的假说中，地壳的密度是个常数，取为p，它漂浮在密度为pi的岩浆上。当然，pi>Po,且向上隆起得越高，则向下陷入得越深。因地壳的物质密度比岩浆的密度小，上部质量起着补偿深部质量不足的作用。

![](images/b6e35c4d1b9705f8e6f164129c49d95a4686eeeb13d3282bf1027af0ee1d13e7.jpg)  
图8.14

如图8.14所示，从地壳割出断面等于一个单位的柱体，上端与海水面相一致的柱体高度称为地壳厚度，用T表示之。对于高出海水面H的柱体，它在海水面以下的那部份质量的厚度必然大于T，其超过部份称为山根，用t表示之。根据漂浮的平衡条件有

$$
{ \mathfrak { t } } \left( { \pmb { \rho } } _ { 1 } - { \pmb { \rho } } _ { 0 } \right) = { \mathrm { H } } { \pmb { \rho } } _ { 0 }
$$

对于海洋地区,相应的条件为

$$
\mathbf { t } ^ { \prime } \left( \boldsymbol { \rho } _ { \mathrm { 1 } } - \boldsymbol { \rho } _ { \mathrm { 0 } } \right) { = } \mathbf { H } ^ { \prime } \left( \boldsymbol { \rho } _ { \mathrm { 0 } } { - } \boldsymbol { \rho } _ { \mathrm { w } } \right)
$$

式中t'为用以抵偿海水部份质量不足的岩浆厚度。由 $( 8 \cdot 9 2 )$ 式可得

$$
\mathbf { t } ^ { \prime } = \frac { \rho _ { \mathrm { 0 } } - \rho _ { \mathrm { w } } } { \rho _ { \mathrm { 1 } } - \rho _ { \mathrm { 0 } } } \mathbf { H } ^ { \prime }
$$

在 20 世纪二三十年代，海斯卡宁将此假说应用於重力测量和大地测量的数据处理工作。它采用的地壳厚度T=30km，岩浆密度取为3.

$2 7 g c m ^ { - 3 }$ ，而地壳密度也是取 $2 . 6 7 g \mathrm { c m } ^ { - 3 }$ 。

由于这一原因，威宁·曼尼兹将艾黎的漂浮学说，在1931年作了修改，引进了区域补偿以代替局部补偿。这两种补偿的差异如图8:15所示。按照威宁·曼尼兹的理论，地形质量是作为一种加在不断裂而有弹性的地壳层上的负荷。此时，可认为整个地块是均衡的，而其各部份不是都均衡的。地块上有山、凹地以及地表面的所有其他地貌。因此，不是补偿地貌的各个零碎部份，而是根据整个地区的负载(以其平均高程决定)使整个地区均衡起来。虽然威宁·曼尼兹对艾黎假说的改进使之更为实际，但比较复、杂，在大地测量中还很少用它。

一般说来，地壳应力求达到静水状的均衡。但可举出许多例子说明完全均衡是不存在的，也是不可能存在的。例如，在原苏联的莫斯科和切尔尼科夫地区，地势平坦，却观测出极大的重力异常。这重力异常表明，深藏在地壳内部而又没有任何外部标记的重质量过多。即使在创立地壳均衡假说的印度，现代的大地测量和重力测量工作的结果，几乎也都没有完全证实这个理论。该地区重力异常的主要因素是隐蔽山脉的引力，这个隐蔽山脉的密度要比周围地区的地壳密度大，并且几乎与喜马拉雅山平行，并极靠南面，而在地表的地形上却毫无显露。上述现象说明，实际的地壳结构及引起重力异常的原因，较之均衡假说中所阐述的要复杂和更多样。虽然如此，均衡理论还是反映了部份真象，因而对于重力测量和大地测量的数据处理还是有一定用处的。

# 第九章 研究地球重力场的现代理论

# 9.1莫洛琴斯基问题

所有的大地测量工作，包括水准测量、三角测量、天文测量和重力测量等都是在地球表面上进行的。而在随后的数据处理时，经典的方法是光归算到比地球表面平滑得多的大地水准面上。如果测量数据分布在较大的范围内，则还需进行第二次归算，即由大地水准面归算到更便于数学运算的椭球面上。此时，求定大地水准面就是一项必不可少的工作。然而,在利用司托克斯理论推求大地水准面时，从理论上说必需知道存在于大地水准面外的地壳物质的密度。但人们掌握的有关信息还不能满足这个要求，而只能采用一些假说来弥补。虽然这些假说的不真实性所造成的影响不是很大，但至少在理论上是不严格的，而且也使精度难于提高。原则上，地面上的大地测量数据可以直接归算到椭球面上，不一定要采用大地水准面作为过渡面。当然，必需知道地球表面与椭球面之间的相对关系。

为此，原苏联的学者莫洛琴斯基(M.C.MoJIogeHkun)在1945年提出了一种直接研究地球表面及其外部重力场的理论，它不需要使用任何有关地壳密度的信息。这一理论具有十分重要的意义，这一发展开创了研究地球形状及其外部重力场的新阶段。因此，在此处将他和此后的一些学者的有关这方面的研究成果称之为研究地球重力场的现代理论。

为了说明利用地面测量数据可以求定地球表面及其外部重力场，需要先证明一个定理：如果已知物体的外表面S、总质量 $\mathbf { M }$ 、物体绕某一固定轴作匀速自转的角速度 $\pmb { \omega }$ 和S面上任一点相对于面上某一固定点的重力位差 $\mathbf { \bar { w } _ { s } } - \mathbf { \bar { w } _ { \rho } }$ ，则S面外和面上的重力位W及其导数可以唯一地确定，而无需已知其内部质量的分布情况。

证明采用反证法。先假定在两种质量分布的情况下存在两个重力位，分别用 $\mathbf { W } ^ { \prime }$ 和W"表示之，然后证明它们之差

$$
\delta \mathbf { W } = \mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime \prime }
$$

在物体表面S上和面外处处为零。

依据已知条件，在S面上任意点相对某一固定点的重力位差 ${ \bf W _ { s } ^ { \prime } } - { \bf W _ { \theta } ^ { \prime } }$ 和 $\mathbf { W _ { s } ^ { ' } } - \mathbf { W _ { \eta } ^ { ' } }$ 都应与已知值相等。因此,在S面上的8W之值由 $( 9 \cdot 1 )$ 式可得

$$
\mathbf { \Delta } \widetilde { \mathbf { d } } \mathbf { W } \vert _ { \mathbf { s } } = \mathbf { W } _ { \mathbf { s } } ^ { \prime } - \mathbf { W } _ { \mathbf { s } } ^ { * } = \mathbf { W } _ { \mathbf { \eta } _ { 0 } } ^ { \prime } - \mathbf { W } _ { \mathbf { \eta } _ { 0 } } ^ { * }
$$

因为 $\mathbf { W } _ { \mathrm { ~ \scriptsize ~ \mathfrak ~ { ~ \mathfrak { ~ . ~ } ~ } ~ } } ^ { \prime }$ 和 $\mathbf { W } _ { \mathrm { ~ 0 ~ } } ^ { \eta }$ 分别为S面上固定点的两种重力位值，其差值为一定值。若用C表示这一定值，则得

$$
\delta \mathbf { W } \mid _ { \mathbf { s } } = \mathbf { C }
$$

该式表明，在S面上任一点的δW值都为同一数值。另一方面，由于质量分布不同只使引力位存在差异，而离心力位不会发生变化。因此，8W只是引力位差，因而它在S面外是个调和函数。在格林第一恒等式(5·7)中，取 $\mathbf { U } = \mathbf { V } = \delta \mathbf { W }$ ，考虑到 $\Delta \delta \mathbf { W } = 0$ ，则可写出

由 $( 5 \cdot 8 0 )$ 和 $( 4 \cdot 8 8 )$ 式，对两个重力位 $\mathbf { w } ^ { \prime }$ 和 $\mathbf { \mathbf { \mathbf { W } } } ^ { \prime \prime }$ 可以分别写出
