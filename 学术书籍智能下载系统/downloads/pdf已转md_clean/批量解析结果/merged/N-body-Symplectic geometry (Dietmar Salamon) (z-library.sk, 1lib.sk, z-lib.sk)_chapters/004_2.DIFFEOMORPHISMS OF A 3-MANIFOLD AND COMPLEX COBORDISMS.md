# 2.DIFFEOMORPHISMS OF A 3-MANIFOLD AND COMPLEX COBORDISMS

By a complex symplectic surface $z$ we mean a differentiable 4-manifold endowed with a complex structure and a holomorphic symplectic form $\pmb \theta \in$ $\Omega _ { z } ^ { 2 , 0 }$ ,i.e. $\pmb \theta$ is holomorphic and $\theta \wedge { \overline { { \theta } } } = 2 \Omega$ is a volume form.If we write $\pmb { \theta } = \pmb { \theta } _ { 1 } + i \pmb { \theta } _ { 2 }$ ，where $\pmb { \theta _ { 1 } } , \pmb { \theta _ { 2 } }$ are real 2-forms, the algebraic constraints imposed at each point are

$$
\theta _ { 1 } \wedge \theta _ { 1 } = \theta _ { 2 } \wedge \theta _ { 2 } \neq 0 , \theta _ { 1 } \wedge \theta _ { 2 } = 0 .
$$

The 2-forms $\pmb { \theta _ { 1 } , \theta _ { 2 } }$ are closed,so they may be viewed as a pair of ordinary symplectic structures on the 4-manifold $\pmb { z }$ . Conversely any pair of closed 2- forms which satisfy the algebraic conditions (2) define a complex symplectic structure. To see this, observe that at each point $\theta _ { 1 } + i \theta _ { 2 }$ is a null 2-form and hence,via the Plucker embedding,defines a subspace $T ^ { 1 , 0 } \subset T Z \otimes { \bf C }$ and so an almost-complex structure on $\pmb { z }$ .If $N$ is the Nijenhius tensor of this structure, a tensor in $T ^ { 1 , 0 } \otimes ( T ^ { 0 , 2 } ) ^ { * }$ ，then $\pmb { d \theta }$ has two components, one $\overline { { \partial } } \theta \in \Omega _ { Z } ^ { 2 , 1 }$ and the other $N \pmb { \theta } \in \Omega _ { Z } ^ { 1 , 2 }$ If $d { \pmb \theta } = { \bf 0 }$ both of these components vanish and this can only happen if $N$ is zero and $\pmb \theta$ is holomorphic.

Now let $z$ be a complex symplectic surface with boundary, i.e. the forms are smooth up to the boundary,and suppose the boundary has a component $\boldsymbol { Y }$ . Suppose we are given a volume form $\pmb { \alpha }$ on $\boldsymbol { Y }$ .Then we can defne a pair of vector fields $v _ { 1 } , v _ { 2 }$ on $Y$ as follows. We restrict the 2-forms $\theta _ { 1 } , \theta _ { 2 }$ to $Y$ and then compose with the isomorphism between 2-forms on $\boldsymbol { Y }$ and vector fields,given to us by the volume form. Thus

$$
i _ { v _ { 1 } } ( \alpha ) = \theta _ { 1 } | _ { Y } , i _ { v _ { 2 } } \alpha = \theta _ { 2 } | _ { Y } .
$$

Another way of saying this is that we choose a Hamiltonian $\pi$ on $\pmb { z }$ ,constant on the boundary component $\boldsymbol { Y }$ and with $d H \wedge \alpha$ equal to the volume form on $\pmb { z }$ ,over the boundary. Then we let $v _ { 1 } , v _ { 2 }$ be the usual Hamiltonian vector fields generated by this function and the two symplectic structures $\pmb { \theta _ { 1 } } , \pmb { \theta _ { 2 } }$ Either way, it is clear that the vector felds are volume-preserving on $\boldsymbol { Y }$ ， that is

$$
\bar { L } _ { v _ { i } } \alpha = 0 ,
$$

where $\pmb { L }$ denotes the Lie derivative. We can put the vector fields together to define a single section $v = v _ { 1 } + i v _ { 2 }$ of the complexified tangent bundle $\pmb { T Y } \otimes$ C. Suppose now that $\pmb { z }$ has two boundary components $Y ^ { + }$ ， $Y ^ { - }$ with volume forms. We get complex vector fields $v ^ { + } \in \Gamma ( T Y ^ { + } \otimes \mathbf { C } ) , v ^ { - } \in \Gamma ( T Y ^ { - } \otimes \mathbf { C } )$ We write, somewhat prematurely, ${ \cal Z } ( v ^ { - } ) = v ^ { + }$ ，and we say that $v ^ { + } , v ^ { - }$ are cobordism related. We will argue that this notion is closely comparable to that of $G ^ { c }$ related complex vector fields, introduced above,where $\pmb { G }$ is the group of volume-preserving diffeomorphisms of $Y$ ，and we use the adjoint representation. It is possible that the two relations are identical if we consider cobordisms which are topological products. In one direction, suppose we start with a solution of equation (1) in this context. That is,we have a 3-manifold $Y$ with a volume form $\pmb { \alpha }$ and volume-preserving vector fields $v _ { 1 } , v _ { 2 } , v _ { 3 }$ on $\boldsymbol { Y }$ depending on a real parameter $t \in [ - a , a ]$ ， and such that

$$
\frac { d } { d t } ( v _ { 1 } + i v _ { 2 } ) = [ i v _ { 3 } , v _ { 1 } + i v _ { 2 } ] .
$$

We suppose also that $v _ { 1 } , v _ { 2 } , v _ { 3 }$ are linearly independent at each point. We will define a complex symplectic structure on $Y \times [ - a , a ]$ using this data. For each “time” $t$ ，let $\epsilon _ { 1 } , \epsilon _ { 2 } , \epsilon _ { 3 }$ be the basis of 1-forms on $\boldsymbol { Y }$ dual to the vector fields ${ \boldsymbol { v } } _ { 1 } , { \boldsymbol { v } } _ { 2 } , { \boldsymbol { v } } _ { 3 }$ . Then we can write

$$
\alpha = f \ { \epsilon } _ { 1 } \wedge { \epsilon } _ { 2 } \wedge { \epsilon } _ { 3 }
$$

for a real-valued function $f \circ \mathbf { n } Y \times [ - a , a ]$ .We define 2-forms on $Y \times [ - a , a ]$ by

$$
\begin{array} { l } { { \theta _ { 1 } = f ( d t \wedge \epsilon _ { 1 } + \epsilon _ { 2 } \wedge \epsilon _ { 3 } ) } } \\ { { \theta _ { 2 } = f ( d t \wedge \epsilon _ { 2 } + \epsilon _ { 3 } \wedge \epsilon _ { 1 } ) . } } \end{array}
$$

It is clear that, just as a matter of algebra,

$$
\theta _ { 1 } ^ { 2 } = \theta _ { 2 } ^ { 2 } = f d t \wedge \alpha ~ , ~ \theta _ { 1 } \wedge \theta _ { 2 } = 0 .
$$

We claim that $\theta _ { 1 } , \theta _ { 2 }$ are closed on $Y \times [ - a , a ]$ if (and only if） the vector fields $v _ { 1 } , v _ { 2 }$ satisfy the equation (3). This is a calculation which is expedited by the use of the following identity.If $\xi , \eta$ are vector felds on some manifold $M$ and $\phi$ is a form on $\pmb { M }$ then

$$
d ( i _ { \xi } i _ { \eta } \phi ) = i _ { [ \xi , \eta ] } \phi + i _ { \eta } L _ { \xi } \phi - i _ { \xi } L _ { \eta } \phi + i _ { \xi } i _ { \eta } d \phi .
$$

This can be seen by applying the formula $L _ { X } = d i _ { X } + i _ { X } d$ twice. In our situation,we take the vector fields $\pmb { \xi } = v _ { 2 } , \eta = v _ { 3 }$ on $\boldsymbol { Y }$ ,at a fixed time $t$ ， and let $\phi = \alpha$ . Then, since the fields are volume-preserving, we get

$$
d ( i _ { v _ { 2 } } i _ { v _ { 3 } } \alpha ) = i _ { [ v _ { 2 } , v _ { 3 } ] } \alpha .
$$

Now note that $i _ { v _ { 1 } } \alpha = f \epsilon _ { 1 } \wedge \epsilon _ { 2 }$ and $i _ { v _ { 2 } } i _ { v _ { 3 } } \alpha = f \epsilon _ { 1 }$ . The form $i _ { v _ { 1 } } \alpha$ is closed since ${ \boldsymbol { v } } _ { 1 }$ is volume-preserving. Let us write, temporarily, $\underline { d }$ for the exterior derivative on forms over the 4-manifold ${ \cal Y } \times { \bf R }$ , and reserve $\pmb { d }$ for the exterior derivative on forms over $Y$ ,with the time regarded as a parameter. In this notation we have

$$
{ \underline { { d } } } \psi = d \psi + d t \wedge { \frac { d \psi } { d t } } .
$$

Now the exterior derivative of $\pmb { \theta _ { 1 } }$ is

$$
\underline { { d } } \theta _ { 1 } = d t \wedge ( d ( f \epsilon _ { 1 } ) + \frac { d } { d t } ( f \epsilon _ { 2 } \wedge \epsilon _ { 3 } ) { \bf \alpha } ) + d ( f \epsilon _ { 2 } \wedge \epsilon _ { 3 } ) .
$$

As we have noted above, $f \epsilon _ { 2 } \wedge \epsilon _ { 3 }$ is closed on $\boldsymbol { Y }$ ，s0 $\underline { { d } } \pmb { \theta _ { 1 } = 0 }$ if and only if

$$
d ( f \epsilon _ { 1 } ) + { \frac { d } { d t } } ( f \epsilon _ { 2 } \wedge \epsilon _ { 3 } ) = 0 ,
$$

that is if and only if

$$
i _ { [ v _ { 2 } , v _ { 3 } ] } \alpha = \frac { d } { d t } ( i _ { v _ { 1 } } \alpha ) .
$$

But the volume form $\pmb { \alpha }$ is independent of time, so

$$
\frac { d } { d t } ( i _ { v _ { 1 } } \alpha ) = i _ { \frac { d v _ { 1 } } { d t } } \alpha .
$$

Thus $\pmb { \theta _ { 1 } }$ is closed if and only if $\begin{array} { r } { \frac { d v _ { 1 } } { d t } = [ v _ { 2 } , v _ { 3 } ] } \end{array}$ ， which is the real part of equation (3). Similarly, $\pmb { \theta _ { 2 } }$ is closed if and only if $\begin{array} { r } { \frac { d v _ { 2 } } { d t } = [ v _ { 3 } , v _ { 1 } ] } \end{array}$ : thus a solution to the equation (3) gives a complex symplectic structure on the cylinder $\textstyle z = Y \times \left[ - a , a \right]$ . If we now apply our construction on the boundary we get precisely the complex vector fields $( v _ { 1 } + i v _ { 2 } ) ( \pm a )$ on the two boundary components,so $Z ( ( v _ { 1 } + i v _ { 2 } ) ( - a ) ) = ( v _ { 1 } + i v _ { 2 } ) ( a )$ . Thus we see that $G ^ { c }$ related vector fields are also cobordism-related

We will now set up the converse problem. We would like to show that any complex symplectic structure on a cylinder arises from this construction. While we will not reach any conclusive results in this direction, we will explain the nature of the partial differential equation which is involved. Observe that in the set-up above the vector fields

$$
v _ { 1 } , v _ { 2 } , v _ { 3 } \ { \mathrm { ~ a n d ~ } } v _ { 4 } = { \frac { \partial } { \partial t } }
$$

on $Y \times [ - a , a ]$ each preserve the volume form α Λ dt. So the rescaled fields $f ^ { - 1 } v _ { i }$ preserve the intrinsic volume form $\Omega = \theta _ { 1 } ^ { 2 } = \theta _ { 2 } ^ { 2 } = f \alpha \wedge d t$ . Now let $\pmb { z }$ be a complex symplectic manifold,diffeomorphic to the cylinder. Suppose we choose a $\mathsf { \Omega } ^ { \mathsf { s t i m e } ^ { \mathfrak { n } } }$ function $\tau : Z \to [ - a , a ]$ ,without critical points,equal to $\pm a$ on the boundary components. This gives a foliation of $\pmb { z }$ into level sets $Y _ { t } = \tau ^ { - 1 } ( t )$ . Using $\tau$ as a Hamiltonian function,for the two symplectic structures we get volume-preserving vector felds ${ \boldsymbol { w } } _ { 1 } , { \boldsymbol { w } } _ { 2 }$ on $z$ , tangent to the leaves $Y _ { t }$ . Suppose we can find another vector field ${ \pmb w } _ { 4 }$ which is transverse to these leaves and such that

(1) ${ \pmb w } _ { 4 }$ and $w _ { 3 } = I w _ { 4 }$ are volume preserving on $\pmb { z }$ (2) ${ \pmb w } _ { \mathbf 3 }$ is tangent to the leaves $Y _ { t }$ ,i.e $\nabla _ { w _ { 3 } } \tau = 0$

Then the complex structure on $\pmb { z }$ arises from a solution of (3) by the construction above. To see this,we define $f = ( \nabla _ { v _ { 4 } } \tau ) ^ { - 1 }$ and set $v _ { \ i } = f w _ { \ i }$ for $i = 1 , \dots 4$ ,and $e _ { i } = f ^ { 1 / 2 } w _ { i }$ . Then

$$
\theta _ { 1 } ( e _ { 1 } , e _ { 4 } ) = f \theta _ { 1 } ( w _ { 1 } , w _ { 4 } ) = f \nabla _ { v _ { 4 } } \tau = 1 ,
$$

and

$$
\theta _ { 2 } ( e _ { 1 } , e _ { 4 } ) = \theta _ { 1 } ( e _ { 1 } , I e _ { 4 } ) = 0 .
$$

So the pair of vectors $e _ { 1 } , e _ { 4 }$ make up a standard complex basis of the tangent space with respect to the complex symplectic form 0. Now the vector field ${ \pmb v } _ { 4 }$ satisfies $\nabla _ { v _ { 4 } } \tau = 0$ ， so we may integrate it to define a product structure on $\pmb { z }$ ,i.e.to fix an identification $Z \cong Y \times [ - a , a ]$ ，such that the function $\tau$ is the time component, and $\begin{array} { r } { v _ { 4 } = \frac { \partial } { \partial \tau } } \end{array}$ . Since the vectors ${ e _ { i } }$ give a standard basis at each point there is a single algebraic model for the situation and we can write

$$
\theta _ { 1 } = f ( d t \wedge \epsilon _ { 1 } + \epsilon _ { 2 } \wedge \epsilon _ { 3 } ) ~ , ~ \theta _ { 2 } = f ( d t \wedge \epsilon _ { 2 } + \epsilon _ { 3 } \wedge \epsilon _ { 1 } ) ,
$$

for certain time-dependent 1-forms $\epsilon _ { \mathrm { i } }$ on $\boldsymbol { Y }$ , dual to the vector fields $v _ { 1 } , v _ { 2 } , v _ { 3 }$ We define a volume form $\pmb { \alpha }$ on $\boldsymbol { Y }$ by $\alpha \wedge d t = f \Omega$ ，Then the vector fields ${ v } _ { i }$ preserve the volume form $\pmb { \alpha }$ ，since the fields ${ \mathfrak { w } } _ { \mathfrak { i } }$ preserve $\pmb { \Omega }$ . Thus we have recovered the set-up we had before,and our calculation shows that the vector fields ${ v } _ { \bar { \mathbf { \nu } } }$ satisfy the evolution equation (3), since the forms $\pmb \theta _ { \dot { \imath } }$ are closed.

We will now translate the conditions on the vector feld ${ \pmb w } _ { 4 }$ into a more handy notation. We will frst make a digression to point out a simple but special construction which can be made on any complex symplectic surface $z$ . We let $w$ be the space of vector fields $\pmb { w }$ on $\pmb { z }$ such that both $\pmb { w }$ and ${ \bf \nabla } _ { I w }$ are volume-preserving. On any $\pmb { n }$ -manifold the volume-preserving vector fields are naturally identified with the closed $( n { - } 1 )$ -forms, so we can identify $w$ with 3-forms $\lambda$ on $\pmb { z }$ such that $d \lambda = d ( \bar { I } \lambda ) = 0$ . If we identify the real 3-forms $\lambda$ with the (2,1)forms $\pmb { \mu }$ by $\lambda = \mu + \overline { { \mu } }$ ，then we identify $w$ with the space of (2,1) forms $\pmb { \mu }$ with $\overline { { \partial } } \mu = 0$ . Now use the symplectic form $\pmb \theta$ to write such a form as a product $\mu = \theta \wedge \beta$ where $\beta$ is of type $( \mathbf { 0 } , 1 )$ . In this way we identify $w$ with the space of (0,1)-forms $\beta$ with $\overline { { \partial } } \beta = 0$ . Now if $\pmb { H }$ is any complex-valued smooth function on $z$ we construct such a form $\beta$ by $\beta = { \overline { { \partial } } } H$ . So we get a map

$$
m : C ^ { \infty } ( Z ) \to \mathcal { W }
$$

which is surjective if $H ^ { 1 } ( Z : { \mathcal { O } } _ { Z } ) = 0$ . This is obviously analogous to the Hamiltonian construction in symplectic geometry. We can take the analogy further by noting that $w$ is naturally a complex Lie algebra. We define a bracket $[ \ , \ ] _ { c }$ in $w$ by

$$
[ X , Y ] _ { c } = [ X , Y ] + [ I X , I Y ]
$$

It is clear from the definition of $w$ that $[ \ , \ ] _ { c }$ does map $w \times w$ to $w$ It satisfies the Jacobi identity because the complex structure is integrable. (The integrability condition precisely asserts that the real vector fields are a subalgebra of the complexified vector fields under the embeddding $x $ $\pmb { \chi } _ { + i I X }$ ,and $[ , ] _ { c }$ is induced from the complexification by this embedding.) Now we define a “Poissson bracket” $\{ \ : , \ : \} _ { c }$ on the complex valued functions by

$$
\{ F , G \} _ { c } \theta \wedge \overline { { { \theta } } } = \overline { { { \partial } } } F \wedge \overline { { { \partial } } } G \wedge \theta .
$$

Some short calculations show that $\{ \ , \} _ { c }$ makes $C ^ { \infty } ( z )$ into a complex Lie algebra,and that $\pmb { m }$ is a Lie algebra homomorphism. Thus we get a kind of complexified Hamiltonian mechanics on a complex symplectic surface. (Which should not be confused with the more standard holomorphic complexification,in which holomorphic functions on $\pmb { z }$ generate holomorphic vector fields preserving $\pmb \theta$ ）

We now return from our digression to the question of finding the transverse vector field ${ \pmb w } _ { 4 }$ satisfying the conditions above. The first condition, that ${ \pmb w _ { 4 } }$ and ${ \bar { I } } w _ { 4 }$ are volume preserving, means that ${ \pmb w _ { 4 } }$ is identifed with a $\overline { { \partial } }$ closed (0,1)-form $\beta$ . The second condition becomes

$$
\mathrm { I m } \left( \beta \wedge \overline { { { \partial } } } \tau \right) = 0 .
$$

We must also consider boundary conditions,on the two boundary components $Y ^ { \pm }$ . We suppose that we have given volume forms $\alpha ^ { \pm }$ on the boundaries,and we want our solution to be compatible with these. This is a condition on ${ \pmb w } _ { 4 }$ only,independent of $\pmb { \tau }$ ， since the volume form induced on the leaves is $\alpha _ { t } = i _ { w _ { 4 } } ( \Omega )$ . Thus $\beta$ should satisfy an extra condition on the boundary. To see this condition, recall that the boundary of a complex manifold has a CR structure,a “complex tangent space” $\pi$ which in this case is just the space spanned by $v _ { 1 } , v _ { 2 }$ with the complex structure induced from $\pmb { z }$ . Thus the volume forms on the boundary and the complex symplectic form give a trivialisation of the complex tangent bundle $\pmb { H }$ . In particular, there is a canonical section $\overline { { \epsilon } } = \epsilon _ { 1 } - i \epsilon _ { 2 }$ of $\overline { { H } } ^ { * }$ and the boundary condition is just that the restriction of $\beta$ to $\pmb { H }$ be equal to ε.

To sum up,suppose $v ^ { \pm }$ are cobordism-related vector fields on $\boldsymbol { Y }$ ,so there is a complex symplectic cobordism $z$ with volume forms on the boundary realising these vector fields. The boundary data gives sections $\overline { { \epsilon } } ^ { \pm }$ of $\overrightharpoon { H } ^ { * }$ over $Y ^ { \pm }$ .If $\pmb { z }$ is a topological cylinder then to show that $V ^ { \pm }$ are $G ^ { c }$ related one has to find a $\overline { { \partial } }$ closed (0,1)-form $\beta$ and a foliating function $\pmb { \tau }$ such that that $\beta$ restricts to $\overline { { \pmb { \epsilon } } }$ on the boundary and $\overline { { \partial } } \tau \wedge \beta \wedge \theta$ is real throughout $\pmb { z }$ The question of the existence of $\beta$ alone is cohomological in character: it is a question of extending the class of ε in the first cohomology of the $\overline { { \partial } } _ { b }$ complex on the boundary to a class in $H ^ { 1 } ( Z , { \mathcal { O } } _ { Z } )$ : if one extension $\beta _ { 0 }$ exists we can modify it to $\beta = \beta _ { 0 } + \overline { { \partial } } H$ ，for any function $H$ which vanishes on the boundary. Suppose now that we fix a $\beta _ { 0 }$ and also an arbitrary foliating function $\pmb { \tau }$ . Then we need to solve the remaining equation

$$
\mathrm { I m } ( \overline { { { \partial } } } H \wedge \overline { { { \partial } } } \tau \wedge \theta ) = - \mathrm { I m } ( \beta _ { 0 } \wedge \overline { { { \partial } } } \tau \wedge \theta ) ,
$$

with $H = 0$ on $\partial \boldsymbol { z }$ . This is an underdetermined linear equation: there is one real equation for two real functions $A , B$ with $\begin{array} { r } { H = A + i B } \end{array}$ . The equation is essentially 3-dimensional, it can be written more explicitly using the vector felds ${ \boldsymbol { w } } _ { 1 } , { \boldsymbol { w } } _ { 2 }$ along the leaves $Y _ { t }$ as

$$
\nabla _ { v _ { 1 } } A + \nabla _ { v _ { 2 } } B = Q ,
$$

with $Q$ given and $A , B$ to be found. It seems likely that there always is a solution to this underdetermined equation and this,along with the extension of ε,would imply that the two complex vector felds are $G ^ { c }$ -equivalent.

3.NAHM'S EQUATIONS,HYPERKAHLER METRICS AND OTHER TOPICS.

We will now explain how one can formulate the analogue of the main result about Nahm's equations in this context.We will begin by recalling the result of Ashtekar. If we have three time-dependent volume-preserving vector feldson $\boldsymbol { Y }$ which satisfy Nahm's equations $\begin{array} { r } { \frac { d v _ { i } } { d t } = [ v _ { j } , v _ { k } ] } \end{array}$ ti $( i , j , k$ cyclic) then $v _ { 1 } , v _ { 2 }$ satisfy (3), and hence define a complex symplectic structure on $\cal Y \times ( - a , a )$ . But the symmetry in the equations means that the same goes for the other pairs $v _ { 2 } , v _ { 3 } \ , \ v _ { 3 } , v _ { 1 }$ . Thus we get a family of complex symplectic structures on the product space and it is clear that these are algebraically compatible with a single Riemannian metric $\pmb { g }$ . It follows straightaway that $\pmb { g }$ is a hyperkahler metric,with the three Kahler forms $\theta _ { 1 } , \theta _ { 2 } , \theta _ { 3 }$ . The vector field ${ \pmb w _ { 4 } }$ now becomes the gradient vector field of the time function $\pmb { \tau }$ . This function must be harmonic, since ${ \pmb w _ { 4 } }$ is volume preserving. Conversely if we are given a hyperkahler manifold $\pmb { z }$ and a harmonic function $\pmb { \tau }$ on $\pmb { z }$ we use the gradient vector field grad $\pmb { \tau }$ , away from critical points, to define a product structure. Now suppose we have cobordism related complex vector fields $v ^ { + }$ ， $v ^ { - }$ on $Y$ ,i.e. we have a complex symplectic manifold $\pmb { z }$ , which we suppose is diffeomorphic to a cylinder,with volume forms on the boundary components, and $v ^ { + }$ ， $v ^ { - }$ are the vector fields induced on these two boundary components. We can ask if $v ^ { + } , v ^ { - }$ are related by a solution of Nahm's equations,as in the finite dimensional case. We can verify this if we can find a suitable hyperkahler structure on the cylinder $z$ . What we need is a kahler metric $\pmb { g }$ on $z$ ，with the given volume form $\pmb { \Omega }$ ，and a harmonic function $\pmb { \tau }$ (with respect to this metric) equal to $\pm a$ on the two boundary components. Of course this harmonic function is uniquely determined, if $z$ is compact by the metric and boundary data. The final condition we need is that the volume forms on the boundary agree with those induced by $\pmb { \tau }$ In the notation we have used above, the volume form on $\pmb { z }$ is

$$
\Omega = f ^ { 2 } ( \epsilon _ { 1 } \wedge \epsilon _ { 2 } \wedge \epsilon _ { 3 } \wedge d \tau ) ,
$$

s0 $f ^ { 1 / 2 } \epsilon _ { i } , f ^ { 1 / 2 } d \tau$ form a standard orthonormal frame of 1-forms at each point.Thus the Riemannian gradient of $\pmb { \tau }$ had norm

$$
| \mathrm { g r a d } \ \tau | ^ { 2 } = f ^ { - 1 }
$$

and the 3-volume form on $\boldsymbol { Y }$ is

$$
\alpha = f \epsilon _ { 1 } \wedge \epsilon _ { 2 } \wedge \epsilon _ { 3 } = | \mathrm { g r a d } \ \tau | \ \mathrm { V o l } _ { 3 } ,
$$

where $\mathrm { \Delta V o l } _ { 3 }$ is the Riemannian volume element on the $\mathbf { 3 - }$ dimensional leaves. This is just the $\widetilde { \mathbf { \Gamma } } \widetilde { \mathbf { \Gamma } } \widetilde { \mathbf { \Gamma } } \widetilde { \mathbf { \Gamma } } \widetilde { \mathbf { \Gamma } } \widetilde { \mathbf { \Gamma } }$ of the gradient vector feld grad $\pmb { \tau }$ through the leaves, or in terms of differential forms $\alpha = * d \tau$ . To sum up, the problem of solv-ing Nahm's equations on $\pmb { z }$ comes down to the following. For any compact Riemannian manifold $( X , g )$ with boundary, whose boundary has two com-ponents $\partial ^ { \pm } X$ we define a measure $\pmb { \eta _ { g } }$ on the boundary by the fux of the unique harmonic function on $\pmb { X }$ with boundary values $\pm \pm$ .Then for our given hypercomplex $z$ ，with volume form $\pmb { \alpha }$ on the boundary，we seek a kahler metric $\pmb { g }$ with the prescribed 4-volume element $\pmb { \Omega }$ in the interior and with $\eta _ { \pmb { g } } = \pmb { \alpha }$ on the boundary.

It seems possible that this may be an interesting kind of boundary value problem to investigate: it is reasonable to hope for existence and uniqueness in the solutions since we can regard the equations in the interior as a second order elliptic equation for 2 real-valued functions (the function $\pmb { \tau }$ and a Kahler potential for $\pmb { g }$ ),and the boundary conditions represent two additional equations over the boundary,which is as one would want.

Finally we should mention another problem where similar constructions seem to be interesting. This is is the case where one takes a real 2- dimensional symplectic manifold $( \Sigma , \omega )$ and the group $\pmb { G }$ to be the central extension of the exact area-preserving diffeomorphisms of $\Sigma$ ，with Lie algebra $C ^ { \infty } ( \Sigma )$ under the Poisson bracket: compare [9], for example. One can look for the structure on a 3-dimensional cobordism $U$ between two such surfaces $\Sigma ^ { \pm }$ which will correspond to the adjoint action of the missing complexified group. The appropriate structure should enable one to find:

(1) a complex 2-form $\Phi$ on $\boldsymbol { U }$ which is real on the boundary， giving symplectic forms $\omega ^ { \pm } \in \Omega ^ { 2 } ( \Sigma ^ { \pm } )$ (2）a complex function $H = H _ { 1 } + i H _ { 2 } : U  \mathbf { C } ;$ such that

$$
d \Phi = d ( H \Phi ) = 0 \mathrm { i n } U .
$$

Identifying the appropriate structure would give a notion of the “adjoint action”of $U$ ,taking $H | _ { \Sigma ^ { - } }$ to $H | _ { \Sigma ^ { + } }$ . One can also look at Nahm's equations in this context,which essentially comes down to studying harmonic functions i $\mathbf { R ^ { 3 } }$ ,sincethe equations $\begin{array} { r } { \frac { d h _ { i } } { d t } = \{ h _ { j } , h _ { k } \} } \end{array}$ for afamilyof maps $\underline { { h } } ( t ) : \Sigma  \mathbf { R } ^ { 3 }$ just assert that the images of $\Sigma$ are the level sets of a harmonic function.

An interesting feature of this second example is that there are many invariants on the Lie algebra. In the ordinary Lie algebra of real-valued functions one knows that for each $\pmb { p }$

$$
I _ { p } ( h ) = \int _ { \Sigma } h ^ { p } \omega
$$

is preserved by the adjoint action. One would like there to be corresponding invariants in the complexification, and this would follow if one could find a definition with the properties mentioned above.For if we have such a structure detailed above on a cobordism $\boldsymbol { U }$ then it follows from Stokes' Theorem that

$$
\int _ { \Sigma ^ { + } } H ^ { p } \omega ^ { + } = \int _ { \Sigma ^ { - } } H ^ { p } \omega ^ { - } .
$$

Something similar occurs for the higher dimensional case we have been studying in this paper. Suppose the 3-manifold $Y$ is a real-homology 3- sphere.Then we can define a quadratic form $Q$ on the Lie algebra of the

group $\pmb { G }$ of volume-preserving diffeomorphisms as follows: we identify the Lie algebra with the closed 2-forms,and for any such form $\pmb { F }$ we choose a 1-form $\pmb { A }$ such that ${ \pmb { F } } = d { \pmb { A } }$ . Then we set

$$
Q ( F ) = \int _ { Y } F \wedge A .
$$

It is easy to see that this does not depend on the choice of A. We would like the complexification of this invariant quadratic form to be preserved by the “action”of a complex-symplectic cobordism $\pmb { z }$ ,and this is the case as long as the symplectic form $\pmb \theta$ is zero in $H ^ { 2 } ( Z , { \bf C } )$ . For then we can write $\pmb \theta _ { i } = d A _ { i }$ over $\pmb { z }$ ,and

$$
Q ( v _ { i } ^ { + } ) - Q ( v _ { i } ^ { - } ) = \int _ { \partial Z } \theta _ { i } \wedge A _ { i }
$$

which is

$$
\int _ { Z } \theta _ { i } \wedge d A _ { i } = \int _ { Z } \theta _ { i } ^ { 2 } = \mathrm { V o l } ( Z ) .
$$

Thus ${ \mathrm { R e } } ( Q ( v ^ { + } ) - Q ( V ^ { - } ) ) = 0$ since $\theta _ { 1 } ^ { 2 } = \theta _ { 2 } ^ { 2 }$ on $z$ and similarly one sees that the imaginary part vanishes since $\theta _ { 1 } \wedge \theta _ { 2 } = 0$

It is worth pointing out that the three diffeomorphism groups we have considered in this paper are related: a diffeomorphism of a circle induces an area-preserving diffeomorphism of the 2-dimensional cotangent bundle, and an exact area-preserving diffeomorphism of a surface lifts to a volumepreserving diffeomorphism of a circle bundle over the surface.
