藏族传统数学不仅有其独特的记数工具，而且采用的方法多种多样，有实物记数、文字数码记数和符号数码记数等。实物记数的工具取材于手指（据藏族天文历算学家桑珠加措等编《天文历算学》记载：“约公元前1世纪吐蕃第九代赞普布德贡加时的藏族古史《亚桑的故事》（）中说到‘纺织老人月算（)。”[2]）、集攒小木棍、小石子、羊粪、绳子、佛教徒念经用的念珠等；文字数码记数为7世纪初松赞干布时吞弥·桑布扎创造了藏族自己通用的文字，随之产生了文字数码方法；符号数码记数产生的时代也比较久远。目前发现的藏族数码分为三大系列：吐蕃时代系列（1984年5月，黄文焕先生在从雍布拉康所藏吐蕃时代藏文写经的若干经页中发现藏族数码，且经考证认为这套数码是8～10世纪的吐蕃数码）；中世纪系列（黄文焕先生1984年底在山南地区乃东县格茹拉康整理经卷时，在一本微残的中世纪藏文目录中发现这套数码并经考证认为是中世纪数码）；近现代系列（1978年9月在甘肃省敦煌县博物馆发现这套数码，经考证认为是近现代数码)。3]藏学家拉巴平措先生认为，世界上著名的记数符号主要有巴比伦记数法、古埃及记数法、中国古代记数法、古希腊记数法、古罗马记数法、古印度记数法、中美洲地区古代记数法。其中，阿拉伯数码已成为当今世界通行的数码。虽然在上述各种记数法中没有列出藏族数码，但通过比较,拉巴平措先生指出藏族数码和阿拉伯数码之间存在着明显的联系，[4]并对藏族数码在学术史上的地位和学术价值作了以下结论：

1、现在全世界通行的阿拉伯数码与藏族数码之间存在着某种非常密切的渊源关系；2、如果现在世界通行的阿拉伯数码来源于印度数码，那么藏族数码和印度数码之间的关系也是密不可分的；3、藏族数码和印度数码之间分不清“源”和“流”的关系，就目前而言，藏族数码保留了更多的历史痕迹；4、对藏族数码及其数学的研究是藏学研究领域尚未开垦的处女地。以上说明藏族数码是一个古老而深奥的数码系统。

# (二)藏族传统数学的数位、数词及表数方法

# 1、数位

藏族传统数学中出现最多的进位制是十进位制，逢十进一（也有百位制）；而数位（以数字记录各个大小数量前后的位置)在藏族古代典籍中记法不同，多寡不等，在《世界数目》中收有19个数位；《阿毗打磨》中收有60个数位；《胜树庄严经》中收有132个数位等；《大方广菩萨藏文殊师利根本仪轨经》（E5舌5）中收有69个数位。[5]现在，藏族数学的数位一般统括为60位，这些数位都是以十进制为单位。事实上从个位到亿位与通用数学进位制的命名相同，但通用数学中更大的数一般用 $1 0 ^ { n } ( n = 1 0 , 1 1 , 1 2 , \cdots )$ 表示，并没有相应的数位名；而藏族数学的独特之处在于数位从亿位（ $1 0 ^ { 9 }$ 位）以后的每一位阿庾多至无量数（）都有其相应的位数命名，而且还在天文历算中继续使用亿位以后（ $1 0 ^ { 1 0 } \sim 1 0 ^ { 6 0 }$ 位)的命名。

# 3、藏族传统数学中的九九乘法口诀

1985年，西北民族学院华侃先生在法国人伯希和（Pual Pelliot） $1 9 0 7 \sim 1 9 0 8$ 年从敦煌劫走、现藏于巴黎国家图书馆的藏文写卷$\mathrm { P } \cdot \mathrm { T } 1 2 5 6$ 号《敦煌吐蕃文书选集》第一辑中发现一篇用藏文音译汉文的九九乘法表。据现代学者研究，原文为藏文，无标题，未署抄写人姓名。全文为横书,共8行，它的次序是从“九一九皆为九”)开始，到“一一如一，圆空零”（可）终结。经研究，华侃先生认为这个写卷可能成书于吐蕃中期或晚期，即8世纪中叶或更后一些。而藏文九九乘法表出自《时轮历精要》，该书写成于第十四“饶迥”之火猪年，即1827年。

最近，我国考古专家在湖南张家界古人堤汉代遗址出土的简牍上发现汉代“九九乘法表”，这枚简牍是木质的，大约22厘米长，残损比较严重。湖南省考古研究所副研究员张春龙先生认为这批简牍大都残损，字迹模糊，难以辨认，但根据推断，这枚九九乘法表的排列顺序应当始于九九八一运算；张春龙先生说，这是他第二次在考古发掘中发现简牍上记载有“九九乘法表”。第一次是2002年6月他在湘西里耶古城出土的一枚秦简上发现了距今2,200多年的乘法口诀表，并被考证为我国现今发现的最早的乘法口诀表实物。中国文物研究所的专家说，除了里耶秦简外，与张家界古人堤遗址发现的这枚简牍样式基本一致的“九九乘法表”还曾在楼兰文书中见到过，那是写在两张残纸上的九九乘法表，为瑞典探险家斯文赫定在上个世纪初期发掘。中国科学院自然科学史研究所所长刘钝说，乘法表在古代并非中国一家独有，古巴比伦的泥版书上也有乘法表，汉字（包括数目字)单音节发声的特点，使之读起来朗朗上口。后来发展起来的珠算口诀也承继了这一特点，对于运算速度的提高和算法的改进起到一定作用。

结合汉藏历史上的文化交流和“九九乘法表”的发现情况，我们可以初步认为藏、汉“九九乘法表”极有可能有历史的渊源关系，但它们又有不同之处。藏族传统数学中的“九九乘法口诀”的主要特点是：（1）从大数开始、小数结束，与现今惯用的乘法表顺序相反；(2）藏族九九乘法口诀是两数双向相乘，与惯用的九九乘法口诀明显不同；（3）藏族九九乘法口诀中同时带有“交换律”概念，“交换律”是通用数学中直到现在都具有重要作用的算律，它在计算简化过程中起着相当重要的作用。

# 4、藏族传统数学中的石子算法和沙盘算法

藏族传统数学从数字产生后，不仅算术理论有了长足的进步，其计算工具上也有重要的发展，从而有利地推动和保证了人们生活、生产实践的发展和需要。

# （1）石子算法

藏族传统数学中的石子算法是世界范围内一种独特的经济应用数学。藏学家诺章·吾坚先生认为藏族传统数学中石子算法的理论体系成熟于第五世达赖喇嘛时期，当时地方政府“孜康”（5)里有一位叫阿南达或称贡嘎的学者，写了一本《八章石子算术》）,被认为是藏族传统数学理论体系的开端。9]这一理论体系也是为便于当时地方政府“孜康”的工作，在计算过程中要靠算具小方黑石、果核、蚕豆、瓦片和小方木棍来分别表示个位数、十位数、百位数、千位数和万位数。其具体表示法是：若集有10个小方黑石就要进一个十位数，用1个果核表示，20个小方黑石就用2个果核，以此类推；集成10个果核就进一个百位数，用1个蚕豆表示；集成10个蚕豆就进一个千位数，用1片瓦片表示；集成10片瓦片便进一个万位数，用1根小方木棍表示，等等。在加、减、乘、除四则运算中的用法是：加：2个果核5个小方黑石加上4个果核6个小方黑石等于7个果核1个小方黑石，即71个小方黑石或7.1个果核；减：8个果核5个小方黑石减去3个果核6个小方黑石等于4个果核9个小方黑石，即49个小方黑石或4.9个果核；乘：8个果核5个小方黑石乘以3，等于2个蚕豆5个果核5个小方黑石，即255个小方黑石或25.5个果核或2.55个蚕豆；除：2个蚕豆4个果核6个小方黑石被3除，等于8个果核2个小方黑石，82个小方黑石或8.2个果核。

总之，藏族石子计算器既可以进行复杂的计算，同时又能将这种算法套用在具体解决问题上，因此民间曾普遍使用该计算器来解决实际生活中的很多问题。

从以上例子，我们可以明显地看出藏族沙盘算法的独特之处：第一，加法运算步骤是从右往左算，而乘除减法从左往右算，这与通用数学中的算法不同；第二，沙盘运算时，心算、口算和书写可同时进行，在计算过程中，写数字和涂数字同步进行，互不影响，使用方便、快速准确，远远超过笔算。它在藏族科学、经济和文化事业的发展中起到独特作用。

# .5、几何图形的出现与发展

和数的概念一样，形的概念也是从外部世界得来的。在昌都、卡若等遗址中发现不同时期的石器和陶器，对这些出土文物从几何图形上分析可知，早在石器时代藏族已有圆、椭圆、矩形、圆柱体等几何形态，但没有真正形成几何理论体系意义上的几何概念；后来这些几何形态在藏族唐卡和壁画中出现，不仅形成了理论概念，而且在唐卡和壁画制作过程中使用诸如中心对称、轴对称、成比例、等腰三角形等几何术语，乃至在现实生活中普遍使用了“三角形的稳定性”等原理。说明藏族传统数学中几何图形的出现及其理论概念及内容形成早、发展快，但真正数学意义上的藏族传统“几何理论体系”是否存在，

尚待进一步研究。

# 三、结语

对藏族传统数学的发展历史和内容进行系统的整理和分析后 我们可以得出以下结论：

以算法为中心 属于应用数学 藏族传统数学不脱离社会生活与生产的实际 以解决实际问题为目标 数学研究是围绕建立算法及提高计算技术而展开的 具有较强的技巧性 比如 藏族传统数学文化中的 石子算法 和 沙盘算法 在应用过程中 若不懂其相应的技巧和口诀 就无从下手 反之 技巧和口诀熟练的人应用起来便快速准确 远远超过笔算 藏族传统数学文化中的 石子算法 和 沙盘算法 专业性强 要靠老师的指点才能学会 寓理于算 理论高度概括 藏族传统数学注重解决实际问题 但这并不意味着藏族传统数学仅停留在经验层次而无理论建树 其实藏族传统数学的几何实践中蕴涵着建立这些几何理论的基础 几何概念与方法隐含在少数几个不证自明 形象直观的数学原理之上 如平面几何中的 三角形的稳定性 原理等
