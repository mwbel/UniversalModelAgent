# 第十二章 后牛顿效应

# \$12.1动力模型

牛顿引力理论是天体力学的基础，从天体力学诞生以来，人们一直用牛顿引力理论去研究天体的运动，前面各章也是在这种框架下讨论人造卫星运动的．本世纪六十年代以来，随着测量技术的发展,观测精度大大提高，以牛顿力学为基础的天体力学理论所达到的精度已不能满足需要，必须以新的引力理论对其加以修正.这里只是说修正，其原因是：到目前为止，在天体力学(或与其有关的领域)问题中所涉及的新引力理论的内容，主要是如何给出相应的引力场中，一个实验质点的运动微分方程，它虽然不同于牛顿引力场中的运动方程，但只是增加一些微小的修正项而已．在此情况下讨论天体力学问题，无论从定量角度还是从定性角度来看，研究方法都无实质性改变．因修正项较小，所以仅仅是在牛顿引力场对应的运动方程中增加一些摄动项，相当于增加一个摄动源.

尽管人卫是在地球附近运动，可以选取局部惯性参考系，但又与日、月运动有关，这就不可避免地涉及到地心参考系和日心参考系以及两者之间的“坐标”(时空坐标)转换问题．因此，讨论人卫运动时，有时不得不采用非惯性地心参考系(这里所说的非惯性是指非牛顿引力效应引起的)，相应的运动方程中的修正项将包含地球史瓦西(Schwarzschild)场的广义相对论效应，地球自转和扁率的广义相对论效应，以及测地岁差等[1-3]．但就修正项的大小而言，最主要的是作为一体广义相对论效应的地球史瓦西场修正项，通常把这些修正项称为后牛顿修正项.
