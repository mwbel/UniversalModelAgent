# 第二章 二体问题

由理论力学可知，若将地球看成一个密度分布均匀的球体，则它对人造卫星的吸引可等效于一个质点，相当于质量全部集中在地球质心这一点上，于是地球和卫星就构成一个简单的二体系统.将坐标系的原点放在地心上，讨论人造卫星相对地心的运动，这是很自然的，卫星运动方程可写成

$$
\ddot { r } = - \frac { \mu } { r ^ { 2 } } \Big ( \frac { r } { r } \Big ) ,
$$

其中 $\pmb { r }$ 是卫星的地心向径， $\mu = G M$ 是地心引力常数，为了方使，根据 $\ S 1 . 1$ 计算单位的选择，可使 $\mu = 1$ ，但为了使读者清楚地看出 $\mu$ 在各种公式中出现的情况，并可与其它书籍文献中出现的同类公式作比较，这一章中暂时保留 $\mu$ 这一常数.

严格地说，人造卫星对地球亦有引力作用，地心坐标系并非惯性参考系，因此，方程(2.1)的原形式应为

$$
\begin{array} { c l c r } { { } } & { { \mathrel { \cdots } } } & { { \displaystyle - \frac { G M } { r ^ { 2 } } \biggl ( \frac { r } { r } \biggr ) - \frac { G M } { r ^ { 2 } } \biggl ( \frac { r } { r } \biggr ) } } \\ { { } } & { { \mathrel { \phantom { = } } } } & { { \displaystyle - \frac { G ( M + m ) } { r ^ { 2 } } \biggl ( \frac { r } { r } \biggr ) , } } \end{array}
$$

方程右端第二项即惯性加速度。 $\pmb { m }$ 是人造卫星的质量．相对地球质量而言它是一个小量，例如一个1吨重的卫星，其质量与地球质量之比只有 $1 0 ^ { - 2 2 }$ ，略去这一小量，对定轨精度的影响显然无需考虑，至少在当前的测量精度意义下是如此．因此，卫星运动方程就.可写成(2.1)的形式.

方程(2.1)或(2.2)是可积的，下面我们首先给出相应的六个 独立积分,在此基础上再进行各种讨论.

# 2.1 二体问题的六个积分

# 1．动量矩积分(或称面积积分）

根据有心力的性质,可直接写出方程(2.1)的动量矩积分。由方程(2.1）亦很容易推出该积分,若记 $h { = } r \times { \dot { r } }$ 为面积速度向量，则由方程(2.1）可得

$$
\frac { d h } { d t } { = } \frac { d } { d t } ( r \times \dot { r } ) = r \times \ddot { r } = 0 ,
$$

这表明 $\pmb { h }$ 为常向量，人造卫星绕地球的运动为一平面运动．相应的动力矩积分可写成

\pmb { h } = \pmb { r } \times \dot { \pmb { r } } = \pmb { h } \hat { \pmb { R } } ,

$$

$$
r \times \dot { r } = r ^ { 2 } \dot { \theta } \hat { k } = r ^ { 2 } \dot { \theta } \hat { R } ,
$$

这表明积分(2.6)就是动量矩积分(2.3)式的标量形式.方程(2.5)和(2.6)构成了平面运动系统对应的二阶常微分方程，需要寻找二个独立积分.

上式方程组的特点是不显含自变量 $\pmb { t }$ ，由常微分方程的基本知识可知，对于这类方程,通过分离 $\pmb { \mathrm { \downarrow } }$ 的方法可使它降一阶，即能够首先讨论 $\pmb { r }$ 对 $\dot { \pmb \theta }$ 的变化规律．为此，记 $r ^ { \prime } { = } d r / d \theta , r ^ { \prime \prime } { = } d ^ { 2 } r / d \theta ^ { 2 } ,$ 由(2.6)式得

方程。但相应的方程比较复杂，仍不便于求解。如果在降阶的同时,再作变量变换

$$

$$

有

$$
\left. \begin{array} { l } { \dot { r } = - h u ^ { \prime } , } \\ { \ddot { r } = - h ^ { 2 } u ^ { 2 } u ^ { \prime \prime } . } \end{array} \right\}

$$

$$

这是可积的，我们顺便提一下,如果将原方程右端作用力 $\mu / r ^ { 2 }$ 改为(或增加) $\mu / r ^ { n }$ 的形式, ${ n } = 0 ,$ 士1,士2,…，尽管经上述变换所得方程不同于(2.11)式，但仍然是可积的，这留给读者作为一道习题.

方程(2.11)给出一轨道积分：

$$
r { = } \frac { 1 } { u } { = } \frac { h ^ { 2 } / \mu } { 1 + e \cos { ( \theta - \omega ) } } ,
$$

$\pmb { e }$ 和 $\pmb { \omega }$ 即两个新积分常数。这是一圆锥曲线，在一定条件下它表示椭圆，地心在其焦点上，我们只讨论这种情况，既然是椭圆，可令

\pmb { \mathscr { p } } = \pmb { a } \left( 1 - e ^ { 2 } \right) = h ^ { 2 } / \mu ,

那么积分(2.6)和(2.12)又可写成

$$
\begin{array} { l } { r ^ { 2 } \Dot { \theta } = \sqrt { \mu p } = \sqrt { \mu a \left( 1 - e ^ { 2 } \right) } , } \\ { r = \frac { a \left( 1 - e ^ { 2 } \right) } { 1 + e \cos \left( \theta - \omega \right) } , } \end{array}
$$

积分常数 $\pi$ 由 ${ \pmb a }$ 代替， $\pmb { p }$ 是椭圆的半通径， $\pmb { a }$ 是半长径， $\pmb { e }$ 是偏心率， $\omega$ 则称为人造卫星过近地点的经度，因 $\theta = \omega$ 时， $\pmb { r }$ 达到最小值. $\omega$ 的几何意义见图2.1,图中 $\pmb { P }$ 是近地点方向，经度 $\pmb \theta$ 是从节点 $N$ 沿卫星运动方向计量.

将 $r = r ( \theta )$ 的关系代入方程(2.14)，原则上可以给出最后一个与时间 $\pmb { \ell }$ 有关的积分，我们暂时放一下,先导出椭圆运动的几个常用关系式。由(2.14)和(2.15)两式,经简单的运算即可得

$$
v ^ { 2 } = { \dot { r } } ^ { 2 } + r ^ { 2 } { \dot { \theta } } ^ { 2 } = \mu \left( { \frac { 2 } { r } } - { \frac { 1 } { a } } \right)
$$

此即活力公式，另外，既然是椭圆运动,那么卫星向径在一个周期$\pmb { T }$ 内扫过的面积就是椭圆的面积 $\pi a ^ { 2 } \sqrt { 1 - e ^ { 2 } }$ 由此可知面积速度$h / 2$ 为

$$
\hbar / 2 = \sqrt { \mu a \left( 1 - e ^ { 2 } \right) } / 2 = \frac { \pi a ^ { 2 } \sqrt { 1 - e ^ { 2 } } } { T } ,
$$

整理后写成下列形式：

$$
\frac { a ^ { 8 } } { T ^ { 2 } } { = } \frac { \mu } { 4 \pi ^ { 2 } } .
$$

若引进平均角速度 ${ \pmb n } = { \pmb 2 } { \pmb \pi } / { \pmb T } _ { \mathrm { i } }$ ，则上式又可写成

$$
n ^ { 2 } a ^ { 3 } = \mu _ { \mathrm { \cdot } }
$$

这两个表达式就是由万有引力定律导出的开普勒（Kepler）第三定律.

# 3．第六个积分—-开普勒方程

为了运算方便,在寻找第六个积分时,不直接引用方程(2.14)按 $d \theta / d t$ 求解,而是利用(2.16)式按 $d r / d t$ 积分，有

$$
\scriptstyle { \dot { r } } ^ { 2 } = \mu \left( { \frac { 2 } { r } } - { \frac { 1 } { a } } \right) - \mu \ { \frac { p } { r ^ { 2 } } }
$$

通过(2.18)式消去 $\mu$ 整理后得

$$
n d t = { \frac { r d r } { a { \sqrt { a ^ { 2 } e ^ { 2 } - ( a - r ) ^ { 2 } } } } } .
$$

对于椭圆轨道, $\pmb { \mathscr { r } }$ 的极大和极小值分别为

$$
\begin{array} { r } { r _ { \operatorname* { m a x } } = a \left( 1 + e \right) , r _ { \operatorname* { m i n } } = a \left( 1 - e \right) , } \end{array}
$$

因此有 $| a - r | \leqslant a e ,$ 故可引人辅助量 $\pmb { \cal E }$ ：

$$
\scriptstyle { a - r = a e \cos E }
$$

或

$$
r { = } a ( 1 { - } e \cos E ) .
$$

代入(2.19)式得

$$
n d t = ( 1 - e \cos E ) d E ,
$$

于是给出第六个积分：

$$
E - e \sin { E } = n ( t - \tau ) ,
$$

这又称为开普勒方程， $\pmb { \tau }$ 是积分常数．当 $\pmb { \hat { t } } = \pmb { \tau }$ 时， $\mathbf { \delta } \mathbf { \mathcal { B } } = 0$ ，相应的$= a ( 1 - e ) = r _ { \operatorname* { m i n } } ,$ 故 $\pmb { \tau }$ 就是卫星过近地点的时刻.

最后引进两个角度 $f$ 和 $M$ ，定义如下：

$$
\scriptstyle f = \theta - \omega , M = n ( t - \dot { \tau } )
$$

$f , M$ 和 $\pmb { \cal E }$ 是三个角度，分别称为真近点角，平近点角和偏近点角，都是从近地点开始计量， $\pmb { \cal E }$ 的几何意义见图2.2,图中 $o$ 是椭圆焦点， $O ^ { \prime }$ 是辅助圆的圆心．显然,在二体问题中，面积积分可简化为

$$
r ^ { 2 } f = h .
$$

采用有摄二体问题作为人造卫星绕地球运动的数学模型时，“需要求出卫星椭圆轨道的摄动变化，而这些变化又将包含周期项与非周期项两类性质截然不同的部分，为了计算和理论分析的需要，有必要把它们分开。如何分解呢？有些量无法直接看出其性质，如 $\scriptstyle ( { \frac { a } { r } } ) , \cos f$ 等,它们是 $f$ 的周期函数，但对时间 $\pmb { \mathrm { \acute { t } } }$ 积分时，在一个卫星运动周期内的累积效果不为零(除非偏心率 $e = 0 ,$ ；也就是说，这种类型的摄动力所引起的卫星轨道变化并不完全是周期性的．为此,我们可用求平均值的方法来加以区分。

任一函数 $F ( t )$ ,在一个卫星运动周期 $\pmb { \mathcal { T } }$ 内的平均值 $\overrightarrow { F }$ 定义为

$$
\widetilde { F } { = } \frac { 1 } { T } { \int } _ { 0 } ^ { \pi } F ( t ) d t .
$$

若记 ${ \pmb F } _ { s }$ 和 ${ { \pmb F } _ { o } }$ 分别为周期项和非周期项，则显然有

$$
\begin{array} { r } { F _ { c } = \widetilde { F } , ~ F _ { s } = F - \overrightarrow { F } . } \end{array}
$$

于是，我们可用对一个卫星运动周期求平均值的方法把周期项分离出来,相应的函数 $\pmb { F }$ 即被分解成两个部分：

$$
\begin{array} { r } { F ( t ) = F _ { c } + F _ { s } ( t ) . } \end{array}
$$

从上述各表达式可清楚地看出，在求积分(2.83)式时，到底采用什么方法，并不影响由(2.84)和(2.85)式表达的函数分解结果的严格性．因此，尽管这一分解是针对后面求卫星轨道摄动变化的需要，可这里计算积分(2.83)时，却能采用椭圆运动关系．当然，考虑摄动时，运动周期 $\pmb { T }$ 及所有椭圆轨道根数均要发生缓慢的变化，但它不会改变周期项 ${ \pmb F } _ { s }$ 的基本特征．上述分解不仅严格,而且仍然保持原分解的意义.

讨论卫星轨道变化时所涉及到的摄动力，对应各种各样的函数，但需要通过求平均值来分离周期项的，基本上有下面四类：

# 2.5 星历表计算

对于椭圆运动，一旦相应的六个积分常数(即轨道根数)被确定,轨道就被确定，由此即可计算任何时刻卫星的空间位置，这就是星历表计算，至于如何确定轨道根数，那是定轨问题，将在第三章和最后一章中专门讨论它.

已知时刻 $\pmb { t }$ 的六个椭圆根数 $a , e , i , \Omega , \omega , M$ ，要计算相应的卫星空间位置，可分三步：

首先也是最重要的一步，由 $M$ 和 $\pmb { e }$ 计算偏近点角 $\pmb { \cal { E } }$ ，即解开普勒方程

$$
\pmb { { \cal E } } = \pmb { { \cal M } } + e \sin \pmb { { \cal E } } .
$$

这虽然是一超越方程，但 $e < 1$ （有时很小)，解此方程的方法显然很多(如简单迭代法，微分改正法等)读者是清楚的,这里不必多述。

得到 $\pmb { \cal E }$ 后,就可利用(2.42)式计算卫星的空间位置向量 $\pmb { r }$ ，即

$$
r { = } a ( \cos E { - } e ) \hat { P } { + } a \sqrt { 1 { - } e ^ { 2 } } \sin E \hat { Q } .
$$

至于P,Q的计算，可利用旋转矩阵(2.44)式，亦可直接用(2.45)式，依需要而定.

最后一步即将 $\pmb { r }$ 转换为站心坐标 $\rho ^ { \prime } ( \rho , \alpha , \delta )$ 或 $\rho ( \rho , A , h )$ ，前者即站心赤道坐标，由

$$
{ \pmb \rho } ^ { \prime } { = } { \pmb r } - { \pmb R }
$$

得

$$
\scriptstyle { \binom { p \cos \delta \cos \alpha } { \rho \cos \delta \sin \alpha } } = { \binom { x - X } { y - Y } }
$$

即

$$
\rho ^ { 2 } = ( x { - } X ) ^ { 2 } + ( y { - } Y ) ^ { 2 } + ( z { - } \mathrm { { Z } } ) ^ { 2 } ,
$$

$$
\alpha = \arg - \imath \bigg ( \frac { y - Y } { x - X } \bigg ) , ~ \delta = \sin ^ { - 1 } \bigg ( \frac { z - Z } { \rho } \bigg ) ,
$$

其中 $\pmb { R } ( X , Y , \pmb { Z } )$ 是测站坐标，若要给出站心地平坐标 $( \rho , A _ { \bullet } ,$ $\pmb { h } )$ ,只需简单的坐标转换,详见第一章\$1.3第2段。
