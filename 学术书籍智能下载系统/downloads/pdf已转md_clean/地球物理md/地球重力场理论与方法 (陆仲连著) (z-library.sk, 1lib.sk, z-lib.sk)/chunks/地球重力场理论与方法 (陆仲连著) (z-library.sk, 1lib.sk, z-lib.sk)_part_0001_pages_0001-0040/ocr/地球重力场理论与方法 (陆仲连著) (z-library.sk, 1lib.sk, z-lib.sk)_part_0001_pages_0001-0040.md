# 绪 论

# \$0.1重 力

牛顿万有引力定律表明：宇宙间的任意两个质点之间都存在相互的吸引力。设 $\dot { \bf m }$ 和 $\mathbf { m ^ { \prime } }$ 分别为两个质点的质量，它们之间的距离为 $\pmb { r }$ ，则引力 $\mathbf { F }$ 的大小与其质量乘积成正比，与其距离的平方成反比，可用式子表示为：

$$
\scriptstyle \mathbf { F = f } { \frac { \mathbf { m } \mathbf { m } ^ { \prime } } { \mathbf { r } ^ { 2 } } }
$$

式中 $\mathbf { f }$ 为一比例系数，它与相互吸引的两质点的质量和距离都无关系，故称为引力常数，它的数值可以通过实验的方法求得。依据国际地球自转服务(IERS)中央局1989年出版的标准[40)为 $\mathbf { f } = 6 . 6 7 2 5 9 \times 1 0 ^ { - 1 1 } \mathbf { m } ^ { 3 } \mathbf { k g } ^ { - 1 } \mathbf { s } ^ { - 2 }$ 。

在本课程中，总是把质量为 $\mathbf { m }$ 的质点称为吸引质点；另一质点称为被吸引质点，并取其质量为一个单位，即 $\mathbf { m } ^ { \prime } { = } \mathbf { 1 }$ 。于是，按照 $( 0 \cdot 1 )$ 式可以写出吸引质量 $\mathbf { m }$ 对单位质点的引力为：

$$
\mathbf { F = f } \frac { \mathbf { m } } { \mathbf { r } ^ { 2 } }
$$

力是一个矢量，它不但有大小，而且有方向。如图0.1所示，引力 $\mathbf { F }$ 的方向是指向吸引点的。如果取$\pmb { \mathrm r }$ 的正方向为由吸引点指向被吸引点，则引力 $\mathbf { F }$ 的方向与 $\pmb { r }$ 的方向恰恰相反。因此，用矢量表示时写为：

![](images/2dd314b95c3d043b70cfc0a9d10aed0c3131bc9b4f783e80389ddc7cfdc9e1f5.jpg)  
图0.1

$$
\vec { \mathrm { \bf F } } { = } { - } { \bf f } \ \frac { \mathrm { m } } { { \bf r } ^ { 2 } } \cdot \frac { \vec { \mathrm { \bf ~ r } } } { \mathrm { \bf ~ r } } ,
$$

式中 $\frac { \overline { { \mathbf { r } } } } { \mathbf { r } }$ 表示 $\pmb { \mathrm { r } }$ 方向的单位矢量。

地球可视为由无穷多个质点组成的质体,它对单位质点的引力是所有质点的引力的合力。因此，地球对单位质点的引力可由下式求得：

![](images/2ee74d924e6647042f0721ccfd7486ab09950db1c59865e36fbcae1b46809414.jpg)  
图0.2

$$
\mathbf { F } = - \mathbf { \nabla } \mathbf { f } \int \frac { \cos \Psi } { \mathbf { r } ^ { 2 } } \mathbf { d m }
$$

式中 $\mathbf { d m }$ 为地球的单元质量， $\mathbf { r }$ 为 $\mathbf { d m }$ 至被吸引点的距离， $\pmb { \psi }$ 为矢量 $\vec { \pmb { { \tau } } }$ 与引力 $\vec { \bf F }$ 的夹角(参见图0.2),积分扩展至整个地球。

众所周知，地球围绕着地轴作自转运动。因而地球表面上的单位质点也随之而绕地轴作圆周运动，必然受到

一个离心力的作用。由力学知识，离心力P总是垂直于旋转轴，而其方向是背向旋转轴的，如

图0.3所示；其大小为：

$$
\mathbf { P } { = } \frac { \mathbf { V } ^ { 2 } } { \rho }
$$

式中V为质点运动的线速度，p为由旋转轴至质点的距离。今以w表示地球绕轴自转的角速度，则

$$
\mathbf { V } { = } { \boldsymbol { \omega } }  p
$$

将 $( 0 \cdot 6 )$ 式代入 $( 0 \cdot 5 )$ 式,得到离心力的表达式：

$$
\mathtt { P } { = } \omega ^ { 2 } \pmb { \rho }
$$

地球自转角速度w的数值可以利用天文测量方法精确求定，在本课程中只取至有限的几位数就够用了，即

$$
\omega { = } 7 . 2 9 2 1 1 5 { \times } 1 0 ^ { - 5 } \mathrm { s } ^ { - 1 }
$$

地球的重力是单位质点受到的地球引力与离心力的合力。如图0.4所示，F表示地球引力， $\vec { \mathbf { p } }$ 表示离心力，g表示重力，则有

$$
\stackrel {  } { \mathbf { g } } = \vec { \mathbf { F } } + \vec { \mathbf { P } }
$$

运动学中的牛顿第二定律指出，物体受力所产生的加速度与其质量的乘积就等于所受的力。据此，对于一个单位质点，作用于其上的重力在数值上等于使它所获得的重力加速度的数值。因此，在本课程中，总是将重力和重力加速度两词通用。测定一点的重力实质是测定该点的重力加速度，说一点的重力多大实质是说这点的重力加速度多大。重力加速度的量纲是 $1 0 ^ { - 2 } \mathrm { m } \mathrm { s } ^ { - 2 }$ ,它被称为“伽”,它是为了纪念世界上第一个测定重力加速度的意大利物理学家伽利略(G.Galilei)。比较实用的单位是“毫伽”,它等于千分之一伽，即 $1 0 ^ { - 5 } \mathbf { m } \mathbf { s } ^ { - 2 }$ 。千分之一毫伽，即 $1 0 ^ { - 8 }$ $\mathbf { m } \mathbf { s } ^ { - 2 }$ ，称为微伽。

![](images/88c51f822e7beb7ccee1eafce22c835586b2550a7c5a29b055bccd1ec009d56f.jpg)  
图0.3

![](images/5d721408aac5eec4aa287b7fe87d7e0bbe8d686dcd8fce206e82b3f068b679e0.jpg)  
图0.4

由于地球接近于一个绕短轴旋转的旋转椭球体，因而极区地面点所受的引力要比赤道处的点所受的引力大；同时极区点所受的离心力接近于零且与引力方向相垂直，故极区的重力值接近于引力的数值，而在赤道处离心力的大小约为引力的三百分之一，且其方向与引力方向相反，这就使得赤道处的重力值明显地比极区的重力值小。因此，在地面上重力值有由赤道向两极逐渐增大的总的变化趋势。实际测量结果表明，在赤道处地面重力值约为978伽，而在极区重力值约为983伽，全球的平均重力值约为980伽。另一方面，重力值又随高程的增加而减小。对于一座山,山顶上的重力值比山谷处的重力值小。

# 0.2用途和本书概要

从前节所述的重力的定义看出，在地球及其周围空间的任意一点都受到重力的作用。因此,人们把地球重力所作用到的整个空间称为地球重力场。在地球上生存的人类,每时每刻都感受到地球重力场的作用。靠着重力场的作用,树上的果实和空气中的水珠才会落到地面上供人类使用。如果没有了重力的作用，就像在宇宙飞船上那样，人和物品随意飘浮，人类的日常生活将会变成何种状态令人难以想像。

地球重力场作为地球本身的一种物理特性，与地球形状及其内部质量的分布密切相关。由于地球形状的不规则和内部质量分布的不均匀，使得地球重力场的结构复杂，这又给人类带来不少的麻烦,大地测量就是受严重影响的领域之一。

大地测量的最终目的是在地面上建立一个高精度的大地坐标系，作为地图测绘的控制并为各种经济建设和国防工程提供地面点的精密坐标。按照传统的方法，确定地面点的水平坐标(经纬度)要进行角度和长度的测量。在进行这些测量时，所用的测量仪器都是以测站铅垂线为基准的，这个铅垂线方向也就是该点的重力方向。当精度要求不高时，在处理测量数据时可以近似地把铅垂线当作平面的垂线,或者是球面的向径，或者椭球面的法线。而在要求高精度结果时就不能这样处理了，必需考虑到铅垂线的实际方向。它并不与椭球面法线重合，两者之间有一个夹角，这就是垂线偏差。对于求定地面点的第三个坐标,即高程，水准测量是一种主要的手段。这种测量的基础是将点之间的水准面看成是相互平行的。但在实际的地球重力场中,水准面之间是互相不平行的，为此必需对水准测量结果加入与重力场有关的改正。同时,还有一个高程异常的问题。垂线偏差和高程异常的大规模推求，都需要应用地球重力场的理论和方法。

大地测量工作者致力于地球重力场理论的研究已有几百年的历史。然而由于掌握的重力数据的不足，一直不能应用于解决实际问题。真正将重力数据应用于大地测量的数据处理，也就是近半个世纪的时间。从本世纪的四五十年代开始，形成了严密的地球重力场理论，但数据的不足仍然影响着这些理论的完全实现。

人造地球卫星的发射成功，为大地测量提供了新的工具,也为人们获取地球重力场信息提供了新的手段。然而，人造地球卫星仍然在地球重力场中运动,要精确求定卫星飞行轨道也离不开地球重力场问题。因此，随着空间技术的发展，地球重力场的研究就显得更为重要了。

本书重点论述利用地面重力数据研究地球外部重力场的理论和方法。

前三章讨论了获取地面重力数据的原理、仪器和方法。其中第一章介绍运用可倒摆和物体自由下落测定地面点的绝对重力值；第二章介绍进行重力差测定的仪器和数据处理过程；第三章对在海面上进行重力测量作了简要的论述。第四、五章分别论述了位理论和调和函数及其边值问题，它们为以下章节的讲述准备了必需的数理基础知识。第六章讨论了水准椭球的重力场，因为水准椭球可以看成是真实地球的一个近似，水准椭球的重力场也就是地球重力场的一个近似。第七至九章讨论了研究地球重力场的理论。在第七章对地球重力场的初步论述的基础上，第八章系统地讨论了传统的研究地球重力场的理论，即司托克斯理论；第九章讨论的是研究地球重力场的现代理论；它们为重力数据的实际应用提供了理论基础。第十章介绍了地球重力场模型的概念、建立方法、现有模型的状况及其应用。第十一、十二章分别论述了利用地面重力数据推求垂线偏差和高程异常的理论和方法，它们对于地面天文大地测量数据的处理是必需的。第十三章阐述了最小二乘配置方法在研究地球重力场中的具体应用，它为地球重力场的研究开辟了一个新的途径。第十四章讨论了利用地面重力数据推求外部重力场信息的理论和方法，这在空间技术和远程武器的发展中有着相当的作用。

期望本书能为承担与地球重力场有关技术的朋友们喜欢。但由于作者知识有限;所写内容不够全面，并且还可能有错误之处，敬请读者批评指正。

# 第一章绝对重力测量

# 1.1历史的回顾

直接测定重力加速度数值的工作称为绝对重力测量。

约在1590年伽利略发现了物体自由下落定律。他以测定下落物体在第一秒钟内所走的路程测定重力加速度的数值。因为，根据等加速运动定律，此路程等于 $\frac { 1 } { 2 } \mathbf { g }$ 。由于一秒钟的时间间隔及物体下落的路程的精确测定在当时都是不可能的，再加上未消除空气阻力等的影响，所获得的结果是很不精确的。

荷兰物理学家惠更斯(Ch.Huygens）在1673年提出的摆动周期、摆长和重力加速度的关系给重力值的测定开创了新的理论基础。人们开始用摆测定重力值。1735年拉康达明(Ch.de.La Condamine)在加依尔岛的珊多明哥所测得的重力值其相对误差约为1/2000。1792年波尔达(J.Borda)和卡西尼(J.Cassini)在巴黎测得的重力值其相对误差为1/13000。法国物理学家比奥(J.B.Biol)改进了波尔达和喀西尼的仪器，于1801-1818年间，在法国、英国和西班牙完成了一系列的重力测定。这些观测结果比较精确，以至在1884年被赫尔默特(R.Helmert)用于推算正常重力公式。

1811年德国天文学家鲍年毕尔盖尔(J.Bohnenberger)阐明了用可倒摆测定重力值的原理,这种摆于1818年由英国物理学家卡德(H.Kater)设计出来。之后很多人用卡德摆测定重力，其中特别著名的是英国测量家沙宾(E.Sabine),他于1822-1824年间在太平洋的许多岛上、在美洲、在格林兰以致于在斯比茨别尔根作了观测。卡德本人在格林尼治所进行的观测，结果精度为 $3 \times 1 0 ^ { - 5 }$ 。 ！

贝 塞尔(F.W.Bessel)针对卡德摆的一些缺点分析了产生误差的原因,提议设计一种避免这些误差或使这些误差消除的摆。这个念头直到19世纪60年代末在雷白索里特型仪器中才得以实现。它允许在两个位置上的：周期不必完全相等，这就缩短了观测的时间并省去了移动一个重锤的工作。在以后的一个较t长的时期内，这种摆在世界绝对重力测量工作中起着很大的作用。先后建立了维也纳基准和菠茨坦基准，作为地球上一切相对重力测定的基础。

# 1.4运用可倒摆测定绝对重力

由于实践中使用的物理摆的形状不可能很规则，这就使得转动惯量Js很难精确求得，进而使按(1·15)式计算归化摆长不可能实现。如果能使归化摆长成为摆上某两点之间的距离，那就可以较容易地量得归化摆长。为此，提出了采用可倒摆的方法。

# \$1.6几项改正

在绪论中已经讲明，所研究的地球重力场中的地球重力的定义是地球的引力和地球绕自转轴旋转所产生的离心力的合力。然而，当用可倒摆或者自由落体作绝对重力测量时，作用于摆或自由落体的力不只是这两个力，还包含有其他的力。如：太阳和月亮的引力、大气的引力等。因此，为了得到所需要的重力(引力和离心力的合力)，必需扣除其他力的影响，这就要加入几项相应的改正。

# 一、日月引力的影响

消去这项影响所加的改正通常称之为"重力的固体潮改正"。

# 三、极移的影响

离心力是重力的一个组成部分,它与地球的自转轴密切相关。地球的自转轴也是地球的极轴，依据天文学的研究，地球自转轴在地球本体内的位置是在不断地变动的。这种运动被称为地极移动，简称为极移。显然，极移将引起离心力的方向和大小发生变化，从而使地面点的重力值随时间而变。为了使得在不同时间测得的重力值归算到同一时刻,必需加入相应的改正。

：当将地球视为半径为 $\mathbf { R } ^ { \prime }$ 的球体时，地面点离心力在重力方向的分量 $\mathbf { P _ { 8 } }$ 为

$$
\scriptstyle \mathbf { P } _ { \mathrm { g } } = \omega ^ { 2 } \mathbf { R } \cos \varphi
$$

对纬度 $\pmb { \mathcal { P } }$ 进行微分，可得纬度变化 $\Delta \varphi$ 引起的重力变化 $\Delta \mathbb { P } _ { \Xi } .$ 的关系式，即

$$
\Delta \bar { \mathbf { P } } _ { \mathrm { g } } = - \omega ^ { 2 } \mathbf { R s i n } 2 \varphi \Delta \varphi
$$

依据天文学知识，极移引起的地面点纬度的变化可按下式求得：

$$
\Delta \phi = \bf { x p c o s } \lambda - \bf { y p s i n } \lambda
$$

式中 $\gimel$ 为点的经度， $\mathbf { x } _ { \mathbf { P } } , \mathbf { y } _ { \mathbf { P } }$ 为在国际地球自转服务(IERS)参考架中的地极坐标,以弧度为单位,它可以在IERS年报中找到。将 $( 1 \cdot 6 \dot { 2 } )$ 式代入 $( 1 \cdot 6 1 )$ 式，则得

$$
\Delta \mathrm { P } _ { \mathrm { s } } { = } { - } { \omega } ^ { 2 } \mathrm { R s i n } 2 \varphi \mathrm { ( } { \bf x } _ { \mathrm { P } } \mathrm { c o s } \lambda { - } { \bf y } _ { \mathrm { P } } \mathrm { s i n } \lambda \mathrm { ) }
$$

考虑到地球不是一个刚体,地面点重力值的极移改正 $\delta _ { \pmb { \mathrm { g } } _ { \pmb { \mathrm { p } } } }$ 应该是

$$
\delta _ { B P } = \delta \Delta P _ { g }
$$

式中8为重力的潮汐因子。

数值计算表明，重力的极移改正的数值一般只能达到10微伽左右。因此，只有在需要应用高精度的重力值时才需考虑此项改正。

# \$1.7重力系统

在地面上进行绝对重力测定，由于所用仪器的体积大、附属设备多、要求的外界条件高等原因，每测一个点化费的观测时间较长，且要化费较多的财力。因此，它不适应在地面上进行大规模重力测量时采用。近百年来，在地面上进行重力测量的主要手段是采用测定两点重力差的相对重力测量方法。为此,在全球至少要有一个或一些进行了绝对重力测量的重力点，才可以推算出其他点的绝对重力值。这种测定了绝对重力值的点被称为国际重力基准（点）。

历史上有过两个国际重力基准，即维也纳重力基准和波茨坦重力基准。

维也纳重力基准指的是以奥地利维也纳天文台的重力点为基准，其值为g=981.290士0.01(伽),它是奥波尔泽(Th.Oppolzer)在1884年测定的。1900 年在巴黎举行的国际大地测量协会第18次会议上通过采用。凡是根据维也纳重力基准推算得的其他点的重力值均属于维也纳重力系统。

波茨坦重力基准指的是以德国波茨坦大地测量研究所摆仪厅的重力值为基准，其值为：g=981274±3(毫伽),该点的位置是φ52°22.86'N、入=13°4.06'E,海拔高H=87.48m。此值是1898至1904年在赫尔默特指导下,由居能(Kuhnen）和富特范格勒(Furtwangler)用五付可倒摆测定的。1909年在伦敦举行的国际大地测量会议上决定采用波茨坦重力基准。凡是根据波茨坦重力基准推算的重力值均属于波茨坦重力系统。属于维也纳系统的重力值，只要减去16毫伽，即可转化为波茨坦系统。

在此后的几十年间，许多国家对绝对重力测量的技术进行了研究，并测定了一些点的绝对重力值。经与波茨坦重力基准点联测，发现波茨坦点的绝对重力值有12到16毫伽的误差。因此,在第二次世界大战结束以后，国际大地测量协会着手建立新的国际重力基准。经过一段时间的工作之后，于1971年在莫斯科举行的国际大地测量和地球物理联合会(IUGG)第15届大会上通过了国际重力基准网1971(IGSN-71)，用以代替原来使用的波茨坦国际重力基准。

国际重力基准网1971由1854个重力点组成，分属于108个国家或地区的494个城市。它们的绝对重力值是根据大量观测数据经过处理后按最小二乘原理整体平差后求得的。参加平差的观测结果包括：10个绝对重力测量结果、1200个摆仪测量结果和23700多个重力仪测量结果。对于这1854个点，每个点的最后重力值的中误差都小于士0.1毫伽。因此,都可以作为推算其他点的重力值的基准。

根据国际大地测量与地球物理联合会第15届大会通过的第11项决议，波茨坦A点的IGSN-71系统的重力值为981260.19±0.02(毫伽)，而它的波茨坦重力系统的数值为

981274.20毫加。因此,该点波茨坦系统值的改正为一14.0毫伽。这就表明，凡属波茨坦重力系统的重力值只要减去14毫伽,即可改化为IGSN-71重力系统。当然，严格说来，对于各个地区的重力值在推算过程中还包含了各种联测误差，所加的改正数应该是有所区别的。

在我国测得的地面点的重力值也存在有新旧两个系统。旧系统是本世纪50年代建立的，通常称它为“57网"系统。由于在建立这个网时，利用了原苏联几个点的重力值，因此它属于波茨坦重力系统。新系统是在80年代重新建立的，通常称它为"85网"系统。由于在建立这个网时,与日本、巴黎和香港地区的23个IGSN-71网点进行了联测，因此它属于IGSN-71系统。新旧系统的重力值之间的差值约为 $+ 1 3 . 5$ 毫伽。

从上面的论述中看出，重力系统的不同会给重力值带来较大的系统差。因此，在使用重力数据时必需弄清它属于那个重力系统，并将其转换至同一系统中。

# 2.2进行相对重力测量的静力法

用摆仪进行重力测量时，观测的是运动着的摆，故被称为动力法。由于仪器构造复杂和笨重，观测的环境条件要求高，而且观测项目多和花费的时间长,数据处理也较繁琐，很难达到高精度和适应在广阔领土上的重力普查工作。

进行相对重力测量的静力法是利用一种力来平衡物体所受的重力，重力的变化将引起平衡位置的变化，只要用相应的方法测出平衡位置的变化，就可算出重力的变化，也就是测定两地的重力差。

早在20世纪初，德国地球物理学家海凯尔（O．Hecker）就曾用水银气压计和沸点温度计一起来测量重力差。虽然测量精度低于士0.02伽，但在当时却解决了海上重力测量的手段，也为用静重力仪测定重力开创了先例。

1913年德国的玛德逊（V．Madsen）在设计以气体的弹力平衡水银所受的重力为基础的气体重力仪时，选择了一种方法，它使温度变化得到补偿，以此代替测量气体的气温。在30年代，据此设计了一批仪器。它们的观测和计算只需几分钟，且能达到毫伽级的精度。图2.1为德国和哈尔克（H.Haalck）所制造的新型气体重力仪的比较详细的略图。它有两个水银柱AA和BB；五个容器V、V、V、V和V，其中V和V;连成一个；各容器内的气压分别为：V，内近于零，V和V内近于一个大气压，V4、V约两个大气压。V和V向上伸出的两个管子，在上部互相连通，但附有一个栓塞L。同时，L的两边弯下一个连通支管，分别有两个栓塞L和L；管内装有一滴液体M；管上刻有分划尺，用于读出液体滴M来回游动的位置。实际测量时，在已知点上将栓塞L、L2和L；都打开，使液滴M约居管的中央，然后将所有栓塞关闭。将仪器移至待测点，设该点的重力比已知点的小，则A,A和BB,两管水银的重量减小，故V内的水银面下降，V3内的水银面上升，由V来的气体压力大于Vz来的气体压力。此时打开栓塞L2和L，液滴M将往左移动。按照其位移量即可推算出两点的重力差。

以弹簧的弹力来平衡重力所制造的重力仪为弹簧重力仪，其理论是由别罗特(A.Berreth)于1932年建立的。野外仪器中最著名的是德国梯逊(St.Y.Thyssen）所创造的。美国的哈特利(K:Hartley）也制造了一个弹簧重力仪，其结构如图2.2所示。从仪器上盖垂直悬挂一个螺旋形弹簧R，它被称为主弹簧。在主弹簧的下端联着一个带有旋转点O的水平杠杆L，重锤P则安在杠杆下方。在杠杆左端有一个辅助弹簧T，它用来平衡主弹簧，其上端和测微螺旋V相联。S是反光镜，它依附在两个相互靠近的刀口上，这两个刀口中有一个是不活动的，与框F.联接;另一个处在杠杆L的一端。一束细细的光线射到镜子上，而后又射到分划尺M上。显然，重锤P和杠杆的重量变化都会引起镜子的转动，并使光线沿分划尺作相应的位移。旋转测微螺旋V，它改变弹簧T的张力，并使光线返回零分划线。这样，螺旋V上的读数差就是重力变化的大小。

$$
\frac { \mathrm { d } \alpha } { \mathrm { d } \mathbf { g } } { = } \frac { - \mathbf { m } l } { \mathrm { f } \mathbf { r } \mathbf { x } _ { \mathrm { l } } + \tau }
$$

式中的负号表示g的增加引起α角减小。从(2.12)式看出，右端的分母值越小，则灵敏度越大。当灵敏系统的构件选定后，f、r和τ三个参数的值就不会再变了，只能通过调整主弹簧上端点的X坐标x来改变frxi+τ的数值。如图2.4所示，主弹簧的上端点总是安置在y轴的左侧，此时x为负值，这样frxi十τ的数值就小了。但也不能使它接近于零,这会使灵敏度太大，给观测带来困难。通常使灵敏度达到每毫伽儿十秒，既有较高的灵敏度，又能稳定地观测。

由于灵敏度是常数，因而只要测出点之间摆杆偏转角的变化△α就可以算出点间的重力差 $\Delta \mathbf { g }$ 。

(2.11)式表明，灵敏度与零位相对于水平位置的夹角0有关。当零位在水平位置上方时，0为正，右端的分母值减小;因而灵敏度变大；而零位在水平位置下方时,0为负，右端的分母值增大,灵敏度减小。这个规律将用于灵敏度数值的调整。

如对平衡方程(2.9)以 $\theta$ 和 $\mathbf { g }$ 为变量进行微分，则得

$$
\begin{array} { r } { \mathrm { d } \mathbf { g } = \mathbf { g } \mathbf { t } \pmb { g } \left( \alpha + \theta \right) \mathrm { d } \theta } \end{array}
$$

当采用零位读数法时，即 $\scriptstyle \mathbf { \tilde { \alpha } } = 0$ 则有

$$
\scriptstyle \mathrm { d } _ { \mathbf { g } } = _ { \mathbf { g } } \mathbf { t } _ { \mathbf { g } } \theta \mathbf { d } \theta
$$

(2·13)式显示了零位的微小偏差d0可能造成所测重力值产生的误差dg，它与零位相对于水平位置的倾角有关。若0=0,则d0将不引起重力值产生误差。这是将零位选定在水平位置的一个理由。

# 2.4ZSM型重力仪的检验和调整

在将每台重力仪用于测量工作之前，必需对仪器作全面的检验和必要的调整，使其各个部件处于正确的位置和相应的工作状态，以确保测量工作的正常进行，取得好的测量结果。对于ZSM型重力仪，必需完成的检验和调整的项目及其方法如下。

# 一、面板位置的检验和调整

面板的正确位置应该是使水准器和脚螺旋之间保持如图2.12所示的状态。此时，在将横水准器调平后，再旋转纵向那个脚螺旋时，横水准器的气泡保持不动。如果发生移动，则说明没有处于正确的位置，应予调整。调整应由两人合作进行：一人用双手分握仪器外壳的两个提手，使仪器立稳不动；另一人用双手手指将仪器面板周圈的橡皮圈掀起，并抓住护圈缓慢地转动面板。但要注意，转动时切勿将仪器上提。操作完成后需再次作检验，直至达到转动纵向脚螺旋时，横水准器气泡居中不动为止。

# 二、测量范围调整

![](images/8b5ea0e5c5d1a2e0be3cc3ab2bedadb888ef19784aeada35cb0d33cca837056d.jpg)  
横向水平调节螺丝  
图 2.12

在上一节中已经提到，ZSM型重力仪的直接测量范围只有约100毫伽。因此，如果在重力值为979800毫伽的地方仪器计数器读数为0.0格，则用它只能在重力值为.979800至979900毫伽的范围内进行重力差测量。如果要在其他的重力值范围内测量，则需作测量范围的调整。例如，将在重力值为979900毫伽处的仪器计数器读数调为0.0格，则测量范围就变为979900至980000毫伽了。如果待测点的重力值与起始点的重力值的差值大于100毫伽，则需在两点间选取一个(或几个）过渡点。先测出过渡点与起始点的重力差，在过渡点测完后作测量范围调整，再测出待测点与过渡点的重力差。估算待测点近似重力值的方法是应用 $\mathfrak { F } \mathfrak { o } . \mathfrak { 1 }$ 中所述的地面重力值的近似规律：点位北移重力值增大，在中纬度地区约每公里1毫伽；点位高程增大1米，重力值约减小0.2毫伽。

测量范围调整的步骤如下：

1.将仪器整平，把计数器读数调到所需要的读数附近。

2.取下图2.10所示的面板上的测程调整孔塞，插入测程调整改刀，旋转测程调节连杆，把亮线调到零线附近。其规律是：当亮线位于零线的右边时，反时针方向转动改刀；当亮线位于零线的左边时，顺时针方向转动改刀。

3.用改刀垂直向下轻压连杆，然后用手指头轻敲面板，使测程部分机械间隙稳定，抽出改刀，将孔塞盖好。
