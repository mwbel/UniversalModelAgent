### 使用方法

#### 1. 初始化系统
```bash
source venv/bin/activate
python main.py --init
```

#### 2. 查看统计
```bash
python main.py --stats
```

#### 3. 搜索书籍（需要网络）
```bash
python main.py --search
```

#### 4. 下载书籍
```bash
python main.py --download
```

#### 5. 验证文件
```bash
python main.py --validate
```

#### 6. 生成报告
```bash
python main.py --report --report-format xlsx
```

#### 7. 查看候选结果
```bash
python main.py --candidates
python main.py --candidates --book-id 1
```

### 技术亮点

1. **模块化设计**: 清晰的模块分离，易于维护和扩展
2. **多源搜索**: 支持多个搜索源，提高成功率
3. **智能评分**: 基于标题、作者、格式的评分算法
4. **数据库管理**: 完整的状态跟踪和日志记录
5. **配置驱动**: YAML配置文件，灵活调整参数

### 已知限制

1. **网络依赖**: 需要稳定的网络连接
2. **反爬虫**: 搜索源有反爬虫机制，需要延迟和代理
3. **资源可用性**: 不是所有书籍都有开放资源
4. **版权问题**: 仅用于学习研究，需遵守版权法
