# 第一章 绪 论

讨论一个力学问题，首先涉及到的就是时空坐标系的选择问题,在人卫定轨精度的要求日益增高的情况下，坐标系的选择问题显得更加突出．本书当然没有必要对这一问题作详尽的阐述，只是对那些与人造地球卫星运动有直接关系的内容作一简略的介绍·读者如有需要，可翻阅有关书籍和文章.

# 81.1计算单位

为了计算和公式表达的方便，选取适当的计算单位是必要的。众所周知,在常用的克、厘米、秒单位系统中，引力常数值为

$$
G { = } 6 . 6 7 2 \times 1 0 ^ { - 8 } \mathrm { c m ^ { 3 } / g \cdot s ^ { 2 } } ,
$$

这种单位在大尺度问题中，用起来显然是不方便的．在讨论太阳系中大行星和小行星的运动时，常采用太阳质量、天文单位 $( \mathbf { A U } ) =$ $1 . 4 9 5 9 7 8 7 0 \times 1 0 ^ { 1 1 } \mathrm { { m } ) }$ 和平太阳日作为基本单位，那么引力常数将变为

$$
G = 2 . 9 5 9 1 2 9 0 8 \times 1 0 ^ { - 4 }
$$

# 81.2时间系统

就运动而言，需要的是一种均匀的时间尺度．过去，这种均匀的时间尺度是以地球自转为基准的．由于地球自转的不均匀性和测量精度的不断提高，上述均匀时间尺度已不适应；但由于种种原因，又必须将时间与地球自转相协调，这就导致了时间系统的复杂化.

现行的时间系统基本上分为四种：恒星时，世界时，历书时和原子时.前两种都是根据地球自转测定的，历书时则是根据地球、月亮和行星的运动来测定的，而原子时是以原子的电磁振荡作为标准的.下面将对这些时间系统作一简单介绍。

# 1．恒星时(ST)

春分点连续两次过中天的时间间隔称为一“恒星日”.那么，恒星时就是春分点的时角，它的数值 $\pmb { \cal S }$ 等于上中天恒星的赤经 $\alpha ,$ 即

$$
\pmb { \mathcal { S } } = \pmb { \mathcal { \alpha } } ,
$$

这是经度为 $\lambda$ 的地方恒星时．与世界时密切相关的格林尼治恒星时 $\delta _ { 0 }$ 由下式给出：

$$
s _ { 0 } = s - \lambda ,
$$

经度 $\lambda$ 规定向东计量．格林尼治恒星时又有真恒星时（或称视恒星时)GAST与平恒星时GMST之分.既然恒星时是由地球自转时角所确定的时间，那么地球自转的不均匀性就可通过它与均匀时间尺度的差别来测定.

格林尼治恒星时主要是在空间坐标系的转换中用到，有关内容将在下一节中介绍.

# 2．世界时(UT)

与恒星时相同，世界时也是根据地球自转测定的时间，它以平太阳日为单位,1/86400平太阳日为秒长．事实上，测定太阳的精度远低于测定恒星的精度，因此，世界时是通过对恒星观测测定的恒星时再根据两种时间的定义转换而给出的.

根据天文观测直接测定的世界时，记为UT0，它对应于瞬时极的子午圈．加上引起测站子午圈位置变化的地极移动（即地球自转轴在地球体内的移动)修正,就得到对应于平均极的子午圈的世界时，记为UT1，即

$$
\mathbf { U T 1 } { = } \mathbf { U T } 0 + \Delta \lambda ,
$$

$\Delta \lambda$ 是极移改正量.

由于地球自转的不均匀性,UT1并不是均匀的时间尺度。而地球自转不均匀性呈现三种特性：长期慢变化（每百年使日长增加1.6毫秒),周期变化(主要是季节变化，一年里日长约有 $0 . 0 0 1$ 的变化;除此之外，还有一些影响较小的周期变化)和不规则变化.这三种变化不易修正，只有周年变化（即上述周期变化的主要部分)可用根据多年实测结果给出的经验公式进行改正，改正值记为$\Delta T _ { s }$ 由此引进世界时UT2:

$$
\mathbf { U T 2 = U T 1 + } \Delta T _ { s } .
$$

相对地说，这是一个比较均匀的时间尺度，但它仍包含着地球自转的长期变化和不规则变化，特别是不规则变化，其物理机制尚不清楚，至今无法改正.

周期项 $\Delta T _ { s }$ 的振幅并不大，而UT1又直接与地球瞬时位置相联系，因此，对于一般人卫工作，就可用UT1作为统一的时间系统；而对于高精度的要求，即使UT2也不能满足，必须寻求更均匀的时间尺度.

# 3.历书时（ET)

由于世界时不能作为均匀的时间尺度，经数次天文会议讨论决定，从1960年起曾引人一种以太阳系内天体公转为基准的均匀时间系统，称为历书时(ET),1960 年到1967年期间，它是世界公认的计时标准.

历书时的定义是：1900年1月0日历书时 $1 2 ^ { \tt b }$ 瞬刻的回归年长度的31556925.9747分之一为一历书秒；起算历元为1900年初太阳平黄经等于 $2 7 9 ^ { \circ } 4 1 ^ { \prime } 4 8 ^ { \prime \prime } . 0 4$ 的时刻，也就是纽康(Newcomb)原先选定的1900年1月0日格林尼治平午的时刻，现在把它作为1900年1月0日历书时 $1 2 ^ { \mathfrak { h } }$ .

历书时是--种由力学定律确定的均匀时间，它是太阳、月亮和行星运动理论中的独立变量，同时也是这些基本历表的时间引数．某一时刻的历书时可以通过对太阳、月亮或行星的观测来得到，而最有效的方法是观测月亮．但对建立一个均匀时间尺度而言，其观测精度仍嫌不够，而且要得到这样的时间又很缓慢．因此,1967年后，计时标准转向原子时，它有更高的精度，而且随时可以直接求得．不过在这期间，历书时仍然作为一个天文常数被保留下来．从1984年开始，历书时才完全被原子时所代替。

# 4.国际原子时（TAI)

这是一种标准频率．1967年10月，第十三届国际计量大会决定引入新的秒长定义，即铯原子 ${ \mathsf { C s } } ^ { 1 8 8 }$ 基态的两能级间跃迁辐射的9192631770周所经历的时间作为一秒的长度，称为国际单位秒(SI).由这种时间单位确定的时间系统称为国际原子时(TAI).取1958年1月0日的UT20时为起算点，这是希望在起始历元有 $\mathbf { T A I } = \mathbf { U T } 2$ ，但事后通过美、英、瑞士等国多台原子钟比对发现,并没有达到此目的,实际上是

$$
\begin{array} { r } { \left( \mathbf { T A I - U T 2 } \right) _ { \textrm { l 9 5 8 , 0 } } = - 0 \cdot 0 0 3 9 , } \end{array}
$$

这就是国际原子时的起始时刻.

随着原子时的引入和某些具体问题的要求，又出现下列两种时间系统.

# 5.协调世界时(UTC)

用原子钟控制时号发播可得到稳定的时号．但由于原子时秒长比世界时秒长略短，世界时时刻将日益落后于原子时；而有很多问题却涉及到计算地球的瞬时位置，这又需要UT1．因此，为了避免发播的原子时与世界时产生过大的偏离，实际的时号发播是寻求TAI与UT1之间的一种协调，称为协调世界时(UTC).

UTC是一种均匀时号,它依据原子时，却又参考世界时．从1972 年起，UTC用原子时秒长发播，但要求它与UT1之差不超过 $0 . 9$ ，为达到此目的，必须调整UTC的整秒数，规定只在1月1日或7月1日将原子钟拨慢1秒,这就是所谓闰秒．在引用UTC时必须注意这一点.

6.动力学时—地球动力学时(TDT)和质心动力学时(TDB)从1984年起,历书时正式被下述力学时所取代.

因原子时TAI是在地心参考系中定义的具有国际单位制秒长的坐标时间基准，它就可以作为动力学中所要求的均匀时间尺度。由此引人一种地球动力学时(TDT),它与原子时TAI的关系为

$$
\mathrm { T D T = T A I + 3 2 ^ { \circ } } , 1 8 4 .
$$

这一关系是根据1977年1月1日 $0 0 ^ { 5 } 0 0 ^ { \textmu } 0 0 ^ { \textmu }$ （TAI）对应TDT为1977年1月10003725而来，此起始历元的差别就是该时刻历书时与原子时的差别．这样定义起始历元就便于用TDT系统代替ET系统．显然,TDT是地心时空标架的坐标时，用作视地心历表的独立变量．在人造地球卫星动力学中，它就是一种均匀时间尺度,相应的运动方程即用它作为自变量，通常以 $\pmb { \mathscr { E } }$ 表示.

除此之外,还定义一种质心动力学时TDB，即太阳系质心时空标架的坐标时.它是一种抽象、均匀的时间尺度,月球、太阳和行星的历表都是以TDB为独立变量的，岁差、章动的计算公式也是依据该时间尺度的。

上述两种动力学时的差别TDB一TDT是由相对论效应引起的,它们之间的转换关系正在讨论中，文[1]曾给出了它们之间的一种转换公式,其中最大的一项为

$$
1 { ^ 8 } 6 5 8 \times 1 0 ^ { - 3 } \ \sin E ,
$$

$\pmb { \mathcal { L } }$ 是地月系质心绕日轨道的偏近点角.

# \$1.3坐标系统

空间坐标系的复杂性主要是由岁差章动和地极移动引起的。

定义一个空间坐标系应包含三个要素：坐标原点，参考平面$( x y$ 平面）和参考平面上的主方向（ $\dot { \pmb x }$ 轴方向)．对于人卫工作而言，所涉及到的主要是地心坐标系和站心坐标系，它们的坐标原点分别为地心和测站，这一点并无问题．但参考平面及其主方向的选择，将会受到岁差章动和地极移动的影响。

日、月和大行星对地球非球形部分的吸引，会产生两种效应.一是作为刚体平动的力的效应，主要是月球对地球扁球部分的作用，将引起一种地球扁率间接摄动．另一种就是作为刚体定点转动的力矩效应，使地球像陀螺那样，出现进动与章动，即自转轴在空间摆动，这就是岁差章动．由于岁差章动，地球赤道面亦随着时间在空间摆动；另外，由地球内部和表面物质运动引起的地球自转轴在其内部的移动(极移)，都将影响坐标系中参考平面的选取问题.

基于上述原因，根据不同要求，就出现了各种空间坐标系统.下面介绍人卫工作中所遇到的几种空间坐标系以及它们之间的转换关系.

# 1．六种地心坐标系和两种站心坐标系

关于轨道坐标系，这里有必要进一步作些说明，在一般人卫工作中，采用分析方法计算卫星轨道的变化显然是方便的．对于这种方法，若引用历元地心惯性系，那么由于岁差章动导致地球赤道面在空间摆动，从而引起地球引力场位函数的变化，使卫星轨道增加一种附加摄动(亦称坐标系摄动,将在第十一章中详细讨论),随着计算时刻与选取历元之间的间隔增长而增大，这将给定轨问题带来麻烦．若引用瞬时真赤道地心系，虽然地球引力场位函数基本不变，但它是非惯性系，需增加一项惯性力，这也是一种坐标系摄动，尽管它比上述附加摄动小（主要一项是由春分点移动对应的赤经岁差章动 $\mu + \Delta \mu$ 所引起的)，但仍嫌不便，兼于上述两种坐标系的优缺点，人卫工作中常采用一种混合地心系，即参考平面为瞬时真赤道，而 $\pmb { x }$ 轴是指向某历元(如1950.0或2000.0)的平春分点（该点实为瞬时真赤道上的一个“假想”点，在真春分点以东 $\mu + \Delta \mu$ 处，赤经岁差和章动 $\mu + \Delta \mu$ 的计算公式后面将会给出），这就是表1.2中所列出的轨道坐标系，在此坐标系中，附加摄动很小,对于一般精度要求，可完全忽略，不过，对于数值方法，只要通过坐标转换即可解决上述附加摄动问题，无须引进轨道坐标系;当然,引进也无妨，仍有可取之处.

除上述各种地心和站心坐标系外，有时涉及到日、月和大行星的坐标，它们的历表是对应某历元的太阳系质心惯性系的，坐标原点是太阳系质心，参考平面是该历元的平赤道， $\pmb { x }$ 轴指向该历元的平春分点.

# 3.测站坐标与参心坐标系

测站的位置矢量 $\pmb { r _ { A } ^ { \prime } }$ 是在地固坐标系中给出的，相应的测站地理经纬度均对应平极(取为CIO).测站的球坐标三分量为大地高H和经纬度λ,φ、关于大地高H,它是从地表面到参考椭球面(地固坐标系就是在这一参考椭球体上建立的，其中心即当作地心)的高度,而从地表面到大地水准面的高度称为正高(或称为海拔高度),它与大地高之差称为高程异常,关于纬度 $\varphi _ { i }$ 有大地纬度与天文纬度之别,大地纬度是参考椭球面的法线与赤道面之间的夹角，而天文纬度是铅垂线方向（垂直于当地的水准面）与赤道面的夹角，一般铅垂线与椭球面的法线并不重合，这就是垂线偏差，对于精度要求较高的问题,必须考虑上述差别;而对于一般人卫工作可不必考虑，即无需区别大地高和正高，天文纬度和大地纬度.

有时测站坐标需要用直角坐标 $X ^ { \prime } , Y ^ { \prime } , \mathbf { \vec { z } ^ { \prime } }$ 来表达，根据椭圆性质，不难给出它们与球坐标的关系如下：

$$
\begin{array} { r l } & { \int _ { \mathbb { Z } ^ { \prime } = [ N +  { H _ { \circ } } ] \cos \varphi \cos  { \lambda } , } ^ { X ^ { \prime } = ( N +  { H _ { \circ } } ) \cos \varphi \cos  { \lambda } , } } \\ & { \left\{ Y ^ { \prime } = (  { N } +  { H _ { \circ } } ) \cos \varphi \sin  { \lambda } , \right. } \\ & { \left. \int _ { \mathbb { Z } ^ { \prime } = [ N ( 1 -  { e _ { c } } ^ { 2 } ) +  { H _ { \circ } } ] \sin \varphi , } ^ { \phantom { \dagger } } \right. } \end{array}
$$

其中

$$
\begin{array} { r } { N = \frac { a _ { e } } { \sqrt { 1 - e _ { c } ^ { 2 } \sin ^ { 2 } \varphi } } \ , } \end{array}
$$

${ \pmb q } _ { \pmb { \imath } }$ 是参考椭球体的赤道半径， $e _ { c }$ 是地球子午圈的偏心率，它与地球扁率 $\pmb { \ 6 }$ 的关系为

$$
e _ { o } ^ { 2 } = 2 \epsilon - \epsilon ^ { 2 } .
$$

若 $\epsilon = 1 / 2 9 8 . 2 5 = 0 . 0 0 3 3 5 2 9 ,$ 财 $e _ { c } ^ { 2 } = 0 . 0 0 6 6 9 4 6 .$ ，对于一般人卫工作,可作近似：由测站到地心作一连线，该连线与参考椭球面有一交点，就把这一交点到测站的距离看作海拔高度 ${ \pmb H }$ ，于是有

$$
( R - H ) \cos \varphi ^ { \prime } = \biggl [ \frac { a _ { \circ } } { \sqrt { 1 - e _ { c } ^ { 2 } \sin ^ { 2 } \varphi } } \biggr ] \cos \varphi ,
$$

$$
( R { - } H ) \sin \varphi ^ { \prime } { = } \biggl [ \frac { \mathcal { Q } _ { e } } { \sqrt { 1 - e _ { c } ^ { 2 } \sin ^ { 2 } \varphi } } \biggr ] \sin \varphi .

其中 $\pmb { R }$ 是测站到地心距离(简称地心距)， ${ \pmb \varphi } ^ { \prime }$ 是测站的地心纬度.根据这一公式，可由 $H , \varphi$ 算出相应的 $R , \varphi ^ { \prime }$ ．若用 $R , \varphi ^ { \prime } , \lambda$ 表示测站的球坐标，则与直角坐标的关系为

$$
\scriptstyle { \left\{ \begin{array} { l l } { X ^ { l } = R \cos \varphi ^ { l } \cos \lambda , } \\ { Y ^ { l } = R \cos \varphi ^ { l } \sin \lambda , } \\ { Z ^ { \prime } = R \sin \varphi ^ { \prime } . } \end{array} \right. }

最后谈谈参心坐标系的问题.在人造卫星上天前，用大地测量方法给出的参考椭球体，其中心(简称参心)与地心相差较大.如我国曾采用过的苏联克拉索夫斯基椭球体以及后来建立的1967年大地参考系都是如此，参心与地心之差达百米的量级.我们不妨称它们为局部参考椭球体，依据这种椭球体建立的参考系称为参心坐标系，过去的测站坐标都是在这种参考系中给出的。

参心坐标系与地心坐标系之间的差别，不仅是坐标原点不一致，三个坐标轴也不平行，而且两个坐标系的长度标准亦不统一（这是由于所引用的基本常数及其有关量不一致引起的).因此，在使用时必须知道它们之间的转换关系．为此，我们将参心坐标系和地心坐标系分别记为 $\pmb { O } ^ { \prime } \mathbf { - x } ^ { \prime } \pmb { y } ^ { \prime } \mathbf { z } ^ { \prime }$ 和 $0 - x y z$ ，参心 $\pmb { O } ^ { \prime }$ 在地心坐标系中的坐标为 $\Delta x , \Delta y , \Delta z ;$ ：参心坐标系经三次旋转 $( x - y$ 平面绕 $\pmb { \mathcal { z } }$ 轴， $y - z$ 平面绕 $\pmb { x }$ 轴和 $z - x$ 平面绕 $y$ 轴)使其三轴平行于地心坐标系的三轴所对应的旋转角(即欧拉角)为 $\omega _ { z } , \omega _ { x } , \omega _ { y } ;$ 两坐标系之间的长度改正因子为 $\pmb { \kappa }$ ，那么，这两个坐标系之间的转换关系就是

$$
{ \left( \begin{array} { l } { x } \\ { y } \\ { z } \end{array} \right) } = \left( \mathbf { 1 } + \kappa \right) R _ { y } \left( \omega _ { y } \right) R _ { x } \left( \omega _ { x } \right) R _ { z } \left( \omega _ { z } \right) { \left( \begin{array} { l } { x ^ { \prime } } \\ { y ^ { \prime } } \\ { z ^ { \prime } } \end{array} \right) } + { \left( \begin{array} { l } { \Delta x } \\ { \Delta y } \\ { \Delta z } \end{array} \right) } .
$$

通常称 $\kappa , \Delta x _ { i } \Delta y , \Delta z , \omega _ { x } , \omega _ { y } , \omega _ { z }$ 七个量为坐标系的转换参数．例如，北美参心坐标系与美国史密森天文台(SAO)所建立的地心坐标系之间的七个转换参数为

$$
\kappa { = } ( 5 . 7 3 { \pm } 2 . 1 ) \times 1 0 ^ { - 6 } ,
$$

$$
\Delta x = - 3 7 . 9 \pm 7 . 2 ( \mathrm { m } ) , \Delta y = 1 6 4 . 1 \pm 7 . 1 ( \mathrm { m } ) ,
$$

$$
\Delta z = 1 7 4 . 4 \pm 7 . 1 ( \mathrm { m } ) ,
$$

$$
\omega _ { x } = 1 ^ { \prime \prime } . 2 4 \pm 0 . 7 , \omega _ { y } = 1 ^ { \prime \prime } . 2 9 \pm 0 . 8 , \omega _ { z } = - 1 ^ { \prime \prime } . 1 7 \pm 0 . 6 ,
$$

只要地面上三个点,既有参心坐标,又有地心坐标,即可按（1.68）式确定七个转换参数．当然，实际上是利用多站来确定的.一般三个旋转参数 $\omega _ { x } , \omega _ { y } , \omega _ { z }$ 和长度改正因子 $\pmb { k }$ 不易测准，但它们的量级比原点改正参数 $\Delta \pmb { x }$ ， $\Delta y$ ， $\Delta z$ 小．略去 $\omega _ { x } , \omega _ { y } , \omega _ { z }$ 的二阶量，(1.68)式即可简化为

$$
\left( \begin{array} { c } { x } \\ { y } \\ { z } \end{array} \right) = ( 1 + \kappa ) \left( \begin{array} { c c c } { 1 } & { \omega _ { z } \ - \omega _ { y } } \\ { - \omega _ { z } } & { 1 } & { \omega _ { x } } \\ { \omega _ { y } } & { - \omega _ { x } } & { 1 } \end{array} \right) \left( \begin{array} { c } { x ^ { \prime } } \\ { y ^ { \prime } } \\ { z ^ { \prime } } \end{array} \right) + \left( \begin{array} { c } { \Delta x } \\ { \Delta y } \\ { \Delta z } \end{array} \right) ,
$$

或写成下列形式：

$$
\begin{array} { r } { \left\{ \begin{array} { l l } { x = \left( 1 + \kappa \right) x ^ { \prime } + \Delta x + \left( \omega _ { z } y ^ { \prime } - \omega _ { y } z ^ { \prime } \right) , } \\ { y = \left( 1 + \kappa \right) y ^ { \prime } + \Delta y + \left( \omega _ { x } z ^ { \prime } - \omega _ { z } x ^ { \prime } \right) , } \\ { z = \left( 1 + \kappa \right) z ^ { \prime } + \Delta z + \left( \omega _ { y } x ^ { \prime } - \omega _ { x } y ^ { \prime } \right) . } \end{array} \right. } \end{array}

# 1.4 卫星的空间坐标

目前对人造地球卫星观测所得到的资料基本上有四种类型，即

(1）光学测角资料(I)—地平坐标 $( A , h )$ 型， (2）光学测角资料(II)一赤道坐标(α,)型, （3）多普勒测速 $\pmb { \rho }$ 型，   
（4）激光或雷达测距 $\rho$ 型.

在取得原始观测资料后,需要经过一系列改正才能提供可用资料。这些改正包括天文系统差(如大气折射、光行差、岁差章动等)，各种物理因素的影响(如大气对流层、电离层的影响等)以及仪器误差等。关于这些改正，请读者参看有关书籍和文献，本书不再介绍.

但有一点需要说明，即所有观测资料对应的空间坐标系必须清楚，显然，测速和测距资料并不存在这个问题,而需要讨论的是光学测角资料.

# 1. $( A , h )$ 资料对应的地心坐标

如果是在地固坐标系中讨论问题,那么根据上一节的讨论,对于这种类型的资料，其站心位置矢量 $\pmb { \rho }$ 对应地固坐标系中的 $\pmb { r } ^ { \prime }$ ，它们的关系为

$$
\begin{array} { r } { \pmb { r } ^ { \prime } { = } \pmb { r } _ { A } ^ { \prime } + ( \pmb { Z } \pmb { G } ) ^ { \tau } \pmb { \rho } . } \end{array}
$$

其中 $r _ { A } ^ { \prime } , \rho$ 和矩阵 $( \mathbf { Z } G )$ 的表达式，请见(1.65)，(1.44)和(1.48)式．如果在轨道坐标系中讨论问题，则有

$$
r { = } R _ { z } ( { \ - } \mathscr { \ - } \theta _ { G } ) ( E P ) ^ { \pi } [ r _ { A } ^ { \prime } + ( Z G ) ^ { \pi } \rho ] .
$$

对于一般精度要求,极移可不考虑,则上式简化为

$$
\pmb { r } { = } \pmb { r } _ { A } { + } ( \pmb { Z } S ) ^ { \pmb { \tau } } \pmb { \rho } ,
$$

其中 $\boldsymbol { r } _ { A }$ 即轨道坐标系中的测站位置矢量，见(1.67)式.矩阵 $( \mathbf { \vec { Z } } \mathbf { \vec { S } } )$ 由下式表达：

$$
( \dot { \mathbb { Z } } \mathcal { S } ) = R _ { y } \bigg ( \varphi - \frac { \pi } { 2 } \bigg ) R _ { z } ( \theta _ { a } + \lambda - \pi ) \mathrm { ~ . ~ }
$$

# 2. $( \alpha , 8 )$ 资料对应的地心坐标

$( \alpha , \delta )$ 型资料一般有两种形式。一是照相资料,定标星的位置由星表给出,通常是对应历元地心惯性系的;而另一种则是直接得到的光学资料，对应站心赤道坐标系（位置矢量即 $\rho ^ { \prime }$ )．对于后一种形式，在地固坐标系和轨道坐标系中，其位置矢量分别由下式表达：

$$
\mathbf { \boldsymbol { r } } ^ { \prime } = \mathbf { \boldsymbol { r } } _ { A } ^ { \prime } + \left( E \mathbf { \boldsymbol { R } } \right) \mathbf { \boldsymbol { \rho } } ^ { \prime } .
$$

和

$$
r { = } R _ { z } ( - \theta _ { G } ) ( E P ) ^ { \pi } [ r _ { A } ^ { \prime } { + } ( E R ) \rho ^ { \prime } ]

同样，若略去极移量，则上式简化为
