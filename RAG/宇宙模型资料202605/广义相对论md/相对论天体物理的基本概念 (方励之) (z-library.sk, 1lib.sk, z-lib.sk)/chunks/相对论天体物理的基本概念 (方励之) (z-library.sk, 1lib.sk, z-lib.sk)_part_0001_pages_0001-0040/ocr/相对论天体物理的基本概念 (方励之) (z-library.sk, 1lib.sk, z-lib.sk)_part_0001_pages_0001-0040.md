# $\frac { 5 } { 3 } y = 1$ 天体物理问题的特点

天依物理的特点之一，是它经常要讨论大的时空尺度，比如，遥远的星系发射光，我们接收光，这是天体物理最常遇到的事件，天体物理所要讨论的常常就是这些相距很远的事件之间的关系

为了描写事件，我们要用一定的参考系，在给定的参考系中，取定坐标之后，每个事件就可以用四个数来加以标志：三个空间坐标 $\sqrt { \frac { x } { y } }$ 表示事件发生的位置：一个时间坐标 $t$ 表示事件发生的时间

为了描写我们接收到光这个事件，最方便的参考系是取相对于我们自己为静止的参考系，或者取相对于银河系、相对于本星系群为静止的参考系，

但是，另一方面，对于遥远的星系 $\frac { \sigma } { T }$ 的发射光这件事来说，最好取相对于G为静止的参考系，或者，取相对于 $G$ 所在的星系团为静止的参考系，在这个参考系中，发光事件的坐标是 $\frac { \mu _ { 1 } } { \mu _ { 2 } } , \frac { \mu _ { 2 } } { \mu _ { 1 } }$ 显然，一个必须解决的问题是：我们如何将一个事件在某一参考系中的坐标变换成另-参考系中的坐标？只有这样，我们才能知道，在我们的坐标系中，发光事件的坐标 $\sqrt { r }$ 及 $\ddagger$ 应当是什么.

适用于牛顿力学的坐标变换是伽利略变换，这就是如下熟知的一组公式：

$$
\left. \begin{array} { l } { { \displaystyle { \vec { x } } ^ { t } = \Re - \vec { x } ^ { t } } } \\ { { \displaystyle { \vec { y } } ^ { t } = \Re } } \\ { { \displaystyle { \vec { z } } ^ { t } = \Xi } } \\ { { \displaystyle { \vec { y } } ^ { t } \cdot \cdot \cdot \frac { x } { c } } } \end{array} \right\} ,
$$

其中 $( \frac { 1 } { 6 } , \frac { 2 } { 3 } , y , \frac { 3 } { 2 } )$ 表示在惯性坐标系 $k ^ { 2 }$ 中一个事件的坐标，而$( t ^ { p } , \phi ^ { p } , y ^ { p } , z ^ { p } )$ 表示同一事件在另一惯性系 $E ^ { \gamma , r }$ 中的坐标． $E ^ { \gamma }$ 相对于 $E$ 以速度 $v$ 沿 $\pmb { \mathcal { D } }$ 轴运动.

但是，在天体物理中-般不能再用伽利略变换，原因有两方面.

其一，在天体物理中经常要遇到高速情况，比如，刚才讨论的星系 $\frac { T } { T }$ 发射光和我们接收光这一问题，如果星系 $\frac { 1 7 } { 1 7 }$ 与我们的距离足够远，一般地说，星系相对于我们是有很大的速度的，这样，我们不能使用仅适用于低速情况的变换.

其二，银河系和星系 $\alpha$ 之间往往还存在着加速度，这种加速度是由引力引起的，因此，我们还必须寻求适用于有引力作用情况的坐标变换.

这后一点，是天体物理的又一个特点：只有在天体物理问题中，引力才起主导作用，反之，在地而实验室范围中的物理学，引力的作用一般是次要的.

在几种基本的相互作用中引力是最弱的一个。一个电子和一个质子之间的静电作用要比它的引力作用约大 $e ^ { \frac { 2 } { 2 } } / [ \frac { \gamma } { 2 } + \gamma \gamma _ { k , g } ^ { 2 } ] ^ { 2 } r \gamma _ { k , g } = 1 0 ^ { \frac { 2 } { 2 } g }$ 倍，因此，在原子物理中，我们完全忽略了引力的影响，类似地，在核物理中，也可以忽略核子之间的引力作用，

在大尺度上，则相反，引力占据了主导的地位，因为引力是长程力，在大范围上核子之间的短程相互作用是完全可以忽略的，原子、分子之间的范德瓦尔斯力，按 $1 _ { i } ^ { i } y ^ { i } ^ { T }$ 规律下降，当原子之间大于10-厘米时，这种力也就很小，可以略去了，所以，对于分布在真空中，相距很远的体系来说，强的短程力都不再起作用。

静电力虽然也是一种长程力，但是静电力是可以屏蔽的，宇宙间有正电荷，也有负电荷，而日，天体平均地说，是电中性的，即正电荷等于负电荷，以致长程的静电力全都被屏蔽掉了，引力却是不可屏蔽的，因为万物皆相互吸引，这与静电力是士分不同的，

总之，对于大尺度时空上的问题，不可避免地要遇到引力，为了丢清楚天体尺度上的事件，首先就需要有正确的引力理论，牛顿引力理论在弱场、低速的问题中，是十分正确的，然ii，相对论天休物理所要讨论的往往是强场，或是高速，或者是二者兼而有之的情况，这就是天体物理的特点所提出的特有的问题。

# $f ^ { \prime } ( 2 )$ 牛顿力学与绝对空间

牛顿第二定律是整个牛顿力学体系的核心，它的最基本的表述是：

$$
H ^ { \prime } = \pi r \mathbf { \bar { \Phi } } _ { 3 }
$$

其中 $m = 7$ 是物体的质量， $\pmb { \Vert }$ 是它相对于某一参考系的加速度， $F$ 是所有其他物体对这个物体的作用力，

从物理上看，牛顿第二定律并不对任何参考系都适用，因为，力 $F$ 是描写宇宙间所有其他物体对该物体的作用的，这个定义不涉及坐标系，它应当是一个不随坐标系变化而变化的量，然而，另一方面： $\pmb { \mathbb { M } }$ 则是相对于某一坐标系的加速度，这样，当我们变换参考系时，一般说 $\pmb { \mathbb { I } }$ 是变化的，而 $F$ 则不变，所以，如果在某个参考系中 $F$ 与 $\sqrt [ n ] { \sqrt { l } } \cdot \frac { \sqrt { l } } { l }$ 相等，则在新的坐标系中一般就不相等，比如，我们选择一个新的参考系，它相对于原来的参考系具有加速度 $A$ ，那么：在新参考系中，物体加速度为 $a ^ { l } = a - 4$ ，所以，在新参考系中$F = \pi r \alpha ^ { n }$ 不再成立，而应当是 $F ^ { \prime } { = } \pi ( \bar { a } ^ { i } + A )$ ，而这并不是牛顿第二定律.

牛顿力学所适用的一类参考系是所谓惯性系，惯性系之间作相对的匀速运动，可见惯性系在牛顿力学体系中占据着极为特殊的地位.

不过，我们要探讨更为深一层的问题：为什么自然界单单挑出这一类参考系是“优越”的？如何确定一个参考系是不是惯性的？有一种常见的看法，认为惯性系本身是不作加速运动的、也没有转动，果然如此的话，我们不禁又要问：这个加速或旋转又是相对于什么来说的呢？

牛顿答复了所有的问题，其关键是引入了绝对空间概念，相对于绝对空间的无转动无加速的参考系才是惯性系，这个绝对空间是一个“与外界任何事物无关的，总保持着同样情况的不变的”空间，按照牛顿的观点，我们可以用简单的实验来确定相对于绝对空间的加速度或转动运动.

例如，如果我们向前加速，则我们看到房屋向后加速，然而，此时并没有发现向后的力作用到房屋上，所以，按照牛顿的说法，观测到的房屋加速并不是牛顿第二定律的结果，而是由于观察者相对于绝对空间的加速运动引起的。

再如，桶水在转动，我们如何来判断它的转动是绝对的（即相对于绝对空间有转动)，抑或是相对的(即由观测者本身的转动而引起的视效应)？牛顿的判据是：如果我们看到桶中的水面是凹的，它就有绝对转动，相反，若水面是平的，它的转动就仅仅是一种视效应，即由观察者本身的绝对转动引起的，

然而，尽管牛顿给出了绝对加速以及绝对转动的判据，但是他未能给出绝对速度的判据，实际上，远在牛顿之前，早已有人意识到我们是找不到"绝对静止”的基准的，远在两千年前，中国的古籍《尚书纬·考灵曜》一书中就有这样一段话：“地恒动不止，而人不觉，臂如人于舟中，闭窗而坐，舟行而人不觉也”，伽利略在论述这个问题时，用了完全相同的比喻，他在《关于托勒密和哥白尼两大世界体系的对话》一书中写道：

“把你和一些朋友关在一条大船的甲板下的主舱里，让你们带着儿只苍蝇、蝴蝶和具他小飞虫，舱内放一只大碗，其中有儿条鱼，然旨，挂上个水瓶，让水一滴一滴地滴到下面的一个宽门罐里，船停着不动时，你留神观察，小虫都以等速向舱内各方向飞行，鱼向各个方随使游动，水滴滴进下面的罐中，你把任何东西扔给你的朋发時，只要距离相等，向这一方向也不必比另一方向用更多的力，你双弱齐跳，无仑向哪个方向跳过的距离都相等，当你仔细地观察这些事情之后，再使船以年何速度前进，只要运动是匀速的，也不忽左忽有地摆动，你将发现，所有上述现象丝毫没变化，你也无法从其中何一个现象米确定，船是在运动还是在停着不动，即使船运动得当快，在跳跃时，你将和以前一样，在船底板上跳过相同的距离，你跳向船尾也不会比跳向船头来得远，虽然，当你跳到空中时，胧下的船底板向着你跳的相反方向移动，你把不论什么东西扔给你的同伴时，不论他是在船头还是在船尾，只要你自已站在对面，你也并不需要用更多的力，水滴将象先前一样，滴进下面的罐子里，一滴也不会滴庖船尾，虽然水滴在空中时，船已行驶了许多柞，鱼在水中游向水碗前部所用的力并不比游向水碗后部来得大，它们一样悠闲地游向放在水碗边缘任何地方的食饵，最后，蝴蝶和苍蝇继续随便地到处飞行，它们也决不会向船尾集中，并不因为它们可能长时间留在空中，脱离开了船的运动，为赶上船的运动而显出累的样子.”

总结这些话就会得到这样的结论：绝对速度是不可测量的。这是第一个否定绝对空问概念的有力的论据。

再有，在牛顿的体系中，绝对空间是一个物理实在，即一个有物理作用的客休，因为它会影响到物体的动力学性质，比如，只有相对于它来说，惯性定律才是成立的，这是一种十分强的影响，但是，物质的运动反过来却不能对绝对空间有任何影响，因为它“外界任何情况无关”，这种没有反作用的单向关系，在物理上是不能令人满意的.

# 51.5 马赫原理

狭义相对论是在惯性系的范围内排除了牛顿绝对空间的概念，即我们不能断定哪一个惯性系是绝对静止的，但是，在惯性系与非惯性系之间似乎仍有明显的区别，因为例如牛顿的水桶给出了什么是绝对加速度的判别法。

莱伯尼兹、贝克莱等一开始就不赞成牛顿的观点，他们不仅在绝对速度不可测量的意义下否定绝对空间的概念，而且认为绝对加速，或绝对转动的概念同样是不正确的，马赫更全面地发展了这种观点，

牛顿认为，即使宇宙间没有任何物质存在，也可以想象一个空的空间存在着，而且可以判断物体相对于这个空间是否有加速运动，但是，如果我们想象一种宇宙，其中只有两个物体存在，我们能不能判断它们是在相互绕质心旋转呢？牛顿认为可以，只要用一弹簧将两物体连结起来，如果体系有绝对转动，弹簧就要被拉长，根据弹簧是否伸长，就可以判断有没有绝对的转动，莱伯尼兹及贝克莱不同意这种观点，他们主张，在这种想象的宇宙中弹簧永远不会有任何伸长，他们认为牛顿关于这种想象宇宙的讨论是基于他对现实宇宙的观测的，然而，现实宇宙与想象宇宙有巨大的差别，这就是现实宇宙中存在着大量的星及星系，它们构成了宇宙的物质背景，当两物体间的弹簧伸长时，我们从观测上发现的只是它们相对于大量的星系在转动，而不是相对于绝对空间.在这里绝对空间的观念间样是无用的，

1872年马赫提出，物体并没有什么绝对加速度，而只有相对于遥远星系的加速度，下面用一个例了来说明这种观点，夜间，我们站在星斗之下，我们看到满天的星星是静止的，这时，我们的两臂自然地下垂，当我们突然转动身躯时，我们看到两件事同时发生了，一是星星开始旋转，二是我们的两臂也被甩向外边，牛顿认为这两件事是没有直接的关联的，而是由于存在第三者一一绝对空间，相对于绝对空间的转动引起惯性离心力，是绝对空间的存在决定了两臂被甩开，相反，马赫认为不存在这个想象中的第三者，关键是上述同时看到的两种现象之间有直接的关系，是转动的星的体系决定了两臂的甩开运动，应当建立一种更正确的动力学，它应当说明转动星体如何作用到手臂上产生了牛顿休系中的“惯性力”

显然,引力的最重要特征之一，就是引力质量和惯性质量之间的不可区分。

在进一步发展这种观点之前，我们该说几句有关物理理论的个有趣特征，这涉及它的方法论，我们知道，对一件事做肯定的判断是不容易的，但是，作否定的判断有时更难，比如，当我们说"这件事可能办到”的时候，只要有一次，或一种情况下实验证明能办到，则就算有了充分的观测根据，可是，当我们说“这件事不可能办到”时，那就必须证明用任何办法也不能成功，显然，不能指望用实验来作这个证明，因为实验总是有限的，不可能把无限多的情况都一一作完，所以，否定判断，实质上总是由有限的实验通过理论抽象而获得的，

# \$1-7 局部惯性系

等效原理保证了在任何个局部范围中，一定存在着其中引力的作用都被消除了的参考系，这种消除引力的参考系，称为局部惯性系，在这种参考系中，没有引力的作用，因此，一切不受外力(指引力以外的其他力)的质点都作直线运动，所以，它符合惯性系的定义.

在通常的牛顿力学中，用惯性定律来判断一个参考系是不是惯性系，即在没有外力的环境中，质点应保持惯性运动，但是，由于引力是不可屏蔽的，它无处不在，所谓“没有外力的环境”实质上是一-个不存在的环境，将惯性系建立在这种想象上，是缺乏实际意义的，只有在局部惯性系中，我们才真正找得到"没有外力的环境”，并且在这个环境中的确仍有惯性定律，因此，局部惯性系

更加接近惯性系的本来的含意.

在任何局部时空中，存在着无限多个局部惯性系，它们之间作匀速相对运动，狭义相对论只能适用于每一个局部范围之中.

局部惯性系比牛帧体系中的惯性系概念更明确也更一般，首先，它说明由于引力场的存在只有在局部范围中才能使用这概念，牛顿体系中的大范出的、甚至全空间统一的惯性系是不存在的，其次，在牛顿休系中不可能理解为什么惯性系是“优越”的；而现在却看到，它之所以优越是因为在这种参考系中消除了引力的作用，再则，在牛顿休系中，惯性系是决定于绝对空间的，它不受物质运动的影响；现在，一个作自由落体运动的实验室才是一个局部惯性系，显然它是决定于物质的分布及运动的，现在我们并不要求局部惯性系相对于某个绝对空间是无加速度的，也不要求各个不同的局部范围内的惯性系之间是无加速度的，例如，围绕地球运行的人造卫星、或者飞向金星的飞船，它们都是局部惯性系，因为它们都是在纯引力的作用下作自由的飞行．尽管它们之间可能是有加速度的.

如上节已经说过的，不能期望在一个大范围上把引力全部消除掉，例如，两个物休之间的相对加速度就不能用选择参考系的方法加以消除，这种加速度反映着两个时空点(两个局部)上的引力作用的差别，反映着两点上的局部惯性系之间的关系

总之，引力的作用使大范围的惯性系不再存在，只能有局部惯性系，这些局部惯性系之间的关系则是由引力所确定的，可以简单地说，引力的作用就在于决定各个局部惯性系，

引力决定着参考系，而参考系是描写物质运动的时空背景。因此，引力实质上是通过决定时空背景的性质来影响物质的运动，与其他的力不同，引力是由时空度规来描写的。
