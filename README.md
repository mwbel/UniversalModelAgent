# 宇宙模型智能体项目

融合现代天文学、古代天文学、数学、物理学等交叉学科知识的智能交互平台。

## 项目文档

- [PRD_宇宙模型智能体.md](./PRD_宇宙模型智能体.md) - 产品需求文档
- [任务执行清单.md](./任务执行清单.md) - 详细任务分解

## 项目结构

```
UnivModel/
├── frontend/                 # 前端应用
│   ├── src/

│   │   ├── styles/          # 样式文件

├── backend/                 # 后端服务（待开发）

```

## 快速开始

### 前端开发

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:3000 查看应用

### 前端
- React 18 + TypeScript
- Vite
- CSS3

### 后端（规划）
- Python + FastAPI
- PostgreSQL + 向量数据库
- LangChain/LlamaIndex

### 3D 可视化（规划）
- Three.js / Babylon.js

## 核心特色

1. **跨学科知识整合** - 融合现代天文学、古代天文学、数学、物理学
2. **时轮金刚体系** - 独特的藏传佛教宇宙观可视化
3. **AI + 3D 联动** - 问答驱动可视化，可视化驱动问答
4. **古今对比** - 展现人类对宇宙认知的演进
