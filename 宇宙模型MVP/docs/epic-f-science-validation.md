# Epic F 可视化科学性校验记录

## 校验原则

- 以教学解释优先，强调现象机制与变量关系。
- 不追求真实比例尺，但避免与基础科学概念冲突。
- 每个模块中的说明文字应与 PRD 中的可视化目标一致。

## 模块校验

### 1. 日食 / 月食

- 日食路径设置为月球位于太阳与地球之间。
- 月食路径设置为地球位于太阳与月球之间。
- 交线偏差与月球轨道倾角示意用于说明“并非每次朔望都会发生食现象”。

### 2. 行星逆行

- 逆行解释采用“地球从内侧轨道超过外行星”这一经典教学机制。
- 右侧曲线明确标注为背景恒星上的表观视运动，避免误解为真实轨道。
- 外行星默认选项为火星、木星、土星，符合常见教学案例。

### 3. 开普勒第三定律

- 使用归一化单位，关系采用 `T = sqrt(a^3)`。
- 说明中明确指出重点是比例关系，不是具体观测数据。
- 同时展示轨道运动与关系曲线，保证“变量变化导致结果变化”可见。

### 4. 哈勃定律

- 趋势线采用 `v = H0 * d` 的线性模型。
- 噪声只用于模拟观测离散，不改变总体线性趋势。
- 文案避免把哈勃定律表述成任何单个星系的真实局域速度定律。
