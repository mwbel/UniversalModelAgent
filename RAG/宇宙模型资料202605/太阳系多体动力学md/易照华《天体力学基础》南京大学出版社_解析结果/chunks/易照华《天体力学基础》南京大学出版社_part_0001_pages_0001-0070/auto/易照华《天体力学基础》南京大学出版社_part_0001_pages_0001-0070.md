# 说 明

1987年，人民教育出版社计算机室按照第三次全国中学计算机教育工作会议上通过的《普通中学电子计算机选修课教学大纲 (试行)》的要求，编写了《程序设计》和《应用软件》供全国各地在高中开设计算机选修课的学校选用。六年来，这套教材印行约 250万册，为我国的中学计算机教育作出了一定的贡献。1991年10月，国家教育委员会基础教育司在山东省济南市召开了第四次全国中小学计算机教育工作会议。大会颁发了新的计算机教学大纲，即《中小学计算机课程指导纲要》，对我国的中小学计算机教育及教材建设提出了新的要求。为了满足计算机教育发展的需要，我室和全国中小学计算机教育研究中心（北京部）组织编写了这套新的高级中学计算机选修课教材。

为了编好这套新教材，使之能适应全国大多数地区中学计算机教学现状，符合教学实际，做到易教、易学，我们除总结上一套教材的使用经验外，还特别邀请了多年从事中学计算机教学工作、有丰富教学经验的教师，召开了计算机教材研讨会。教师们总结了用旧教材教学时的经验，指出了有关的缺点和不足，提出了很多宝贵的意见和建议。在此基础上，经过多次研究与讨论，确定了新教材的整体结构、教学内容的选取和教学顺序的安排等有关问题，编写了这套新教材。

# 第一节信息与计算机

# 一、信息

近十几年来，人们越来越多地通过各种传播媒介接触到信息这个名词，如信息传输、信息检索、信息处理、信息论、…。在这里，信息指的是声音、图形、信号、数据等各式各样的载体中所包含的实际意义。例如，古代烽火台上的狼烟烽火信号传递的信息是“有来犯之敌”；消防车行驶中呼啸的警笛发布的信息是“有火警，本车正赶赴火场”；计算机商店价目表中的4500.00这组字符表示的信息可能是“这台计算机的售价是四千五百元人民币”等。

信息在现代社会中的地位与作用越来越重要。政府机关、工商企业、金融部门、文化教育等社会的各个领域，每日每时都在收集、发送大量的有关信息，通过对信息的加工处理与分析作出各自的决策，指导整套机构的运转。例如，一个工厂要生产一种产品，必须收集市场需求、原材料供应、资金来源等多方面的信息，通过对这些信息的分析确定一整套方针与策略，才能调动本厂的有关部门进行一系列的业务操作。可以说，没有信息就没有整个社会的正常运行。

随着社会的进步与发展，需要处理的信息量也越来越大，一些传统的处理手段已经无法满足人们的需求。例如，气象部门做天气预报时要处理成千上万个数据，如果靠人拉计算尺或按计算器来处理数量如此巨大的数据，待到搞出结果恐怕早已时过境迁变得毫无意义了。于是，能高速、精确、自动地处理大量信息的电子计算机就应运而生了。

# 二、计算机

电子计算机简称计算机，也叫电脑，是一种能按照人们的要求自动接收、处理、输出信息的电子与机电设备构成的复杂装置。早期的计算机一般用于科学计算，或者说用于处理数值信息。现代计算机的应用已经扩展到科研、生产以及社会生活的各个方面，大量地用于对图形、文字、声音或其他形式的非数值信息的处理，已经成为能代替人进行多种繁重脑力劳动的工具。依照规模和处理信息的能力等指标划分，计算机有巨型机、大型机、小型机与微型机之分。由于微型计算机具有体积小、功耗低、使用方便、价格低廉等方面的优点，应用更为普遍，已经深入家庭应用领域。本书将以微型计算机为例，介绍有关计算机的初步知识，如果没有特别说明，本书所说的计算机一般指的就是微型计算机。

# 1．计算机的组成

这里所说的计算机，指的是整个微型计算机系统，它包括硬件和软件两大部分。

# (1）硬件

硬件是计算机的实体部分，如主机、键盘、显示器、打印机、磁盘驱动器等。图1.1是常见的微型计算机系统硬件的外观示意图。图中的计算机有一个带键盘的主机箱、一个显示器、一台打印机和放在一起的两个磁盘驱动器。键盘上有字母、数字、符号键和一些功能键，用这些键可以向计算机发命令指挥它工作。从外表看，显示器很像一台电视机。显示器的屏幕可以显示汉字、字母、数字、符号、图形等信息。打印机可以把汉字、字母、数字、符号、图形等打印在纸上。磁盘驱动器的作用与录音机类似。用录音机可以把声音录在磁带上，也可以把磁带上录下的声音放出来；用磁盘驱动器可以把文字、图形等信息“录制”在磁盘上，需要时还可以从磁盘上调出来。主机的内部是一些复杂的电子线路，例如，运算器、存储器、控制器等。可以把主机看作整个计算机系统的中枢，它的作用是控制其他部分和主机一起完成信息处理任务。例如，主机可以接受从键盘上输入的信息并对它们进行适当的处理、指挥显示器显示信息、指挥打印机输出信息、指挥磁盘驱动器把信息存放到磁盘上或从磁盘中取出信息，等等。

# (2）软件

只有硬件的计算机还不能用来处理信息，这就像婴儿尽管有大脑可以思考，但还不能解数学题一样。要想让孩子解数学题，必须教给他们有关的知识。同样地，要想用计算机处理信息，还必须告诉它们如何处理。一般说来，人们是通过为计算机编程序的方式来告诉计算机如何进行具体的处理操作的。所谓程序，是指为解决某一问题而设计的一系列计算机能够执行的指令（程序、指令等有关知识将在以后的章节中介绍)。习惯上把计算机运行所需的程序以及与之相关的资料、手册等叫做软件。软件（包括程序、资料、使用说明等）可以存放在磁盘里，使用时用磁盘驱动器把它们调入主机。有的软件在机器出厂时已固化在机内。一个计算机系统配备的软件品种越丰富，这个系统能处理的问题也就越多。例如，配备了本书第二章所介绍的字处理软件，就可以用计算机写信、编书、发文件；配备了叫做“数据库”的软件，就可以用计算机处理学校、机关、企业等单位的人事、财务以及其他一些日常事务；配备了教育软件，就可以用计算机辅导学生学习；配备了语音识别软件，就可以把人对着话筒讲话的内容转换成文字；…。软件是计算机系统不可缺少的重要组成部分。

# 2.计算机的特点

计算机是人类发明的一种信息处理工具,与传统的处理方式相比,它有以下几个重要的特点:

# (1）处理速度快

计算机处理信息的速度非常快。例如，用计算机进行数值计算时，我国研制的“银河Ⅱ”巨型计算机每秒能进行十亿次运算，更高速的巨型计算机的运算速度已经达到每秒上千亿次，速度较慢的微型计算机，每秒也能进行数十万次运算。

# (2）计算精度高

计算机的计算精度很高。像中华学习机和AppleⅡ机这样较低档次的微型计算机，就可以输出小数点后9位有效数字，使计算结果精确到十亿分之一。

# (3）具有记忆能力

计算机一般都有很大容量的存储器。例如，常见的微型计算机主机的存储器，可以存储几万至几百万个字符。用磁盘还能存储更多的信息，例如，在一片直径不到 $1 4 \mathrm { c m }$ 的薄薄的软磁盘上，可以存储几万到几十万字的一本书中的全部内容。把能用来解决各类问题的程序、数据等存放在计算机的存储器里，就如同人用脑子记忆处理问题的方法、步骤和原始数据。使用一些命令可以让计算机按照记忆下的方法、步骤反复地进行处理。人脑记忆时，会由于某些原因发生差错，计算机记下的内容可以丝毫不差地随时重现出来。

# (4）能进行逻辑判断

计算机具有逻辑判断功能。例如，可以对两个数据进行比较看看它们是否相同、哪个更大些，也可以从众多数据中选出具备某种属性的一类数据来。因此，可以用计算机进行逻辑推理和定理证明等处理。

# (5）高度自动化

利用计算机解决问题时，把编好的程序输人计算机，告诉它如何进行具体的操作后，计算机就可以自动完成人们交给它们的任务，一般不再需要人来进行干预。

# 三、计算机的广泛应用

计算机的应用非常广泛，在这里我们举一些例子来做一些简单的介绍。

# 1．数值计算

在科研与生产中，经常会遇到各种各样的数学问题。这些数学问题往往需要进行大量、繁琐的计算才能得到结果。例如，在工程设计中，有时要解含有几百个未知数的方程组；要确定宇宙火箭飞行的轨道，需要进行非常复杂的计算；要预报5-7天以后的天气情况，必须在很短的时间内用来自世界各地的数百万个数据进行计算；。在有计算机之前，这类计算只能靠人工来完成，往往要投入很多人力、花费大量的时间，出错的概率相当高。用计算机解决这类问题时，科技人员可以把计算步骤与计算时所需的数据一起输人计算机，计算机能高速、自动地进行复杂的计算，并输出准确的结果，从而大大提高了工作效率。

# 2.自动控制

计算机经常用于连续不断地监测、控制整个实验或生产过程。例如，在化学工业领域，可以用计算机通过检测设备随时收集有关气体、液体等原料或产品的流量、压力、温度等参数的信息，一旦某些参数值偏离了预定的标准，计算机就会立即指挥气泵、液泵、阀门、加热装置等设备做出相应的调整，使原料或产品的参数达到预定的标准值，从而使生产过程得到及时的控制。必要时，计算机还可以通过声光等手段发出警报，通知管理人员。由于计算机处理信息的速度非常快并且可以长时间稳定地运行，用于自动控制可以大大增加生产过程的安全性、提高产品的质量、降低原材料的消耗，使生产效率大幅度地提高。又如，巡航导弹里的计算机存储器中存放着到达攻击目标沿途的地形图，导弹飞行时，计算机根据飞行方向、速度等参数进行计算，随时确定自身的位置，并根据地形图及时调整导弹的飞行高度以避开山峰、建筑物等，飞抵预定目标时，计算机会指挥引爆装置启动从而摧毁目标。

# 3.事务处理

事务处理涉及面很宽，如机关办公、商业活动、交通运输、银行帐目、招生考试、公共安全、图书流通、…等等，不一而足。在这些方面，有大量的事务处理任务可以用计算机来完成。例如，每年的初中升学考试后，考试管理部门把考生各科的成绩、所报志愿的情况、各校招生的名额等必要的信息输入计算机，计算机可以高速、自动地统计出每个考生的总分、按分数排出录取顺序、确定各类学校的录取分数线、逐学校输出各高中录取新生的名单。同时，计算机还可以为教育管理部门提供本次考试的统计数据，作为进一步分析、研究的依据。又如，银行的储蓄所办理存取业务时，把储户的账号、存入或取出的款数输入计算机，计算机就会自动记账、算利息、打印存款单。各储蓄所的计算机连在一起构成网络，储户可以在一个储蓄所开一个存款户头后，随时在网络中的任意一个储蓄所办理存款、取款手续。

# 4.辅助设计

利用计算机可以帮助人设计服装、机器零件等各式各样的产品，还可以进行汽车、船舶、飞机、建筑物等大规模工程的设计。例如，用计算机设计服装时，量好身长、胸围、肩宽等几个尺寸后，把数据输入计算机，计算机会提供不同的式样供人们选择，选好以后，计算机就可以立即画出服装的纸样。利用计算机辅助设计技术，使设计过程实现自动化或半自动化，可以缩短设计周期，节省人力、物力，降低成本，保证质量，极大地提高工作效率。在设计大规模、高精度项目时尤其如此。目前，计算机或其他电子产品所用的集成电路，都是用计算机辅助人来设计的。设计新型号计算机的过程，也离不开计算机辅助设计技术。

# 5.辅助教学

计算机辅助教学始于六十年代。随着计算机技术的发展，这个领域的应用越来越普遍。用计算机可以模拟一些难于在实验室里进行的实验过程，让学生更加深入地认识其本质。计算机可以帮助教师辅导学生，为不同水平的学生提供不同的教学内容，根据从学生那里反馈的信息，自动安排进一步的辅导过程。利用计算机还可以把录音、录像等多种媒体综合起来用于教学，从而提高学生的学习兴趣和积极性，方便教师讲课，提高教学效率。在技术培训的教学过程中也有很多应用计算机的例子。例如，培训驾驶员时，学员可以坐在教室里的模拟装置上，计算机会在学员面前的屏幕上显示出前方路面的图像，并根据学员操纵方向盘、油门、刹车、离合器等装置的情况，不断变换图像的运动速度、方向，并能发出逼真的音响效果，使学员如身临其境，获得与实际驾驶时相同的感受与经验，既训练了学员又避免了初学者用真车训练时可能发生的事故。

# 6.辅助决策

计算机辅助决策的例子也很多。例如有的医院把著名医生的行医经验存人计算机，病人就医时，把症状输人计算机，计算机会自动依据所存储的“经验”为病人诊断病情、打印医嘱、开出处方，使病人得到正确的治疗。又如，把计算机与X光透视技术结合起来的“计算机X射线断层造影”（简称CT)技术，可以准确地确定人体中的肿瘤、血栓或异物的具体位置，为顺利实施手术提供精确的图像与数据。

# 7.信息检索与传输

计算机用于信息检索也越来越普遍。例如，在现代化的图书馆里，读者已不必像过去那样根据分类卡片去找所要借阅的图书，而只需把书名输入计算机，计算机就会在显示器屏幕上显示出书名、作者、出版者、出版日期、内容摘要、分类号、索书号以及这本书在书库中的位置等信息，读者可以很快找到所需的图书。一个地方的计算机可以通过电话线、通讯卫星等设施与世界各地的计算机相连，这又使得信息检索的范围扩展到了全世界。人们不仅可以利用计算机在极短的时间内查阅到国内外与之相连的计算机存储的各类资料、把所需的内容显示或打印出来，还可以把文件、图像、数据等传到需要这些信息的计算机上去。

# 8．出版印刷

当前，许多有名的作家都开始用计算机写作。全国各大报社中绝大多数已经使用计算机排版，许多出版社也用计算机来处理图书的录人、编辑、排版等业务。用计算机写作可以节省大量的稿纸，写字的速度也比用笔快得多，而且非常便于修改。用计算机排版的速度非常快，可以在几分钟到几十分钟之内完成几十万字的分页、加页码、安排不同的字体字号等非常复杂的操作，使印刷厂的排字工人从繁重的捡铅字、组版的体力劳动中解放出来，大大地缩短了图书、报刊的出版周期，提高了生产效率。

# 四、学习计算机知识的意义和方法

计算机科学技术的发展非常迅速，功能越来越强，应用也越来越普遍，将成为人类学习、工作、生活中必不可少的工具。计算机在各个领域中的广泛应用减轻了人们脑力与体力劳动的强度，极大地提高了工作效率，同时，也将改变人们工作与生活的方式。作为即将走向社会或进人高等院校学习的高中学生，学习并掌握一些有关计算机的基础知识和技能无疑是非常必要的。

今后的几十年，将是经济、科学、技术高度发展的时期。为把我国建设成一个高度现代化的强大国家，在21世纪成为全世界的政治、经济、文化中心，需要一代又一代的青少年学生掌握现代化科学技术的基础知识。在学校中进行计算机教育已势在必行。目前，教育部门已经制定了规划，全国各地都在积极创造条件，有步骤、有计划、分层次地在中小学逐步发展计算机教育事业，让更多的青少年学生有机会学习计算机知识。

学习计算机知识，不但可以掌握一种现代化的实用技术，而且在扩展知识的广度、培养分析问题和解决问题的能力等方面都是非常有益的。这些技术、知识与能力在以后的学习、工作和生活中将起到非常重要的作用。

计算机是用来进行信息处理的工具，与其他传统工具不同，它是一种集许多现代科学技术为一身的高科技产品。计算机课的内容涉及到语文、数学、外语以及其他许多学科的基础知识。计算机课是一门实践性很强的课程，需要结合上机实习才能真正领会、掌握有关的概念、思想、方法与技能。因此，要学好计算机知识、掌握这种工具，需要首先学好其他各门课程，为今后的学习打下良好的基础。同时，应珍惜上机实习的机会、认真预习、认真操作、认真总结，注意培养自己动脑、动手，应用计算机解决实际问题的能力。

# 第二节计算机的基本设备

目前，我国中学用于教学的微型计算机主要有两类，一类是以中华学习机CEC－I和AppleⅡ 型为代表的较低档次的微型机，另一类是“长城”、IBMPC等型号的较高档次的微型计算机。中华学习机和AppleⅡ 机的功能、使用方法等基本相同。下面我们以中华学习机为例，简单介绍一下它们的基本设备。

# 第三节计算机的基本操作

# 一、启动

如果只有主机和显示器，启动时只要先接通显示器的电源，然后再接通主机的电源，计算机系统就可以工作了。这时，中华学习机显示器的屏幕上会显示出

ZHONG HUA XUE XI JI

VERSION 1.0

#

等字符。其中第一行是“中华学习机”的汉语拚音。第二行是软件版本的编号说明。]形符号叫做提示符，表示可以用BASIC语言指挥计算机工作了（有关BASIC语言的知识将从第三章开始介绍)。提示符后面有一个闪动着的矩形光标，它指出了从键盘上输入字符时，那些字符在屏幕上显示的位置。屏幕上每行可以显示40个字符，一共可以显示24行。显示的内容超过 24行时，屏幕会向上滚动，最上面一行退出屏幕，新的一行显示在屏幕的最下面。

如果还连有磁盘驱动器，启动前应把存有某种软件程序的磁盘插入磁盘驱动器，关好驱动器门再进行上述操作(连有两个磁盘驱动器时，把磁盘插入1号驱动器)。这时，显示器的屏幕上会显示出与磁盘上存储的软件有关的信息，具体内容将在本书后面的章节中介绍。

# 二、关机

使用完毕，应切断计算机系统的电源。操作顺序与启动时相反，即先切断主机的电源，然后再切断显示器的电源。如果连有磁盘驱动器，在切断主机的电源之前先打开驱动器门，把驱动器中的磁盘取出来放好。如果较长时间不用计算机，应把电源线的插头从供电的插座上拔下来。

# 三、键盘与指法

键盘是用来向计算机输入信息的主要设备。要想熟练地使用计算机，除了必须记住键盘上各种键的位置和功能以外，还应掌握正确的击键指法。只有这样，才能提高输入信息的速度，用好计算机。

# (1）字母数字键

在键盘上，从上面数第二行的中间部分，有一行标有 $1 \sim 9$ 和0的十个数字键。每按下一个数字键，屏幕上会显示出键帽上所标的数字。第三、四、五行的中间部分，有26个标有大写英文字母的字母键。每按下一个字母键，屏幕上也会显示出相应的英文字母。

# (2）符号键

键盘的右半部，有11个符号键，每个键上标有上下两种不同的符号。每按下一个符号键，屏幕上会显示出键帽上下半部所标的符号。例如，按下标有±的键时，屏幕上会显示 $=$ 号。

与符号键类似，每个数字键所标数字的上方也标有一个符号。这21个键又称为双档键。单独按任一双档键，只能输入键帽下半部所标的符号或数字。

# (3）字型变换键

键盘上第五行左右两侧各有一个标有 Shift的键，叫做字型变换键，也叫上下档切换键，简

称 Shift键。按住Shift键再按任一双档键，屏幕上会显示出键帽上半部所标的符号（例如，按住Shift键再按标有5的数字键，屏幕上会显示出一个百分号 $\%$ );按住Shift键再按字母键，屏幕上会显示出该字母的大写形式。

# (4）大写字母锁定键

键盘的左下角，标有 $\mathbf { \Pi } _ { \mathrm { L o c k } } ^ { \mathrm { C a p } }$ 的键是大写字母锁定键。按一下这个键，它就会锁定在按下去的位置上，使英文字母的输人处于大写字母锁定状态。这时，再按字母键，不管是否按住了Shift键，屏幕上显示的都是大写字母。再按一下这个键，它会重新弹起，取消大写字母锁定状态。

# (5）空格键

键盘上最下面一行中间的长条形键是空格键。按一下空格键可以使屏幕上的光标向右移动一个字符的位置，使键入的字符中间留有空格。

# (6）回车键

键盘右方标有Retum的键叫回车键。按一下回车键，屏幕上的光标会跳到下一行。应用计算机解决问题时，从键盘上向计算机发一次命令或输入一次信息时，一般都要按一次回车键。

除以上介绍的六种键以外，键盘上还有其他一些常用键，以后用到时再介绍。

# 第一节字处理软件简介

# 一、什么是字处理软件

在人们的日常工作、学习和生活中，会经常接触到文字处理工作，例如，书写信函、修订文稿、印发文件、保管资料等。在计算机得到广泛应用之前，这类工作基本上要靠手工来完成，往往需要花费大量的时间和人力，工作效率比较低。

随着经济与科学技术的发展，计算机的应用已经扩展到文字处理领域。人们可以用计算机编写、保存、印发公文和信件；记者可以用计算机撰写消息、报导并通过电话线把它们传送到远方的编辑部；作家可以用计算机写小说，把几十万乃至几百万字的书稿存放在磁盘里，随时可以调出来加工润色而不必浪费一张稿纸；……。

用计算机进行文字处理时，需要给它们配上字处理软件。字处理软件是一种能够让人用计算机输入、输出、存储、编辑文稿，从而提高工作效率的软件。目前，为微型计算机设计的字处理软件有许多种，常用的有WORDSTAR、CCED、WPS、WORDPERFECT等。

# 二、字处理软件的主要功能

字处理软件一般具有输入、编辑、存储、输出等几方面的功能。例如，可以用计算机键盘输人汉字、英文或其他文字信息；可以非常容易地找到文章或书稿中的某一个位置，方便地进行插入、删除、修改、替换等编辑处理，还可以把文章或书稿中的句子、段落等部分复制或搬移到所需的位置上，然后重新整理，使版面整齐、美观；可以把已经编辑好的文稿或文稿的某些部分存人磁盘长期保存，需要时再把它们调出来；可以根据需要设定打印格式、打印字体，计算机自动地按要求把文稿打印在纸上。

此外，功能较强的英文字处理软件还备有一部存储了数万条词汇及各种语法规则的电子词典，用来自动地检测、修改文稿中的拚写或语法错误。

# 三、中文字处理软件

早期的字处理软件只能处理英文文稿。70年代中期，我国的计算机科技工作者开始研究汉字信息处理技术。目前，已经研制开发出了许多种适合不同机型的中文字处理软件。这些中文字处理软件除具有处理英文文稿的功能之外，还可以处理中文文稿。经过多年来的改进，中文信息处理的水平不断提高，用键盘输人汉字的速度已经大大超过英文输入的速度。中文字处理软件的功能也越来越强，输出的文字有各种漂亮的字体、不同大小的字号，可以满足办公、出版、印刷等众多方面的要求。因此，中文字处理软件应用也越来越广泛，逐渐改变了中国人进行文字处理的工作方式，从而大幅度地提高了中文文稿处理的工作效率。许多政府机关、工矿企业都用中文字处理软件管理各种文件资料。越来越多的公司在招聘职员时，把是否会使用字处理软件当作录取与否的重要条件之一。

(1)在编辑状态下，按下 $\mathrm { C t r l } + \mathbf { W }$ 键，按照屏幕提示把当前正在编辑或输入的文件存入磁盘，存储完毕后返回到CWS主菜单状态。

(2)如果文件不需要存盘，则在编辑状态下按下 $\mathbf { C _ { } } \mathbf { f r l } + \mathbf { K } + \mathbf { Q _ { } }$ 键，放弃当前正在编辑或输入的文件，返回CWS主菜单。

# 第三节字块操作

在上一节中，我们介绍了文稿的编辑命令。利用这些命令，可以对文稿中的文字进行修改和调整。用CWS软件编辑文稿时，还可以很方便地处理一些更复杂的问题，例如，把文稿中的某一部分内容向前或向后移动、把某一部分内容复制到文章的某个位置上、把一篇文章中的一部分内容复制到另一篇文章中去、把文章中的某一部分内容删除，等等。处理这类问题时，需要使用CWS 软件的字块操作功能。顾名思义，字块就是文稿中成块的文字。字块可以是一句话、可以是一段文字、也可以是整篇文章。字块由操作人员来确定，确定好之后就可以对它进行移动、复制、删除、存储、读取等操作。

习惯上把对字块进行的操作叫做块操作。块操作是应用字处理软件与手工文字处理的最显著区别之一。利用块操作功能，可以避免重复输人，简化编辑过程，极大地提高用计算机进行文字处理的效率。

# 一、定义字块

要进行块操作，首先要在文稿中定义字块，即告诉计算机，哪里是字块的开始，哪里是字块的结尾。定义字块时，先把光标移到要定义成字块的那些文字的第一个字上，然后按$\mathbf { C } \mathbf { t r l } + \mathbf { K } + \mathbf { B }$ 键。这时，在光标前会出现一个 ${ \bf { \bar { \tau } } } < { \bf { B } } >$ 。此后，再把光标移到要定义成字块的那些文字最后的那个字后面，按下 $\mathbf { C _ { [ r ] } } + \mathbf { K } + \mathbf { K }$ 键。这时，光标前会出现一个 ${ \bf { < K > } }$ 。这样就完成了定义字块的操作。屏幕上显示的 ${ \bf { \bar { \tau } } } < { \bf { B } } >$ 是字块的块首标志，简称首标； $\mathbf { < K > }$ 是字块的结尾标志，简称尾标。首标与尾标之间的文字就是字块。在CWS 软件中，每次只能定义一个字块。

# 二、块复制

用块复制操作可以把整块文字复制到从光标处开始的一片区域里。实际操作时，在定义好字块以后，把光标移到要复制字块内容的起始位置上，按下 $\mathbf { C } \mathbf { t r l + K + C }$ 键，定义好的字块就会被复制到当前光标处，光标所在处以及后面的文字会自动向后顺序移动，定义好的原始字块不变。

# 三、块删除

在编辑过程中，对于文稿中大段无用的文字，可以用块删除操作把它们删掉。实际操作时，按下 $\mathbf { C _ { [ r ] } } + \mathbf { K } + \mathbf { Y }$ 键，定义好的字块就会被从屏幕上删除，但字块的首尾标志还留在屏幕上，这时，再用Esc键把字块的首尾标从屏幕上删掉。

要把定义好的字块移到某个位置上，可以先把字块复制到指定的位置上，然后再用块删除操作删去原来的字块。

# 四、块写文件

编辑文稿时，可以用块写文件操作把字块中的文字存人磁盘备用。具体操作步骤如下。

$\textcircled{1}$ 把要存盘的文字定义成字块;

$\textcircled{2}$ 按下 $\mathbf { \Lambda } _ { \mathbf { C } \mathbf { t } \mathbf { r } 1 + \mathbf { K } + \mathbf { W } }$ 键；

$\textcircled{3}$ 计算机询问存人磁盘时给文件取个什么名。当屏幕上会显示出“文件名：_”时，输入一个新的文件名，并按回车键；

$\textcircled{4}$ 屏幕上又提示“请插入工作盘于第一驱动器，按任一键”时，按照提示信息的要求把字库盘从驱动器中取出，插人工作盘，按下任一键。

这时，计算机会把字块中的文字存入磁盘中指定的文件中。

# 五、取消块定义

CWS 软件中每次只能定义一个字块。要定义新字块就必须先使旧字块的定义无效。在这种情况下，需要清除旧字块的首、尾标。具体做法是：

$\textcircled{1}$ 把光标分别移到首标和尾标的右侧，用 Esc 键删除屏幕上的字块的首标<B>和尾标$\mathbf { < K > }$ ；

$\textcircled{2}$ 按 $\mathbf { C _ { [ r ] } } + \mathbf { K } + \mathbf { H }$ 键。这时，字块的首尾标会从内存中消失，原字块定义被取消了，可以再定义新的字块。

# 六、读取文件

编辑文稿时，还可以利用读文件操作，从磁盘上读入一个文件，把这个文件的内容加到文稿中光标指定的位置上。操作步骤如下。

$\textcircled{1}$ 把光标移到要添加内容的起始位置;

$\textcircled{2}$ 按下 $\mathbf { C _ { i r l } + K + R _ { \alpha } }$ 键；

$\textcircled{3}$ 屏幕上显示“请插入工作盘于第一驱动器，按任一键”时，依提示插入工作盘，按下任一键;

$\textcircled{4}$ 屏幕上显示“文件名：”时，输入要读入的文件名。按回车键后，计算机会把该文件读入

内存并在光标处显示出来，然后就可以继续进行编辑。

在第二章里，我们学习了字处理软件。在字处理软件的支持下，可以很方便地用计算机进行文字的编辑工作。目前，各式各样的软件在各种计算机上得到了广泛的应用。在不同软件的支持下，人们可以用计算机高效地处理不同领域的问题。软件的核心是计算机工作者用各种程序设计语言编制的一些程序。这些程序控制着计算机，使之有效地完成人们交给它们的任务。为了较深人地了解与应用计算机，学习一些程序设计的基础知识是很有必要的。

# 第一节BASIC语言简介

# 一、什么是BASIC语言

BASIC是英语 Beginner's All-purpose Symbolic Instruction Code 的缩写，意思是“初学者通用符号指令代码”。同时，basic又是一个英语单词，含“基本的、基础的”之义。BASIC 语言是编程序应用计算机解决实际问题时常用的一种语言。用BASIC语言编程序时，可以使用某些简单的英文单词、专用的符号和数字来表达意思。在计算机里，特别是微型机里，一般都配有厂家提供的BASIC语言的解释程序。BASIC解释程序把这些单词、符号和数字表达的意思“翻译”成计算机能执行的指令，指挥计算机工作。BASIC语言有它自已的表示法与规则，我们将在介绍程序设计方法时，陆续介绍BASIC语言的有关表示法和规则。

学校里常用的中华学习机和Apple机，在出厂时就配备了BASIC解释程序。开机时屏幕上显示出的]叫做BASIC提示符，它表示“BASIC解释程序已经开始工作，可以使用BASIC语言了”。

使用BASIC语言有两种方式，一种叫命令方式，也叫立即执行方式；另一种叫程序方式，也叫延迟执行方式。

# 二、命令方式

从键盘上输人BASIC语言中的某些命令，计算机会立即执行命令所规定的操作，例如，计算、显示信息等等。

利用 BASIC命令，可以让计算机计算代数式的值。

例如，要让计算机计算 $3 4 0 8 2 + 9 1 7 5$ 的值，可以在屏幕上显示］提示符时，键入PRINT和算术式 $3 4 0 8 2 + 9 1 7 5$ 。按回车键后，计算机会计算这个式子的值并显示出结果来。整个过程的屏幕显示如下：

]PRINT 34082+9175← 43257

]

BASIC 语言有许多命令，这里使用的是PRINT命令。PRINT的英文原意是印刷、打印。在]提示符后面输入PRINT和一些表达式，计算机就会计算表达式的值并在屏幕上输出结果。这里的 $3 4 0 8 2 + 9 1 7 5$ 是比较简单的BASIC表达式。实际应用时，表达式可能会更复杂，而且BASIC表达式与数学中的式子也略有不同。由于键盘的限制等技术原因，BASIC语言规定，在表达式中:

例3.4 分别用手工和计算机计算1234的3次幂。

$$

在中华学习机或Apple机上，一般用科学记数法表示绝对值不小于99999999.2的数和绝对值小于0.01的数，否则用日常的记数方式来表示。用科学记数法表示数的形式是

$$
\pm \mathrm { \bf ~ N E } \pm \mathrm { \bf ~ n n }
$$

其中， $\pm \mathrm { ~ \bf ~ N ~ }$ 叫做尾数，可以是非零整数或整数部分是一位非零数字的小数，小数点后面最多可有8位数字。表示正数时，正号省略掉；E是英文exponent 的第一个字母的大写形式，表示指数。后面的 $\pm$ nn 叫阶码，阶码有两个整数位，表示尾数所乘的10的幂指数。绝对值小于10的阶码，输出时十位上补0。阶码为正时， $^ +$ 号不省略。

因此，例3.4中计算机输出的结果 $1 . 8 7 9 0 8 0 9 \mathrm { E } + 0 9$ 表示 $1 . 8 7 9 0 8 0 9 \times 1 0 ^ { 9 }$ ，即1879080900。又如，让计算机执行PRINT-55/334455661时，屏幕上会显示出 $- 1 . 6 4 4 4 6 3 1 2 \mathrm { E } - 0 7 _ { : }$ ，表示$- 1 . 6 4 4 4 6 3 1 2 \times 1 0 ^ { - 7 }$ ,即－0.0000000164446312。

计算机能处理的数有一定的范围。中华学习机和Apple机能处理的数的绝对值不能大于$1 . 7 0 1 4 1 1 8 3 \times 1 0 ^ { 3 8 }$ ，否则，计算机不能正确计算。例如，让计算机计算18的38次幂时，屏幕上不会输出计算结果，主机扬声器发出“嘟”的一声后，会显示出错信息：

]PRINT $1 7 . 1 \sim 3 8  \nu$ ？OVERFLOW ERROR ]

屏幕上显示的OVERFLOWERROR表示运算过程中发生了“溢出错误”，说明运算结果的绝对值太大了。习惯上把这种现象叫做上溢。如果一个数的绝对值小于 $2 . 9 3 8 7 5 8 8 \times 1 0 ^ { - 3 9 }$ ,计算机也不能正确计算。例如，让计算机计算2.93与10的－39次幂的积时，屏幕上会给出错误结果：

]PRINT 2.93\*1E-39   
0   
]

这种现象叫做下溢。实际上，发生下溢时，计算机把绝对值小于 $2 . 9 3 8 7 5 8 8 \times 1 0 ^ { - 3 9 }$ 的数近似地当作0来处理。对于下溢，计算机不会显示出错信息，扬声器也不会发出声响。

# 2.让计算机输出字符

用 PRINT命令，可以让计算机把用双引号括起来的一串字符照原样输出。

例如，在PRINT后面键入”HELLO!IAMACOMPUTER.",按回车键后，屏幕上下一行会显示出双引号中的那串字符，但不会显示双引号本身：

]PRINT \*HELLO! | AM A COMPUTER:'

HELLO! IAM A COMPUTER.

]

在BASIC语言里，把两个双引号括起来的字符叫做字符串。PRINT命令中如果有双引号括起来的字符串，计算机就会照原样输出字符串的内容。字符串可以由汉字、字母、数字、空格以及其他符号组成。字符串中可以有多个字符，字符的个数叫做字符串的长度。长度为0的字符串叫做空字符串或空串。例如，计算机执行 PRINT""后，会输出一个空行。空串不同于只由空格组成的字符串。

PRINT命令可以有多个输出项。输出项之间可以用分号或逗号隔开。例如，在PRINT命令中顺序键入” $\mathbf { A } = "$ ； $4 ~ * ~ 5$ ， $- 9 9$ 时，屏幕显示如下：

]PRINT $\mathbf { \tilde { \rho } } \mathsf { A } = \mathbf { \tilde { \rho } }$ ;4\*5.-99   
$\mathsf { A } = 2 0$ 199

]

可以看到，屏幕上先显示的是 $\mathbf { A } =$ ,紧接着的是20,空几个格以后显示的是－99。

在这里，PRINT命令有三个输出项" $\mathbf { A } { = } { \boldsymbol { \ " } }$ $4 { * } 5$ 和数－99。其中 $\mathbf { A } =$ 是双引号括起来的字符串，计算机执行时，照原样输出。 $4 { * } 5$ 是表达式，计算机执行时先计算它的值然后输出20这个计算结果。 $- 9 9$ 是一个数，是表达式的一个特例。对于PRINT命令中的这种由一个数构成的表达式，计算机直接输出该数。因此，这里输出的是－99。

${ \bf \ " } \mathbf { A } = \bf \ " $ 与 $4 { * } 5$ 之间是用分号隔开的，而 $4 { * } 5$ 与－99之间是用逗号隔开的。PRINT 命令中用来把输出项隔开的分号和逗号控制着输出的格式。计算机输出用分号隔开的输出项时，会在屏幕上把它们一个紧接着一个地显示出来。我们把这种输出格式叫做紧凑格式；输出用逗号隔开的输出项时，每个输出项会在屏幕上的一些标准位置上开始显示，我们把这种输出格式叫做标准格式或分区格式。

在中华学习机和Apple机的显示器上，每行可以显示40个字符，其中第1、第17、第 33个字符的位置是标准位置。一行中用逗号隔开的输出项多于3个时，从第4项开始会自动转到下一行继续输出。输出的字符显示在一行中第40个字符的位置时，紧接在后面的字符也会自动转到下一行继续输出.

# 三、程序方式

用计算机解决的问题往往很复杂，需要先编好程序，告诉计算机如何一步一步地进行操作，然后再让计算机执行。请看一个例子。

例3.5让计算机接受从键盘上输入的任意两个数，然后分别计算并输出这两个数的和与积。

操作：先输入程序，操作及屏幕显示如下。

]10 INPUT X,Y←   
]20 PRINT“和 $\mathbf { \Phi } = \mathbf { \tilde { \Phi } } ; \mathbf { X } + \mathbf { Y } \longleftrightarrow$   
]30 PRINT“积 $\mathbf { \Phi } = \mathbf { \tilde { \Phi } } ; \mathsf { X } ^ { \ast } \mathsf { Y } \ \gets \pmb { y }$   
]40 END   
]

操作完毕，屏幕上显示出]。这时，计算机不执行什么操作，而是等待进一步的指示。从键盘上输入RUN并按回车键后，屏幕上会显示出一个？号。这时，输入两个数，例如987和 343后(输入时用逗号把两个数隔开)，计算机就会进行一系列的操作，并输出运算结果。过程如下。

]RUN亻

]

RUN是让计算机运行程序的命令。计算机接到RUN命令后，就开始执行已经输入的程序。

这种先输入一个BASIC程序，然后键人RUN并按回车键让计算机自动运算的方式，叫做BASIC语言的程序方式。

# 第二节BASIC程序

# 一、BASIC程序的组成

在例3.5的操作中，我们输入的是一个简单的BASIC程序。一般说来，BASIC程序是由若干语句行组成的。语句行由行号、语句、行结束符（回车）组成。语句的作用是告诉计算机进行哪些操作。每个语句由语句定义符和语句体构成。这个简单程序可以列表分析如下：

<table><tr><td>行号</td><td>语句定义符</td><td>语句体</td><td>行结束符</td><td>功 能</td></tr><tr><td>10</td><td>INPUT</td><td>XY</td><td>←</td><td>接收从键盘上输入的两个数</td></tr><tr><td>20</td><td>PRINT</td><td>&quot;和=&quot;;X+Y</td><td>←</td><td>输出两个数的和</td></tr><tr><td>30</td><td>PRINT</td><td>&quot;积 =&quot;; X*Y</td><td>←</td><td>输出两个数的积</td></tr><tr><td>40</td><td>END</td><td></td><td>←</td><td>结束</td></tr></table>

每个语句行开头的数字是行号，也叫标号。BASIC语言规定，行号可以是0到63999之间的无符号整数。它确定了语句在程序中的位置。相邻的两个语句行的行号可以不连续，以便需要时插入新的语句行。计算机按行号由小到大的顺序执行各语句行所规定的操作。

语句中的 INPUT、PRINT、END等英文单词叫做语句定义符。BASIC语言的语句定义符是该语句的名称，它规定了语句的功能。不同的语句定义符使计算机进行不同的操作。

语句定义符后面，一般是语句体。语句体为计算机指出操作对象、操作内容等。有些语句定义符（如例3.5中的END)后面没有语句体。这样的语句定义符可以独自确定要计算机执行的具体操作。

# 二、变量

例 3.5的这个程序中，X和Y是变量名。

BASIC 程序中常常用到变量。可以把变量看作计算机存储器里存储数据的一些存储单元。向存储单元中存人一个数据后，代表这个存储单元的变量就获得了该数据的值。变量的值是可变的，存人一个新数据后，变量中的旧值就被新值所取代，因此称之为变量。例如，存人一个数3以后，变量的值是3,再存入另一个数－99.99,变量的值就变成－99.99。

不同的变量有不同的变量名，就像旅馆里不同的房间有不同的房间号一样。根据房间号，旅馆服务员可以很快地找到不同房间的客人。根据变量名，计算机可以找到不同的存储单元，然后把数据放到里面或者从里面读出先前存放的数据。

变量名一般要用一、两个英文字母或者一个英文字母后面跟着一个数字组成。

为便于认读，也可以用两个以上字符作变量名，但计算机只识别前两个字符。例如，在计算机看来，MENU和MEMBER这两个变量名指的是同一个变量。

变量名不能是BASIC语言专用的英文单词，如INPUT、PRINT等，而且，变量名中也不能含有这些英文单词，例如，BEND(含有 END）CPRINTS(含有PRINT)都不能作变量名。变量名中不能使用 $+ - * / \ \backslash$ 等字符。

# 三、几个基本语句

# 1. INPUT语句

在例3.5的程序里，第10语句行中的 语句是键盘输入语句也叫 INPUT语句。INPUT语句的格式是

在英语中，INPUT的意思是“输入”。INPUT语句的功能是接收从键盘上输入的数据。计算机执行INPUT语句时，先显示一个？号，等待从键盘上输入的数据。输人数据后，计算机把输入的数据存储在变量名表指定的变量里，以备进一步处理之用。

INPUT语句中可以有一个或多个变量名，它们构成变量名表。如果有多个变量名，要用逗号把它们隔开。例如，

INPUT X,Y,Z

如果分隔多个变量名的不是逗号，例如用了分号，输人数据以后，扬声器会发出声响，并显示出？ SYNTAX ERROR IN xX

]

指出程序的语句有错误，并指出有错误语句的行号。

在 BASIC程序里，用INPUT语句除了可以为变量提供数值以外，还可以提供字符串。请看下面的程序。

10 PRINT "HOW OLD ARE YOU? "

# 2.PRINT语句

例3.5的程序中，20、30这两行语句是PRINT语句。PRINT语句的格式是

PRINT $<$ 输出项表

其中，输出项可以是数、字符串、变量或其他表达式。有多个输出项时，要用分号或逗号隔开。例如,

PRINT $" { \mathbf A } = "$ ； A, $\mathbf { \ " B } = \mathbf { \ " }$ ; B

例3.5中的这两个PRINT语句输出项的表达式中，出现了变量名。计算机执行时，先从变量名指定的变量中读出两个数据的值，然后用这些具体的值代入表达式中的变量进行计算并输出结果。

在PRINT语句的末尾，可以使用逗号或分号。PRINT语句末尾的逗号或分号不影响本语句的输出格式，但对程序中这个语句后面再次出现的第一个PRINT语句起作用。例如，计算机执

100 PRINT 39，40 ;   
110 PRINT 41

这两个语句后，屏幕上会显示出39 4041

又如，计算机执行

200 PRINT 1，17,  
210 PRINT 33

这两个语句后，屏幕上的输出情况如下：

1 17 33

这说明，计算机执行相邻的两个PRINT语句时，如果前面的PRINT语句末尾有分号，后一个语句输出的第一项会紧接着前一个语句所输出的最后一项；如果前面的PRINT语句末尾有逗号，后一个语句输出的第一项会位于前一个语句输出的最后一项的下一个标准位置。

# 3.END语句

例3.5的程序中，语句40是END语句。END语句没有语句体。计算机遇到END语句时，就停止执行这个程序。

# 4.赋值语句

在BASIC程序中，可以在程序运行时，用INPUT语句把数据存储在变量里。这样做的优点是，编程序时可以不知道程序要处理的是什么数据，程序运行时，再根据需要输入具体的数据，比较灵活。缺点是机器等待输入数据时，占用了一些时间。INPUT语句没有计算功能，它只能把数值或字符串提供给变量。

在许多情况下，常用赋值语句完成为变量提供数据的操作。

例3.7计算半径为15的圆的面积的程序。

10 LET $\mathrm { P I } { = } 3 . 1 4 1 5 9$   
20 LET $\scriptstyle \mathbf { A } = \mathbf { P I } \ * \ 1 5 \ \hat { \mathbf { \Omega } } 2$   
30 PRINT A  
40 END  
键入RUN并按回车键以后，屏幕上会显示出

706.85775

在这个程序里，语句10、20叫做赋值语句。赋值语句的格式是其中， $^ { * } = { } ^ { n }$ 叫做赋值号。赋值语句的功能是把赋值号右边表达式的值存放到赋值号左边变量名指定的变量中去。如果表达式需要计算才能确定它的值，计算机会先进行计算。例如，计算机执行

时，先计算出 $3 5 + 4 4$ 的值 79,然后再把79存放到变量Y中去。

变量被赋予一个值以后，这个值就会被保存在该变量里，直到用LET或 INPUT语句给它一个新值，才可能发生变化。在BASIC程序里，用LET或 INPUT语句为变量提供具体值之前，数值型变量中的值是0；字符串型变量为空串（其他型号的机器可能有例外)。

英文单词LET有“让……怎么样”的意思。赋值号可以读做“变成”。因此，从字面上看，语句10的意思是：“让变量PI的值变成3.14159”。计算机执行语句10这个赋值语句时，把赋值号左边的变量PI原来的值0变成了赋值号右边的值3.14159;语句20的意思是“让变量A的值变成赋值号右边表达式 $\mathbf { P I } * 1 5 \hat { \mathbf { \Omega } } 2$ 的值”。计算机执行语句20时，先计算出表达式的值706.85775，然后把这个值存放在变量A中。

比较一下这个例子与例3.5,可以发现，用PRINT语句和LET语句都可以计算表达式的值。但是，它们之间是有很大区别的。用PRINT语句计算时，计算完了马上输出，不保存计算结果；用LET语句计算时，计算完以后不输出结果，而是把结果保存在一个变量里，供以后处理时使用。

赋值语句的语句定义符LET可以省略，例如，LET $\mathbf { Y } = 3 5 + 4 4$ 可以写成 $\mathbf { Y } = 3 5 + 4 4$ 。

# 四、BASIC程序的输入、修改与运行

输入BASIC程序时，要一行一行地输入每个语句行的所有字符 (包括空格符)，每输人完一行，按一次回车键。按回车键的作用是告诉计算机这一行语句已经输入完毕。计算机接到回车键的信号后，就把回车键前面输入的内容存入内存储器。等到所有的语句都输入完以后，计算机内存储器里就存好了一个完整的BASIC程序。

# 一、顺序结构的程序

至此，我们已经接触了一些BASIC程序。这些程序运行时，计算机按行号从小到大的顺序逐行执行各个语句所规定的操作。我们说这样的程序是顺序结构的程序，简称顺序程序。下面再看几个顺序程序的例子。

例3.8某代销点出售《无线电》、《计算机》和《看世界》三种杂志，它们的定价分别是1.20元、1.55元和2.00元。利用下面的这个程序可以在输入各种杂志的订购数量后，立即输出一个货单。

# 二、赋值语句的应用

用LET语句可以为一个变量重复赋值。每次赋予变量一个新值以后，变量中的旧值就被新值所取代。请看一个例子。

例3.9某粮库3月4日存粮 ${ 5 0 0 0 0 0 } \mathrm { { k g } } ,$ ，3月5日调进粮食 $3 0 0 0 0 \mathrm { k g }$ ，3月6日调出全部存粮的二分之一，3月7日调出 $1 5 0 0 0 \mathbf { k g }$ 。编一个程序，让计算机输出每天的库存数。

分析：粮库的库存数是逐日变化的，可以设置变量来存放每天的库存数。为了节省存储空间，可以只设置一个变量。处理时，每次把当天的库存数统计好存人变量里，然后输出适当的文字和变量的当前值，来说明当天的库存数。

程序：

10REM 粮库的库存数  
20 LET $\mathbf { X } { = } 5 0 0 0 0$   
30 PRINT $^ { \prime \prime } 3$ 月4日库存数:"；X;"KG"  
40 LET $\mathbf { X } { = } \mathbf { X } { + } 3 0 0 0 0$   
50 PRINT $" 3$ 月5日库存数:"；X;"KG"  
60 LET ${ \bf X } = { \bf X } / 2$   
70 PRINT $" 3$ 月6日库存数:"；X;"KG"  
80 LET $\mathbf { X } { = } \mathbf { X } { - } 1 5 0 0 0$   
90 PRINT $^ { \prime \prime } 3$ 月7日库存数:"；X;"KG"  
100 END  
运行这个程序，屏幕上会输出  
3月4日库存数：50000KG  
3月5日库存数：80000KG  
3月6日库存数：40000KG  
3月7日库存数：25000KG

在数学里像 $\scriptstyle \mathbf { X } = \mathbf { X } + 3 0 0 0 0$ 、 $\mathbf { X } { = } \mathbf { X } { - } 1 5 0 0 0$ 这样的式子是不成立的，这是因为数学里的 ${ \bf \omega } ^ { \prime \prime } = { \bf \omega } ^ { \prime \prime }$ 是等号，表示的是相等关系。BASIC语言的赋值语句里， $^ { \ast } = { } ^ { \ast }$ 是赋值号，表示的是计算机把表达式的值赋给变量的一种操作。这样，从字面上看，语句20的意思是“让X的值变成 $5 0 0 0 0 ^ { \prime }$ ；语句40的意思是“让X的值变成X原来的值加上 $3 0 0 0 0 ^ { \prime }$ ；语句60的意思是“让X的值变成X原来值的二分之一"；语句80的意思是“让X的值变成X原来的值减去 $1 5 0 0 0 ^ { \prime }$ 。实际上，计算机执行这四个语句时就是按上述意思去操作的，即执行语句20时，让变量X的值变成 50000;执行语句40时，先取来变量X的值50000,与表达式中的数 30000相加后把结果再存人变量X,使变量X的值变成了80000;执行语句60时，先取来变量X的值 80000,除以2后把商再存人变量X使变量X的值又变成了40000;执行语句80时，先取来变量X的值40000,减去15000后把差存人变量X,使程序结束时变量X的值变成了25000。

# 四、程序设计的几个步骤

应用计算机解决实际问题时，一般需要经过 $\textcircled{1}$ 提出问题、 $\textcircled{2}$ 确立数学模型、 $\textcircled{3}$ 确定算法、$\textcircled{4}$ 画出框图、 $\textcircled{5}$ 编写程序、 $\textcircled{6}$ 上机调试等几个步骤。下面我们举一个简单的例子初步说明一下这几个步骤。

# 1．提出问题

例如，有甲、乙两个杯子分别装着果汁和咖啡，要求把两个杯子中的饮料交换过来。试模拟这个交换过程。

# 2．确立数学模型

计算机不能代替我们做实物的移动和交换，但可以模拟这个过程。如果用变量A和变量B分别模拟甲、乙两个杯子，用代码1表示果汁，用代码2表示咖啡，就可以把交换饮料的物理问题转换成交换两个变量值的数学问题。后者（交换两个变量的值）可以看作前者（交换两个杯子中的饮料）的数学模型。

# 3.确定算法

为解决一个问题而采取的方法和步骤，通常叫做算法。算法不仅指数值计算的方法和步骤，而且也包括非数值运算的方法和步骤。例如，棋谱可以看作下棋的算法，菜谱可以看作做菜的算法等等。

对于这个例子，可以把交换甲、乙两个杯子中的饮料的算法表示如下：

$\textcircled{1}$ 将甲杯中的果汁倒入空杯丙中;  
$\textcircled{2}$ 将乙杯中的咖啡倒入甲杯;  
$\textcircled{3}$ 将丙杯中的果汁倒入乙杯。

实现这个算法的关键是使用第三个杯子 (丙杯)。同样地，可以确定交换变量A和变量B的值的算法：

1.BASIC语言有两种执行方式：命令方式和程序方式。许多BASIC语言的单词既可以用于命令方式又可以用于程序方式，例如，PRINT、LET、HOME等。有些单词，如INPUT,只能用于程序方式。还有些单词，一般情况下只用于命令方式，如LIST、NEW等。习惯上把通常用于命令方式的BASIC单词叫做命令。本章学习了四个BASIC命令：

RUN 运行程序  
LIST 列程序清单  
NEW 清除内存中的程序  
HOME 清屏并使光标定位于屏幕的左上角

构成程序语句并决定该语句功能和名称的BASIC单词叫做语句定义符。本章学习了五个语句定义符：

INPUT 键盘输入

LET 赋值 PRINT 显示输出 REM 注释 END 结束

2.BASIC程序的描述性定义：BASIC程序由若干语句行构成。一个语句行由行号、一条语句或由冒号隔开的若干条语句、行结束符（回车）三部分组成。每条语句由语句定义符和语句体组成 (END语句例外)。语句定义符决定了该语句的名称和功能。

3．变量：BASIC程序中用来存放数据的是变量。计算机为程序中的变量分配出一定的存储单元，供存放数据用。计算机根据变量名访问指定的变量，进行有关的操作。变量有数值变量与字符串变量，分别用来存放数值型数据和字符型数据。在每一时刻，变量有一确定的值，但随着程序的运行，变量的值可以改变。

4．函数：本章学习了两个BASIC函数

ABS(X) 绝对值函数SQR(X) 算术平方根函数

5．数：计算机能够处理一定范围内的数。对于很大或很小的数，计算机用科学记数法来表示。

6.BASIC表达式：BASIC表达式由常数、变量、函数、运算符号组成。有些BASIC表达式的表示方式与数学表达式略有不同。

7．顺序结构的程序：计算机按行号从小到大的顺序执行各个语句行的程序是顺序结构的程序。

8．程序设计的基本步骤：程序设计的基本步骤包括 $\textcircled{1}$ 提出问题、 $\textcircled{2}$ 确立数学模型、 $\textcircled{3}$ 确定算法、 $\textcircled{4}$ 画出框图、 $\textcircled{5}$ 编写程序、 $\textcircled{6}$ 上机调试。对于较复杂的问题，确定算法与画出框图往往要由粗到细逐步完成。

# 第四章 循环程序设计

应用顺序程序，让计算机按顺序执行一系列的语句，可以进行一些处理。但在实际应用中，会经常遇到需要反复进行一系列处理才能解决的问题，在这种情况下，本章要介绍的循环程序是一种非常得力的工具。

# 第一节循环结构的程序

我们知道，用BASIC语言的 SQR(X)函数可以输出一个数的算术平方根。要求出1到9这9个自然数的算术平方根，可以编出下面这段程序。

10 PRINT SQR(1)   
20 PRINT SQR(2)   
30 PRINT SQR(3)   
40 PRINT SQR(4)   
50 PRINT SQR(5)   
60 PRINT SQR(6)   
70 PRINT SQR(7)   
80 PRINT SQR(8)   
90 PRINT SQR(9)

照此办理，要想求出1到100这100个自然数的算术平方根，就得写出100条类似的语句来。类似地，在上一章的例3.11中，如果要统计输出全班40个学生的考试成绩，同样的程序要分别运行40次才行。这样做太麻烦了。BASIC语言提供了循环语句，可以使程序变得非常简明。请看下面的程序。

例4.1 输出自然数1到100的算术平方根的程序。

20 PRINT SQR()  
30 NEXT I  
40 END

1  
1.41421356  
1.73205081  
2

… (从略) 9.89949495 9.94987438 10

这个程序里只有一个PRINT语句，为什么能输出 100个结果呢？好像计算机执行了100 次PRINT语句。实际上确实如此，这是因为10和30两个语句起了关键性的作用。

# 一、FOR-NEXT 语句

例4.1中的语句10叫做循环初始语句，也叫FOR语句。FOR语句中的变量I叫做循环控制变量，也叫循环变量；循环变量I后面的数1叫初值；TO后面的数100叫终值；STEP后面的数1叫步长值。语句30叫做循环终端语句，也叫NEXT语句。FOR语句的格式是

NEXT语句的格式是

# NEXT $<$ 循环变量名 $>$

在程序中，FOR语句和NEXT语句必须成对出现，二者使用的循环变量名必须相同。FOR语句中循环变量的初值、终值、步长值可以是数，也可以是变量，还可以是表达式。步长值是1时，FOR语句中的 STEP1可以省略不写。例如，例4.1中的语句10可以写成

10 FOR $\mathbf { I } = 1$ TO 100

计算机执行FOR语句时，首先把初值赋给循环变量，其次记下终值和步长值为以后的处理做好准备，然后开始执行FOR语句下面的语句。执行到NEXT语句时，计算机先让循环变量增加一个步长值，然后用增值后的循环变量值与记下的终值比较，如果没超过终值，就再次执行FOR 语句后面的语句，如果循环变量的值超过了终值，就执行NEXT语句后面的语句。

运行例4.1中的程序，计算机执行语句10时，首先让循环变量I变成初值1,然后记录下终值100 和步长值1就完成了语句10规定的操作;接下来计算机执行语句20,由于变量I的值已变成1,屏幕上显示出1的算术平方根；再接下来，计算机执行语句30。首先让循环变量I增加一个步长变成 $_ { \mathbf { I } + 1 }$ ,即变成2,然后把I的新值与记录下来的终值100比较，看看是否超过了终值，显然还没超过，于是就返回去第二次执行语句20;第二次执行语句20时，变量I的值变成了2,屏幕上显示出的是2的算术平方根；再次执行语句30时，先把变量I的值变成3,由于还没超过终值100,就再次返回去第三次执行语句20,，如此反复，直到第 100次执行语句30时，循环变量I的值变成101,超过了记录下来的终值100,这次计算机不再返回去执行语句20,而是向下执行这个NEXT语句下面的语句40。由于语句40是END语句，屏幕上输出了100的算术平方根后就结束了。

步长值不是1时，FOR 语句中必须给出 STEP和具体的步长值。例如，用

100 FOR $\mathbf { K } = \mathbf { 0 }$ TO 1000 STEP 3

120 PRINT K,

130 NEXT K

140 END

可以输出1000以内能被3整除的所有非负整数。

# 二、数据的预置与读取

应用循环程序可以处理成批的数据。在此之前，提供数据的任务一般靠LET和INPUT语句来完成。要处理的数据很多时，使用LET和INPUT语句会很不方便。在这种情况下，可以使用DATA语句预先设置好数据，处理时再用READ语句把数据读入指定的变量中。请看下面的程序。

10 READ A

语句 70叫做DATA语句，也叫数据语句或置数据语句。DATA语句的格式是

# DATA $<$ 数据表 $>$

其中的数据表是用逗号隔开的一组数据，数据表的末尾没有标点。BASIC程序输人后，如果程序里有DATA语句，计算机会设置一个数据指针，使这个指针指向程序中第一个DATA语句的第一个数据。

语句10,30,50叫做READ语句，也叫读数据语句。READ语句的格式是

# READ $<$ 变量名表 $>$

其中的变量名表可以是一个变量名，也可以是用逗号隔开的多个变量名，变量名表的末尾也不能有标点。

计算机执行READ语句时，从数据指针指出的那一个数据开始读取数据（变量名表中有几个变量，就读取几个数据)，并按照读取的顺序把数据值依次赋给变量表中的变量。与此同时，每读取一个数据，计算机就把数据指针移到DATA语句中的下一个数据上，为读取下一个数据指明位置。

# 本章小结

1.FOR-NEXT语句是构成循环程序的BASIC语句对，这一对语句控制着夹在它们之间的循环体的执行次数。设计循环程序时，应注意正确设置循环变量的初值、终值和步长值。

2．累加运算时，应在进入循环之前，为存放累加和的变量赋初始值0;累乘运算时，应在进入循环之前，为存放累乘积的变量赋初始值1。

3．用READ/DATA语句可以较方便地为变量提供批量数据。应注意使READ语句中的变量与DATA语句中相应的数据的类型保持一致。

4．取整函数 $\mathbf { I N I } ( \mathbf { X } )$ 的功能是取不大于自变量 $\mathbf { X }$ 的最大整数；随机数函数RND(X)的功能是产生一个大于0小于1的随机数。

5.利用公式 $\mathbf { R N D } ( 1 ) * ( \mathbf { M } - \mathbf { N } ) + \mathbf { N }$ 可以产生随机数R $( \mathbf { N } < \mathbf { R } < \mathbf { M } )$ ；利用公式 $\mathbf { I N T } ( \mathbf { R N D } ( 1 ) * ( \mathbf { M } - \mathbf { N } + 1 ) ) + \mathbf { N }$ 可以产生随机整数 $\mathbf { K } \left( \mathbf { N } < \mathbf { K } < \mathbf { M } \right)$ 。
