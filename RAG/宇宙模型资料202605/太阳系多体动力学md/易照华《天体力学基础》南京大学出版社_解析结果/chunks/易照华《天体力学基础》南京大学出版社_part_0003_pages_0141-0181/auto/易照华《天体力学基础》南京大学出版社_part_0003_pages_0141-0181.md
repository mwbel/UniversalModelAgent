# 第十章 磁盘操作系统

# 第一节磁盘操作系统概述

# 一、计算机存储器

计算机的存储器有内存储器与外存储器之分。内存储器又分只读存储器和随机存取存储器两类。只读存储器简称ROM(英文Read Only Memory的缩写)，随机存取存储器简称RAM(英文Random Access Memory 的缩写)。计算机出厂时，ROM中就存好了一些控制计算机进行基本操作的程序。机器运行时可以从ROM中取出信息，但不能把信息存人ROM。关机后，ROM中存储的信息仍然存在，所以，ROM也叫做永久性存储器。利用计算机解决具体问题时，程序、原始数据、运算结果等都存放在RAM里。内存储器存取信息的速度很快，但容量有限，不能满足大型程序的需要，也难以存放很多原始数据，而且一旦关机，RAM里面存储的信息就会丢失。

计算机外存储器（也叫辅助存储器）可以弥补内存储器的这些不足。它有以下优点：（1)能存储大量信息 (包括程序和数据)；(2)在断电情况下能长期保存信息；(3)能够用来较方便地在多台计算机之间传递信息。因此，外存储器一般用来存放大批的原始数据、备用的程序等，需要时把它们调入内存储器。

中华学习机和APPLEⅡ机的外存储器是盒式磁带机和软磁盘驱动器。磁带机价格较低，但速度慢，使用也不方便，所以，在有条件的地方一般都用软磁盘驱动器作外存储器。

# 二、软磁盘和磁盘驱动器

软磁盘简称软盘或磁盘(disk),是用塑料制成的圆形薄片，表面均匀地涂着一层磁性材料，封装在一个方形纸套中。按照直径的大小划分，软磁盘有3.5英寸、5.25英寸和8英寸等不同规格。APPLE机和中华学习机所用的软盘一般是5.25英寸的（见图10.1）。

软磁盘中心的大圆孔叫做中心定位孔;封套下方的长孔叫做读写孔；封套右边的小缺口叫做写保护口；中心定位孔旁边的小圆孔叫做检测孔。

软磁盘表面的磁性材料是信息的磁记录媒体。可以把磁性材料的每一个微小区域看作一个小磁铁，用与录音磁头类似的器件通过不同方向的电流可以使这些小磁铁产生不同的磁化方向。正反两种不同磁化方向可以

用来表示0和1这两种二进制信息，于是就把信息存储到了磁盘上。反过来，当具有不同磁化方向的小磁铁通过磁头时，会使磁头的线圈里产生不同方向的电流，这样就把存储在磁盘上的信息取了出来。实际上，信息的存储与取出的过程要复杂得多，这里只是简单地介绍一下基本的原理。

为了便于在磁盘上存储信息和从中取出信息，人们把整个磁盘划分成许多叫做磁道(track)的同心圆，每个磁道再划分成若干个存储区，这些存储区叫做扇区(sector)。存储在磁盘上的信息，由所在的扇区号和那些扇区所在的磁道号来确定它们的位置（见图10.2)。

磁盘驱动器(drive)是计算机对磁盘进行读、写操作的工具.把磁盘上存储的信息读人计算机内存储器的过程叫做读操作。把计算机内存储器中的信息存入磁盘的过程叫做写操作。读操作与写操作都要通过磁盘驱动器来完成。

进行读写操作时，磁盘驱动器的驱动轴穿过软盘的中心定位孔把盘片夹紧并使它高速旋转。驱动器的磁头通过读写孔与软盘表面接触，并可以沿着软盘半径的方向滑动，对软盘上不同的区域进行读写操作。磁盘上的检测孔可以使磁盘驱动器中的光电检测装置产生检测信号，指出磁道的起始位置。对于封上写保护口的软盘，磁盘驱动器只能从中读取信息，不能把信息写进去，这样可以避免由于操作的失误把已有的信息破坏，从而对磁盘上存储的信息起到保护作用。

# 三、磁盘操作系统

操作系统是为了提高计算机的工作效率、方便用户使用计算机等目的配备的一种软件。它能自动、高效地管理中央处理机、内存储器、外存储器、打印机以及其他外部设备等硬件资源，各种程序的运行也是在操作系统的控制下进行的。

在微型计算机上使用的操作系统叫做磁盘操作系统，即DOS（英文Disk Operating System的缩写)。DOS 的程序存放在磁盘中，需要时调入内存储器。APPLE－Ⅱ 和中华学习机上可以运行 DOS 3.3, $\mathbf { C P / M }$ ，PRODOS 等多种不同的磁盘操作系统，其中DOS3.3 应用最广。

DOS 3.3把磁盘分为35个磁道，从外向内依次编号为0,1,，34。每个磁道又被分成16个扇区，每个扇区可以存放 256个字节（byte)的信息。每个字节占8个二进制位(bit)。因此，磁盘上每一面的最大存储容量是 $3 5 \times 1 6 \times 2 5 6 = 1 4 3 3 6 0$ 字节。习惯上把 $1 0 2 4 ( 2 ^ { 1 0 } )$ 个字节叫做1KB(英文千字节kilo byte 的缩写)，这样，143360 字节就是140KB（ $1 4 3 3 6 0 \div 1 0 2 4 = 1 4 0 )$ 。通常， $0 \sim 2$ 号磁道用来存放DOS系统程序，17号磁道是目录索引区。余下的31个磁道用来存放其他信息，容量是127KB。

在DOS 3.3的管理下，信息是以文件(file)的形式存储在磁盘上的。磁盘上文件的内容可以是程序，也可以是文字，还可以是数。广义地讲，这些存储在计算机存储设备里的程序、文字和数等信息都是数据。因此，这里所说的文件，指的是存储在计算机存储器中的数据的集合。不同的文件有不同的文件名(filename)，DOS3.3规定，文件名可以由以字母开头的1到30个字符串组成。文件名中不允许有逗号。DOS根据文件名来管理文件在磁盘上的存取操作。

# 第二节DOS 3.3的常用命令

# 一、DOS 3.3 的引导

利用DOS3.3可以进行文件的存取、删除、加锁、解锁、更改文件名、查磁盘目录、等项管理操作。这些操作都是通过向计算机发布DOS命令来完成的。要让计算机执行DOS命令，需要先把DOS 3.3系统装入内存储器，这个过程叫做DOS的引导。引导DOS3.3的方法一般有两种，即开机引导和用命令引导。

# 1.开机引导

在开机之前先把存有DOS3.3系统的磁盘插入第一驱动器，关好驱动器的门，然后接通主机电源。习惯上把这种开机时引导DOS的方式叫做冷启动引导。

# 2.BASIC 状态下用命令引导

主机电源打开时驱动器中没有磁盘或者虽然已经把磁盘放人驱动器，但磁盘上没有DOS 3.3系统或者系统程序已被破坏，驱动器会不停地转动。这种情况下，应同时按下Ctrl和Reset键进行强迫中断，让驱动器停下来，进人BASIC状态。在BASIC 状态下，把DOS3.3系统盘插入驱动器，然后键入

$$
\mathbf { P } \mathbf { R } \# s \gets
$$

(其中 $\pmb { s }$ 是驱动器控制卡所插入的槽口号，习惯上把驱动器控制卡插在第6号槽口上)

无论用哪种方法引导，都会看到驱动器的指示灯亮起来，听到磁盘在驱动器中旋转和嗒嗒……的驱动器读取磁盘上信息时发出的声响。指示灯熄灭、驱动器停转以后，屏幕上会出现如下信息:

APPLE IDOS VERSION 3.3 SYSTEM MASTERJANUARY 1. 1983COPYRIGHT APPLE COMPUTER，INC.1980，1982BE SURE CAPS LOCK IS DOWN]

表明DOS3.3已装人内存。使用不同的系统磁盘引导后，屏幕上显示的信息会有所不同。

引导DOS的工作完成以后，就可以使用DOS命令了。这时，可以像没有DOS 时那样输人或运行BASIC程序，也可以使用DOS命令进行磁盘文件的管理操作。

DOS一旦引导成功，就会保存在内存中，直到关机。所以，一般把引导DOS的工作放在开机时进行。需要注意的是，在BASIC状态下引导DOS后，机内原有的BASIC程序会丢失。

# 二、DOS 3.3 的常用命令

# 1．新磁盘的初始化

新磁盘不能直接用来存储信息，需要用DOS命令建立每面35个磁道，每个磁道16个扇区的格式。这种建立格式的过程叫做磁盘的初始化。初始化的方法如下：

(1）引导DOS,把DOS3.3装人内存储器;

(2）取出DOS3.3系统盘，把新磁盘插入驱动器，关上驱动器门;

(3）键入初始化命令，对新磁盘进行初始化。

初始化命令的格式是

# INIT 文件名

其中文件名是用户自己指定的，是用大写字母起头的字符串 (不能含逗号)，长度在1到30个字符之间。例如 HELLO,BOOT,LI XIAO MING,AMS#等都是合法的文件名。

键入INIT命令以后，驱动器的指示灯就亮了，这时可以听到驱动器转动的声音。略等片刻，指示灯熄灭，驱动器停止转动，屏幕重新出现提示符和光标，初始化操作完成。

在使用DOS命令时，键人命令后一按回车键，光标就会消失，同时驱动器上的指示灯就会亮起来，表示驱动器在工作。驱动器工作时，键盘好像被“锁”起来了，一般说来，按哪一个键都没有反应，直到DOS命令执行完，提示符]和光标重新出现以后，才能够再次输人BASIC和DOS 命令。

初始化后的新磁盘称为用户盘，它可以存放用户的程序和数据。DOS3.3在进行初始化工作时，不但在磁盘上建立格式，还把DOS3.3系统程序写人磁盘，因此，用户盘也可以用于引导DOS。

在进行新磁盘初始化之前，可以先输入一个BAISC程序，用来在屏幕上输出用户自己喜欢的信息。例如，输入以下程序：

20 PRINT TAB(16) ；" WELLCOME! "

30 PRINT TAB(20)；" LI XIAO MING "

40 NEW

进行调试、修改，能正常运行后键入命令

INIT HELLO

把新磁盘初始化。此后，用这张磁盘引导DOS时，计算机装入DOS后会自动清屏，并显示出如

下信息。

# WELLCOME!

LI XIAO MING

这说明进行新磁盘初始化时，如果内存储器中有程序，INIT命令在建立格式、把DOS系统程序写人磁盘的同时，还会把内存储器中的程序以INIT命令中的文件名存人磁盘。在用这个用户盘引导DOS时，计算机会自动把那个程序装入内存并运行它。因此，我们一般把初始化时存进磁盘的程序叫做磁盘的欢迎程序。

# 2．磁盘文件目录的查询

信息在磁盘上都是以文件形式存储的，DOS3.3为存入磁盘的文件建立了文件目录。目录中记录了所有文件的加锁标记、文件类型、文件大小和文件名等信息。要查询磁盘上存储了哪些文件，可以使用命令

CATALOG ←

键人CATALOG命令后，屏幕上会按下列格式显示出磁盘上文件目录的清单。

DISKVOLUME254 A 006 HELLO \*| 018 ANIMALS \*T 003 APPLE PROMS \*A 003 MAKE TEXT \*B 009 MASTER CREATE B006 LOADER.OBJ0

]

（屏幕上没显示提示符]时，说明文件目录清单尚未显示完，按一下空格键就会继续显示。）

其中第一行是磁盘卷号，从第二行开始是磁盘文件的目录表。目录表的每一行提供了文件的四个信息，从左到右依次是加锁标记、文件类型、文件长度和文件名，例如,

\*A003 MAKE TEXT

中的\*号表示这个文件已经加锁；A表示它是BASIC程序文件；003表示这个文件在磁盘上占用了3个扇区；MAKE TEXT表示文件名是MAKE TEXT。

# (1）加锁标记

加锁的文件在列目录时第一列显示出 $^ *$ 号，而未加锁的文件目录第一列为空格。加锁的文件只能读出，不能删除或改写。

# (2）文件类型

一个文件可以是一个BASIC语言源程序，也可以是二进制的机器语言程序或数据，还可以是其它类型的文件。文件的类型是在存人磁盘时由DOS自动确定的。DOS3.3用不同的字母标识不同的类型。常见的类型标识符有A、I、B、T等，它们的含义是

A:浮点BASIC语言程序文件;  
I:整数BASIC语言程序文件;  
B:二进制代码文件(机器语言程序文件、二进制代码组成的数据文件，如图形等);  
T:文本文件 (也叫数据文件)。

# 3.BASIC程序的存取

# (1)BASIC 程序存盘

把内存储器中的BASIC程序存入磁盘的命令是 SAVE命令，命令格式为

# SAVE 文件名

键入命令后，内存储器中的BASIC程序以命令中给出的文件名存人磁盘。

在程序存盘时，如果给出的文件名与磁盘中已有的文件名不重复，则磁盘上将增加一个新文件；如果与磁盘上的某个文件名重复，而那个文件又是未加锁的，那么，程序就存入盘内，覆盖同名的旧文件 (旧文件内容丢失)；如果那个文件是加锁的，屏幕上会显示

FILE LOCKED

表示由于同名文件已加锁，程序不能存人。这时要修改SAVE命令中的文件名，重新键入SAVE命令，才能将程序存人磁盘。

执行 SAVE命令后，无论程序是否顺利地存人磁盘，内存储器中的程序都不会丢失，依然保留在主机里。

# (2）把磁盘上的BASIC程序调入内存储器

把磁盘上存储的BASIC程序调入内存储器的命令是LOAD命令，命令格式为

# LOAD 文件名

键入命令后，计算机就在磁盘上寻找命令指定的文件，如果找到，而且它是A型文件，就把这个BASIC程序调人内存储器，覆盖机内原有的BASIC程序；如果没有找到指定的文件，就显示信息

FILE NOT FOUND

表示这个文件没有找到；如果指定文件不是A型文件，就显示信息

FILE TYPE MISMATCH

表示类型不符，不能取出这个文件。

用LOAD命令把BASIC程序从磁盘上调入内存储器后，可以对它进行修改、运行、存盘等操作。

# (3）运行磁盘上的BASIC程序

在 BASIC状态下，RUN命令可以运行内存储器中的程序。在DOS状态下，用RUN命令可以直接运行磁盘上的BASIC程序。命令格式为

# RUN 文件名

计算机执行RUN命令时，先把程序调人内存储器，然后立即运行该程序。所以，键人RUNLX1 命令相当于键人LOADLX1和 RUN这两个命令。

LOAD命令和RUN命令执行后对磁盘上的文件都没有影响。

# (4）在 BASIC 程序中使用 DOS 命令

DOS 命令在 BASIC程序中不能直接使用。如果需要在程序运行当中执行DOS命令，必须把DOS 命令作为字符串放在PRINT语句中，并且在字符串的前面加上 ASCII码是4的控制字符$\mathrm { C t r l } { + } \mathbf { D }$ 作为PRINT 语句中的第一个输出字符。

例如，若想让程序A1运行后自动从磁盘调入另一个BASIC程序A2继续运行，可以在程序A1的最后加上

XXXX PRINT CHR\$(4) ； "RUN A2"

这条语句 (其中的 xXXx表示该语句的行号)，就能让两个程序连续运行了。

注意，如果含有DOS命令的PRINT语句前面也是PRINT语句，则前面的那个PRINT语句不能以逗号或分号结束。

# 5.文件的加锁与解锁

# (1）文件的加锁

为了避免由于操作失误而破坏重要的或需要长期保存的文件，可以给文件加锁。给文件加锁的命令是

# LOCK 文件名

文件加锁后，用CATALOG命令查看文件目录时，会看到加了锁的文件目录信息中增加了加锁标记“\*”号。对于加锁的文件，只能从磁盘上读出，不能用SAVE等命令改写或者用DELETE 命令删除。

# (2）文件的解锁

已经加锁的文件需要改写，或者确实没有保存的必要而需删除时，还可以进行解锁操作，解锁操作的命令是

# UNLOCK 文件名

解锁后，就可以对文件进行改写或删除操作了。

# 6.更改文件名

存储在磁盘上的文件，可以用RENAME命令更改文件名，命令格式为

# RENAME旧文件名，新文件名

计算机执行RENAME命令后，以新文件名代替旧文件名，文件内容不变。加锁的文件不能更改文件名。

# 7．文件的删除

对于没有必要再保存的文件，可以用DELETE命令把它们删除，以便利用那些文件占据的磁盘空间来存储新文件。命令格式为

DELETE命令可以把任何类型的文件删除。命令正常执行后不显示任何信息。如果要删除的是加了锁的文件，屏幕上会显示出

FILE LOCKED

# \*第十一章计算机原理

计算机有非常强的信息处理功能，在现代社会中应用非常广泛。与此同时，人们往往对它产生一种神秘感。实际上，计算机并不神秘，它只是科技工作者为减轻人类脑力劳动强度而发明的一种电子装置，是一种工具。

在生活与生产实践中，人类创造了各式各样的工具，增强了人类自身的能力，减轻了劳动强度。一般说来，各种工具都是人类自身某些功能的扩展与加强。例如，起重机械扩展并加强了人类手的功能；望远镜、显微镜和雷达扩展并加强了人类眼睛的功能；各种交通工具扩展并加强了人类腿脚的功能等等。

计算机也是一种工具。与其他工具不同，计算机扩展并加强了人类大脑的功能，可以按照人们事先编排好的程序，高速、精确地完成计算、逻辑判断以及其他信息处理方面的任务，还能超越人体机能的限制，进行人类大脑力所不及的处理操作。

为了更好地应用计算机这种工具，解决生活与工作中的实际问题，学习一些计算机原理方面的基础知识，无疑是非常必要的。

# 第一节计算机的工作原理

# 一、计算机的组成

为了更好地认识计算机的组成，首先看一下人动脑筋解决问题的过程。

# 1．人解决问题的过程

在小学珠算课上，一位老师告诉学生：“计算23加上18再乘以5所得的积，把结果写在纸上”。学生们听清了题目的要求，记住了参加运算的三个数值，根据已经记熟的珠算口诀与拨珠方法进行计算，然后按照算盘上的得数在纸上写下 $( 2 3 + 1 8 ) \times 5 = 2 0 5$ 这个结果。

在上述过程中，学生用耳朵听老师提出的要求；用大脑记忆运算所需的数值、珠算口诀和解题方法；用手拨动算盘珠进行实际运算；用笔在纸上写出运算结果。学生的大脑还控制着整个过程的完成，即指挥耳朵去听，指挥手去拨珠，指挥手去用笔写。

# 2.计算机的基本组成部分

计算机处理信息的情况与上述过程是很相像的。就像学生要用耳朵听题目的要求一样，计算机在处理信息之前要先接受信息。计算机接受信息的部分叫做输入设备；接受信息后，要记住它们，还要记忆处理信息所用的有关方法和步骤。计算机中负责记忆工作的部分叫做存储器，它们相当于人类用以进行记忆的那部分脑组织；计算机实际进行信息处理的部分叫做运算器，相当于学生解题过程中所用的算盘；与上述过程中书写运算结果所用的笔和纸相对应的，是计算机的输出设备，它们负责把处理的结果展示给使用计算机的人；计算机中还有一部分叫做控制器，它能指挥计算机的各部分协调工作，高效地完成信息处理任务。

# (1）输入设备

输入设备的主要功能是把程序、原始数据、声音、图像等信息转换成计算机能够处理的电信号，并把它们送入计算机的存储器中。常见的输入设备有键盘、扫描仪、语音输入装置等。键盘可以用来输人字母、数字、汉子或其他符号等信息。用扫描仪可以输人图形、文字等信息。利用语音输人装置，可以通过人对计算机讲话的形式把汉语或某些其他国家的语言转换成相应的文字信息并输入计算机，供计算机做进一步的处理。

# (2）存储器

存储器的主要功能是保存信息。存储器又分为内存储器（也叫主存储器）和外存储器（也叫辅助存储器)。下面分别介绍两种不同的存储器。

a.内存储器内存储器的存储容量较小，存取速度较高，主要用来存放正在运行的程序、正在处理的数据等信息。

微型计算机的内存储器由集成电路芯片组成，内部分为许多存储单元，就好像一幢大楼内有许多房间一样。每一个存储单元都有各自的编号，类似于大楼内的房间有各自的门牌号。存储单元的编号叫做地址。计算机根据存储单元的地址准确地向某一存储单元存入信息或从某一存储单元取出信息。

每个存储单元又分为若干位，每一位是一个微小的半导体元件。半导体元件输出电位的高低表示着这一位上所存储的二进制数字的值。若干位二进制数字组成一个存储单元中的二进制数，这个二进制数可以表示指令或者字符、数值等信息。一个存储单元所存储的内容叫做字，存储单元的位数叫做字长。常见的微型计算机的字长有8位、16位和32位等不同规格。这表示存储单元中的信息是由8位、16位或32位二进制数字组成的。一般的微型计算机的内存储器容量有几十KB到几MB(兆字节， $1 \mathbf { M } = 1 0 ^ { 6 }$ )。还可以把微型计算机的内存储器进一步分为随机存取存储器(RAM)和只读存储器 (ROM)两类。RAM中存储的信息可以更改，而且，一旦切断计算机的电源，所存储的信息就不复存在了。ROM中存储的信息是计算机生产厂家确定的，不能由用户更改，也不受电源有无的影响。

b．外存储器外存储器的存储容量较大，一般由几百KB到几百MB不等，存取速度较低，可以常期保存信息，主要用来存放暂时不用的或内存储器一时放不下的大批信息，如暂不运行的程序、大量的原始数据、运算结果、汉字字库、图形信息等等，需要时可以把它们调入内存储器。外存储器主要有磁盘存储器和磁带存储器，是利用磁记录原理存储信息的设备。在“磁盘操作系统”那一章里，已对磁盘存储器作了介绍，这里就不再重复了。磁带存储器的工作原理与磁盘存储器大同小异，所不同的是磁带存储器把信息存储在像录音磁带那样的介质上。磁带存储器存取信息的操作是按顺序进行的，存取速度要比磁盘存储器来得慢。

近年来，外存储器家族中又增加了一名新成员，就是光盘存储器。光盘存储器是利用激光束存取信息的存储设备。光盘存储器的存储容量极大，一个盘片就可以存储几百MB信息，相当于上千片软磁盘的存储容量。存储在光盘存储器中的信息不会丢失，因此，光盘存储器可以用来存储需永久保存的信息。

# (3）运算器

运算器是计算机进行算术与逻辑运算的设备，所以有的书上称之为算逻单元。算术运算指的是加、减、乘、除等运算。逻辑运算指的是对二进制数位的比较、选择、分类、合并、移位、转移等操作。在计算机里，算术运算中的减法用加上负数来完成；乘、除法用一系列的加、减来完成。因此，运算器的核心部分是一个加法器。较高级的计算机为了提高运算速度,还配有乘除部件。

# (4）控制器

控制器的功能是指挥计算机的各组成部分协调地工作，保证信息处理任务能按预先规定的步骤有条不紊地进行。控制器是计算机的“神经中枢”，计算机是在控制器的指挥下高速进行工作的。控制器可以向计算机的其他部分发出控制信号，控制信息处理的整个过程，同时，控制器还接收计算机其他部分送来的信号，以便调整控制器对其他部分的控制。图11.1中的虚线所表示的就是控制信号的流向。

运算器和控制器合起来叫做中央处理机，即 CPU(英文Central ProcessingUnit的缩写)。

# (5）输出设备

输出设备的任务是把计算机进行信息处理时产生的结果，以人能够识别的形式记录下来或者显示出来，供人们分析、使用或保存。常见的输出设备有显示器、印字机、绘图仪以及语音输出设备等。从外表看，显示器与电视机很相像，有彩色与单色之分，它可以在屏幕上输出汉字、字符和图形。常用的印字机有点阵式打印机、激光印字机、喷墨印字机、热敏印字机等许多不同的品种，可以用来把要输出的汉字、字符和图形印到纸或胶片等介质上。绘图仪一般用来输出高质量的图形，如工程图、电路图、卫星轨迹图、建筑图等各种各样平面及立体图形。使用语音输出设备，可以把处理结果通过扬声器用讲话的方式告诉用户。输入设备和输出设备合起来统称输入输出设备，或者称之为I/O(英文Input 和Output 的缩写）设备。外存储器既可用于保存信息，也可用于读取信息，因此也属于I/O设备。

# 3.计算机的硬件与软件

计算机的存储器、运算器、控制器和输人输出设备都是电子或机电产品，是计算机的实体部分，习惯上叫做计算机的硬件或硬设备。顾名思义，硬件是看得见摸得着的“硬东西”。

我们知道，尽管婴儿有大脑可以思考，有眼睛、耳朵可以感知外部世界，有手可以拿起笔来，但他们还不能解任何数学题。要想让孩子解数学题，必须教给他们如何识数、如何计算、如何书写等有关的知识。一个人掌握了某一方面的知识，就可以解决这方面的问题。掌握的知识越多，能解决的问题也就越多。人的知识记在大脑里或记在书、笔记本里，可以随时回忆或查阅。

与此相仿，只具有存储器、运算器、控制器和输入输出设备等硬件的计算机也不能工作，必须给它们配备软件。配备软件相当于教给计算机知识。软件是相对于硬件而言的，是指计算机运行所需的程序及与之相关的资料，如操作系统、BASIC解释程序、汇编程序、编译程序、应用程序以及有关的手册等。软件也叫做软设备。一台计算机配备了一些软件，就能用来解决一些问题。配备的软件越丰富，能用它解决的问题就越多。计算机的软件除了固化在ROM中的以外，主要存放在外存储器里，使用时再调入内存储器。

一般说来，只有在配上足够的软件以后，硬设备才能用来解决实际问题。换句话说，一个计算机系统是由硬件与软件结合起来共同组成的。

# 二、计算机中的数

在计算机内部，程序中的指令、数据、汉字和其他字符、声音、图形等都是以数的形式来体现的，用不同的数代表不同的信息。数在计算机里是如何表示的呢？请先看一个日常生活中的例子。

一个使用四位密码的号码锁有四个排在一起的数码轮，每个数码轮上顺序标有 $0 , \ 1 , \cdots , 9$ 等十个数字。开锁时要转动每个数码轮，使四个数码轮的数字组成的数恰好是这个锁的密码。这种号码锁实际上是一种以机械方式表示十进制数的设备。四个数码轮表示个、十、百、千等四个数位。每个数码轮上不同的十种位置状态表示相应数位上的数值。

计算机中表示数的器件也是根据类似的原理设计的。用能够输出不同电位的电子元件构成一个个数位，不同的电位表示这一数位上的数值。若干个这样的元件排在一起就构成了表示整个数的器件。根据这个原理，如果用电子元件构成十进制的一个数位，就需要有十种不同电位的输出状态，这在实际生产中是极难实现的。但是，电子元件最容易实现电路的通断、电位的高低等两种稳定状态，因此，计算机内部就采用了每一位只用0和1两种数字表示数的方法，用高电位表示1,低电位表示0（或者反过来)。这种只使用0和1两种数字的记数法，叫做二进制记数法。用二进制记数法表示的数叫做二进制数。

# 1.二进制数

二进制数的每一位上只有0或1两种数字，如101，100.111,11111等。为了与常用的十进制数区别，可以把上述二进制数写成 $^ { 1 0 1 , }$ ， $1 0 0 . 1 1 1 _ { _ 2 }$ ,11111。数中右下角的脚标2表示这个数是二进制数，在不致产生混淆时，可以不标出。二进制数的 $" 0 "$ 读作“零”，“1”可以读作“一”，也可以读作“幺”。例如，101可以读作“幺零幺”。

# 三、程序存储原理

# 1．计算机的工作过程

计算机的输入输出设备可以接收原始数据、输出运算结果；存储器可以存放数据；运算器可以进行算术与逻辑运算；控制器可以协调各部分的工作。下面我们以计算机计算 $2 5 6 \times 3 4 3 + 1 2 7 5$ 为例，大致说明一下计算机的工作过程。

用计算机解决这个问题时，需要把256、343、1275这三个数输入计算机，然后才能进行实际计算。计算机的运算器每次只能完成一步基本运算，因此要把计算细分为许多基本的操作，让计算机一步一步地去完成。最后,还要让计算机输出计算结果。这个过程可由以下几个步骤来实现:

$\textcircled{1}$ 从输入设备接收256这个数据，把它放在存储器里;  
$\textcircled{2}$ 从输入设备接收343这个数据，把它放在存储器里;  
$\textcircled{3}$ 从输入设备接收1275这个数据，把它放在存储器里；  
$\textcircled{4}$ 从存储器里取第一个数据送到运算器;  
$\textcircled{5}$ 从存储器里取第二个数据送到运算器与运算器中的数据进行乘法运算得到中间结果87808;  
$\textcircled{6}$ 从存储器里取第三个数据送到运算器与运算器中的数据进行加法运算得到最终结果89083;  
$\textcircled{7}$ 把最终结果送到存储器里;

$\textcircled{8}$ 从存储器里取出最终结果送到输出设备上输出。

与每个步骤相对应，可以向计算机发出一条或几条命令，让计算机完成这个步骤规定的操作。能让计算机完成一定操作的命令，叫做指令。计算机接到指令后，控制器就指挥计算机的有关部分完成指令所规定的操作。计算机按顺序把所有指令都执行了以后，解决问题的任务也就完成了。

为解决问题而设计的指令的有序集合，叫做程序。

# 2.计算机语言

一般说来，程序中的每一条指令在计算机里表现为一个二进制数。计算机根据这些二进制数每一位上0和1的不同组合来确定要执行哪种操作。因此，人们可以直接用二进制指令来编写程序，然后通过一定的输入设备把程序存入存储器，并使计算机运行这些程序。这相当于人用计算机能懂的语言与计算机对话，教计算机如何按照人们的要求去完成指定的任务。由于计算机可以直接识别并执行二进制指令，我们把这类指令叫做机器语言代码，把这种用二进制指令编写的程序叫做机器语言程序。机器语言程序是一些0和1组成的二进制代码的组合，如

000010000001000000 000011000001000010 001001011000000000 ····

机器语言代码很难阅读与记忆，书写与输入也很不方便，而且，不同类型的计算机往往只能识别和执行各自的机器语言代码，通用性较差。为此，人们设计了用符号、数字来代替二进制指令的汇编语言。汇编语言中有与机器语言代码对应的汇编指令，机器语言中的每一条二进制指令对应于一条汇编指令。汇编指令常常采用英语单词的缩写形式来帮助人们记忆其规定的操作，如，用ADD表示加、用SUB表示减、用MUL表示乘、用DIV表示除等等。用汇编指令编写的程序比较便于阅读、记忆、书写与输入。例如，要记忆类似

JMP L3   
ADD 0, 3   
MOV 3, 2

这样用汇编指令编写的程序，就比记上面那些0和1的组合要容易得多了。但是，计算机不能识别汇编指令。为此，人们设计了汇编程序。汇编程序就好像一个翻译，用它可以自动地把汇编指令“翻译”成机器语言指令，供计算机识别、执行。

由于每一条机器语言代码对应于一条汇编指令，用汇编指令编程序仍是很繁重的，而且汇编语言通常也因机型不同而有很大差异。为此，人们又设计出了便于使用的高级语言。高级语言接近自然语言，编程序时使用语句来描述要让计算机进行什么样的操作。语句一般由一些简单的英文单词、字母数字和数学式子等来表达意思，例如，

PRINT " Welcome to computer world! "

# 3.程序存储原理

用计算机解决问题时，首先要编好程序。计算机的工作就是执行程序。为了让计算机按顺序自动地高速执行程序中的指令，必须把编好的程序输人计算机，以二进制数码的形式存放在存储器里一片连续的存储单元中。同时，还要把程序运行时所需的数据以二进制数码的形式输人计算机，存放在称之为“数据区”的一片连续的存储单元里。计算机工作时，从存储器中取出第一条指令，分析这条指令要求计算机进行哪一种操作、操作所需的数据存放在哪个存储单元，然后执行指令规定的操作。执行完以后，再从存储器里取出第二条指令，再分析、再执行，，这样反复进行，直到完成程序所规定的全部操作。

计算机进行取指令、分析、执行等操作是在极短的时间内完成的，一般只需不到千万分之一秒。所以，计算机能高速地完成非常复杂的计算或其他处理任务。

# 第二节计算机系统

从上一节里我们了解到，计算机系统是由硬件与软件组成的。硬件是存储程序和数据、执行程序指定操作的电子与机电设备的实体，是由复杂的电子线路和机电部件组成的。组成这些复杂电子线路的基本元件，是一些叫做门电路的基本电路。把基本电路组合起来构成具有某种功能的电路时，有一些独特的规律，就是逻辑代数中的规律。

# 4.寄存器

在计算机里，有一种用来存储指令、数据、存储单元地址等二进制信息的装置，叫做寄存器。每个寄存器由若干个能保存一位二进制数的触发器组成。触发器可以用两个与非门按图11.9的线路连接而成。它有两个输入端，S和R,两个输出端Q和 $\overline { { \mathsf { Q } } }$ 。当Q端输出1信号（高电位）时，表示这一位上存储的是二进制数“1"；Q端输出0信号（低电位）时，表示这一位上存储的是二进制数 ${ } ^ { \mathfrak { a } } 0 ^ { \mathfrak { n } }$ 。

在计算机内部，触发器的两个输入端S和R平时总是保持在高电位状态，即 $\mathbf { S } = 1$ ， $\mathbf R = 1$ 。

![](images/bb22cf257c03c400ef8b20c99bc2df59127a5f36babcb486802edb7190a76a3a.jpg)  
图11.9触发器

让触发器存储二进制数 $^ { \mathfrak { a } } 1 ^ { \mathfrak { n } }$ 时，在S端输人一个0信号，然后再恢复到1信号状态。根据与非门所实现的逻辑运算可知，S端输入的这个0信号，使Q端产生一个1信号。Q端产生的这个1信号通过导线传送到触发器右边那个与非门的一个输入端，和另一个输人端R的1信号共同作用使Q端产生一个0信号。端产生的这个0信号通过导线传送到触发器左边那个与非门的一个输入端。这个0信号的作用很重要，它使触发器左边那个与非门的输出端Q在S端的0信号消失、恢复到1信号状态后，仍保持在输出1信号的状态上。

如果要让触发器存储二进制数 $" 0 "$ ，就在R端输入一个0信号，然后再恢复到1信号状态。这时，R端输入的这个0信号，会使 $\overline { { \mathbf { Q } } }$ 端产生一个1信号。 $\overline { { \mathbf { Q } } }$ 端产生的这个1信号传送到触发器左边那个与非门，和输入端S的1信号一起使Q端产生一个0信号。这样，就完成了存储二进制数 $\ " { } ^ { \mathfrak { a } } _ { \mathbf { 0 } ^ { \mathfrak { n } } }$ 的操作。同样，Q端产生的这个0信号也会反回来控制右边那个与非门，使得在R端的0信号消失、恢复到1信号状态时,Q端仍然输出1信号，从而使Q端保持0信号的输出状态。

可以看出，这种触发器有两种稳定状态：一种是 $\scriptstyle Q = 1$ ， $\overline { { { \mathsf { Q } } } } = 0 ;$ 另一种是 $\scriptstyle { Q = 0 }$ ， $\overline { { \mathsf { Q } } } = 1$ 。因此，这种触发器也叫双稳态触发器。双稳态触发器常用图11.10所示的符号表示。

# 二、计算机系统的层次结构

在计算机系统中，看得见摸得着是它的硬设备，如显示器、键盘、打印机、主机箱。打开主机箱还可以看到中央处理机等各种集成电路芯片、电路板以及电源等设备。中央处理机配有一套机器语言指令，熟悉机器语言的人员才可以进行操作。

要想使用计算机，需要让各种硬件资源发挥它们的功能，如用键盘接受命令、用显示器显示字符及图形等信息、用磁盘存取信息、用打印机输出等等。这项任务可以由操作系统来完成。配上操作系统后，计算机的功能有了很大的扩充，使用起来也比较方便了，操作人员可以用操作系统命令管理计算机的各种软硬件资源。

要用计算机解决生产、生活中的实际问题，需要先编程序。要让计算机能执行用各种语言编制的程序，需要给计算机配上相应的“翻译”软件，如汇编程序、BASIC语言解释程序、Pascal语言编译程序等等。配上这类软件以后，计算机就可以按照程序设计人员编好的程序去进行实际的处理了。

在实际应用中，能够自己编程序解决问题的人员在计算机使用者中所占的比例很小。绝大多数人是使用现成的应用软件来完成各自的信息处理任务的。应用软件种类繁多，如文字处理软件、统计软件、办公自动化软件、计算机辅助设计软件、教育软件、……，等等。配上应用软件以后，计算机系统的功能得到了更进一步的扩充，非计算机专业人员也能直接使用计算机了，因而计算机系统的应用得到了广泛的普及。

需要指出，操作系统的作用非常重要。它有效地管理着计算机系统的硬件资源，而且，只有在它的控制下，汇编程序、解释程序、编译程序以及应用软件才能完成各自的任务。

综上所述，在硬件的基础上，配备了不同的软件后计算机就具备了不同层次的功能。这相当于形成了不同层次的计算机系统。例如，配备了操作系统就形成了操作系统级的计算机系统；再配上汇编程序，就形成了汇编语言级的计算机系统；再配上高级语言解释程序或编译程序，就形成了高级语言级的计算机系统；在操作系统级的层次上配备应用软件后，就形成了应用级的计算机系统。

# 第三节计算机发展简史

# 一、计算机的诞生与发展

计算机是进行信息处理的工具，而“计算”只是它广泛应用的一个方面。但是，最初人们设计计算机时，确实是为了解决数值计算方面的问题。“计算机”也就因此而得名，并且一直延用至现在。为了回顾一下计算机发展的历史，首先看一下计算工具的发展。

# 1．早期的计算工具

早期的计算工具主要是人类自身的某些器官（如手指、脚趾）或周围可数的物体（如石子、绳结、小棒)。这些工具一般用来计数。经过加工制造出来的第一种计算工具是我国早在唐朝就出现了的算盘。算盘的发明为世界科学技术的进步作出了光辉的贡献，目前仍在世界上许多国家和地区广泛使用着。用算盘能快速进行加、减、乘、除等运算，即使在普遍使用了计算器的今天，仍能显示出它独到的优点。随着生产与科学研究的发展，计算工作日趋复杂，于是人类发明了越来越先进的计算工具，如对数计算尺、机械式计算器等等。然而，这些计算工具的共同弱点是不能自动进行连续运算，也不能存放大量的数据和运算结果，运算速度较慢，能处理的问题也是有限的。因此，产生了计算工具自动化的需求。研制自动化计算工具的第一人是英国的查尔斯·巴贝奇（Charles Babbage,1791-1871)。他用了几十年的时间设计了一台分析机，准备用刻有数字的轮子来存储数据，通过齿轮的转动来进行计算，用齿轮和杠杆构成的装置来传送数据，用穿孔卡片输入程序和数据，用穿孔卡片和打印机输出计算结果。由于当时的技术条件所限，这台分析机未能制成，但是巴贝奇的设计思想是不朽的，它与现代计算机的设计思想完全吻合。

# 2.计算机的诞生

计算工具的现代化是从本世纪开始的。计算机的诞生有它的物质基础和客观需求的背景。第二次世界大战期间，由于军事上的需要，电子工业及科研领域硕果累累，如脉冲电路、电子开关及雷达等。相应的理论与电子产品为以后设计与制造计算机打下了一定的基础。另一方面，科学技术特别是军事科学技术的发展，要求能够发明一种能以极高的速度和精度进行计算的工具。于是，电子计算机的研制就提到日程上来了。1943年，由美国军事部门出资在宾夕法尼亚大学开始了第一台电子计算机ENIAC的研制。这台计算机在1945年12月投入运行，到1946年2月正式交付使用。ENIAC用了18000个电子管，1500多个继电器，占地170平方米，重30吨，耗电150千瓦，每秒可进行5000次运算。

# 3．计算机的发展

从第一台计算机诞生到现在，无论在技术上、性能上，还是在应用方面，计算机的发展都是非常迅速的。计算机家族不断增添新的成员，如大型机、中型机、小型机、微型机、巨型机等。各种型号的计算机在各个不同的应用领域发挥着巨大的作用。在此期间，计算机的运算速度每过几年就会提高十倍，制造成本会降低到原来的十分之一，体积也可以缩小到原来的十分之一。目前广泛应用的微型机的功能已远远超过了当初的中小型机，而价格只是它们的几十分之一。从所采用的电子元器件看，几十年来，计算机经历了电子管、晶体管、集成电路、大规模集和超大规模集成电路等四代的变化。

1946年到1958年左右研制的计算机是以电子管为基本元件的，可称之为第一代计算机。第一代计算机体积庞大、存储容量小、功耗较大。这一代计算机确立了计算机发展的技术基础，主要用于科学计算。

大约从1958年开始，晶体管取代了电子管，成为计算机的基本元件。这一时期的计算机是计算机家族的第二代产品。第二代计算机在体积、重量、功耗等方面有了相当大的改进。与第一代计算机相比，这一代计算机的性能提高了数十倍。第二代计算机除了用于科学计算以外，在数据处理和工业控制方面也显示了非凡的能力。

大约从1964年开始，集成电路取代晶体管成了计算机的基本元件，计算机进人了第三代产品时期。所谓集成电路，就是把许多晶体管、电阻、电容及它们之间的连线制作在一块很小的半导体基片上构成的整体电路。集成电路的应用，使计算机的体积、功耗有了明显的减小，计算速度、存储容量和可靠性都有了较大的提高。第三代计算机的应用扩展到了许多科学技术领域。

从70年代开始，相继出现了大规模和超大规模集成电路。人们把用这类集成电路制造的计算机叫做第四代计算机。大规模集成电路和超大规模集成电路的元件密度非常高，目前，每平方毫米半导体基片上可以制作几十万个晶体管构成的电路。第四代计算机的体积与功耗更小了，性能更高了，运算速度提高到了每秒几百万次、几十亿次甚至更高。这一时期，出现了体积小、重量轻、功能很强、价格低廉的微型计算机。计算机的应用得到了迅速的发展，已进入了人类社会的各个领域。

# 二、我国计算机的发展

早在1956年，我国就开始了计算机的研制。1958年和1959年，先后制成了“103”和$" 1 0 4 "$ 型电子管计算机。这些计算机在天气预报、大型水坝设计、油田开发和我国第一颗原子弹的研制以及人工合成胰岛素等生产、科研项目中发挥了重要的作用。

1965年，我国研制了“109乙”型晶体管计算机。“109乙”计算机的运算速度达到了每秒6万次。之后，又陆续研制成功了“013”、“DJS8”等十多种不同型号的晶体管计算机。

进入70年代以后，我国先后研制成功了“TQ16”、“DJS100”系列、“DJS200”系列等集成电路计算机，并形成了成批生产的能力。国产计算机的运算速度已超过每秒百万次。

自 80年代以来，我国的计算机科学技术水平有了飞速的发展，除了许多种国产微型计算机大批投入市场以外，在1983年先后研制出了“757”大型计算机和“银河－I”巨型计算机。运算速度分别达到了每秒1000万次和每秒1亿次。1992年10月，我国又研制成了每秒能进行10亿次以上运算的“银河－Ⅱ”巨型计算机，并于1992年11月19日通过了整机技术鉴定。“银河－Ⅱ”型计算机的研制成功，标志着我国的计算机科学技术已达到了世界先进水平。

随着计算机科研与生产的发展，我国的计算机应用领域也在不断地扩大。近十年来，微型计算机进入了中小学教育领域，每年都有几十万中小学生接受不同程度的计算机教育，为我国的计算机科研、生产与应用培养了大批后备力量。目前，计算机的应用正在逐步向家庭扩展。

# 本章小结

这一章介绍了计算机的组成、计算机中的数和计算机的工作原理，简单介绍了逻辑代数和逻辑电路的初步知识，以及计算机系统的分层结构概念，回顾了一下计算机的发展历史。

1．计算机是能高速、自动进行信息处理的电子装置。它由输入输出设备、存储器、运算器、控制器等硬件和使这些硬件发挥其功能的软件组成。计算机通过执行程序来完成人们交给它的任务。程序、数据等以二进制数的形式存放在存储器里，计算机自动按顺序执行程序中的指令。二进制是计算机内部采用的数制。用“除二取余法”可以把十进制整数转换成二进制数。十六进制数的每一位与四位二进制数相对应，按顺序把十六进制数的每一位变成相应的二进制数，就可以把十六进制数转换成二进制数，反之亦然。

2．逻辑电路是计算机电子线路的基本组成部分，其输入输出关系体现了逻辑代数中的运算关系。在计算机内部，二进制数 $" 0 "$ 和“1”是用电路输出信号电位的高低来表示的。“或”、“与”、“非”是逻辑代数中的三种基本运算，体现这三种运算关系的逻辑电路是或门、与门、非门。用或门、与门、非门等基本逻辑电路组合起来可以构成寄存器、加法器等计算机硬件电路；以计算机系统的硬件为基础，配上操作系统和各种软件可以构成不同层次的计算机系统。

3．本世纪40年代，人类发明了电子计算机。随着科学技术水平的提高与人类社会实践的需要，计算机科学技术的发展速度非常快，计算机的应用越来越普遍，已经渗透到科研、生产、教育以至家庭生活等各个领域。
