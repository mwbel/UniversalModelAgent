# Epic F 首批可视化模块说明

| 模块 | 核心教学目标 | 主要交互参数 | 主要输出 |
| --- | --- | --- | --- |
| 日食 / 月食演示 | 理解太阳、地球、月球相对位置如何产生不同遮挡现象 | 现象模式、交线偏差、月球轨道倾角示意、动画速度 | 遮挡路径、满足条件提示、关键说明 |
| 行星逆行演示 | 理解逆行是视运动现象而非真实倒退运动 | 外行星选择、地球速度倍率、外行星速度倍率、回环放大 | 轨道平面、视线连线、背景恒星上的回环轨迹 |
| 开普勒第三定律 | 理解半长轴增加时轨道周期如何按幂律变化 | 目标半长轴、对比半长轴、说明视图、动画速度 | 轨道对比、周期曲线、T² 与 a³ 对应关系 |
| 哈勃定律图表 | 理解距离与退行速度近似线性相关，H0 决定斜率 | 哈勃常数、最远样本距离、观测噪声、样本数量 | 趋势线、散点分布、当前斜率说明 |

## 问答联动约定

- 页面支持 `module` URL 参数直接定位模块。
- 支持将当前控件值编码到 URL，例如：
  - `index.html?module=eclipses&mode=solar&alignment=5`
  - `index.html?module=hubble&hubbleConstant=67&sampleSize=20`
- 后续 Epic G 可在回答结果中返回：
  - `visualization_id`
  - `visualization_title`
  - `visualization_url`
  - `visualization_params`
