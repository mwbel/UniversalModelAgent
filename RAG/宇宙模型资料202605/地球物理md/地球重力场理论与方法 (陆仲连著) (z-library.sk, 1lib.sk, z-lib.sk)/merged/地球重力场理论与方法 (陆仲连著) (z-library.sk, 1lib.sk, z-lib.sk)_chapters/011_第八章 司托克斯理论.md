# 第八章 司托克斯理论

# $\ S 8 . 1$ 司托克斯定理

如果已知一个水准面。及其内部物质的总质量M和整个物质绕一固定轴均匀旋转的角速度w,则a面外空间的重力位和面上的重力值都唯一地被决定，而与物体的质量分布情况无关。这就是司托克斯(StokesG.G.）定理。

为了证明这个定理，先假定在两种质量分布情况下，o面上分别有两种重力分布g'和g"，在o面外空间有两个重力位W'和W";然后导出这两者没有差别。

依据 $( 5 \cdot 8 2 )$ 式，可以分别写出

$$
\begin{array} { r } { \int \mathrm { g } ^ { \prime } \mathrm { d } \sigma = 4 \pi \mathbf { f } \mathbf { M } - 2 \omega ^ { 2 } \tau } \end{array}
$$

$$
\begin{array} { r } { \int \mathrm { g } ^ { \prime \prime } \mathrm { d } \sigma = 4 \pi \mathrm { f } \mathrm { M } - 2 \omega ^ { 2 } \tau } \end{array}
$$

式中τ为质体的体积。显然,这两个等式的右端是完全一样的，左端也应当相等，因而有

$$
\begin{array} { r } { \underset { ( \sigma ) } { \int } ( \mathbf { g } ^ { \prime } - \mathbf { g } ^ { \prime \prime } ) \mathbf { d } \sigma = 0 } \end{array}
$$

再利用力和位的关系，(8·1)式可改写为

$$
\int _ { ( \sigma ) } \frac { \partial } { \partial \mathbf { n } } ( \mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime \prime } ) \mathrm { d } \sigma = 0
$$

式中 $\pmb { \nbar { \mathbf { n } } }$ 为 $\pmb { \sigma }$ 面的外法线方向。

重力位由离心力位和引力位两部分组成，离心力位与质量分布情况无关。因此,W'与W"要是不一样,也只是因为引力位不一样,即W'一W"就是质量分布不同而引起的引力位差，因而它是个调和函数。若以W。和W。分别表示两个重力位在o面上的值，那末调和函数W-W"在o面上是个常数W一W。根据调和函数的性质,W-W在无穷远处的值应为零,并在整个外空间不能有极值,那末调和函数W'一W必然是由α面上的值W一W。往外逐渐趋向于零。这样一来，在 $\pmb { \sigma }$ 面上的 $\frac { \partial } { \partial \mathbf { n } } ( \mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime } )$ 必然到处都是一个符号：若 $\mathbf { W } _ { 0 } ^ { \prime } - \mathbf { W } _ { 0 } ^ { \prime } < 0$ ，则 $\frac { \partial } { \partial \mathbf { n } } ( \mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime \prime } ) \vert _ { \sigma }$ 都为正号：若 $\mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime \prime } { > } 0$ ，则 $\frac { \partial } { \partial \mathbf { n } } ( \mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime \prime } ) \vert _ { \circ }$ 都为负号。但它们又应满足(8.2)式，即在面上的总和应为零，这就只能是在 $\pmb { \sigma }$ 面上 $\frac { \partial } { \partial \mathbf { n } } ( \mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime \prime } )$ 的值处处为零,即

$$
\frac { \partial } { \partial \mathbf { n } } ( \mathbf { W } ^ { \prime } - \mathbf { W } ^ { \prime \prime } ) \mid _ { \mathbf { \theta } } = 0
$$

也可写为

$$
\mathbf { g } ^ { \prime } - \mathbf { g } ^ { \prime \prime } { = } 0
$$

这表明 $\pmb { \sigma }$ 面上只有一种重力分布。

再依据 $( 5 \cdot 8 5 )$ 式可以写出

$$
\begin{array} { r l } & { \mathbf { W } _ { p } ^ { \prime } = \frac { 1 } { 4 \pi } \underset { \mathfrak { o } } { \int } \underset { \mathbf { r } } { \mathbf { g } } ^ { \prime } \mathrm { d } \sigma + \frac { \omega ^ { 2 } } { 2 } ( \mathbf { x } ^ { 2 } + \mathbf { y } ^ { 2 } ) + \frac { \omega ^ { 2 } } { 2 \pi } \underset { \mathfrak { o } } { \int } \frac { \mathrm { d } \tau } { \mathbf { r } } } \\ & { \mathbf { W } _ { p } ^ { \prime } = \frac { 1 } { 4 \pi } \underset { \mathfrak { o } } { \int } \frac { \mathbf { g } ^ { \prime \prime } } { \mathbf { r } } \mathrm { d } \sigma + \frac { \omega ^ { 2 } } { 2 } ( \mathbf { x } ^ { 2 } + \mathbf { y } ^ { 2 } ) + \frac { \omega ^ { 2 } } { 2 \pi } \underset { \mathfrak { o } } { \int } \frac { \mathrm { d } \tau } { \mathbf { r } } } \end{array}
$$

式中W和W分别为外空间任一点p(包括o面上)的两种重力位值。将两式相减得到

$$
\mathbf { W } _ { \mathfrak { p } } ^ { \prime } - \mathbf { W } _ { \mathfrak { p } } ^ { \prime } = \frac { 1 } { 4 \pi } \int \ d \frac { \mathbf { g } ^ { \prime } - \mathbf { g } ^ { \prime \prime } } { \mathbf { r } } \mathrm { d } \sigma
$$

由 $( 8 \cdot 3 )$ 式， $( 8 \cdot 4 )$ 式右端为零。最后得到

$$
\mathbf { W _ { p } ^ { ' } } { - } \mathbf { W _ { p } ^ { ' } } { = } 0
$$

这说明，在o面外空间只有一种重力位。这就完成了司托克斯定理的证明。

第六章研究水准椭球的重力场时就应用了此定理。已知水准椭球的表面就是外水准面，它的形状和大小用两个参数(椭球的长半径a和扁率α)表示，再给定椭球的总质量M和绕短轴旋转的角速度…，它的外部重力场就唯一地决定了，而并没有对其质量分布作任何的假定。但在实质上，外水准面形状与质量分布是有内在的联系的。给定了外水准面形状，质量分布的任意性也就受到很大的限制。

# $\ S 8 . 2$ 大地水准面

上一节讲述的司托克斯问题是已知外水准面o、总质量M和旋转角速度w求外部重力位W和a面上的重力g。而实践中往往不知道o面，却能知道o面上的重力g。因此，要解的是司托克斯反问题，即已知g、M、ω求定α面及其外部重力位W。

然而，对于真实的地球，在研究外部重力场时没有一个真正的外水准面可用。因为真正的外水准面应该是把地球的最高点都包在里面的等位面,它与地表的大部分都相距约9公里，研究这样一个面的形状及其外部重力场就缺乏实际价值了，况且在这个面上的重力值也是很难求得的。百余年来，人们都以大地水准面作为 $\pmb { \sigma }$ 面。所谓大地水准面是地球重力场的一个水准面，它在海洋上与静止的海水面重合，在陆地则是它在大陆底部的延伸。若以 ${ \bf W } _ { 0 }$ 表示大地水准面上的重力位值，则 $ { \mathbf { W } } { = }  { \mathbf { W } } _ { 0 }$ 就是大地水准面的方程。

$$

$$

式中 $\mathbf { w } _ { \mathfrak { p } }$ 为 $\mathbf { P }$ 点的重力位值。设 $\bar { \pmb { g } }$ 为沿线段 $\mathbf { P } _ { 0 } \mathbf { P }$ 重力 $\pmb { \mathrm { \delta } }$ 的积分中数，即

$$

$$

以 $( 8 \cdot 5 )$ 式代入可得

$$

$$

该式表明，要计算正高必需知道两个量：地面点与大地水准面的重力位差和地面至大地水准面之间的平均重力。

设O点为平均海水面上一点，即大地水准面上的一点，一般在海岸附近。在O点与P点之间进行水准测量，并沿水准路线作一定密度的重力测量，类似于(8·5)式可以按下式求得位差

$$

在地球内部重力位的二次导数不得而知,现以正常重力位的二阶导数U+Uy代替W+W,而-2w+U.+U就是椭球外Q点处的正常重力的垂直梯度。因而，由(8·10)式可得

# $\ S 8 . 3$ 司托克斯公式

如前所述，大地水准面的外部尚有地球的部分质量存在。为了使大地水准面满足外水准面的条件，必需对这部分质量进行处理;另一方面，重力值只能在地球表面上通过测量求得，还要通过一定的手段才能求出大地水准面上的重力值。这些问题将在以后进行论述。在本节，将暂时认为大地水准面是外水准面，且已知这个面上的重力值，导出推求大地水准面及其外部重力位的公式。

在第七章中已经提到，在解此类问题时，先引入一个已知的正常重力场,通常采用水准椭球的重力场作为正常重力场，作为实际地球重力场的近似,然后求出它们之间的差别。这样，实际重力位与正常重力位之差是扰动位 $\mathbf { T }$ ,它在外空间(包括界面上)满足微分方程 $( 7 ~ \bullet ~ 4 7 )$ 。

在研究大地水准面及其外部重力场时，大地水准面上的 $\mathbf { P } _ { \mathrm { 0 } }$ 点就是 $( 7 ~ \cdot ~ 4 7 )$ 式中的A点。如图8.2所示，过P。点作正常椭球面U=U。的法线,与椭球面相交于Q。点，则Q。点就相当于(7·47)式中的N点。距离PQ。称为大地水准面高，用N表示之。为了讨论简单,先假定W。${ \bf \Pi } = { \bf U } _ { 0 }$ ，并令

$$
\Delta \mathsf { g } _ { \mathsf { p } _ { 0 } } = \mathsf { g } _ { \mathsf { p } _ { 0 } } - \mathsf { \Upsilon } \mathsf { q } _ { 0 }
$$

称为重力异常，则由 $( 7 ^ { \circ } \bullet \pmb { 7 } )$ 式可得

$$

$$

它是扰动位T在界面W一W。上应满足的条件。由于在边界条件中有函数及其一阶导数，因此是一个第三外部边值问题。

二、重力的局部地形改正

所谓局部地形改正是过研究点P作一水平面,去除水平面以上的质量m,m等和补上水平面以下缺少的质量m2,m4等所引起的P点重力值的变化，见图8.6。由于在水平面上的质量(m,,m,)对P点的引力在重力方向上的分量为负值，去掉这些质量后将使P.点的重力值增加;在水平面下填补质量(m2,m,),也会使P点的重力值增加。因此，局部地形改正总是为正值。

由此人们想到，若在去除大地水准面以外的质量的同时，在大地水准面内部填补一些质量，以减小大地水准面上P,点的重力位的变化。今以Wc表示填补质量对P。点的引力位，则间接影响的计算式变为

$$
\delta \mathsf { N } { = } \frac { \mathbb { W } _ { \Gamma } { - } \mathbb { W } _ { \Gamma } } { \mathbb { \gamma } _ { \mathfrak { g } } }
$$

在1905年鲁茨基(RudzkiM.P.)提出-一种逆点法，即对去除的大地水准面以外的任--个单元质量dm，在大地水准面的内部填补上一个单元质量dm'，它们对P点的引力位相等，而两个单元质量在同一向径上。这样一来，间接影响就为零了。然而，虽然调整大地水准面和真正的大地水准面是重合了，但大地水准面外部的重力场却变化了。因此，也不是一种可行的方法。

如果只对地面观测重力值g加空间改正0.3086H，减去正常椭球面上相应点的正常重力值 $\gamma _ { _ { \scriptscriptstyle ( j ) } }$ ，所得的数值称为空间异常。若以 $\Delta \mathbf { g }$ 表示空间异常，则有

$$

$$

当用空间异常代入司代克斯公式(8·33)推求大地水准面高时，间接影响只有1米左右。这是因为：对地面观测重力值加空间改正相当于将大地水准面以上的物质垂直下移H这一距离，如图8.12所示。由于在大地水准面下部填补了质量，因而使间接影响大为缩小。但由于空间异常与地形的关系明显，用内插方法推求未知点的空间异常值要达到好的精度比较困难。

人们倾向于根据地壳均衡理论来填补大地水准面以下的质量，由此所得的异常称为均衡异常，用它来计算大地水准面高较为适当。认为这种模型能最好地适应地质的实际情况；异常值很小，变化平缓，和地形无关，既适合于内插也可以外推；间接影响适中而不太大，约10m的数量级。因此，在下两节中将对地壳均衡理论进行论述。

# $\$ 8.8$ 地壳均衡假说

完全布格异常在山区是系统的负值，而且数值相当大，平均每1000m的高度绝对值增大约100毫伽。唯一的可能解释是山底下的质量有所欠缺，高出海平面的地形质量以某种方式被补偿。

垂线偏差的大小也有类似的现象。实际的垂线偏差值比可见的地形质量所造成的影响要小得多。这也只能用内部质量有某种补偿来解释。

几乎是在同一时间，有两种不同的理论产生，被用于说明这种补偿现象。这就是普拉特(PrattJ.H)在1854年和爱黎(AiryG.B)在1855年各自提出的地壳均衡理论。现分别叙述于下。

# 一、普拉特-—海福特系统

这种补偿系统的概念是由普拉特提出的，后来由海福特引进数学公式，并系统地用于大地测量。

根据普拉特的假说，地壳的下界面在平均海水面下的深度在各处都是-一样的,通常用D表示这个深度。在地壳下有岩浆，要使它达到静力平衡，就必需保持地壳对岩浆的压力在地壳下界面的各点上都是相等的。也就是地壳的下界面应是一个均衡面，因而D就是均衡面的深度，简称均衡深度。起伏的地形表明，地壳的厚度不是一个常数。若以H表示地表在海水面上的高程，则地壳厚度为D十H。为了保证在下界面上的压力处处相等，必需使一定切面的各个垂直柱体中的质量为一常数。

![](../chunks/地球重力场理论与方法 (陆仲连著) (z-library.sk, 1lib.sk, z-lib.sk)_part_0005_pages_0161-0200/ocr/images/05a033d01364c44490c1713ff84e0a2682d48cc01f365cd37800abb438ef5459.jpg)  
图8.13

因此，海底地壳密度的增量为

如图8.13所示，设厚度为D的柱体的物质密度为P，而厚度为D+H的柱体的质量密度为p，依据普拉特假说，有如下方程成立：

$$
( \mathrm { D } + \mathbf { H } ) \varrho = \mathrm { D } \varrho _ { \mathrm { \scriptscriptstyle t } }
$$

因此，对于 $\mathsf { H }$ 为正的大陆地区，地壳的实际密度ρ一定小于标准密度p。由(8·85)式可得，密度差为

$$
\Delta \rho { = } \rho _ { \mathrm {  i } } - \rho _ { \mathrm { 6 } } { = } \frac { \mathrm { H } } { \mathrm { D } { + } \mathrm { H } } \rho _ { \mathrm { i } }
$$

而在海洋地区，满足平衡的条件为

$$
( \mathrm { D } { - } \mathrm { H } ^ { \prime } ) \rho { + } \mathrm { H } ^ { \prime } \rho _ { \mathrm { w } } { = } \mathrm { D } \rho _ { \mathrm { 0 } } \ .
$$

式中 $\mathbf { H ^ { \prime } }$ 是海深， $\rho _ { \mathbb { W } }$ 是海水的密度 $( 1 . 0 2 7 g { \tt c m } ^ { - 3 } )$ 。

$$
\rho - \rho _ { \mathrm { 0 } } = \frac { \mathrm { H } ^ { \prime } } { \mathrm { D } - \mathrm { H } ^ { \prime } } \rho _ { \mathrm { 0 } } - \rho _ { \mathrm { w } }
$$

根据这种假说，山岳不是某种不相干的附属物需要消除，而是地壳的组成部份，是由地壳下层膨胀而成的。那些高出海水面的物质是由深处升起的，上部质量的多余正好等于下部质量的不足。只有保持这个条件地壳才均衡，单纯消除外部质量就会破坏这种均衡性。

20世纪初期，海福特在将普拉特假说运用于美国重力测量和大地测量的数据处理时，对其稍作了改变。他把高出海水面的那部份地壳的密度都取为标准密度值p。此时，山岳的质量仅仅被地下部份密度的减少所抵偿，基本方程 $( 8 \cdot 8 5 )$ 变为

$$
\mathrm { D } \boldsymbol { \rho } \boldsymbol { + } \mathrm { H } \boldsymbol { \rho } _ { \mathrm { 0 } } \boldsymbol { = } \mathrm { D } \boldsymbol { \rho } _ { \mathrm { 0 } }
$$

下部密度的差值为

$$
\Delta \rho { = } \frac { \mathbf { H } } { \mathbf { D } } \rho _ { \mathrm { 0 } }
$$

这给实际计算带来了方便。在实际计算时他采用p=2.67gcm-3，而D曾用过122km、113.7km和 $1 0 2 \ k m$ 等不同的数值。

# 二、艾黎—-海斯卡宁系统

这个系统的假说是由艾黎提出的，而由海斯卡宁给出实际应用于大地测量的计算公式。

在艾黎的假说中，地壳的密度是个常数，取为p，它漂浮在密度为pi的岩浆上。当然，pi>Po,且向上隆起得越高，则向下陷入得越深。因地壳的物质密度比岩浆的密度小，上部质量起着补偿深部质量不足的作用。

由于这一原因，威宁·曼尼兹将艾黎的漂浮学说，在1931年作了修改，引进了区域补偿以代替局部补偿。这两种补偿的差异如图8:15所示。按照威宁·曼尼兹的理论，地形质量是作为一种加在不断裂而有弹性的地壳层上的负荷。此时，可认为整个地块是均衡的，而其各部份不是都均衡的。地块上有山、凹地以及地表面的所有其他地貌。因此，不是补偿地貌的各个零碎部份，而是根据整个地区的负载(以其平均高程决定)使整个地区均衡起来。虽然威宁·曼尼兹对艾黎假说的改进使之更为实际，但比较复、杂，在大地测量中还很少用它。

一般说来，地壳应力求达到静水状的均衡。但可举出许多例子说明完全均衡是不存在的，也是不可能存在的。例如，在原苏联的莫斯科和切尔尼科夫地区，地势平坦，却观测出极大的重力异常。这重力异常表明，深藏在地壳内部而又没有任何外部标记的重质量过多。即使在创立地壳均衡假说的印度，现代的大地测量和重力测量工作的结果，几乎也都没有完全证实这个理论。该地区重力异常的主要因素是隐蔽山脉的引力，这个隐蔽山脉的密度要比周围地区的地壳密度大，并且几乎与喜马拉雅山平行，并极靠南面，而在地表的地形上却毫无显露。上述现象说明，实际的地壳结构及引起重力异常的原因，较之均衡假说中所阐述的要复杂和更多样。虽然如此，均衡理论还是反映了部份真象，因而对于重力测量和大地测量的数据处理还是有一定用处的。

# $\ S 8 . 9$ 重力的均衡改正

根据地壳均衡假说，单纯去掉大地水准面以外的质量将破坏地壳均衡，应该把这部份质量填补在大地水准面以下适当的位置。依据普拉特—海福特均衡模型，地形质量应填补在大地水准面与均衡面之间，使其物质的密度由 $\pmb \rho$ 增加到标准值 $\pmb { \rho _ { 0 } }$ ;若依据艾黎——海斯卡宁均衡模型，地形质量用于填补在大陆的根处，使其物质密度从 ${ \pmb \rho } _ { 0 }$ 增加到 $\pmb { \rho } _ { 1 }$ 。最后结果都是有一个想像的有一定厚度且密度为常数 $\pmb { \rho } _ { 0 }$ 的地壳，对于普拉特-海福特模型厚度为D,而对于艾黎——海斯卡宁模型厚度为 $\mathrm { \Delta T }$ 。地面重力观测值根据这种质量调整归算到大地水准面上所需加的改正称为均衡改正，这项改正可分解为如下三个部份：
