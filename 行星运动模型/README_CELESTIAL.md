# 天球坐标系可视化应用使用指南

## 🌍 应用简介

这是一个基于地平坐标系的天球可视化应用，参考MATLAB视运动程序实现，用于展示太阳的周年视运动轨迹（光行迹）。

## 🚀 启动方式

```bash
streamlit run celestial_sphere_app.py
```

访问地址：http://localhost:8502（如果8501被占用）

## ✨ 核心功能

### 1. 天球坐标系展示

**坐标系定义**：
- **X轴**：指向东方
- **Y轴**：指向北方
- **Z轴**：指向天顶

**主要元素**：
- 🌐 **天球**：半透明球面
- ⭕ **地平圈**：灰色圆圈，表示地平线
- 🔵 **天赤道**：青色虚线，与地球赤道平行
- 🟡 **黄道**：黄色虚线，太阳在天球上的年运动轨迹
- ⚪ **天轴**：白色虚线，连接南北天极
- ⭐ **北天极**：标记点，仰角=纬度

### 2. 太阳周年视运动轨迹（光行迹）

**定义**：
在同一地点，每天相同时间（如12:00）记录太阳位置，一年形成的轨迹。

**特点**：
- 轨迹呈"8"字形（日行迹/analemma）
- 标记四个节气点：
  - 🟢 **春分**：约3月20日
  - 🔴 **夏至**：约6月21日
  - 🟠 **秋分**：约9月23日
  - 🔵 **冬至**：约12月22日

### 3. 观测地点配置

**预设地点**：
- 北京（116.39°E, 39.90°N）
- 上海（121.47°E, 31.23°N）
- 拉萨（91.13°E, 29.65°N）
- 广州（113.26°E, 23.13°N）
- 乌鲁木齐（87.62°E, 43.82°N）
- 自定义（可输入任意经纬度）

**参数说明**：
- **经度**：-180° 到 180°（东经为正，西经为负）
- **纬度**：-90° 到 90°（北纬为正，南纬为负）
- **时区**：UTC偏移量（北京时间为UTC+8）

### 4. 显示选项

- ✅ **显示地平面**：灰色半透明平面
- ✅ **显示天赤道**：青色虚线圆圈
- ✅ **显示黄道**：黄色虚线圆圈（倾斜23.5°）
- ✅ **显示太阳周年轨迹**：黄色轨迹线

## 🎯 使用场景

### 场景1：观察不同纬度的太阳轨迹

1. 选择不同城市（如北京、拉萨、广州）
2. 观察太阳周年轨迹的变化
3. 理解纬度对太阳高度角的影响

**现象**：
- 纬度越高，太阳轨迹越靠近地平线
- 纬度越低，太阳轨迹越接近天顶

### 场景2：观察不同时间的太阳位置

1. 调整"固定观测时间"滑块（0-23小时）
2. 观察不同时刻的太阳轨迹
3. 理解太阳的日变化

**现象**：
- 正午12:00：太阳轨迹最高
- 早晨6:00：太阳轨迹偏东
- 傍晚18:00：太阳轨迹偏西

### 场景3：观察节气变化

1. 查看标记的四个节气点
2. 理解太阳赤纬的年变化
3. 观察"8"字形轨迹的形成

**现象**：
- 春分/秋分：太阳在天赤道上
- 夏至：太阳最北（赤纬+23.5°）
- 冬至：太阳最南（赤纬-23.5°）

## 🔬 技术原理

### 坐标转换流程

```
JPL DE440星历表
    ↓
地心坐标系（x, y, z）
    ↓
赤道坐标系（赤经RA, 赤纬DEC）
    ↓
地平坐标系（方位角Az, 高度角Alt）
    ↓
笛卡尔坐标（x, y, z）
    ↓
3D可视化
```

### 关键公式

**赤道坐标 → 地平坐标**：
```
高度角 Alt = arcsin(sin(φ)·sin(δ) + cos(φ)·cos(δ)·cos(H))
方位角 Az = arctan2(sin(H), cos(H)·sin(φ) - tan(δ)·cos(φ))
```

其中：
- φ = 观测地纬度
- δ = 太阳赤纬
- H = 时角

**地平坐标 → 笛卡尔坐标**：
```
x = cos(Alt) · sin(Az)  # 东-西方向
y = cos(Alt) · cos(Az)  # 南-北方向
z = sin(Alt)            # 天顶-天底方向
```
